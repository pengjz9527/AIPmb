# 持有贷款查询

从 `coredatas/持有贷款信息表.xlsx` 中查询用户持有的贷款。

## 触发场景

- "我有哪些贷款？"
- "我的房贷还剩多少？"
- "帮我算一下还款计划"
- "我想提前还一部分贷款"

## 数据字段

| 字段 | 说明 |
|------|------|
| 贷款类型 | 如个人一手住房贷款 |
| 贷款本金(元) | 原始贷款金额 |
| 未还贷款本金(元) | 剩余未还本金 |
| 已还本金(元) | 已偿还本金 |
| 年利率 | 当前执行利率 |
| LPR调整 | LPR加减基点 |
| 下一个重定价日 | 利率调整日期 |
| 还款方式 | 等额本息/等额本金 |

## API 接口

- `GET /api/v1/held-products/loans?user_name=xxx` — 列出贷款
- `GET /api/v1/held-products/loans/{loan_id}` — 详情
- `GET /api/v1/held-products/loans/{loan_id}/repayment-plan` — 还款计划
- `POST /api/v1/held-products/loans/{loan_id}/prepay` — 提前还款

## 回答模板

```
您目前持有 {count} 笔贷款：

**{贷款类型}** ({贷款编号})
- 贷款本金: ¥{贷款本金}
- 未还本金: ¥{未还贷款本金}
- 年利率: {年利率} ({利率定价方式})
- LPR调整: {LPR调整}
- 下一个重定价日: {下一个重定价日}
- 还款方式: {还款方式}
```

## 还款计划计算

根据还款方式生成未来12期还款计划：
- 等额本息：每月还款额固定
- 等额本金：每月本金固定，利息递减

## 提前还款流程

1. 确认贷款支持提前还款
2. 调用 prepay 接口提交申请
3. 告知用户预计扣款时间和新的还款计划
