# 养老金查询

从 `coredatas/养老金信息表.xlsx` 中查询用户个人养老金账户。

## 触发场景

- "我的养老金账户有多少钱？"
- "个人养老金怎么缴存？"
- "养老金有什么税收优惠？"
- "养老金投资方案推荐"

## 数据字段

| 字段 | 说明 |
|------|------|
| 账户类型 | 个人养老金账户 |
| 总金额(元) | 账户总资产 |
| 闲置金额(元) | 未投资金额 |
| 年度缴存上限(元) | 每年最高缴存额（通常12000元） |
| 剩余缴存额度(元) | 当年还可缴存金额 |
| 已缴存金额(元) | 当年已缴存金额 |
| 税收优惠 | 税优政策说明 |
| 自动缴存状态 | 是否设置自动缴存 |
| 推荐投资方案 | 系统推荐方案 |
| 参考收益率 | 预期收益率 |

## API 接口

- `GET /api/v1/held-products/pensions?user_name=xxx` — 列出养老金账户
- `GET /api/v1/held-products/pensions/{pension_id}` — 详情

```
- 推荐方案: {推荐投资方案}
- 参考收益率: {参考收益率}
```

## 缴存建议

1. 若剩余缴存额度 > 0，建议充分利用年度额度享受税优
2. 闲置金额较多时，建议根据推荐方案进行投资
3. 可设置自动定期缴存，避免遗漏
