# 持有理财查询

从 `coredatas/持有理财信息表.xlsx` 中查询用户持有的理财产品。

## 触发场景

- "我持有哪些理财产品？"
- "我的理财收益怎么样？"
- "交银理财稳享灵动慧利6个月2号的详情"
- "我想赎回理财产品"

## 数据字段

| 字段 | 说明 |
|------|------|
| 产品名称 | 理财产品名称 |
| 产品分类 | 如半年宝、定期存款 |
| 持仓金额(元) | 当前持仓市值 |
| 持仓收益(元) | 累计收益 |
| 收益率 | 产品收益率 |
| 可赎回日期 | 可赎回时间 |
| 可赎回金额(元) | 可赎回金额 |
| 赎回状态 | 预约赎回/到期自动等 |

## API 接口

- `GET /api/v1/held-products/wealth?user_name=xxx` — 列出持有理财
- `GET /api/v1/held-products/wealth/{product_id}` — 详情
- `POST /api/v1/held-products/wealth/{product_id}/redeem` — 赎回

## 回答模板

```
您目前持有 {count} 只理财产品，总持仓金额 ¥{total}：

1. **{产品名称}** ({产品分类})
   - 持仓金额: ¥{持仓金额}
   - 持仓收益: ¥{持仓收益}
   - 收益率: {收益率}
   - 可赎回日期: {可赎回日期}
   - 赎回状态: {赎回状态}
```

## 赎回流程

1. 确认产品处于可赎回状态
2. 调用 redeem 接口提交申请
3. 告知用户预计到账时间（T+1）
