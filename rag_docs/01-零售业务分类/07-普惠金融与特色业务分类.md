# 普惠金融与特色业务分类

## 概述

普惠金融（Inclusive Finance）是指以可负担的成本为有金融服务需求的社会各阶层和群体提供适当、有效的金融服务。普惠金融的核心目标是让金融服务覆盖传统金融体系难以触达的小微企业、农民、城镇低收入人群、老年人、残疾人等特殊群体，是金融服务实体经济、促进社会公平的重要体现。

城商行和农商行是普惠金融的天然主力军。相较于国有大型银行，中小银行扎根地方、深耕基层，在普惠金融领域具有不可替代的地缘优势和信息优势。截至2025年末，全国普惠小微贷款余额已超过32万亿元，其中城商行和农商行贡献了约40%的份额。普惠小微贷款利率已进入"3%时代"，银行如何在低利率环境下实现普惠金融的商业可持续，成为行业面临的核心挑战。

2025年，金融监管总局与人民银行联合发布《银行业保险业普惠金融高质量发展实施方案》，对普惠金融的发展方向、考核指标、风险管理等提出了新要求，推动普惠金融从"量的扩张"转向"质的提升"。

## 各子类型详解

### 一、农户金融服务

#### 1. 农户小额信用贷款

**定义**：银行根据农户的信用状况，在核定的额度内发放的无需担保的小额贷款。

**核心要素**：
- **贷款额度**：一般为1-30万元，部分地区可达50万元
- **贷款期限**：与农业生产周期匹配，一般1-3年
- **利率定价**：享受普惠金融优惠利率
- **审批方式**：基于农户信用档案和评级结果

**业务特点**：
- 无需抵押和担保，纯信用放款
- 审批流程简便，部分产品可"秒批秒贷"
- 随借随还，按实际使用天数计息
- 是农商行最基础、最核心的普惠金融产品

**风控要点**：
- 建立完善的农户信用档案
- 通过"整村授信"批量获取客户信息
- 关注农户的生产经营状况和自然灾害风险
- 利用农业保险分散贷款风险

#### 2. 农户联保贷款

**定义**：由3-5户农户组成联保小组，成员之间互相担保，银行向小组成员发放的贷款。

**核心特点**：
- 成员之间承担连带保证责任
- 无需额外抵押物
- 额度较单户信用贷款更高
- 利用农村熟人社会的信用约束机制

**适用客群**：信用状况一般、无法提供抵押物但有联保条件的农户

#### 3. 土地承包经营权抵押贷款

**定义**：农户以承包土地的经营权作为抵押物向银行申请的贷款。

**核心特点**：
- 需在当地农村产权交易平台办理抵押登记
- 抵押物价值评估需参考当地土地流转价格
- 贷款期限与土地剩余承包期匹配
- 是农村"两权"抵押贷款试点的重要品种

**风控要点**：
- 土地经营权流转市场的成熟度直接影响抵押物变现能力
- 需关注自然灾害和政策变动风险
- 建立土地经营权抵押物处置机制

#### 4. 农业产业链贷款

**定义**：围绕农业产业化龙头企业或专业合作社，向产业链上下游农户发放的贷款。

**核心特点**：
- 依托产业链核心企业的信用和订单关系
- 贷款资金封闭运行，降低挪用风险
- 还款来源为农产品销售回款
- 实现"龙头企业+农户+银行"的三方共赢

#### 5. 农机具抵押贷款

**定义**：农户以购买的农业机械作为抵押物向银行申请的贷款。

**核心特点**：
- 抵押物为农业机械（如拖拉机、收割机等）
- 需在农机监理部门办理抵押登记
- 贷款额度一般为农机价格的50%-70%
- 适合购置大型农机的资金需求

### 二、小微企业金融服务

#### 1. 小微企业信用贷款

**定义**：银行根据小微企业的经营状况和信用记录，发放的无需抵押担保的贷款。

**核心要素**：
- **贷款额度**：一般为10-300万元
- **贷款期限**：1-3年
- **利率定价**：普惠小微企业贷款利率目前多在3%-5%之间
- **审批依据**：纳税数据、银行流水、水电用量、工商登记等

**业务特点**：
- 无需抵押物，降低小微企业融资门槛
- 线上审批、快速放款
- 随借随还，灵活使用
- 是普惠金融考核的核心指标

**风控模型**：
- 银税互动：以纳税信用和纳税数据作为授信依据
- 银商互动：以工商注册和经营数据作为授信依据
- 流水贷：以银行流水数据作为授信依据
- 电力贷：以用电量数据辅助判断经营状况

#### 2. 小微企业抵押贷款

**定义**：小微企业以房产、土地等不动产作为抵押物向银行申请的贷款。

**核心特点**：
- 抵押率一般为评估价值的50%-70%
- 贷款额度较高，可达数百万元
- 期限灵活，1-5年不等
- 利率低于信用贷款

#### 3. 供应链金融

**定义**：围绕核心企业，向其上下游小微企业提供的综合金融服务。

**主要模式**：
- **应收账款融资**：以上游企业对核心企业的应收账款为质押获取融资
- **预付款融资**：核心企业为下游经销商提供担保，帮助其获取采购融资
- **存货质押融资**：以库存商品作为质押物获取融资

**城商行/农商行优势**：
- 深耕本地产业链，熟悉区域内核心企业和上下游关系
- 可为本地供应链提供定制化金融方案
- 通过核心企业批量获客，降低获客成本

#### 4. 科创企业贷款

**定义**：面向科技型小微企业提供的专项贷款产品。

**核心特点**：
- 关注企业的知识产权、研发能力和成长性
- 可采用知识产权质押、股权质押等创新担保方式
- 部分产品与政府风险补偿基金结合
- 审批模型纳入技术创新指标

### 三、社区银行服务

#### 1. 社区支行

**定义**：设立在居民社区内或周边的轻型化银行网点，以零售业务为主。

**核心特点**：
- 面积小（50-100平方米），人员精简（2-4人）
- 以智能柜台和自助设备为主
- 提供基础存取款、转账、理财购买、贷款申请等服务
- 营业时间和业务范围相对灵活

**渠道定位**：
- 城商行深耕社区的核心触点
- "最后一公里"金融服务延伸
- 社区客户建立信任关系的场所

#### 2. 网格化营销

**定义**：将服务区域划分为若干网格，由客户经理负责特定网格内客户的全面金融服务。

**核心特点**：
- 每个网格配备专属客户经理
- 客户经理深入网格内社区、商圈、企业
- 建立网格内客户的详细信息档案
- 实现"人对人、面对面"的精准营销

**城商行/农商行实践**：
- 网格化是中小银行最核心的客户经营方法
- 通过"扫街""扫楼""扫村"等方式实现全覆盖
- 客户经理与社区居民建立长期信任关系

### 四、三农服务

#### 1. 农村支付服务

**定义**：在农村地区为农户提供的基础支付结算服务。

**主要形式**：
- 助农取款服务点：在村级设立取款点，解决取款难问题
- 流动银行车：配备移动设备的车辆定期进村服务
- 手机银行推广：提升农村地区手机银行使用率
- 农村支付环境建设：推进农村地区支付基础设施覆盖

#### 2. 农村保险代理

**定义**：银行在农村地区代理销售农业保险和人身保险产品。

**核心特点**：
- 农业保险：种植险、养殖险等，部分享受财政补贴
- 人身保险：意外险、健康险等
- 银行与保险公司合作，提供"保险+信贷"综合服务

#### 3. 助农取款服务

**定义**：银行在农村地区设立服务点，通过POS机或智能终端为农户提供小额取款、转账等基础金融服务。

**核心特点**：
- 服务点设在村级便利店、卫生室等
- 单笔取款限额一般为2000元
- 可办理转账、缴费等基础业务
- 是解决农村金融服务"最后一公里"的重要手段

### 五、特殊群体金融服务

#### 1. 新市民金融服务

**定义**：面向因创业就业、子女教育等原因进城但尚未完全融入城市的新市民群体提供的金融服务。

**服务内容**：
- 创业贷款：为新市民创业提供融资支持
- 住房租赁贷款：解决新市民租房资金需求
- 教育培训贷款：支持新市民技能提升
- 专属存款产品：设计符合新市民特点的储蓄产品
- 基础银行服务：简化开户流程，提供便捷服务

#### 2. 老年客户金融服务

**定义**：面向老年人群体提供的适老化金融服务。

**服务内容**：
- 适老化网点改造：无障碍设施、大字版资料、爱心座椅等
- 手机银行适老化：推出"关怀版""大字版"App
- 上门服务：为行动不便的老人提供上门办理服务
- 养老金代发：代发养老金是服务老年客户的基础业务
- 防诈宣传：针对老年客户开展防诈骗宣传教育

### 六、乡村振兴金融服务

#### 1. 乡村产业贷款

**定义**：支持乡村产业发展的贷款产品，涵盖农业现代化、乡村旅游、农村电商等领域。

**核心特点**：
- 支持特色农业、现代农业产业园
- 支持乡村旅游和休闲农业
- 支持农村电商和物流体系
- 支持农产品加工和品牌化建设

#### 2. 数字乡村金融

**定义**：利用数字技术为乡村振兴提供的综合金融服务。

**核心内容**：
- 农村信用信息平台建设：整合政务、商务、金融数据
- 线上贷款产品：基于大数据和AI的线上农贷
- 农村电商金融服务：为农村电商提供支付、结算、融资一体化服务
- 智慧农业金融：为智慧种植、智慧养殖提供金融支持

## 城商行/农商行特色实践

### 1. 普惠金融差异化战略

- **做小做散**：坚持服务小微和农户的市场定位，避免与大型银行正面竞争大客户
- **深耕本地**：利用地缘优势和信息对称性，建立对本地客户的深度了解
- **整村授信**：农商行以行政村为单位，对全村农户进行批量授信
- **产业链金融**：围绕本地特色产业和核心企业，构建产业链金融服务体系

### 2. 普惠金融数字化转型

- **移动展业**：客户经理携带移动终端进村入户，现场办理业务
- **线上信贷**：基于大数据和AI技术，开发线上信用贷款产品
- **数字化风控**：整合税务、工商、司法、征信、农业等多维数据构建风控模型
- **智慧农贷**：利用卫星遥感、物联网等技术辅助农业资产评估和监控

### 3. 政银合作模式

- **银税互动**：与税务部门合作，以纳税信用换取融资额度
- **银担合作**：与政府性融资担保机构合作，分担小微企业贷款风险
- **风险补偿基金**：与地方政府合作建立小微企业贷款风险补偿基金
- **创业担保贷款**：与人社部门合作，为创业人员提供贴息贷款

### 4. 典型案例

- **某农商行"整村授信"工程**：以行政村为单位，由客户经理联合村两委对全村农户进行信息采集和信用评级，实现"一次授信、循环使用、随借随还"，覆盖辖内300余个行政村，授信农户超过5万户
- **某城商行"小微企业快贷"**：整合税务、工商、征信等多维数据，构建线上审批模型，小微企业主通过手机银行即可完成贷款申请，从申请到放款最快10分钟，单户额度最高200万元
- **某农商行"乡村振兴贷"**：围绕当地特色农业产业（如茶叶、中药材），联合龙头企业和农业保险公司，为种植户提供"贷款+保险+销售"一体化服务，解决农户融资难、销售难问题
- **某城商行"新市民金融服务站"**：在工业园区、建筑工地等新市民集中区域设立金融服务站，提供开户、贷款、保险、理财等一站式金融服务，配备专门的金融顾问

### 5. 普惠金融可持续发展

- **利率定价平衡**：在普惠金融低利率环境下，通过批量获客和数字化运营降低成本，实现商业可持续
- **风险分担机制**：通过银担合作、银政合作、银保合作等方式分散贷款风险
- **综合收益考量**：普惠金融不仅看贷款利差收入，还要考虑带来的存款、中间业务等综合收益
- **马太效应应对**：面对大型银行下沉竞争，通过差异化服务和更深入的本地化经营保持竞争优势

## 监管要求与合规要点

### 1. 普惠金融考核指标

- **两增两控**：普惠小微企业贷款增速不低于各项贷款增速、有贷款余额的户数不低于上年水平；合理控制资产质量和综合融资成本
- **涉农贷款考核**：农商行涉农贷款占比需达到监管要求
- **小微企业贷款考核**：普惠小微贷款增速、户数等指标纳入监管考核
- **首贷户考核**：鼓励银行拓展首次获得贷款的小微企业和个体工商户

### 2. 贷款利率管理

- 普惠小微企业贷款利率鼓励在LPR基础上合理定价
- 不得违规收费，不得以贷转存、借贷搭售
- 综合融资成本需透明化披露
- 对小微企业减费让利

### 3. 农村金融合规

- 农户贷款需严格执行"三查"（贷前调查、贷时审查、贷后检查）
- 涉农贷款资金用途需合规，不得挪作他用
- 助农取款服务点需符合安全管理要求
- 土地承包经营权抵押需依法合规办理

### 4. 消费者权益保护

- 普惠金融客户多为金融知识薄弱群体，需加强金融知识普及
- 贷款合同和条款需用通俗易懂的语言说明
- 不得诱导过度借贷
- 加强金融消费者投诉处理机制

### 5. 反洗钱合规

- 农村地区的客户身份识别需严格执行
- 关注涉农贷款和小额信贷中的洗钱风险
- 助农取款服务点需纳入反洗钱监测范围
