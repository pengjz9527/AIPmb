# 零售业务分类体系

## 概述

零售银行业务（Retail Banking）是商业银行面向个人客户和家庭客户提供的各类金融服务的总称，是商业银行最基础、最核心的业务板块之一。在利率市场化深入推进、金融科技加速渗透、客户需求日益多元化的背景下，零售业务已成为商业银行战略转型的主战场，被誉为"银行零售转型的下半场"。

对于城商行和农商行而言，零售业务具有特殊战略意义：一方面，零售业务是中小银行抵御经济周期波动、稳定息差收入的压舱石；另一方面，依托本地化经营优势和社区深耕能力，中小银行在零售客群经营上具有天然的客户黏性和信息对称优势。

## 各分类之间的关系

零售业务七大分类之间存在紧密的业务关联：

- **存款是基础**：个人存款业务是零售业务的负债端核心，为贷款和投资提供资金来源
- **贷款是利润引擎**：个人贷款是零售业务的主要收入来源，与存款形成资产负债的平衡
- **银行卡是载体**：银行卡是客户使用各类零售服务的物理和虚拟介质
- **理财是增值服务**：财富管理类业务帮助客户实现资产保值增值，是AUM增长的核心驱动力
- **中间业务是轻资产方向**：中间业务不占用资本金，是零售转型的轻资本发展方向
- **渠道是触达方式**：电子渠道和物理渠道是客户接触和使用零售服务的通路
- **普惠金融是社会责任**：体现银行服务实体经济、践行社会责任的特色业务领域
