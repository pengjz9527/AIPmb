# 零售业务产品介绍

## 分类说明

本分类系统梳理了城商行/农商行零售业务五大产品线的详细介绍文档，旨在为从业人员提供专业、全面的产品知识参考。每篇文档均按照统一结构编写，涵盖产品概述、分类、适用对象、核心要素、办理渠道、政策法规、城商行/农商行特色、常见问题及关联知识等模块。

## 产品体系总览

| 产品线 | 篇数 | 核心产品 |
|--------|------|----------|
| 存款产品 | 8篇 | 活期存款、定期存款、大额存单、结构性存款、通知存款等 |
| 贷款产品 | 9篇 | 住房贷款、消费贷款、经营贷款、信用贷款、农户贷款等 |
| 银行卡产品 | 4篇 | 借记卡、信用卡、联名卡、虚拟卡 |
| 理财产品 | 6篇 | 银行理财、代销基金、代销保险、贵金属、个人养老金等 |
| 中间业务产品 | 4篇 | 代理收付、代销国债、资金托管、外汇兑换 |

## 使用说明

1. 每篇文档均按照统一模板结构编写，便于快速查阅和对比
2. 利率、费率等数据以文档标注的参考时点为准，实际以银行最新公告为准
3. 各产品的"城商行/农商行特色"章节着重介绍中小银行的差异化经营策略
4. "关联知识"章节提供相关文档的相对路径链接，方便交叉阅读

## 数据参考时点

本文档体系中的利率、费率等数据参考时点为2024-2025年，具体以各银行最新公告为准。
