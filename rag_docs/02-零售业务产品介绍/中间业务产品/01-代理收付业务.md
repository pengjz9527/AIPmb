# 代理收付业务

## 产品概述

代理收付业务是指商业银行接受政府机关、企事业单位或个人委托，代为办理各类资金的收取和支付的业务。代理收付业务是银行最基础的中间业务之一，涵盖了代发工资、代收水电费、代扣社保/公积金、代付养老金/补贴等众多场景，是银行连接客户、沉淀资金、拓展业务的重要纽带。

代理收付业务虽然单笔收益不高，但具有客户覆盖面广、资金沉淀效应强、交叉销售潜力大的特点。对于城商行和农商行而言，代理收付业务是获客活客的基础性业务——代发工资客户是银行最优质的零售客户来源，代收代付业务是维护客户关系、提升客户黏性的重要工具。

## 产品分类

### 按业务方向分类

#### 代理收入类（代发/代付）
- **代发工资**：代企业向员工发放工资、奖金、补贴。
- **代付养老金**：代社保机构向退休人员发放养老金。
- **代付补贴/救助金**：代政府部门发放各类财政补贴、社会救助金。
- **代付拆迁补偿款**：代政府部门发放拆迁补偿款。
- **代付分红/利息**：代企业向股东发放分红或利息。

#### 代理支出类（代收/代扣）
- **代收水电燃气费**：代公用事业单位收取水费、电费、燃气费。
- **代收通讯费**：代电信运营商收取话费、宽带费。
- **代扣社保/公积金**：代社保/公积金管理中心从个人账户扣缴社保和公积金。
- **代扣保险费**：代保险公司从个人账户扣缴保险费。
- **代扣贷款还款**：银行内部贷款还款的自动扣款。
- **代收物业费**：代物业管理公司收取物业费。
- **代收学费**：代学校收取学费、住宿费。
- **代收税款**：代税务部门收取个人所得税等。

### 按委托方类型分类
- **政府类**：代发养老金、代付补贴、代收税款等。
- **企事业单位类**：代发工资、代收水电费等。
- **金融机构类**：代扣保险费、代收基金申购款等。
- **个人类**：个人之间的代付、代收服务。

### 按办理方式分类
- **批量处理**：委托方提交批量数据，银行批量处理。
- **实时处理**：单笔实时处理（如柜台代收水电费）。
- **定期处理**：按约定周期自动处理（如每月代扣社保）。

## 适用对象与准入条件

### 适用对象

#### 委托方（B端）
- 政府机关、事业单位
- 各类企业（含小微企业、个体工商户）
- 公用事业单位（水、电、燃气、通讯）
- 保险公司
- 学校、医院
- 物业管理公司

#### 被服务方（C端）
- 企业员工（代发工资）
- 退休人员（代付养老金）
- 城乡居民（代收各类费用）
- 社保/公积金参保人员
- 贷款客户（代扣还款）

### 准入条件
| 条件 | 具体要求 |
|------|----------|
| 委托方 | 具有合法经营资质的企事业单位或政府机构 |
| 签约协议 | 委托方与银行签署代理收付协议 |
| 账户要求 | 被服务方需持有该银行结算账户 |
| 数据格式 | 委托方按银行要求的格式提交数据 |
| 资金保障 | 委托方账户需有足够资金（代付业务） |

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 业务范围 | 代发工资、代收代付各类费用、代扣社保/公积金等 |
| 收费标准 | 委托方按笔数/金额支付手续费；被服务方通常免费 |
| 处理时效 | 批量代发：T+0至T+1；实时代收：即时到账 |
| 数据安全 | 符合个人信息保护和数据安全法规要求 |
| 对账方式 | 电子对账/纸质对账，按月/按季 |
| 失败处理 | 失败交易退回或重新处理 |

### 代发工资业务详解

| 要素 | 说明 |
|------|------|
| 签约流程 | 企业与银行签署代发工资协议 |
| 数据提交 | 企业每月提交工资数据（Excel/API对接） |
| 资金划转 | 企业将工资总额划入银行指定账户 |
| 批量发放 | 银行批量发放至员工个人账户 |
| 到账时间 | 通常T+0到账（本行）或T+1到账（跨行） |
| 费用 | 本行代发免费/低收费；跨行代发可能收取少量费用 |
| 附加服务 | 免费开通手机银行、短信通知、信用卡优先审批 |

### 代收代付费用参考

| 业务类型 | 委托方费用 | 被服务方费用 |
|----------|-----------|-------------|
| 代发工资（本行） | 免费或0.5-2元/笔 | 免费 |
| 代发工资（跨行） | 2-5元/笔 | 免费 |
| 代收水电费 | 按协议定价 | 免费 |
| 代扣社保 | 按协议定价 | 免费 |
| 代付养老金 | 按协议定价 | 免费 |

## 办理渠道

| 渠道 | 可办理业务 | 备注 |
|------|-----------|------|
| 营业网点柜面 | 签约、数据提交、查询 | 全功能 |
| 企业网银 | 数据上传、批量处理、对账 | 企业端主力 |
| 手机银行APP | 签约、缴费、查询 | 个人端 |
| 网上银行 | 缴费、签约管理 | 个人端辅助 |
| 自助终端 | 缴费 | 社区/农村服务点 |
| 微信支付/支付宝 | 部分代收业务 | 第三方渠道 |
| 电话银行 | 查询、签约 | 语音自助 |

## 政策法规依据

1. **《商业银行中间业务暂行规定》**（中国人民银行〔2001〕）：中间业务基本规范。
2. **《商业银行服务价格管理办法》**（发改委、银监会〔2014〕）：服务收费规范。
3. **《中国人民银行关于改进个人银行账户服务加强账户管理的通知》**：账户管理规范。
4. **《个人信息保护法》**（2021年）：个人信息保护法律。
5. **《数据安全法》**（2021年）：数据安全管理法律。
6. **《商业银行操作风险管理指引》**：操作风险管理规范。

## 城商行/农商行特色

### 代发工资业务优势
- **本地企业深耕**：与本地中小企业、事业单位建立长期代发工资合作关系。
- **政府代发业务**：承接当地政府部门的养老金代发、补贴代发等业务。
- **农民工工资代发**：配合建筑工程项目，提供农民工工资代发服务。
- **代发+综合服务**：代发工资客户同步推荐信用卡、理财、贷款等产品。

### 农村场景
- **惠农补贴代发**：农商行是各类惠农补贴（种粮补贴、农机补贴、低保金等）的主要代发银行。
- **社保代发**：代发城乡居民养老保险金，覆盖广大农村退休居民。
- **村级代收**：通过村级金融服务站代收水电费、通讯费等。
- **新型农村合作医疗代收**：代收缴新农合保费。

### 数字化创新
- **API对接**：与企业ERP/HR系统对接，实现工资数据自动传输。
- **智能对账**：电子化对账替代纸质对账，提升效率。
- **线上签约**：企业网银/手机银行在线签署代发协议。
- **实时到账**：本行代发工资实时到账，提升员工体验。

### 社区化服务
- **社区缴费**：在社区网点提供水电燃气费等一站式缴费服务。
- **适老化服务**：为老年客户提供柜台代缴服务，避免线上操作困难。
- **流动服务**：流动银行车定期到偏远地区提供代收代付服务。

## 常见问题

### Q1：代发工资对员工有什么好处？
代发工资对员工的好处包括：一是工资到账快速安全，无需现金领取；二是可享受银行提供的专属权益（如免费短信通知、跨行取款优惠、信用卡优先审批等）；三是工资流水是申请贷款的重要收入证明；四是方便自动理财（如工资自动转入协定存款或理财产品）。

### Q2：企业办理代发工资需要哪些手续？
主要流程包括：一是在银行开立企业结算账户（如未开立）；二是与银行签署代发工资协议；三是提供企业营业执照、法人身份证等基本材料；四是首次代发时提供员工名单和银行卡号信息；五是每月按约定时间提交工资数据和资金。整个流程通常在1-3个工作日内完成。

### Q3：代收水电费等费用怎么办理自动扣款？
办理自动扣款的方式：一是在手机银行APP中设置"自动缴费"功能，绑定水电燃气等缴费户号；二是在柜台签署自动代扣协议；三是在微信银行/网上银行中设置。设置自动扣款后，每月费用将自动从账户中扣除，避免因忘记缴费而停水停电。

### Q4：代发工资可以发到不同银行的卡上吗？
可以。企业可以选择将工资发至员工在不同银行的银行卡上（跨行代发），但跨行代发可能需要支付少量手续费（2-5元/笔），且到账时间可能略慢（T+1）。多数企业鼓励员工在代发银行开户，以减少费用和提高效率。

### Q5：农商行代发养老金是什么时候到账？
养老金到账时间因地区而异，通常为每月10-20日之间（具体以当地社保部门公布的时间为准）。养老金发放到社保卡金融账户后，可在农商行网点、ATM、手机银行等渠道查询和使用。建议开通短信通知功能，养老金到账即时收到提醒。
