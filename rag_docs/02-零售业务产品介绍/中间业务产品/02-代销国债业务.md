# 代销国债业务

## 产品概述

代销国债业务是指商业银行作为国债承销团成员或代销机构，代理财政部向个人投资者销售储蓄国债（凭证式/电子式）和记账式国债的业务。国债以国家信用为担保，是风险最低的固定收益类投资品种，被誉为"金边债券"。

代销国债是银行中间业务的重要组成部分，虽然直接收入不高（手续费率约0.1%-0.5%），但对提升银行品牌形象、吸引保守型客户、沉淀资金具有重要作用。对于城商行和农商行而言，国债销售是服务中老年客户、维护客户关系的重要工具，也是展示银行公信力的窗口。

## 产品分类

### 按国债类型分类

#### 储蓄国债（凭证式）
- **性质**：不可流通的固定利率国债，以纸质凭证记录。
- **期限**：3年期和5年期。
- **购买方式**：银行柜面购买。
- **兑付方式**：到期一次还本付息。
- **提前兑取**：可提前兑取，按持有时间靠档计息。

#### 储蓄国债（电子式）
- **性质**：不可流通的固定利率国债，以电子方式记录。
- **期限**：3年期和5年期。
- **购买方式**：银行柜面或网上银行。
- **兑付方式**：按年付息，到期还本。
- **提前兑取**：可提前兑取，按持有时间靠档计息。
- **自动兑付**：到期自动转入银行账户。

#### 记账式国债
- **性质**：可在银行间债券市场和交易所市场交易的国债。
- **期限**：1年-50年不等。
- **购买方式**：银行柜台/网银、交易所。
- **兑付方式**：按半年付息，到期还本。
- **交易特性**：可在二级市场买卖，价格随市场波动。

### 按发行方式分类
- **承销团成员销售**：银行作为国债承销团成员直接销售。
- **代销方式**：银行作为代销机构代理销售。

## 适用对象与准入条件

### 适用对象
- 风险偏好极低的保守型投资者
- 中老年客户（偏好确定性收益）
- 大额资金短期管理需求的客户
- 养老金/退休金配置需求的客户
- 有国债质押融资需求的客户

### 准入条件
| 条件 | 具体要求 |
|------|----------|
| 身份 | 中国境内自然人（个人投资者） |
| 身份证明 | 有效身份证件 |
| 购买渠道 | 凭证式需柜面购买；电子式可柜面或网银 |
| 账户要求 | 电子式需开立国债托管账户 |
| 购买限额 | 单人单期最高购买限额（通常500万元） |
| 购买时间 | 发行期内购买（通常每月10日起，额度有限） |

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 储蓄国债利率 | 3年期约2.0%-2.5%，5年期约2.2%-2.7%（以各期发行为准） |
| 记账式国债利率 | 随市场波动，参考国债收益率曲线 |
| 发行时间 | 储蓄国债通常每月10日发行（3-11月） |
| 购买起点 | 100元起购，以100元整数倍递增 |
| 付息方式 | 凭证式：到期一次还本付息；电子式：按年付息 |
| 提前兑取 | 持有满6个月后可提前兑取，扣除一定天数利息 |
| 质押功能 | 可作为质押物办理国债质押贷款 |

### 储蓄国债提前兑取规则

| 持有时间 | 计息规则 |
|----------|----------|
| 不满6个月 | 不计利息 |
| 满6个月-2年 | 按实际天数计息，扣除180天利息 |
| 满2年-3年 | 按实际天数计息，扣除90天利息 |
| 满3年（3年期到期） | 按票面利率全额付息 |
| 满3年-5年（5年期） | 按实际天数计息，扣除180天利息 |
| 满5年（5年期到期） | 按票面利率全额付息 |

### 储蓄国债与银行定期存款对比

| 对比项 | 储蓄国债 | 银行定期存款 |
|--------|---------|-------------|
| 信用等级 | 国家信用（最高） | 银行信用（存款保险50万保障） |
| 利率水平 | 通常略高于同期限定存 | 各行定价不同 |
| 付息方式 | 电子式按年付息 | 到期一次还本付息 |
| 提前支取 | 靠档计息（更友好） | 按活期计息（损失较大） |
| 购买便利性 | 发行期购买，可能抢不到 | 随时可存 |
| 免税 | 国债利息免个人所得税 | 存款利息无需缴税 |
| 额度限制 | 单人单期限额 | 无限制 |

## 办理渠道

| 渠道 | 可办理业务 | 备注 |
|------|-----------|------|
| 营业网点柜面 | 购买凭证式/电子式国债、兑取 | 全功能 |
| 网上银行 | 购买电子式国债、查询 | 线上渠道 |
| 手机银行APP | 购买电子式国债、查询、兑取 | 推荐渠道 |
| 银行柜台（记账式） | 购买/卖出记账式国债 | 柜台债券业务 |
| 交易所（记账式） | 购买/卖出记账式国债 | 需证券账户 |

## 政策法规依据

1. **《中华人民共和国国债条例》**（国务院令第41号）：国债管理基本法规。
2. **《储蓄国债（电子式）管理办法》**（财政部、人民银行〔2013〕）：电子式储蓄国债管理。
3. **《储蓄国债（凭证式）管理办法》**（财政部、人民银行〔2021〕）：凭证式储蓄国债管理。
4. **《国债承销团管理办法》**：国债承销团管理。
5. **《财政部关于国债发行计划的通知》**：年度国债发行安排。

## 城商行/农商行特色

### 客户服务
- **中老年客户专属服务**：为中老年客户提供国债购买指导、优先排队等服务。
- **发行提醒**：国债发行前通过短信、微信等方式提醒客户。
- **额度保障**：对VIP客户提供国债购买额度预约服务。
- **到期提醒**：国债到期前提醒客户续购或安排资金使用。

### 营销策略
- **国债+存款组合**：将国债与银行定期存款组合，为客户提供安全的资产配置方案。
- **国债到期转化**：国债到期时推荐理财产品、大额存单等替代产品。
- **社区宣传**：在社区开展国债投资知识宣传活动。
- **农村推广**：农商行在乡镇网点和村级服务站推广国债购买服务。

### 国债质押服务
- **国债质押贷款**：以持有的储蓄国债作为质押物申请贷款，额度可达国债面值的90%。
- **灵活融资**：避免提前兑取国债造成的利息损失。
- **利率优惠**：国债质押贷款利率通常低于信用贷款。

## 常见问题

### Q1：储蓄国债怎么买？什么时候买？
储蓄国债通常在每年3月至11月的每月10日开始发行（具体以财政部公告为准），发行期为10天左右。购买方式：凭证式国债需到银行柜台购买；电子式国债可通过柜台或网上银行购买。由于国债热销，发行首日往往额度紧张，建议提前关注发行公告，发行当日尽早购买。

### Q2：储蓄国债利率比银行定期存款高吗？
通常情况下，储蓄国债利率略高于同期限的国有大行定期存款利率，但可能低于城商行/农商行的上浮后定期利率。以2025年为例，3年期储蓄国债利率约2.0%-2.5%，而大行3年期定存利率1.25%。但需注意，国债利息免个人所得税（虽然当前存款利息也免税），且国债信用等级最高（国家信用）。

### Q3：储蓄国债可以提前兑取吗？
可以。储蓄国债持有满6个月后可提前兑取，但需扣除一定天数的利息作为手续费。相比银行定期存款提前支取按活期计息，国债的提前兑取规则更友好（靠档计息）。但建议在资金规划时尽量避免提前兑取，以获得完整收益。

### Q4：电子式国债和凭证式国债哪个好？
电子式国债更方便——可通过网上银行/手机银行购买，到期自动转入银行账户，按年付息可享受复利效果。凭证式国债适合习惯纸质凭证的老年客户，到期一次还本付息。两者利率相同，安全性相同。建议优先选择电子式国债，购买和管理更便捷。

### Q5：国债额度抢不到怎么办？
国债额度有限，常出现"秒光"现象。建议：一是发行首日尽早购买（开门前排队或网银提前登录）；二是关注多家银行的额度分配，部分银行额度较充裕；三是选择不太热门的期限（如5年期通常比3年期容易买到）；四是如确实买不到，可考虑银行大额存单或结构性存款作为替代。
