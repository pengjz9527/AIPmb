# 活期存款

## 产品概述

活期存款是指不约定存期、客户可随时存取、按结息期实际存款天数计付利息的个人存款产品。活期存款是商业银行最基础的负债业务之一，也是银行零售业务产品体系的基石。对于城商行和农商行而言，活期存款是维护客户关系、沉淀低成本资金的重要工具。

活期存款具有流动性最强、起存门槛最低的特点，是客户日常资金管理的"蓄水池"。客户通过活期账户实现工资入账、消费支付、转账结算、代扣代缴等日常金融需求，同时也是购买理财、基金、保险等其他金融产品的基础账户。在银行AUM（管理客户资产规模）管理中，活期存款是客户资产配置的基础层。

## 产品分类

### 按账户性质分类
- **个人结算账户（I类户）**：全功能账户，支持存取现金、转账、消费，无额度限制。每人每家银行只能开立一个I类户。
- **个人储蓄账户（II类户）**：限制功能账户，日累计限额1万元，年累计限额20万元。可通过电子渠道开立。
- **个人电子账户（III类户）**：小额支付账户，余额上限2000元，日累计限额2000元，年累计限额5万元。

### 按币种分类
- **人民币活期存款**：基础本币活期储蓄。
- **外币活期存款**：支持美元、欧元、港币等主要外币币种的活期储蓄，利率按各币种挂牌利率执行。

## 适用对象与准入条件

### 适用对象
- 年满16周岁具有完全民事行为能力的中国公民
- 持有有效身份证件的境外个人（含港澳台居民）
- 代发工资客户群体
- 日常消费结算需求客户
- 个体工商户用于日常经营结算

### 准入条件
- 持有效身份证件（居民身份证、临时身份证、户口簿、护照等）
- 未满16周岁需由法定监护人代理开户，并提供监护关系证明
- 同一客户在同一银行只能开立一个I类户
- 反洗钱身份识别要求（需留存联系方式、职业、住址等信息）

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 起存金额 | 人民币1元起存，无上限限制 |
| 基准利率 | 0.05%（2025年5月中国人民银行挂牌利率） |
| 计息方式 | 按日计息、按季结息 |
| 结息日 | 每季末月20日（3月20日、6月20日、9月20日、12月20日） |
| 计息公式 | 利息 = 累计计息积数 × 日利率 |
| 日利率换算 | 日利率 = 年利率 ÷ 360 |
| 存取方式 | 随存随取，无存取限制 |
| 存款凭证 | 存折、借记卡、电子账户 |
| 利率浮动 | 各银行可在基准利率基础上自主浮动 |

### 计息规则详解
1. **累计积数计息法**：按实际天数每日累计账户余额，以累计积数乘以日利率计算利息。
2. **利息入息**：每季度结息后，利息自动转入活期账户本金，下季度起息时按新本金计息（复利效果）。
3. **销户计息**：客户销户时，按上一结息日至销户日的实际天数计付利息。
4. **利率调整**：如遇利率调整，按结息日挂牌利率执行（不分段计息）。

## 政策法规依据

1. **《储蓄管理条例》**（国务院令第107号，2011年修订）：储蓄存款的基本法规。
2. **《人民币银行结算账户管理办法》**（中国人民银行令〔2003〕第5号）：银行结算账户管理规范。
3. **《关于改进个人银行账户服务 加强账户管理的通知》**（银发〔2015〕392号）：I/II/III类账户分类管理制度。
4. **《存款保险条例》**（国务院令第660号）：50万元限额偿付保障。
5. **《中国人民银行关于人民币存贷款计结息问题的通知》**（银发〔2005〕129号）：计结息规则。
6. **《金融机构客户身份识别和客户身份资料及交易记录保存管理办法》**：反洗钱身份识别要求。

## 城商行/农商行特色

### 差异化利率策略
城商行和农商行在活期存款经营中，通常采取以下差异化策略：
- **利率上浮**：部分城商行活期存款利率在基准基础上适度上浮，以吸引客户资金沉淀。
- **协议存款结合**：针对代发工资客户，提供"活期+协定"组合方案，提升资金收益。
- **活期增值产品**：如"智能活期""余额理财"等，实现活期资金自动归集至货币基金或通知存款，兼顾流动性与收益性。

### 场景化获客
- **代发工资场景**：通过与当地企业、机关事业单位合作，批量获取代发工资客户，沉淀活期资金。
- **社保卡场景**：农商行依托社保卡代发养老金、惠农补贴等，形成稳定的活期存款来源。
- **商户收单场景**：通过聚合支付、POS收单业务沉淀商户日常经营资金。

### 数字化转型
- **无卡存款**：支持手机银行扫码存款、无卡ATM存款。
- **智能提醒**：账户余额变动实时推送，大额变动二次确认。
- **电子对账**：线上电子对账单替代纸质对账单。

## 常见问题

### Q1：活期存款利率是多少？利率会变化吗？
当前活期存款基准利率为0.05%（2025年5月挂牌利率）。各银行可在基准利率基础上自主浮动。利率由中国人民银行根据宏观经济形势调整，如遇调整，按结息日当天银行挂牌利率执行，不进行分段计息。

### Q2：活期存款的利息怎么计算？什么时候到账？
活期存款采用按日计息、按季结息的方式。每日根据账户余额计算积数，每季末月20日为结息日，21日利息自动入账。计算公式为：利息 = 累计计息积数 × 日利率（年利率÷360）。例如，账户日均余额10万元，按0.05%年利率计算，季度利息约为12.50元。

### Q3：同一银行可以开几个活期账户？
根据央行账户分类管理规定，同一客户在同一家银行只能开立一个I类全功能结算账户。如需开立多个账户，可开立II类或III类限制功能账户。I类户无额度限制，II类户日累计限额1万元，III类户余额上限2000元。

### Q4：活期存款受存款保险保障吗？
是的。根据《存款保险条例》，活期存款属于受保障的存款类型，最高偿付限额为人民币50万元。同一存款人在同一家投保机构的存款本息合计在50万元以内的，实行全额偿付。

### Q5：长期不动的活期账户会被怎么处理？
根据监管规定，银行对长期不动户（通常指一年以上无主动交易且余额较低的账户）会进行限制管理，可能暂停非柜面业务。客户需持有效身份证件到网点进行身份重新确认后恢复使用。建议定期清理不使用的"睡眠账户"，防范信息安全风险。
