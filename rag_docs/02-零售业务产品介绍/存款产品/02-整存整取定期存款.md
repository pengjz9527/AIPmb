# 整存整取定期存款

## 产品概述

整存整取定期存款是指客户与银行约定存款期限，一次性存入本金，到期一次性支取本息的定期储蓄存款产品。作为商业银行最传统的定期存款品种，整存整取以其安全性高、收益确定的特点，长期以来是我国居民最主要的储蓄方式之一。

整存整取在银行负债结构中占据重要地位，是银行中长期资金的主要来源。对于城商行和农商行而言，定期存款（尤其是整存整取）是稳定负债来源的核心组成部分，对流动性管理和资产负债匹配具有关键作用。在当前利率下行周期中，合理引导客户配置不同期限的定期存款，是中小银行负债管理的重要策略。

## 产品分类

### 按期限分类
整存整取定期存款设有六个标准期限档次：
- 三个月期
- 六个月期（半年）
- 一年期
- 两年期
- 三年期
- 五年期

### 按存款凭证分类
- **存单式**：以纸质定期存单为凭证，到期凭存单支取。
- **卡内定期**：在借记卡内开立定期子账户，电子化管理。
- **存折式**：在定期一本通存折中记录，适合习惯纸质凭证的老年客户。

### 按利率类型分类
- **普通定期**：按银行挂牌利率执行。
- **利率上浮定期**：部分银行对大额或特定客户提供利率上浮。
- **特色定期**：银行推出的品牌化定期产品（如"定活通"等），利率更具竞争力。

## 适用对象与准入条件

### 适用对象
- 有中长期闲置资金的个人客户
- 风险偏好极低、追求稳定收益的保守型投资者
- 养老金、子女教育金等有明确时间规划的资金
- 大额资金短期闲置客户（可搭配大额存单）
- 代发工资客户结余资金

### 准入条件
- 持有效身份证件的自然人
- 人民币整存整取起存金额：50元
- 外币整存整取起存金额：等值50元人民币的外币
- 未满16周岁需监护人代办

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 起存金额 | 人民币50元起存 |
| 期限档次 | 3个月、6个月、1年、2年、3年、5年 |
| 存款利率（2025年5月挂牌） | 3个月0.65%、6个月0.85%、1年0.95%、2年1.05%、3年1.25%、5年1.30% |
| 计息方式 | 到期一次还本付息 |
| 计息公式 | 利息 = 本金 × 年利率 × 存期（年） |
| 自动转存 | 支持约定到期自动转存（本息转存或仅本金转存） |
| 提前支取 | 可部分提前支取一次，剩余部分按原利率计息 |
| 逾期支取 | 逾期部分按支取日活期利率计息 |

### 利率体系详解

#### 挂牌利率与执行利率
- **挂牌利率**：银行对外公示的标准利率，各银行根据市场情况自主定价。
- **执行利率**：实际办理时的利率，部分银行对大额存款（如5万元以上）执行上浮利率。
- **城商行/农商行利率**：通常高于国有大行挂牌利率10-30BP，以吸引存款。

#### 各期限利率对比（2025年5月参考）
| 期限 | 国有大行参考利率 | 城商行/农商行参考利率 |
|------|-----------------|---------------------|
| 3个月 | 0.65% | 0.70%-0.85% |
| 6个月 | 0.85% | 0.90%-1.05% |
| 1年 | 0.95% | 1.05%-1.25% |
| 2年 | 1.05% | 1.15%-1.40% |
| 3年 | 1.25% | 1.35%-1.70% |
| 5年 | 1.30% | 1.40%-1.75% |

### 提前支取规则
1. **全额提前支取**：按支取日活期存款利率（0.05%）计付利息。
2. **部分提前支取**：可部分提前支取一次（部分银行支持多次），提前支取部分按活期利率计息，留存金额若不低于起存金额则按原利率、原期限继续计息。
3. **提前支取次数**：传统规定只允许部分提前支取一次，部分银行已放宽至不限次数。
4. **注意事项**：提前支取需持有效身份证件到柜面办理，部分银行支持手机银行线上提前支取。

### 自动转存规则
1. **本息转存**：到期后本金和利息合并作为新的本金，按转存日当天同期限挂牌利率续存。
2. **仅本金转存**：到期后利息转入活期账户，仅本金按原期限续存。
3. **转存次数**：无次数限制，每次转存视为一笔新的定期存款。
4. **取消转存**：客户可随时到银行取消自动转存约定。
5. **利率风险**：转存时按转存日利率执行，如利率下行则收益减少。

## 政策法规依据

1. **《储蓄管理条例》**（国务院令第107号，2011年修订）：定期存款基本法规依据。
2. **《中国人民银行关于人民币存贷款计结息问题的通知》**（银发〔2005〕129号）：明确定期存款计息规则，提前支取按活期计息。
3. **《存款保险条例》**（国务院令第660号）：定期存款纳入存款保险保障范围。
4. **《商业银行存款利率管理办法》**：银行自主定价的法律依据。
5. **《中国人民银行关于完善商业银行存款利率市场化形成机制的通知》**：取消存款利率浮动上限。

## 城商行/农商行特色

### 利率优势
城商行和农商行在定期存款定价方面通常具有以下优势：
- **利率上浮空间大**：相比国有大行，中小银行定期存款利率普遍高出10-30个基点，三年期、五年期差距更为明显。
- **阶梯利率**：部分银行根据存款金额设置阶梯利率，如5万元以下执行标准利率，5万元以上上浮10BP，20万元以上上浮20BP。
- **季节性优惠**：开门红（1-3月）、年中、年末等时点推出利率上浮活动。

### 特色产品创新
- **"定活通"类产品**：定期存款与活期存款智能联动，活期账户余额超过设定金额自动转定期，定期到期自动转活期。
- **靠档计息产品**（已基本停售）：提前支取按实际存期靠档计息，兼顾收益和流动性。2020年后监管要求整改，目前多数银行已停止新发。
- **分期付息定期**：按月或按季支付利息，适合需要定期现金流的退休客户。
- **"孝心存""教育储蓄"等主题定期**：面向特定客群，提供专属利率和增值服务。

### 揽储策略
- **社区营销**：依托社区网点开展定期存款营销活动，重点维护中老年客群。
- **积分回馈**：存款积分兑换生活权益（超市购物券、体检套餐等）。
- **存贷联动**：定期存款客户可享受贷款优惠利率、信用卡优先审批等权益。

## 常见问题

### Q1：整存整取可以提前取款吗？利息怎么算？
可以提前支取。全额提前支取按支取日活期利率计息；部分提前支取时，支取部分按活期利率计息，留存部分仍按原定期利率计息。建议尽量避免提前支取，或仅部分支取以减少利息损失。目前部分银行手机银行支持线上提前支取操作。

### Q2：定期存款到期忘记取款怎么办？
如已设置自动转存，到期后会自动按同期限续存，利率按转存日挂牌利率执行。如未设置自动转存，到期后的逾期部分按活期利率计息。建议客户在存款时即开通自动转存功能，避免遗忘造成收益损失。

### Q3：定期存款可以质押贷款吗？
可以。未到期的定期存单可作为质押物申请存单质押贷款，贷款额度一般不超过存单面额的90%-95%，贷款利率通常低于信用贷款。这是短期资金周转的有效方式，避免提前支取造成的利息损失。

### Q4：为什么城商行/农商行的定期利率比大银行高？
城商行和农商行由于网点覆盖和品牌影响力相对较弱，需要通过较高的利率吸引存款。同时，中小银行的存款主要投放于当地信贷市场，贷款利率相对较高，因此有能力支付较高的存款利率。但需注意，存款利率较高也反映了银行负债成本压力，客户可放心的是，所有正规银行均受存款保险制度保障。

### Q5：五年期定期存款还值得存吗？
五年期定期存款利率（挂牌1.30%）在当前利率下行环境中仍具有一定吸引力，尤其对风险偏好极低的客户。但需注意：一是流动性风险，五年内提前支取按活期计息损失较大；二是利率风险，如未来利率回升可能错失更高收益。建议结合个人资金规划，采取"阶梯存款法"分散配置不同期限的定期存款。
