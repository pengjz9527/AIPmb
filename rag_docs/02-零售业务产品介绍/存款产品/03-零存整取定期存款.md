# 零存整取定期存款

## 产品概述

零存整取定期存款是指客户与银行约定每月固定存入一定金额，到期一次支取本息的定期储蓄存款产品。该产品旨在帮助客户养成定期储蓄的习惯，实现"积少成多"的财富积累目标，是银行面向工薪阶层、年轻客户和青少年群体推出的基础储蓄产品。

零存整取定期存款虽然单户金额较小，但具有客户黏性强、资金稳定性好的特点。对于城商行和农商行来说，零存整取是培养年轻客户、建立长期客户关系的有效工具，同时也有利于银行稳定月度资金流入。在"强制储蓄"理念逐渐回归的背景下，零存整取作为帮助客户实现消费控制的工具，重新受到部分年轻客群的关注。

## 产品分类

### 按期限分类
- **一年期零存整取**：每月存入，存满12个月后一次支取。
- **三年期零存整取**：每月存入，存满36个月后一次支取。
- **五年期零存整取**：每月存入，存满60个月后一次支取。

### 按存入方式分类
- **手动存入**：客户每月到银行或通过手机银行手动存入约定金额。
- **自动扣款**：约定从活期账户每月自动扣转固定金额，无需客户操作。

### 按用途主题分类
- **普通零存整取**：无特定用途限制。
- **教育储蓄**：面向在校学生，用于积累教育资金，部分银行提供利率优惠。
- **养老零存整取**：面向即将退休人群，到期正好进入退休阶段。
- **梦想储蓄**：面向年轻客群，用于旅行、购车等目标储蓄。

## 适用对象与准入条件

### 适用对象
- 月收入稳定的工薪族，希望养成储蓄习惯
- 青少年学生，培养理财意识（监护人代办）
- 有明确中期储蓄目标的客户（如攒首付、购车、旅游）
- 自制力较弱、需要"强制储蓄"机制的消费人群
- 农村地区的定期储蓄客户（农商行重点客群）

### 准入条件
- 持有效身份证件的自然人
- 起存金额：每月固定存入金额最低5元，最高不超过2万元（各行标准略有差异）
- 每月存入金额须为固定数额，一经约定不可变更
- 同一客户可开立多笔零存整取账户
- 未满16周岁需由监护人代理开户

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 起存金额 | 每月5元起存，最高月存2万元 |
| 期限档次 | 1年、3年、5年 |
| 存款利率（2025年5月挂牌） | 1年0.65%、3年0.85%、5年0.85% |
| 存入方式 | 每月固定存入约定金额 |
| 计息方式 | 积数计息法（月积数法） |
| 到期支取 | 一次支取全部本息 |
| 漏存补存 | 次月可补存上月漏存金额，未补存按活期计息 |
| 提前支取 | 按实际存期和活期利率计息 |

### 计息规则详解

#### 月积数计息法
零存整取采用月积数计息法，计算公式为：

**利息 = 月存金额 × 累计月积数 × 月利率**

其中，累计月积数 = (存入次数 + 1) ÷ 2 × 存入次数

例如：每月存1000元，存期1年（12个月）：
- 累计月积数 = (12 + 1) ÷ 2 × 12 = 78
- 月利率 = 0.65% ÷ 12 = 0.05417%
- 利息 = 1000 × 78 × 0.05417% ≈ 42.25元
- 到期本息 = 12000 + 42.25 = 12042.25元

#### 漏存处理规则
1. **允许补存**：某月漏存后，次月可以补存（即次月存入双倍金额）。
2. **未补存处理**：未补存的月份，该笔存入金额按活期利率计息。
3. **多次漏存**：如连续多个月漏存，部分银行规定视为违约，剩余月份按活期计息。
4. **建议**：开通自动扣款功能可有效避免漏存问题。

## 政策法规依据

1. **《储蓄管理条例》**（国务院令第107号，2011年修订）：零存整取的基本法规依据。
2. **《中国人民银行关于人民币存贷款计结息问题的通知》**（银发〔2005〕129号）：计息规则规范。
3. **《存款保险条例》**（国务院令第660号）：纳入存款保险保障。
4. **《商业银行服务价格管理办法》**：零存整取不收取开户费和管理费。

## 城商行/农商行特色

### 产品创新
- **"梦想储蓄计划"**：部分城商行推出主题化零存整取产品，客户可设定储蓄目标（如旅行基金、数码产品），手机银行可视化展示进度，增加趣味性。
- **"亲子账户"**：家长为子女开立的零存整取账户，用于积累教育金、成长基金，部分银行提供利率上浮或积分奖励。
- **"惠农零存整取"**：农商行面向农村客户推出，结合农时特点，允许按季或按收获季存入（灵活变通月度存入要求）。

### 营销策略
- **利率优惠**：部分中小银行零存整取利率略高于大行，尤其是三年期和五年期。
- **积分奖励**：连续12个月按时存入可获额外积分或权益奖励。
- **联动营销**：零存整取到期自动转为定期存款或推荐理财产品，实现客户资产升级。
- **社区活动**：组织"储蓄达人"评选、"攒钱打卡"等活动，增强客户参与感。

### 普惠金融
- **低门槛覆盖**：5元起存的超低门槛，确保低收入群体也能参与。
- **农村储蓄服务**：农商行通过村级金融服务站、流动银行车等渠道，为偏远农村客户提供零存整取服务。
- **社保联动**：农商行将零存整取与城乡居民养老保险缴费结合，帮助农民积累养老金。

## 常见问题

### Q1：零存整取每月可以存不同金额吗？
不可以。零存整取要求每月固定存入约定金额，金额一经确定不可变更。如果某月漏存，次月可以补存（存入双倍金额）。如需灵活存入金额，建议选择活期存款或开放式理财产品。

### Q2：零存整取和整存整取哪个更划算？
从收益角度看，整存整取利率更高。以一年期为例，整存整取利率0.95%，零存整取利率0.65%。但两者的定位不同：整存整取适合一次性有较大闲置资金的客户；零存整取适合每月有固定结余、希望养成储蓄习惯的客户。如果每月有1000元结余，一年积累后选择整存整取也是可行的方案。

### Q3：零存整取可以提前支取吗？
可以提前支取，但提前支取部分按活期存款利率计息，利息损失较大。建议尽量避免提前支取，如有紧急资金需求，可考虑部分提前支取（部分银行支持），留存部分继续按原约定执行。

### Q4：自动扣款失败了怎么办？
如约定的活期账户余额不足导致自动扣款失败，该月视为漏存。建议确保扣款日前活期账户有足够余额。次月可补存上月漏存金额（存入双倍）。如连续多月扣款失败，部分银行可能暂停自动扣款协议，需到银行重新办理。

### Q5：零存整取适合什么人群？
零存整取最适合以下人群：一是月收入稳定但结余不多的工薪族，通过强制储蓄积累资金；二是想培养储蓄习惯的年轻人，控制消费冲动；三是为孩子积累教育金的家长；四是有明确中期消费目标（如旅行、购车首付）的人群。对于有大额闲置资金的客户，建议选择整存整取或大额存单，收益更高。
