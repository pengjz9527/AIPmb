# 大额存单

## 产品概述

大额存单（Certificate of Deposit, CD）是指由银行业存款类金融机构面向非金融机构投资人发行的、以人民币计价的记账式大额存款凭证，是银行存款类金融产品，属一般性存款。大额存单由中国人民银行于2015年6月正式推出，是利率市场化改革的重要产物。

大额存单兼具定期存款的安全性和相对较高的收益性，同时具备可转让、可质押的特点，是中高端客户资产配置的重要组成部分。对于城商行和农商行而言，大额存单是吸收大额资金、稳定中长期负债的关键产品，也是与国有大行竞争优质存款客户的重要工具。

## 产品分类

### 按付息方式分类
- **到期一次还本付息型**：最常见类型，到期一次性支取本金和利息。
- **按月付息型**：每月支付利息，到期归还本金，适合需要定期现金流的客户。
- **按季付息型**：每季度支付利息，到期归还本金。

### 按是否可转让分类
- **可转让大额存单**：到期前可通过银行平台转让给其他投资者，提升流动性。
- **不可转让大额存单**：只能持有到期或提前支取（按活期计息）。

### 按期限分类
大额存单期限品种丰富，包括：1个月、3个月、6个月、9个月、1年、18个月、2年、3年、5年共9个标准期限。

### 按发行方式分类
- **标准化产品**：按统一格式发行，利率公开透明。
- **定制化产品**：针对特定客户群体定制发行，金额和期限可灵活约定。

## 适用对象与准入条件

### 适用对象
- 有大额闲置资金的个人客户（20万元以上）
- 追求稳健收益的中高端客户
- 有资产配置需求的高净值客户
- 企业、事业单位、社会团体等机构客户（机构版大额存单起存金额1000万元）
- 有资金流动性管理需求的客户（选择可转让型）

### 准入条件
- 个人投资者：起存金额不低于20万元人民币
- 机构投资者：起存金额不低于1000万元人民币
- 持有效身份证件（个人客户）
- 已开立银行结算账户
- 部分银行要求提前预约（额度有限时）

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 起存金额 | 个人20万元起，机构1000万元起 |
| 期限品种 | 1个月、3个月、6个月、9个月、1年、18个月、2年、3年、5年 |
| 利率类型 | 固定利率为主，部分银行发行浮动利率 |
| 利率水平 | 通常高于同期限普通定期存款10-30BP |
| 计息方式 | 按实际天数计息，ACT/365 |
| 付息方式 | 到期一次还本付息/按月付息/按季付息 |
| 提前支取 | 可提前支取，靠档计息或按活期计息（各行规则不同） |
| 转让功能 | 部分产品支持到期前转让（通过银行平台） |
| 质押功能 | 可作为质押物办理贷款 |

### 转让功能详解
1. **转让方式**：通过发行银行的手机银行或网上银行平台进行转让，买卖双方在线完成交易。
2. **转让价格**：由卖方自主定价（通常按持有天数和票面利率计算应收本息，可溢价或折价转让）。
3. **转让费用**：部分银行收取少量转让手续费（通常不超过转让金额的0.1%）。
4. **受让方要求**：受让方需在该银行开立账户，且购买金额需满足起存门槛。
5. **转让优势**：解决了定期存款提前支取按活期计息的痛点，兼顾收益和流动性。

### 提前支取规则
- 大额存单允许提前支取，但规则因银行而异。
- **靠档计息**（部分银行）：按实际存期对应的普通定期利率计息（监管已要求整改，逐步取消）。
- **按活期计息**（多数银行现行规则）：提前支取部分按支取日活期利率计息。
- 部分银行规定大额存单不允许部分提前支取，只能全额提前支取。

## 政策法规依据

1. **《大额存单管理暂行办法》**（中国人民银行公告〔2015〕第13号）：大额存单的基本制度规范。
2. **《关于完善商业银行存款利率市场化形成机制的通知》**：大额存单发行利率市场化定价依据。
3. **《存款保险条例》**（国务院令第660号）：大额存单纳入存款保险保障范围，50万元以内全额偿付。
4. **《关于进一步规范大额存单业务的通知》**：规范大额存单提前支取靠档计息等行为。
5. **中国人民银行利率市场化相关政策**：大额存单作为利率市场化重要工具的政策定位。

## 城商行/农商行特色

### 利率竞争优势
- **利率溢价明显**：城商行和农商行大额存单利率通常比国有大行高出20-50BP，对大额资金客户吸引力较强。
- **灵活定价策略**：根据本行资金需求和市场竞争情况，灵活调整各期限品种的利率水平。
- **定制化发行**：针对高端客户、企业客户定制发行专属大额存单，提供更有竞争力的利率。

### 产品差异化
- **按月付息产品**：中小银行更积极推出按月付息型大额存单，满足退休客户、经营客户的现金流需求。
- **可转让产品**：部分城商行开发了自有的大额存单转让平台，提升产品流动性。
- **拼团购买**：部分银行推出"拼团"模式，多位客户联合认购可享受更高利率。

### 营销策略
- **季节性营销**：在"开门红"、年中、年末等关键时点集中发行高利率大额存单，吸引资金回流。
- **客户分层**：对不同等级客户提供差异化利率，如私行客户专属大额存单利率更高。
- **额度控制**：通过限量发行制造稀缺性，引导客户抢购，提升品牌影响力。
- **存贷联动**：大额存单客户可享受贷款优惠利率、信用卡高额度等综合权益。

### 挑战与应对
- **负债成本压力**：高利率大额存单增加了银行负债成本，需平衡存款增长与盈利能力。
- **监管约束**：监管要求规范大额存单利率定价，避免恶性竞争。
- **5年期停发趋势**：部分大行已停发5年期大额存单，中小银行也在逐步压缩长期限产品。

## 常见问题

### Q1：大额存单和普通定期存款有什么区别？
主要区别有五点：一是起存金额不同，大额存单个人20万起，定期存款50元起；二是利率不同，大额存单利率通常高于同期限普通定期10-30BP；三是流动性不同，大额存单可转让、可质押；四是付息方式更灵活，大额存单支持按月/按季付息；五是发行方式不同，大额存单按期发行，有额度限制，定期存款随时可存。

### Q2：大额存单安全吗？受存款保险保障吗？
大额存单是一般性存款产品，受《存款保险条例》保障，同一存款人在同一家银行50万元以内的存款本息全额偿付。但需注意，如购买金额超过50万元，超出部分在银行破产时不享受全额保障。建议大额资金分散存放于多家银行。

### Q3：大额存单可以转让是怎么回事？
可转让大额存单允许持有人在到期前通过银行平台将存单转让给其他投资者。转让价格由卖方自主确定，通常按持有天数和票面利率计算合理价格。转让后卖方提前收回资金（可能获得高于活期的收益），买方获得剩余期限的存单收益。这一功能有效解决了提前支取按活期计息的问题。

### Q4：为什么有些银行买不到大额存单？
大额存单采用限额发行方式，每期有总额度限制。由于中小银行大额存单利率较高，市场需求旺盛，经常出现"秒杀"现象——发行当天额度即被抢完。建议关注银行发行公告，在发行首日尽早购买。部分银行对VIP客户提供优先购买权。

### Q5：大额存单利率还会继续下降吗？
从2024-2025年的趋势看，大额存单利率整体呈下行态势。这主要受以下因素影响：一是央行持续引导降低银行负债成本以支持实体经济；二是市场利率整体下行；三是银行净息差收窄压力。建议有大额闲置资金的客户尽早锁定较高利率的中长期大额存单。
