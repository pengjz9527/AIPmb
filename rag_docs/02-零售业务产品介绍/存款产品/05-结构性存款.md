# 结构性存款

## 产品概述

结构性存款是指商业银行吸收的嵌入金融衍生产品的存款，通过与利率、汇率、指数等的波动挂钩或者与某实体的信用情况挂钩，使存款人在承担一定风险的基础上获得相应的收益。结构性存款本质上仍属于存款类产品，纳入存款保险保障范围（保本部分），但同时嵌入了金融衍生品，使收益具有不确定性。

结构性存款是银行在利率市场化背景下推出的创新型产品，满足了客户在保本前提下追求更高收益的需求。对于城商行和农商行而言，结构性存款是丰富产品线、提升竞争力的重要工具，有助于吸引风险偏好略高于纯存款客户的客群。

## 产品分类

### 按挂钩标的分类
- **利率挂钩型**：收益与LIBOR、SHIBOR、LPR等利率指标挂钩。
- **汇率挂钩型**：收益与美元/人民币、欧元/人民币等汇率波动挂钩。
- **指数挂钩型**：收益与沪深300、中证500、标普500等股票指数挂钩。
- **商品价格挂钩型**：收益与黄金、原油、铜等大宗商品价格挂钩。
- **信用挂钩型**：收益与特定企业或债券的信用事件挂钩。

### 按收益结构分类
- **保本固定+浮动型**：保证最低收益（如1%），浮动部分取决于挂钩标的表现。
- **保本不保收益型**：保证本金安全，但收益完全浮动，最低可能为零收益。
- **阶梯收益型**：挂钩标的落在不同区间对应不同收益率。
- **二元/鲨鱼鳍型**：挂钩标的触及某个触发点则获得高收益，否则获得低收益。

### 按期限分类
- **短期产品**：7天至3个月
- **中期产品**：3个月至1年
- **长期产品**：1年以上（较少见）

## 适用对象与准入条件

### 适用对象
- 风险承受能力为稳健型及以上的零售客户
- 有保本需求但希望获取更高收益的投资者
- 对金融市场有一定了解的投资者
- 资产配置中需要低风险衍生品敞口的客户
- 企业客户的闲置资金管理需求

### 准入条件
- 起存金额：一般1万元起（部分产品5万元或更高）
- 风险承受能力评估：需完成风险测评，结果需达到"稳健型"及以上
- 持有效身份证件
- 已开立银行结算账户
- 需签署结构性存款产品协议和风险揭示书
- 部分复杂产品可能要求客户具备投资经验

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 起存金额 | 一般1万元起，部分产品5万元/10万元起 |
| 期限 | 7天至1年为主，部分产品可达2年 |
| 本金保障 | 保本（纳入存款保险） |
| 预期收益 | 通常为1%-5%（浮动，非承诺） |
| 最低收益 | 一般为0%-1.5%（视产品而定） |
| 计息方式 | 到期一次还本付息 |
| 提前支取 | 一般不允许提前支取 |
| 挂钩标的 | 利率、汇率、指数、商品价格等 |
| 收益确定 | 到期根据挂钩标的实际表现确定 |

### 收益结构示例

以一款挂钩沪深300指数的结构性存款为例：

| 沪深300涨跌幅 | 年化收益率 |
|--------------|-----------|
| 涨幅 ≥ 5% | 4.50% |
| 0% ≤ 涨幅 < 5% | 3.00% |
| -5% ≤ 跌幅 < 0% | 1.50% |
| 跌幅 < -5% | 0.50% |

### 风险等级
结构性存款虽然保本，但收益具有不确定性，风险等级通常为：
- **R1（低风险）**：保证最低收益的产品（如最低1%）。
- **R2（中低风险）**：保本但不保最低收益，收益完全浮动。
- 注意：结构性存款不同于结构性理财产品，后者不保本。

## 政策法规依据

1. **《关于进一步规范商业银行结构性存款业务的通知》**（银保监办发〔2019〕204号）：结构性存款业务的专项监管规定，要求银行具备衍生品交易资格。
2. **《商业银行理财业务监督管理办法》**：结构性存款参照理财产品进行销售管理。
3. **《存款保险条例》**（国务院令第660号）：结构性存款本金纳入存款保险保障。
4. **《银行业金融机构衍生产品交易业务管理办法》**：银行开展结构性存款业务的资格要求。
5. **《商业银行资本管理办法》**：结构性存款的风险权重和资本要求。

## 城商行/农商行特色

### 产品设计特点
- **简单易懂**：城商行/农商行的结构性存款产品设计相对简单，挂钩标的以利率、汇率为主，避免过于复杂的衍生品结构。
- **收益区间透明**：明确展示不同情景下的收益率，帮助客户理解产品。
- **短期产品为主**：以1个月至6个月期限为主，降低客户资金锁定时间。
- **本地化标的**：部分产品挂钩与本地经济相关的指标，如地方政府债利率等。

### 竞争策略
- **较高预期收益**：相比国有大行，中小银行的结构性存款通常提供更高的预期收益率，以吸引客户。
- **低门槛**：起存金额设置较低（1万元），扩大客户覆盖面。
- **组合营销**：将结构性存款作为客户从存款向理财过渡的"桥梁产品"，逐步引导客户接受净值型产品。
- **企业客户拓展**：针对本地企业客户的闲置资金，定制结构性存款方案。

### 挑战与风险
- **衍生品能力不足**：中小银行的衍生品交易能力相对较弱，部分银行通过与其他金融机构合作（如大型券商）进行风险对冲。
- **合规风险**：需确保产品设计符合监管要求，不得将结构性存款与理财产品混淆销售。
- **"假结构"问题**：监管严格禁止"假结构"——即实际上保本保收益但通过虚假衍生品结构规避监管的产品。

## 常见问题

### Q1：结构性存款保本吗？安全吗？
结构性存款的本金是受保障的。根据监管规定，结构性存款属于存款类产品，本金纳入存款保险保障范围（50万元以内全额偿付）。但需注意，保本不意味着保收益，浮动收益部分取决于挂钩标的的实际表现，可能获得较低收益甚至零收益。

### Q2：结构性存款和银行理财有什么区别？
两者的核心区别在于：一是保本与否——结构性存款保本（本金受存款保险保障），净值型理财产品不保本；二是产品性质——结构性存款是存款类产品，理财是资产管理类产品；三是收益确定性——结构性存款收益虽有浮动但有区间，理财收益完全取决于市场表现；四是监管要求——结构性存款要求银行具备衍生品交易资格。

### Q3：结构性存款可以提前支取吗？
一般不允许提前支取。结构性存款在起息后即锁定期限，到期才能支取。这是由于其内嵌的衍生品结构需要锁定到期日才能确定收益。如果客户有流动性需求，建议选择通知存款、短期定期存款或开放式理财产品。

### Q4：为什么结构性存款的实际收益经常低于预期？
结构性存款的预期收益通常展示的是"最高可能收益"或"中间情景收益"，而实际收益取决于挂钩标的在观察日的真实表现。如果市场走势不利于挂钩条件，实际收益可能落在较低区间。建议购买前仔细阅读产品说明书中的收益计算规则，理解不同情景下的收益情况。

### Q5：购买结构性存款需要做风险测评吗？
需要。根据监管要求，结构性存款参照理财产品进行销售管理，客户首次购买前必须完成风险承受能力评估，评估结果需达到"稳健型"及以上才能购买。风险测评通常在柜面或手机银行完成，有效期一般为一年。
