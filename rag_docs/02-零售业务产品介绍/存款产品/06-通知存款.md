# 通知存款

## 产品概述

通知存款是指客户在存入款项时不约定存期，支取时需提前通知银行，约定支取日期和金额方能支取存款的一种存款方式。通知存款兼具活期存款的灵活性和高于活期的收益性，是大额短期闲置资金管理的理想选择。

通知存款填补了活期存款与定期存款之间的产品空白，特别适合资金使用时间不确定但有一定沉淀期的客户。对于城商行和农商行而言，通知存款是吸收短期大额资金的重要工具，也是企业客户和高端个人客户短期资金管理的常用产品。在当前利率环境下，通知存款以相对灵活的特性，成为定期存款和活期存款之间的重要补充产品。

## 产品分类

### 按通知期限分类
- **一天通知存款**：支取前需提前1天通知银行。
- **七天通知存款**：支取前需提前7天通知银行，利率高于一天通知。

### 按客户类型分类
- **个人通知存款**：面向个人客户，起存金额5万元。
- **单位通知存款**：面向企业客户，起存金额50万元。

### 按产品形态分类
- **传统通知存款**：需客户主动发出支取通知。
- **智能通知存款**：银行系统自动识别满足通知期限的资金，自动按通知存款利率计息，无需客户手动操作（部分银行提供）。

### 按计息方式分类
- **普通通知存款**：按通知期限对应利率计息。
- **定活两便型**：部分银行推出的变体，按实际存期靠档计息。

## 适用对象与准入条件

### 适用对象
- 有大额短期闲置资金的个人客户（5万元以上）
- 资金使用时间不确定的客户
- 等待投资时机的客户（股市、房市等待窗口期资金）
- 企业短期周转资金管理
- 大额交易结算前的过渡资金
- 代发工资等批量资金短期沉淀

### 准入条件
- 个人客户起存金额：5万元人民币
- 单位客户起存金额：50万元人民币
- 持有效身份证件
- 已开立银行结算账户
- 部分银行要求首次办理需到柜面签约

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 起存金额 | 个人5万元，单位50万元 |
| 通知期限 | 1天通知、7天通知 |
| 存款利率（2025年5月挂牌） | 1天通知0.10%、7天通知0.30% |
| 计息方式 | 按日计息，支取时结息 |
| 计息基础 | 实际存款天数 |
| 最低支取金额 | 个人5万元，单位10万元 |
| 留存金额 | 不低于起存金额（5万元/50万元） |
| 通知方式 | 柜面、手机银行、电话银行等 |
| 自动转存 | 部分银行支持自动续存 |

### 利率对比

| 存款类型 | 利率 | 适用场景 |
|----------|------|----------|
| 活期存款 | 0.05% | 日常结算资金 |
| 1天通知存款 | 0.10% | 随时可能使用的大额资金 |
| 7天通知存款 | 0.30% | 短期闲置（1周以上） |
| 3个月定期 | 0.65% | 确定3个月不用的资金 |

### 支取规则详解
1. **通知义务**：支取前必须提前发出通知，一天通知需提前1个工作日，七天通知需提前7个自然日。
2. **约定金额**：通知时需约定支取金额，实际支取金额与约定金额不符的，按活期利率计息。
3. **取消通知**：通知后可以取消，但需重新计算通知期。
4. **未通知支取**：未经通知直接支取的，按活期存款利率计息。
5. **部分支取**：支持部分支取，但支取后留存金额不得低于起存金额。
6. **最低支取金额**：个人每次最低支取5万元。

### 智能通知存款
部分银行推出"智能通知存款"产品，特点如下：
- 无需客户主动发出支取通知
- 系统自动识别账户中满足通知期限要求的资金
- 满足7天未动的资金自动按7天通知利率计息
- 满足1天未动的资金自动按1天通知利率计息
- 兼顾了活期的便利性和通知存款的收益性

## 政策法规依据

1. **《储蓄管理条例》**（国务院令第107号，2011年修订）：通知存款的基本法规依据。
2. **《中国人民银行关于人民币存贷款计结息问题的通知》**（银发〔2005〕129号）：计息规则规范。
3. **《人民币单位存款管理办法》**：单位通知存款管理规范。
4. **《存款保险条例》**（国务院令第660号）：通知存款纳入存款保险保障。
5. **各商业银行通知存款业务管理办法**：各行内部业务规范。

## 城商行/农商行特色

### 产品优化
- **智能通知存款升级**：部分城商行推出"智能存款管家"类产品，自动将活期账户中超过约定金额的资金归集至通知存款账户，支取时自动赎回，实现"零操作"的通知存款管理。
- **T+0到账**：部分银行支持通知存款支取实时到账，提升资金周转效率。
- **自动通知功能**：客户存入即自动发出通知，满通知期后可随时支取，无需客户手动操作。

### 利率策略
- **利率上浮**：部分中小银行通知存款利率高于大行挂牌利率，如7天通知存款可达0.35%-0.45%。
- **阶梯利率**：根据存款金额设置阶梯利率，如50万元以上享受更高利率。
- **组合方案**：将通知存款与活期账户、理财产品组合，为客户提供流动性管理综合方案。

### 场景应用
- **房产交易过渡资金**：二手房交易中的资金监管期，使用通知存款管理过渡资金。
- **经营周转资金**：个体工商户和小微企业主的短期经营周转资金管理。
- **拆迁补偿资金管理**：城商行/农商行在城镇化过程中，为拆迁户提供大额补偿资金的通知存款管理服务。
- **年终奖/分红管理**：春节前后的年终奖、企业分红等大额资金短期管理。

## 常见问题

### Q1：通知存款和活期存款有什么区别？
主要区别有三点：一是利率不同，通知存款利率高于活期存款（如7天通知0.30% vs 活期0.05%）；二是支取方式不同，通知存款需提前通知银行约定支取日期和金额，活期存款可随时支取；三是起存金额不同，通知存款个人5万元起存，活期存款1元起存。通知存款适合大额短期闲置资金，活期存款适合日常结算。

### Q2：通知存款忘记通知就直接取款会怎样？
未经通知直接支取的，该部分存款按活期存款利率（0.05%）计息，利息损失较大。建议养成提前通知的习惯，或使用智能通知存款产品，避免遗忘。目前多数银行的手机银行支持在线发出通知，操作便捷。

### Q3：智能通知存款和普通通知存款哪个好？
智能通知存款的优势在于无需客户主动发出通知，系统自动按规则计息，操作更便捷。但部分银行的智能通知存款利率可能与普通通知存款略有差异。建议优先选择智能通知存款产品，既享受通知存款的利率，又避免操作遗忘带来的损失。

### Q4：通知存款可以部分支取吗？
可以部分支取，但需注意：一是每次支取金额不低于5万元（个人）；二是支取后账户留存金额不得低于起存金额（5万元）；三是实际支取金额需与通知约定金额一致，否则按活期利率计息。如需支取全部资金，建议发出全额支取通知。

### Q5：通知存款受存款保险保障吗？
是的。通知存款属于一般性存款产品，纳入存款保险保障范围，同一存款人在同一家银行50万元以内的存款本息全额偿付。对于超过50万元的大额资金，建议分散存放于多家银行。
