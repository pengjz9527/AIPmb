# 个人协定存款

## 产品概述

个人协定存款是指个人客户与银行签订协定存款合同，约定账户中保留一定金额的备付金（留存金额），超过留存金额的部分按协定存款利率计付利息的存款产品。协定存款本质上是活期存款的升级版，在保证资金流动性的同时，为超额资金提供高于活期的利息收益。

个人协定存款最初主要面向企业客户（单位协定存款），近年来部分银行将这一产品延伸至个人客户领域，尤其是面向代发工资客户、个体工商户等高净值个人客户。对于城商行和农商行而言，个人协定存款是维护优质客户关系、提高资金沉淀的重要手段。

## 产品分类

### 按客户类型分类
- **个人协定存款**：面向个人客户，留存金额起点较低（通常5万元起）。
- **单位协定存款**：面向企业客户，留存金额起点较高（通常50万元起）。

### 按产品结构分类
- **标准协定存款**：超过留存金额部分按协定利率计息。
- **阶梯协定存款**：根据超额资金规模设置阶梯利率，资金越多利率越高。
- **智能协定存款**：系统自动归集和计算，无需客户手动操作。

### 按留存金额类型分类
- **固定留存型**：留存金额固定不变。
- **动态留存型**：留存金额根据账户历史流水自动调整。

## 适用对象与准入条件

### 适用对象
- 代发工资客户（月工资入账后有一定结余）
- 个体工商户（经营资金频繁进出但有阶段性沉淀）
- 高净值个人客户（大额资金短期管理）
- 拆迁补偿客户（大额补偿资金短期管理）
- 有稳定现金流但资金使用不规律的客户

### 准入条件
- 持有效身份证件的个人客户
- 与银行签订《个人协定存款合同》
- 约定留存金额（一般不低于5万元）
- 已开立活期结算账户
- 部分银行要求为VIP客户或达标客户

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 留存金额 | 由客户与银行约定，一般5万元起 |
| 协定利率（2025年5月挂牌） | 0.10%（各行可能有差异） |
| 活期利率 | 0.05%（留存金额部分按活期利率计息） |
| 计息方式 | 留存金额按活期利率计息，超额部分按协定利率计息 |
| 结息方式 | 按季结息（每季末月20日） |
| 起存金额 | 无额外起存要求，在活期账户基础上签约 |
| 存取方式 | 随存随取，与活期存款一致 |
| 合同期限 | 一般为1年，到期可续约 |
| 费用 | 不收取管理费 |

### 计息规则详解

#### 双层利率结构
- **留存金额以内**：按活期存款利率计息（0.05%）。
- **超过留存金额部分**：按协定存款利率计息（0.10%）。

#### 计息示例
假设客户约定留存金额5万元，协定利率0.10%，活期利率0.05%：

| 账户余额 | 留存部分利息 | 超额部分利息 | 合计利息（季度） |
|----------|-------------|-------------|-----------------|
| 5万元 | 5万×0.05%÷4 ≈ 6.25元 | 0 | 6.25元 |
| 10万元 | 5万×0.05%÷4 ≈ 6.25元 | 5万×0.10%÷4 ≈ 12.50元 | 18.75元 |
| 50万元 | 5万×0.05%÷4 ≈ 6.25元 | 45万×0.10%÷4 ≈ 112.50元 | 118.75元 |

#### 特殊情况处理
1. **余额低于留存金额**：全部按活期利率计息。
2. **账户销户**：销户时按实际天数计息，留存部分按活期、超额部分按协定利率。
3. **利率调整**：如遇利率调整，按调整后的利率执行，分段计息。

## 政策法规依据

1. **《人民币银行结算账户管理办法》**（中国人民银行令〔2003〕第5号）：协定存款账户管理规范。
2. **《人民币单位存款管理办法》**：单位协定存款管理规范（个人协定存款参照执行）。
3. **《中国人民银行关于人民币存贷款计结息问题的通知》**（银发〔2005〕129号）：计息规则。
4. **《存款保险条例》**（国务院令第660号）：协定存款纳入存款保险保障。
5. **各商业银行个人协定存款业务管理办法**：各行内部业务规范。

## 城商行/农商行特色

### 差异化利率
- **协定利率上浮**：部分城商行/农商行个人协定存款利率高于大行挂牌利率，如可达0.15%-0.20%。
- **阶梯协定利率**：根据超额资金规模设置阶梯利率，如超额10万以下0.12%、10-50万0.15%、50万以上0.20%。
- **VIP客户专属利率**：对达标VIP客户提供更优惠的协定利率。

### 组合营销
- **代发工资联动**：代发工资客户自动签约协定存款，提升客户体验和资金留存率。
- **商户资金管理**：为收单商户提供协定存款服务，经营资金入账自动享受协定利率。
- **存贷联动**：协定存款客户可享受贷款利率优惠或信用卡优先审批。

### 智能化管理
- **自动归集**：客户名下多个账户的资金自动归集至协定存款账户。
- **智能提醒**：账户余额低于留存金额时自动提醒。
- **定期报告**：每月推送协定存款收益报告，展示与普通活期的收益对比。

### 农村场景应用
- **惠农协定存款**：农商行面向农村种植大户、养殖大户推出的专属协定存款，留存金额设置更灵活。
- **季节性调整**：根据农时季节调整留存金额，农忙季节降低留存金额，农闲季节提高留存金额。

## 常见问题

### Q1：个人协定存款和普通活期存款相比收益高多少？
协定存款的超额部分利率（约0.10%）是活期利率（0.05%）的2倍。以账户常年保持30万元余额、留存金额5万元为例：普通活期季度利息约37.50元，协定存款季度利息约68.75元，多赚约31.25元，年化多赚约125元。虽然绝对金额不大，但完全无额外操作和风险。

### Q2：签约协定存款后资金还能随时取吗？
可以。协定存款的资金流动性与活期存款完全一致，可以随时存取、转账、消费。系统自动按日计算留存部分和超额部分的利息，客户无需任何额外操作。但如果余额低于留存金额，全部按活期利率计息。

### Q3：协定存款的留存金额可以修改吗？
可以修改。客户可通过柜面或手机银行修改留存金额，但修改后的留存金额需满足银行规定的最低标准（一般5万元）。修改即时生效或次日起生效（各行规定不同）。建议根据账户资金常态水平合理设置留存金额。

### Q4：个人协定存款适合什么样的客户？
最适合以下客户：一是代发工资客户，每月工资入账后有一定结余但不确定何时使用；二是个体工商户，经营资金频繁进出但有阶段性沉淀（如日均余额常在10万元以上）；三是高净值客户，大额资金需要流动性管理但又不想只享受活期利率。

### Q5：协定存款受存款保险保障吗？
是的。协定存款属于一般性存款产品，纳入存款保险保障范围，同一存款人在同一家银行50万元以内的存款本息全额偿付。协定存款的安全性与普通活期存款、定期存款完全相同。
