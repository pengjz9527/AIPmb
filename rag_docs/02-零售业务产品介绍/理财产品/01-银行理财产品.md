# 银行理财产品

## 产品概述

银行理财产品是指商业银行通过发行理财产品募集资金，按照约定方式和比例投资于债券、货币市场工具、非标准化债权、权益类资产等的资产管理产品。2022年资管新规全面落地后，银行理财产品已全面实现净值化管理，打破刚性兑付，产品收益与底层资产表现直接挂钩。

银行理财产品是居民资产配置的重要组成部分，也是银行中间业务收入的核心来源。对于城商行和农商行而言，理财产品是留住客户资金、提升AUM（管理资产规模）的关键产品。在净值化时代，中小银行面临投研能力、产品创新等方面的挑战，部分银行选择通过理财子公司或代销方式参与理财市场。

## 产品分类

### 按投资性质分类（资管新规分类）
- **固定收益类产品**：80%以上资产投资于债券、存款等固收资产，风险较低。
- **权益类产品**：80%以上资产投资于股票等权益资产，风险较高。
- **商品及金融衍生品类产品**：投资于商品、衍生品等，风险特殊。
- **混合类产品**：不符合以上三类标准的产品，资产配置灵活。

### 按运作方式分类
- **封闭式产品**：有固定期限，到期前不可赎回，收益相对稳定。
- **开放式产品**：无固定期限，可在开放日申购/赎回，流动性较好。
- **定期开放产品**：定期开放申购/赎回（如每季度、每半年开放一次）。

### 按风险等级分类（银行产品风险评级）

| 风险等级 | 名称 | 适合客户 | 产品类型 |
|----------|------|----------|----------|
| R1 | 低风险 | 保守型 | 货币市场类、现金管理类 |
| R2 | 中低风险 | 稳健型 | 固收类、部分固收+ |
| R3 | 中风险 | 平衡型 | 固收+、混合类 |
| R4 | 中高风险 | 进取型 | 混合类、权益类 |
| R5 | 高风险 | 激进型 | 权益类、衍生品 |

### 按发行主体分类
- **银行自营理财**：银行自主发行的理财产品。
- **理财子公司产品**：银行理财子公司发行的产品。
- **代销理财产品**：代销其他机构（如基金、保险、信托）的产品。

## 适用对象与准入条件

### 适用对象
- 有一定风险承受能力的个人投资者
- 追求超越存款收益的稳健型投资者
- 有资产配置需求的中高端客户
- 对流动性有不同需求的客户（短期/中期/长期）

### 准入条件
| 条件 | 具体要求 |
|------|----------|
| 年龄 | 18周岁以上（部分产品65岁以上有限制） |
| 风险测评 | 完成风险承受能力评估，结果匹配产品风险等级 |
| 起购金额 | R1/R2产品：1元起；R3/R4产品：1万元起；R5产品：更高 |
| 账户要求 | 已开立银行结算账户 |
| 合格投资者 | 私募类产品要求合格投资者（金融资产≥300万或年收入≥50万） |

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 起购金额 | 公募产品：1元起（部分1万元起）；私募产品：100万元起 |
| 产品期限 | 开放式（随时申赎）至封闭式（最长5年以上） |
| 预期收益 | 非承诺收益，参考业绩比较基准（如3.0%-5.0%） |
| 净值管理 | 每日/定期公布净值（单位净值） |
| 申购赎回 | 开放式产品可在开放日申赎；封闭式不可提前赎回 |
| 费用 | 管理费（0.1%-0.5%）、托管费（0.02%-0.1%）、申赎费 |
| 信息披露 | 定期公布产品报告、净值、持仓信息 |

### 固收+产品详解
"固收+"是近年来银行理财的主流策略：
- **固收底仓**：70%-90%投资于债券、存款等低风险资产，提供基础收益。
- **"+"策略**：10%-30%投资于权益、衍生品等高弹性资产，博取超额收益。
- **风险控制**：通过严格的风险预算和止损机制控制回撤。
- **目标收益**：年化3%-6%（非承诺）。

### 现金管理类产品
- **定义**：投资于货币市场工具的开放式理财产品。
- **特点**：T+0或T+1赎回，流动性接近活期存款。
- **收益**：年化约1.5%-2.5%（2025年参考）。
- **风险等级**：R1（低风险）。
- **监管要求**：2022年后监管要求现金管理类产品对标货币基金，投资范围和久期受限。

| 渠道 | 可办理业务 | 备注 |
|------|-----------|------|

| 微信银行 | 产品推荐、购买入口 | 引流渠道 |
| 理财经理 | 专属推荐、资产配置建议 | 高端客户 |

## 政策法规依据

1. **《关于规范金融机构资产管理业务的指导意见》**（银发〔2018〕106号，"资管新规"）：资产管理业务的顶层制度。
2. **《商业银行理财业务监督管理办法》**（银保监会令2018年第6号，"理财新规"）：银行理财业务的专项监管。
3. **《商业银行理财子公司管理办法》**（银保监会令2018年第7号）：理财子公司管理制度。
4. **《关于规范现金管理类理财产品管理有关事项的通知》**（2021年）：现金管理类理财专项规范。
5. **《商业银行理财产品销售管理办法》**：理财产品销售规范。
6. **《理财公司内部控制管理办法》**（2022年）：理财公司治理要求。

## 城商行/农商行特色

### 产品策略
- **固收类产品为主**：中小银行理财产品以R1-R2级固收类产品为主，风险相对可控。
- **短期产品优势**：开放式短期产品（如7天/14天/1个月）流动性好，适合中小银行客户的资金管理需求。
- **本地化资产配置**：投资本地城投债、本地企业债等，发挥本地信息优势。
- **代销补充**：自身投研能力不足时，通过代销大型理财子公司产品补充产品线。

### 理财子公司发展
- **独立理财子公司**：部分规模较大的城商行已设立独立理财子公司（如杭银理财、宁银理财等）。
- **联合设立**：部分中小银行通过参股方式联合设立理财子公司。
- **代销模式**：多数农商行选择代销大型理财子公司的产品，补充自身产品线。

### 客户服务
- **理财经理制度**：配备专职理财经理，为客户提供一对一的资产配置建议。
- **投资者教育**：通过讲座、沙龙、短视频等方式开展投资者教育，帮助客户理解净值化产品。
- **净值波动应对**：在市场波动时主动沟通客户，解释净值变动原因，避免客户恐慌赎回。

## 常见问题

### Q1：银行理财产品还保本吗？
根据资管新规，银行理财产品已全面打破刚性兑付，不再保本保收益。产品净值随市场波动，可能出现浮亏。但需注意，风险等级R1-R2的固收类产品风险较低，历史上出现大幅亏损的概率较小。如果追求绝对保本，建议选择存款类产品（如大额存单、定期存款）。

### Q2：什么是理财产品的"业绩比较基准"？
业绩比较基准是银行根据产品投资策略和历史表现设定的参考收益指标，不是承诺收益。实际收益可能高于或低于业绩比较基准。例如，业绩比较基准4.0%的理财产品，实际年化收益可能是3.5%或4.5%，取决于底层资产表现。

### Q3：理财产品出现亏损怎么办？
理财产品净值波动是正常现象。建议：一是理性看待短期波动，不要恐慌赎回（"追涨杀跌"往往造成实际亏损）；二是评估亏损原因（是市场系统性风险还是产品本身问题）；三是如果风险承受能力不足，可在开放日赎回并转向风险更低的产品。

### Q4：理财产品的费用有哪些？
理财产品费用主要包括：一是管理费（0.1%-0.5%/年，从产品净值中扣除）；二是托管费（0.02%-0.1%/年）；三是销售服务费（0.1%-0.5%/年）；四是申赎费（部分产品收取，部分免收）。这些费用已从产品净值中扣除，投资者看到的净值是扣费后的结果。

### Q5：如何选择适合自己的理财产品？
建议从以下维度选择：一是风险承受能力（完成风险评估，选择匹配的产品风险等级）；二是投资期限（根据资金使用计划选择产品期限）；三是收益预期（理解净值化产品的收益不确定性）；四是费用对比（选择费用较低的产品）；五是发行主体（选择信誉良好、管理能力强的银行或理财子公司）。
