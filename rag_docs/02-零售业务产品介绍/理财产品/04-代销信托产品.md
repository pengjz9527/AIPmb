# 代销信托产品

## 产品概述

代销信托产品是指商业银行作为销售渠道，代理销售信托公司发行的信托计划。信托产品是信托公司接受投资者委托，以投资者利益最大化为目标，对信托财产进行管理和处分的金融产品。信托产品以其灵活的制度优势和较高的预期收益，长期以来是高净值客户资产配置的重要选项。

信托产品在银行财富管理产品体系中定位于高端产品，面向合格投资者（高净值客户）。对于城商行和农商行而言，信托产品代销是服务高净值客户、提升中间业务收入的重要手段。但近年来信托行业经历深度调整，部分信托产品出现违约风险，对银行代销业务的合规管理和风险控制提出了更高要求。

## 产品分类

### 按信托资金投向分类
- **融资类信托**：资金投向基础设施、房地产、工商企业等融资项目。
- **投资类信托**：资金投向债券、股票、基金等标准化金融资产。
- **事务管理类信托**：提供资产托管、家族信托等服务。

### 按信托功能分类
- **集合资金信托计划**：多位投资者共同参与的标准化信托产品。
- **单一资金信托**：单一投资者定制化的信托产品。
- **财产权信托**：以财产权（如应收账款、股权）设立的信托。

### 按期限分类
- **短期信托**：1年以内。
- **中期信托**：1-3年。
- **长期信托**：3年以上。

### 按风险等级分类
- **低风险信托**：投资于国债、高等级债券等。
- **中风险信托**：投资于城投债、优质企业债等。
- **高风险信托**：投资于房地产、工商企业融资等。

## 适用对象与准入条件

### 适用对象
- 合格投资者（高净值客户）
- 有较高收益预期的投资者
- 有资产配置多元化需求的高端客户
- 有家族信托、财富传承需求的超高净值客户
- 有资产隔离需求的企业主

### 合格投资者标准
| 条件 | 具体要求 |
|------|----------|
| 金融资产 | 家庭金融净资产≥300万元 |
| 或金融资产 | 家庭金融资产≥500万元 |
| 或年收入 | 近3年本人年均收入≥40万元 |
| 或年收入 | 近3年家庭年均收入≥50万元 |
| 投资经验 | 具有2年以上投资经历 |

### 一般准入条件
- 年满18周岁具有完全民事行为能力
- 满足合格投资者认定标准
- 已完成投资者风险承受能力评估
- 签署信托合同和风险揭示书

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 起投金额 | 一般100万元起（部分产品300万元起） |
| 投资期限 | 1-3年为主 |
| 预期收益率 | 4.0%-7.0%（非承诺收益，资管新规后净值化管理） |
| 收益分配 | 按季/半年/年分配，或到期一次分配 |
| 转让限制 | 一般不允许转让，流动性较差 |
| 投资门槛 | 合格投资者认定 |
| 信息披露 | 定期公布信托报告、净值 |

### 信托产品与其他产品对比

| 对比项 | 信托产品 | 银行理财 | 债券基金 |
|--------|---------|---------|---------|
| 起投金额 | 100万元起 | 1元-1万元 | 10元-1000元 |
| 投资者要求 | 合格投资者 | 普通投资者 | 普通投资者 |
| 投资范围 | 广泛（非标+标品） | 以标品为主 | 以债券为主 |
| 流动性 | 差（封闭期不可赎回） | 中等（开放式可赎回） | 好（T+1赎回） |
| 预期收益 | 4%-7% | 3%-5% | 3%-5% |
| 风险等级 | R3-R5 | R1-R4 | R2-R3 |

### 家族信托（高端服务）
- **定义**：以家族财富保护和传承为目的的信托安排。
- **起投金额**：一般1000万元起。
- **功能**：资产隔离、财富传承、税务规划、子女教育基金。
- **期限**：长期（10年以上或永续）。
- **服务对象**：超高净值家族客户。

## 政策法规依据

1. **《信托法》**（2001年）：信托行业基本法律。
2. **《信托公司管理办法》**（银监会令2007年第2号）：信托公司管理规范。
3. **《信托公司集合资金信托计划管理办法》**（银监会令2009年第1号）：集合信托管理规范。
4. **《关于规范金融机构资产管理业务的指导意见》**（"资管新规"）：信托产品净值化管理要求。
5. **《商业银行代理保险业务管理办法》**（参照适用）：代销业务管理。
6. **《信托公司资金信托管理暂行办法》**（征求意见稿）：资金信托最新规范。

## 城商行/农商行特色

### 产品策略
- **精选合作信托公司**：选择评级较高（AA级以上）、风控能力强的信托公司合作。
- **标品信托为主**：在信托行业调整期，重点推荐投资于标准化资产的标品信托。
- **谨慎推荐非标**：对非标融资类信托严格审核底层资产和还款来源。
- **家族信托服务**：为本地高净值客户提供家族信托咨询和对接服务。

### 高净值客户服务
- **私人银行联动**：将信托产品作为私行客户资产配置的重要组成部分。
- **专属顾问**：配备私行顾问提供一对一的信托产品推荐和售后服务。
- **综合方案**：结合信托、保险、基金等产品为客户提供综合财富管理方案。
- **定期回访**：定期向客户报告信托产品运作情况，做好投资者关系管理。

### 风险管理
- **严格准入**：对代销信托产品实行严格的内部准入审核。
- **底层穿透**：深入了解信托产品的底层资产和风险状况。
- **适当性管理**：确保产品风险等级与客户风险承受能力匹配。
- **风险预警**：对存量代销信托产品持续跟踪，发现风险及时告知客户。

## 常见问题

### Q1：什么是合格投资者？普通客户可以买信托吗？
信托产品面向合格投资者销售。合格投资者需满足以下条件之一：家庭金融净资产≥300万元、家庭金融资产≥500万元、近3年年均收入≥40万元（个人）或≥50万元（家庭），且具有2年以上投资经历。不满足以上条件的普通投资者不能购买信托产品，建议选择银行理财、基金等其他产品。

### Q2：信托产品安全吗？会不会亏本？
资管新规后信托产品已打破刚性兑付，不保本不保收益。信托产品的安全性取决于底层资产质量：投资于国债、高等级债券的信托产品风险较低；投资于房地产、工商企业融资的信托产品风险较高。近年来部分信托产品出现违约，投资者需审慎评估底层资产风险。建议选择评级较高的信托公司和投资于标准化资产的产品。

### Q3：信托产品到期后能延期兑付吗？
如果信托产品底层资产未能按时回收，可能出现延期兑付的情况。这是信托行业打破刚兑后的正常现象。投资者应在购买前了解项目的还款来源和风控措施，并关注信托公司的历史兑付记录。如遇延期兑付，投资者有权要求信托公司披露延期原因和后续处置方案。

### Q4：信托产品的收益是怎么计算的？
资管新规后信托产品实行净值化管理，收益取决于底层资产的实际表现。投资者购买时看到的"业绩比较基准"不是承诺收益，实际收益可能高于或低于基准。净值型信托产品每日或定期公布净值，投资者赎回时按实际净值计算收益。

### Q5：家族信托和一般信托有什么区别？
家族信托是以家族财富保护和传承为目的的特殊信托安排，起投金额通常为1000万元以上。与一般信托的区别在于：一是目的不同，家族信托侧重财富传承和资产隔离，一般信托侧重投资收益；二是期限不同，家族信托期限更长（10年以上或永续）；三是功能更丰富，家族信托可设定受益人、分配条件等个性化条款。
