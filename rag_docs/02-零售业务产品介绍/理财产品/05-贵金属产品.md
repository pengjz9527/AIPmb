# 贵金属产品

## 产品概述

贵金属产品是指商业银行面向个人客户提供的黄金、白银等贵金属投资产品，包括实物贵金属、账户贵金属（纸黄金/纸白银）、黄金积存等多种形式。贵金属产品具有避险保值、对抗通胀的功能，是客户资产配置中重要的另类投资品种。

贵金属业务是银行零售业务的特色产品之一，满足了客户多元化的投资和保值需求。对于城商行和农商行而言，贵金属产品是丰富产品线、服务中高端客户的重要工具。在金价波动加大的市场环境下，合理的贵金属配置有助于客户分散投资风险。

## 产品分类

### 按产品形态分类

#### 实物贵金属
- **投资金条/银条**：标准化的投资型贵金属，溢价较低。
- **纪念币/纪念章**：具有收藏价值的贵金属纪念品。
- **贵金属饰品**：黄金首饰等消费型贵金属（银行较少涉及）。

#### 账户贵金属（纸黄金/纸白银）
- **纸黄金**：以黄金价格为基础的电子化交易产品，不涉及实物交割。
- **纸白银**：以白银价格为基础的电子化交易产品。
- **双向交易**：部分银行支持做多和做空两种方向。

#### 黄金积存
- **定期积存**：每月固定金额或克数购买黄金。
- **主动积存**：客户自主决定购买时间和金额。
- **可提取实物**：积存到一定数量后可申请提取实物黄金。

#### 黄金ETF联接基金
- **基金形式**：投资于黄金ETF的联接基金，间接投资实物黄金。
- **门槛低**：10元起购。
- **流动性好**：T+1赎回。

### 按交易方式分类
- **T+0交易**：当日买卖，实时结算（账户贵金属）。
- **T+1交易**：次日结算（黄金ETF联接基金）。
- **现货交易**：购买实物贵金属，即时交割。

## 适用对象与准入条件

### 适用对象
- 有避险保值需求的投资者
- 有资产配置多元化需求的客户
- 看好贵金属长期价值的投资者
- 有黄金收藏和馈赠需求的客户
- 希望通过定投方式投资黄金的客户

### 准入条件
| 条件 | 具体要求 |
|------|----------|
| 年龄 | 18周岁以上 |
| 风险测评 | 完成风险承受能力评估 |
| 起购金额 | 实物金条：按克购买（约10克起）；积存金：1克或100元起；纸黄金：1克起 |
| 账户要求 | 已开立银行结算账户 |
| 签约协议 | 账户贵金属需签署专项交易协议 |

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 实物金条 | 按当日金价 + 加工费（约10-30元/克） |
| 纸黄金 | 按银行报价买卖，买卖价差约0.5-1元/克 |
| 黄金积存 | 按积存日金价购买，管理费约0.01%-0.05%/月 |
| 黄金ETF联接 | 申购费0.1%-0.5%，管理费约0.5%/年 |
| 交易时间 | 账户贵金属：工作日9:00-15:00（部分延长至凌晨） |
| 最小交易单位 | 实物金：1克起；纸黄金：1克起；积存金：1克或100元起 |
| 回购 | 实物金：按当日回购价（低于销售价）；纸黄金：按银行买入价 |

### 各产品对比

| 对比项 | 实物金条 | 纸黄金 | 黄金积存 | 黄金ETF联接 |
|--------|---------|--------|---------|------------|
| 起购门槛 | 高（10克起） | 中（1克起） | 低（100元起） | 低（10元起） |
| 流动性 | 低（需到银行回购） | 高（T+0交易） | 中（可赎回或提取） | 高（T+1赎回） |
| 交易成本 | 较高（加工费+回购价差） | 较低（买卖价差） | 低（管理费） | 较低（申购费+管理费） |
| 是否可提实物 | 本身就是实物 | 不可提取 | 可提取 | 不可提取 |
| 适合人群 | 收藏/馈赠 | 短期交易 | 长期定投 | 便捷投资 |
| 风险 | 保管风险 | 价格波动风险 | 价格波动风险 | 价格波动风险 |

### 黄金积存定投策略
- **定投频率**：每周/每月定期购买。
- **定投金额**：建议占月收入的5%-10%。
- **长期持有**：黄金定投适合3-5年以上的长期投资。
- **逢低加仓**：金价回调时适当增加购买量。
- **目标提取**：积存到一定数量可提取实物金条或金饰。

## 政策法规依据

1. **《中华人民共和国金银管理条例》**：贵金属管理基本法规。
2. **上海黄金交易所交易规则**：黄金交易基础设施规范。
3. **《商业银行贵金属业务管理办法》**（各行内部制度）：贵金属业务管理。
4. **《关于规范商业银行黄金及黄金衍生品业务的通知》**：黄金业务规范。
5. **《商业银行互联网贷款管理暂行办法》**（参照适用）：线上交易管理。

## 城商行/农商行特色

### 产品策略
- **黄金积存为主**：门槛低、操作简单，适合中小银行客户群体。
- **实物金代销**：与大型金矿企业或上海黄金交易所会员单位合作代销实物金。
- **节日营销**：在春节、中秋等传统节日推出纪念金条、生肖金条等主题产品。
- **婚庆金饰**：与当地金店合作，为婚庆客户提供黄金购买优惠。

### 本地化服务
- **贵金属回购**：提供便捷的实物金回购服务，解决客户变现需求。
- **贵金属展示**：在网点设立贵金属展示柜，提升客户体验。
- **投资讲座**：定期举办贵金属投资分析讲座。
- **VIP专属**：为高端客户提供贵金属专属价格和定制服务。

### 创新产品
- **"黄金+存款"组合**：将黄金积存与定期存款组合，兼顾保值和增值。
- **数字黄金**：基于区块链技术的数字黄金产品（部分银行试点）。
- **黄金质押贷款**：以实物黄金作为质押物的个人贷款。

## 常见问题

### Q1：纸黄金和实物黄金有什么区别？
纸黄金是以黄金价格为基础的电子化交易产品，不涉及实物交割，买卖便捷、交易成本低，适合短期交易和投资。实物黄金是看得见摸得着的金条/金币，具有收藏和馈赠功能，但存在保管风险和较高的交易成本（加工费+回购价差）。如果纯粹为了投资获利，建议选择纸黄金或黄金ETF联接基金；如果有收藏或送礼需求，选择实物金条。

### Q2：黄金积存适合什么样的投资者？
黄金积存最适合以下投资者：一是看好黄金长期价值但不知道怎么买的客户；二是希望通过定投方式分散买入时点风险的客户；三是小额投资的客户（100元起即可参与）；四是有"攒金子"习惯的客户。黄金积存兼具定投的纪律性和黄金的保值性，是普通投资者参与黄金投资的便捷方式。

### Q3：金价波动大，投资黄金风险高吗？
黄金价格确实存在波动风险，但黄金作为避险资产，长期来看具有保值功能。影响金价的因素包括：美元走势、地缘政治、通胀预期、央行政策等。建议将黄金作为资产配置的一部分（通常建议占总资产的5%-15%），而非全部投资，以分散风险。通过定投方式可以有效降低短期波动的影响。

### Q4：银行买的金条可以回购吗？
可以。多数银行提供本行销售金条的回购服务，回购价格按当日回购价（通常低于当日销售价5-15元/克）。回购时需携带金条、购买凭证和身份证件到银行网点办理。部分银行也回购非本行销售的金条，但可能需要额外的成色检测。建议在银行购买金条时保留好购买凭证，便于日后回购。

### Q5：黄金ETF联接基金和纸黄金哪个好？
两者各有优劣。黄金ETF联接基金门槛更低（10元起）、费率较低（管理费约0.5%/年），但赎回到账时间较长（T+1）；纸黄金交易更便捷（T+0交易）、可双向交易（做多和做空），但买卖价差成本可能较高。如果是长期投资，建议选择黄金ETF联接基金；如果是短期交易，建议选择纸黄金。
