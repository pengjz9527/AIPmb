# 个人养老金产品

## 产品概述

个人养老金是指政府政策支持、个人自愿参加、市场化运营的补充养老保险制度。2022年11月，个人养老金制度在全国36个城市（地区）先行实施，2024年起逐步推广至全国。参加人每年可向个人养老金资金账户缴存最高12000元，享受税收优惠，并自主选择购买储蓄、理财、基金、保险等个人养老金产品。

个人养老金是我国多层次养老保障体系的"第三支柱"，与基本养老保险（第一支柱）和企业/职业年金（第二支柱）共同构成养老保障体系。对于城商行和农商行而言，个人养老金业务是获客、活客的重要战略产品，也是服务客户全生命周期财富管理的重要载体。

## 产品分类

### 按产品类型分类（个人养老金可投资的四类产品）

#### 个人养老金储蓄
- **特定养老储蓄**：5年/10年/15年/20年期限的定期存款。
- **利率**：通常高于同期限普通定期存款10-30BP。
- **保本保息**：存款类产品，受存款保险保障。

#### 个人养老金理财产品
- **养老目标理财**：以退休日期或风险策略命名的理财产品。
- **固收+策略**：以固收资产为主，适度配置权益资产。
- **风险等级**：R2-R3为主。

#### 个人养老金基金
- **养老目标日期基金（TDF）**：以退休日期命名（如"养老2050"），随时间自动调整资产配置。
- **养老目标风险基金（TRF）**：按风险等级命名（如"稳健养老""平衡养老"），资产配置比例相对稳定。
- **费率优惠**：Y份额（个人养老金专属份额）管理费、托管费减半。

#### 个人养老金保险
- **专属商业养老保险**：保证利率+浮动收益，可终身领取。
- **年金保险**：退休后定期领取年金。
- **两全保险**：满期给付+身故保障。

### 按运作方式分类
- **目标日期策略**：随着退休日期临近，逐步降低风险资产比例。
- **目标风险策略**：维持相对稳定的风险水平。
- **固定收益策略**：主要投资于固收资产。

## 适用对象与准入条件

### 适用对象
- 在中国境内参加城镇职工基本养老保险或城乡居民基本养老保险的劳动者
- 有补充养老需求的在职人员
- 希望享受个税优惠的纳税人
- 有长期储蓄和投资习惯的客户
- 距离退休有一定年限的年轻和中年客户

### 准入条件
| 条件 | 具体要求 |
|------|----------|
| 参保条件 | 已参加基本养老保险（职工或居民） |
| 年龄 | 未满法定退休年龄 |
| 年度缴存上限 | 12000元/年 |
| 开户要求 | 在银行开立个人养老金资金账户 |
| 账户唯一性 | 每人只能有一个个人养老金资金账户 |
| 税收优惠 | 缴存金额在个税应纳税所得额中扣除 |

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 年度缴存上限 | 12000元/年 |
| 税收优惠 | EET模式（缴存免税、投资免税、领取征税） |
| 领取税率 | 领取时按3%缴纳个人所得税 |
| 领取条件 | 达到法定退休年龄/完全丧失劳动能力/出国（境）定居 |
| 投资范围 | 储蓄、理财、基金、保险四类产品 |
| 账户管理 | 封闭运行，达到领取条件前不可提取 |
| 继承 | 参加人身故后，个人养老金资产可继承 |

### 税收优惠详解

| 年收入区间 | 适用边际税率 | 缴存12000元节税金额 |
|-----------|------------|-------------------|
| 不超过36000元 | 3% | 0元（领取时也是3%，无节税效果） |
| 36000-144000元 | 10% | 840元 |
| 144000-300000元 | 20% | 2040元 |
| 300000-420000元 | 25% | 2640元 |
| 420000-660000元 | 30% | 3240元 |
| 660000-960000元 | 35% | 3840元 |
| 超过960000元 | 45% | 5040元 |

**注**：年收入在个税起征点（6万元）以下的客户，缴存个人养老金无节税效果，反而在领取时需缴纳3%的税，建议谨慎参与。

### 四类产品收益风险对比

| 产品类型 | 风险等级 | 预期收益 | 适合人群 |
|----------|----------|---------|---------|
| 储蓄存款 | 极低 | 1.5%-2.5%（确定） | 极保守型 |
| 理财产品 | R2-R3 | 3%-5%（浮动） | 稳健型 |
| 基金产品 | R3-R4 | -10%-15%（波动大） | 平衡-进取型 |
| 保险产品 | 低-中 | 2.5%-4%（部分确定） | 稳健-平衡型 |

## 办理渠道

| 渠道 | 可办理业务 | 备注 |
|------|-----------|------|
| 营业网点 | 开户、缴存、购买产品 | 全功能 |
| 手机银行APP | 开户、缴存、购买产品、查询 | 主力渠道 |
| 网上银行 | 缴存、购买产品 | 辅助渠道 |
| 国家社保平台 | 开户信息查询 | 官方平台 |
| 微信银行 | 产品推荐、开户入口 | 引流渠道 |

### 开户流程
1. **资格确认**：确认已参加基本养老保险。
2. **开户**：在银行开立个人养老金资金账户（一个身份证只能开一个）。
3. **缴存**：向资金账户缴存资金（年度上限12000元）。
4. **投资**：自主选择购买储蓄/理财/基金/保险产品。
5. **抵税**：在个税APP中确认扣除信息。
6. **领取**：达到领取条件后申请领取。

5. **《个人养老金投资公开募集证券投资基金业务管理暂行规定》**：基金产品管理规范。

## 城商行/农商行特色

### 获客策略
- **代发工资联动**：向代发工资客户推广个人养老金开户，实现"工资到账→养老缴存"的无缝衔接。
- **企业团办**：与本地企业合作，批量为员工办理个人养老金开户。
- **社区推广**：深入社区开展个人养老金政策宣讲和开户活动。
- **积分激励**：开户缴存赠送积分，积分兑换生活权益。

### 产品策略
- **储蓄产品优势**：中小银行养老储蓄利率通常高于大行，具有竞争力。
- **稳健产品为主**：重点推荐养老储蓄和稳健型养老理财，符合客户风险偏好。
- **精选基金**：精选养老目标基金，降低客户选择难度。
- **保险产品对接**：与保险公司合作提供养老年金产品。

### 服务创新
- **养老规划服务**：为客户提供退休收入测算和养老资产配置建议。
- **一站式服务**：在一个账户内实现缴存、投资、查询、领取全流程服务。
- **适老化服务**：为临近退休客户提供大字版界面、语音操作等适老化服务。
- **定期报告**：定期推送个人养老金账户报告，展示投资收益和账户余额。

### 农村场景
- **城乡居民养老保险联动**：农商行将个人养老金与城乡居民养老保险结合推广。
- **灵活就业人员覆盖**：面向农村灵活就业人员推广个人养老金。
- **惠农特色**：结合惠农补贴、土地流转收入等场景推广养老储蓄。

## 常见问题

### Q1：个人养老金值不值得参加？
对于有缴纳个人所得税的客户来说，个人养老金是值得参加的。以年收入15万元的客户为例，每年缴存12000元可节税约840元，长期累积效果可观。但对于年收入低于6万元（不需缴纳个税）的客户，缴存个人养老金不仅无节税效果，领取时还需缴纳3%的税，建议谨慎参与。

### Q2：个人养老金的钱什么时候能取出来？
个人养老金账户实行封闭管理，只有满足以下条件之一才能领取：一是达到法定退休年龄；二是完全丧失劳动能力；三是出国（境）定居；四是其他符合国家规定的情形。领取时按3%缴纳个人所得税。未达到领取条件前，资金不可提取。

### Q3：个人养老金账户里的钱可以投资什么？
可以投资四类产品：一是储蓄存款（保本保息，风险最低）；二是理财产品（净值型，R2-R3风险）；三是基金产品（养老目标基金，波动较大）；四是保险产品（储蓄型保险，部分保本）。投资者可根据自身风险偏好和距退休年限选择合适的产品组合。

### Q4：可以更换个人养老金开户银行吗？
可以。个人养老金资金账户可以在不同银行之间转移。如需转户，在新银行开立个人养老金资金账户后，原账户中的资金和产品将转移到新账户。转移过程不影响已缴存金额和税收优惠。但需注意，部分基金产品可能不支持跨行转移。

### Q5：距离退休还有30年，应该怎么配置个人养老金？
距离退休时间较长，可以适当增加权益类资产的配置比例。建议：一是选择养老目标日期基金（如"养老2055"），基金会自动随时间调整资产配置（年轻时多配权益、临近退休多配固收）；二是也可选择"固收+理财+基金"的组合方式；三是每年保持12000元的满额缴存，最大化税收优惠。
