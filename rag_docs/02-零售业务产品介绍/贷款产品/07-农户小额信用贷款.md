# 农户小额信用贷款

## 产品概述

农户小额信用贷款是指农村信用社（农商行）和村镇银行以农户的信用为基础，在核定的信用额度和期限内，向农户发放的无需抵押担保的小额贷款。该产品是农商行服务"三农"的核心产品，也是普惠金融在农村地区的重要实践形式。

农户小额信用贷款是农商行最具特色和竞争力的产品之一，体现了农商行"扎根农村、服务三农"的市场定位。通过"信用评定、整村授信"等模式，农商行将金融服务延伸至广大农村地区，有效解决了农户"贷款难、贷款贵"的问题。在乡村振兴战略背景下，农户小额信用贷款正从传统农业融资向综合金融服务转型升级。

## 产品分类

### 按信用评定方式分类
- **传统信用评定贷款**：通过村级信用评定小组逐户评定信用等级。
- **整村授信贷款**：对全村农户进行批量授信，"一次核定、随用随贷"。
- **大数据授信贷款**：基于涉农数据（土地确权、农业补贴、社保等）进行线上授信。

### 按用途分类
- **农业生产贷款**：用于种植、养殖等农业生产活动。
- **农业经营贷款**：用于农产品加工、运输、销售等环节。
- **生活消费贷款**：用于建房、装修、子女教育等生活消费。
- **综合用途贷款**：不限定具体用途，农户自主支配。

### 按额度分类
- **小额信用贷款**：5万元以下，纯信用。
- **中额信用贷款**：5-30万元，信用为主。
- **大额信用贷款**：30-50万元，需额外担保或更高信用等级。

### 按期限分类
- **短期贷款**：1年以内，适合季节性农业生产。
- **中期贷款**：1-3年，适合养殖、果园等周期较长的项目。
- **长期贷款**：3-5年，适合基础设施建设、大型设备购置。

## 适用对象与准入条件

### 适用对象
- 从事农业生产经营的农户
- 农村种植户、养殖户
- 农村个体工商户
- 农村合作社成员
- 返乡创业的农民工
- 新型农业经营主体（家庭农场、农民专业合作社）

### 准入条件
| 条件 | 具体要求 |
|------|----------|
| 户籍/居住地 | 在农商行服务辖区内有固定住所或经营场所 |
| 年龄 | 18-65周岁，具有完全民事行为能力 |
| 信用记录 | 无不良信用记录（或已清偿并合理解释） |
| 生产经营 | 有明确的生产经营项目和还款来源 |
| 信用评定 | 通过村级信用评定或系统自动评级 |
| 品行要求 | 品行良好、无赌博吸毒等不良嗜好 |
| 家庭状况 | 家庭关系和睦、有劳动能力 |

| 要素 | 内容 |

### 整村授信模式

整村授信是农户小额信用贷款的核心业务模式，流程如下：

1. **信息采集**：客户经理联合村委会采集全村农户基本信息（家庭成员、收入来源、资产状况、信用记录等）。
2. **信用评定**：由村两委、村民代表和银行人员组成的信用评定小组逐户评定信用等级。
3. **额度核定**：根据信用等级核定信用额度，公示评定结果。
4. **签约授信**：与授信农户签订授信合同，发放"信用证"或开通手机银行。
5. **随用随贷**：农户在授信额度内随时申请用款，无需再次审批。
6. **动态管理**：每年重新评定一次，根据还款记录和经营情况调整额度和等级。

## 政策法规依据

1. **《农村信用社农户小额信用贷款管理暂行办法》**（中国人民银行〔2001〕）：农户小额信用贷款的基本制度。
2. **《中国银监会关于银行业金融机构做好扶贫开发金融服务工作的指导意见》**：金融扶贫政策。
3. **《关于金融支持乡村振兴的指导意见》**：乡村振兴金融支持政策。
4. **《中国人民银行关于用好支农支小再贷款 助力乡村振兴的通知》**：支农再贷款政策。
5. **《中国银保监会办公厅关于推进普惠金融高质量发展的实施意见》**：普惠金融政策。
6. **《个人贷款管理办法》**（2024年修订）：最新个人贷款管理要求。

## 城商行/农商行特色

### 整村授信深化
- **"党建+金融"模式**：将信用评定与基层党建结合，村党支部书记参与信用评定，提升公信力。
- **信用村/信用户建设**：对信用评定达标的村授予"信用村"称号，村民享受更高额度和更低利率。
- **数字化整村授信**：利用卫星遥感、农业大数据等技术辅助信用评定，提升效率和准确性。
- **"整村授信+产业"模式**：围绕村域主导产业（如茶叶、果蔬、养殖），整村授信与产业扶持结合。

### 科技赋能
- **移动展业**：客户经理手持移动终端上门采集信息、现场授信、即时放款。
- **大数据风控**：整合土地确权、农业补贴、社保缴纳、水电费等涉农数据构建风控模型。
- **卫星遥感辅助**：利用卫星遥感技术评估农作物种植面积和长势，辅助贷款决策。
- **线上全流程**：从申请到放款全流程线上化，农户"足不出村"即可获得贷款。

### 普惠金融实践
- **利率优惠**：利用央行支农再贷款资金发放贷款，利率可低至3.50%-4.00%。
- **减费让利**：免收评估费、合同工本费等费用。
- **灵活还款**：根据农时特点设计还款计划，如"春借秋还"。
- **容错机制**：对因自然灾害等不可抗力导致逾期的农户，给予展期或续贷支持。
- **金融扶贫**：对建档立卡贫困户提供免担保、低利率的扶贫小额信贷。

### 典型案例
- **某农商行"整村授信"模式**：全辖120个行政村完成整村授信，授信农户3.2万户，授信总额18.5亿元，户均授信5.8万元。
- **"信用变金"工程**：信用村农户享受利率下浮20BP、额度上浮30%的优惠，有效激励了农村信用环境建设。

## 常见问题

### Q1：农户小额信用贷款不需要任何抵押物吗？
是的，农户小额信用贷款是纯信用贷款，不需要任何抵押物或担保人。银行通过信用评定（包括个人信用记录、家庭资产状况、生产经营能力、品行口碑等）确定信用等级和授信额度。信用等级越高，授信额度越大、利率越优惠。

### Q2：什么是"整村授信"？和普通贷款有什么区别？
整村授信是农商行对全村农户进行批量信用评定和授信的业务模式。与普通贷款的区别在于：一是不需要农户逐户到银行申请，银行主动上门采集信息和评定；二是"一次核定、周转使用"，核定额度后可随时用款；三是审批效率高，无需逐笔审批；四是利率更优惠，因为是批量业务降低了银行运营成本。

### Q3：农户小额信用贷款的利率是多少？
农户小额信用贷款的利率通常在3.85%-6.00%之间，具体取决于：一是信用等级（高信用等级享受利率优惠）；二是资金来源（使用央行支农再贷款资金的利率更低，可低至3.50%）；三是贷款用途（农业生产贷款利率可能低于消费贷款）；四是当地政策（部分地区有财政贴息）。

### Q4：贷款到期还不上怎么办？
如遇特殊困难（如自然灾害、疫病等不可抗力），可以采取以下措施：一是申请展期（延长还款期限）；二是申请无还本续贷；三是申请调整还款计划（如"春借秋还"）。农商行通常对因不可抗力导致的逾期有较灵活的处置机制。但如果是自身原因（如经营不善、挥霍浪费），则需承担逾期罚息和信用记录受损的后果。

### Q5：不在农村居住的城里人可以申请农户贷款吗？
一般不可以。农户小额信用贷款的适用对象是在农商行服务辖区内有固定住所和生产经营活动的农户。如果您不在农村地区居住或不从事农业生产经营，建议选择个人消费贷款或个人信用贷款产品。但如果您是农村户籍且家庭仍在农村从事生产经营，即使本人在城镇工作，仍可能符合申请条件。
