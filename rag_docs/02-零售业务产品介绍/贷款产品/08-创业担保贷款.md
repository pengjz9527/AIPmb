# 创业担保贷款

## 产品概述

创业担保贷款是指由政府出资设立担保基金，由担保机构提供担保，由经办银行发放，用于支持个人创业或小微企业吸纳就业的政策性贷款产品。创业担保贷款是国家扶持创业就业的重要政策工具，具有"政府贴息、门槛低、额度适中"的特点，是普惠金融的典型代表。

创业担保贷款体现了金融支持就业、服务民生的政策导向。对于城商行和农商行而言，参与创业担保贷款业务既是履行社会责任的重要方式，也是拓展零售客户群体、深化与地方政府合作关系的有效途径。在当前"稳就业、促创业"的政策环境下，创业担保贷款覆盖面不断扩大，支持力度持续增强。

## 产品分类

### 按借款人类型分类
- **个人创业担保贷款**：面向个人创业者，包括个体工商户和自主创业者。
- **小微企业创业担保贷款**：面向吸纳就业的小微企业。

### 按创业类型分类
- **自主创业贷款**：个人自主创办企业或个体经营。
- **合伙创业贷款**：多人合伙创业，额度可适当提高。
- **组织起来共同创业贷款**：组织多人共同创业。

### 按担保方式分类
- **担保基金担保**：由政府出资的担保基金提供担保（主流方式）。
- **免担保贷款**：对信用良好、符合条件的借款人免除反担保。
- **组合担保**：担保基金+抵押/质押组合。

### 按贴息方式分类
- **全额贴息**：政府全额贴息，借款人实际不付利息。
- **部分贴息**：政府贴息一部分，借款人承担一部分。
- **LPR-150BP以下自付**：借款人承担LPR减150BP以下部分，以上由政府贴息。

## 适用对象与准入条件

### 适用对象
| 人群类别 | 说明 |
|----------|------|
| 城镇登记失业人员 | 持有《就业创业证》的城镇失业人员 |
| 就业困难人员 | 低保家庭、零就业家庭、残疾人等 |
| 退役军人 | 退出现役的军人 |
| 高校毕业生 | 毕业5年内的高校毕业生（含留学回国） |
| 返乡创业农民工 | 从城镇返回农村创业的人员 |
| 网络商户 | 通过网络平台从事经营活动的人员 |
| 建档立卡贫困人口 | 已脱贫但仍在巩固期的人口 |
| 农村自主创业农民 | 从事农业生产经营的农村创业者 |
| 小微企业 | 当年新招用符合条件人员达到一定比例的小微企业 |

### 准入条件
| 条件 | 具体要求 |
|------|----------|
| 身份证明 | 属于上述适用人群之一 |
| 创业项目 | 有具体的创业项目或经营计划 |
| 营业执照 | 已取得营业执照（个体工商户或企业） |
| 经营年限 | 一般要求注册经营不超过5年（部分不限） |
| 信用记录 | 无不良信用记录 |
| 就业带动 | 小微企业需吸纳一定比例的符合条件人员 |
| 反担保 | 部分产品需要提供反担保措施 |

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 贷款额度 | 个人：最高30万元；合伙创业：最高400万元；小微企业：最高400万元 |
| 贷款期限 | 最长3年 |
| 贷款利率 | 不超过LPR+150BP（即最高约4.50%） |
| 贴息方式 | 贷款利率减去LPR-150BP部分由政府贴息 |
| 借款人实际利率 | 约1.50%-2.00%（贴息后） |
| 担保方式 | 政府担保基金担保 |
| 反担保 | 部分免除反担保；部分需提供 |
| 还款方式 | 等额本息、等额本金、按月付息到期还本 |
| 续贷次数 | 可续贷，累计不超过3次 |

### 贴息机制详解
以2025年政策为例（1年期LPR为3.00%）：

| 项目 | 计算方式 |
|------|----------|
| 贷款利率上限 | LPR + 150BP = 4.50% |
| 借款人承担部分 | LPR - 150BP = 1.50% |
| 政府贴息部分 | 贷款利率 - (LPR - 150BP) = 约3.00% |
| 借款人实际负担 | 约1.50%（年化） |

**注**：贴息比例和计算方式因地区政策可能有所差异，部分地区实行全额贴息。

### 额度与期限

| 借款人类型 | 最高额度 | 最长期限 | 续贷 |
|-----------|----------|----------|------|
| 个人创业 | 30万元 | 3年 | 累计不超过3次 |
| 合伙创业 | 400万元（按合伙人数×个人额度） | 3年 | 累计不超过3次 |
| 小微企业 | 400万元 | 2年 | 累计不超过3次 |

## 办理渠道

| 渠道 | 可办理业务 | 备注 |
|------|-----------|------|
| 人社部门 | 资格审核、贴息申请 | 前置审核 |
| 担保机构 | 担保审批 | 政府担保基金 |
| 经办银行网点 | 贷款申请、签约、放款 | 主力渠道 |
| 手机银行APP | 申请、进度查询、还款 | 线上化趋势 |
| 创业服务中心 | 政策咨询、一站式办理 | 综合服务 |
| 社区/村委会 | 推荐、初审 | 基层渠道 |

### 办理流程
1. **资格审核**：向当地人社部门提交申请，审核创业身份和项目资格。
2. **担保审批**：担保机构审核担保条件（是否需要反担保）。
3. **贷款审批**：经办银行审核贷款条件，确定额度和利率。
4. **签约放款**：签订贷款合同，发放贷款。
5. **贴息申请**：贷款发放后向人社部门申请贴息。
6. **还本付息**：按合同约定还本付息。
7. **贴息返还**：贴息资金到账。

1. **《普惠金融发展专项资金管理办法》**（财金〔2023〕75号）：创业担保贷款贴息资金管理。
2. **《关于进一步加大创业担保贷款贴息力度全力支持重点群体创业就业的通知》**（财金〔2020〕21号）：创业担保贷款最新政策。
3. **《中国人民银行 财政部 人力资源和社会保障部关于实施创业担保贷款支持创业就业工作的通知》**（银发〔2016〕202号）：基本制度框架。
4. **《就业促进法》**：就业促进基本法律。
5. **各省市创业担保贷款管理办法**：地方实施细则。

## 城商行/农商行特色

### 服务下沉
- **社区化服务**：城商行依托社区网点，为城市创业者提供就近服务。
- **村级服务**：农商行通过村级金融服务站，将创业担保贷款延伸至农村。
- **一站式办理**：与人社部门、担保机构在银行网点设立"一站式"服务窗口。
- **政策宣讲**：定期开展创业担保贷款政策宣讲会，帮助创业者了解政策。

### 创业生态服务
- **创业培训联动**：与人社部门合作，为创业者提供创业培训和金融知识教育。
- **创业孵化对接**：与创业园区、孵化器合作，为入园创业者提供融资服务。
- **后续跟踪服务**：贷后持续跟踪创业项目，提供经营建议和资源对接。
- **创业生态圈**：搭建本地创业者交流平台，促进经验分享和资源互通。

### 产品创新
- **"创业贷+经营贷"组合**：创业初期使用创业担保贷款，发展壮大后无缝衔接经营性贷款。
- **线上申请**：部分地区实现创业担保贷款线上申请、审核、放款全流程。
- **免反担保**：对信用良好、创业项目优质的借款人免除反担保要求。
- **"首贷户"培育**：对从未获得过银行贷款的创业者，提供首贷培育和辅导。

### 典型案例
- **某城商行创业担保贷模式**：与当地人社局合作，在社区网点设立"创业金融服务窗口"，年均发放创业担保贷款2亿元，惠及创业者1500余人。
- **某农商行"返乡创业贷"**：针对春节后返乡创业人员，提供创业担保贷款+经营指导+市场对接的综合服务。

## 常见问题

### Q1：创业担保贷款真的利息很低吗？
是的。创业担保贷款最大的优势是政府贴息，借款人实际承担的利率很低。以2025年政策为例，借款人实际负担利率约1.50%（远低于商业贷款利率3.00%-5.00%），差额部分由政府财政贴息。部分地区甚至实行全额贴息，借款人无需支付利息。

### Q2：申请创业担保贷款需要担保吗？
创业担保贷款由政府出资设立的担保基金提供担保，借款人一般不需要提供额外的担保物。但部分情况下可能需要"反担保"措施（如公务员担保、房产抵押等）。对于信用良好、创业项目优质的借款人，多地已实行免除反担保。具体要求因地区而异。

### Q3：哪些人可以申请创业担保贷款？
主要包括：城镇登记失业人员、就业困难人员、退役军人、毕业5年内高校毕业生、返乡创业农民工、网络商户、建档立卡贫困人口、农村自主创业农民等。此外，吸纳符合条件人员就业的小微企业也可以申请。具体资格条件请咨询当地人社部门。

### Q4：创业担保贷款可以用于什么？
创业担保贷款主要用于创业项目的经营资金需求，包括：购买原材料、设备、工具；支付房租、装修费用；流动资金周转等。严禁用于个人消费、购房、炒股等非经营用途。银行和人社部门会进行贷后检查，确保资金合规使用。

### Q5：创业失败了贷款怎么办？
创业担保贷款仍需按时偿还。如遇经营困难，可采取以下措施：一是向银行申请展期或调整还款计划；二是向人社部门申请贴息延期；三是如果符合担保代偿条件，由担保基金代偿（但借款人仍有偿还义务）。建议创业前充分评估项目可行性，合理规划资金，降低创业风险。
