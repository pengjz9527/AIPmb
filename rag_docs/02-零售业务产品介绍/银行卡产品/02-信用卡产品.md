# 信用卡产品

## 产品概述

信用卡（Credit Card）是商业银行向个人客户发行的、给予一定信用额度、允许持卡人在信用额度内先消费后还款的支付工具。信用卡兼具具备消费支付、现金提取、分期付款、预借现金等多种功能，是现代零售金融业务中收入贡献最大的产品之一。



信用卡业务是银行零售业务的的重要"利润奶牛"之一，利息收入（分期利息、循环信用利息）和手续费收入（年费、分期手续费、商户回佣等）构成丰厚。。对于城商行和农商行而言，信用卡业务是拓展零售客户、提升交叉销售、增强客户黏性的重要战略。

互联网银行的冲击下，中小银行信用卡业务面临差异化转型，从权益设计、场景创新、本地化运营等方面寻求突破。

## 产品分类

### 按卡等级分类
- **普卡**：入门级信用卡，额度一般在5000--万元，年费较低或免年费。
- **金卡**：中端信用卡，额度一般在1-5万元，享有一定权益和费用减免。
- **白金卡**：高端信用卡，额度一般在5-100万元，享受丰富的权益和专属服务。
- **钻石卡/无限卡**：超高端信用卡，额度极高或无限额，享受全方位尊贵权益。

### 按功能分类
- **标准信用卡**：具备基本消费支付和循环信用功能。
- **分期信用卡**：主打化的分期还款功能。
- **现金分期卡**：支持将额度转为现金分期到账。
- **虚拟信用卡**：无实体卡，线上使用和使用。

### 按合作模式分类
- **标准卡**：银行独立发行的标准信用卡。
- **联名卡**：银行与第三方（如航空公司、商场、互联网平台）联合发行。
- **主题卡**：围绕特定主题（如生肖、、文化）设计的信用卡。

## 适用对象与准入条件

### 适用对象
- 有消费信贷需求的个人客户
- 有一定收入和还款能力的工薪族
- 有信用记录的个人借款人
- 商旅出行频繁的客户
- 线上消费活跃群体

| 要素 | 内容 |

| 信用额度 | 普卡5000-10万，金卡1-50万，白金卡5-100万 |
| 免息期 | 最长50-56天（通常从消费日起计算至还款日） |
| 年费 | 普卡100-100元，金卡200-500元，白金卡800-3000元 |
| 循环利率 | 日利率万分之五（年化约18%25%） |

| 分期手续费 | 月费率0.60%-0.80%（折合年化12%-16%） |
| 超限费 | 部分银行收取（已逐渐取消 |

### 免息期详解
- **定义**：从银行记账日至到期还款日之间的时间段内，消费金额不产生利息。
- **常见天数**：50天或56天（各行不同）。
- **起算方式**：从消费入账日的次日开始计算。
- **还款宽限期**：部分银行提供1-3天宽限期（容时服务）。
- **全额还款**：在免息期内全额还款不产生任何利息费用。
- **最低还款**：还款金额≥最低还款额（账单金额×10%），剩余部分按日计息。

## 政策法规依据

1. **《银行卡业务管理办法》**（中国人民银行〔1999〕267号）：银行卡业务基本管理制度。
2. **《商业银行信用卡业务监督管理办法》**（银监会令2011年第2号）：信用卡业务专项监管。
3. **《中国银保监会 中国人民银行关于进一步促进信用卡业务规范健康发展的通知》**（2022年）：信用卡业务最新监管要求。
4. **《中国人民银行关于信用卡业务有关事项的通知》**（2016年）：信用卡利率市场化改革。
5. **《中国人民银行公告〔2021〕第3号》**：要求明示贷款年化利率。
6. **《征信业管理条例》**：征信信息查询和使用规范。

## 城商行/农商行特色

### 差异化权益设计
- **本地生活权益**：与本地商超、餐饮、加油站合作，提供本地本地生活场景优惠。
- **特色积分体系**：积分兑换本地商户优惠券、公共服务缴费、话费充值等。
- **低门槛权益**：降低权益享受门槛，让更多普通客户也能享受优惠。
- **地方特色卡**：结合本地文化特色设计卡面和权益（如美食卡、旅游卡等）。

### 发卡策略
- **代发工资联动**：代发工资客户自动推荐信用卡，提高发卡效率。
- **场景化营销**：在商场、超市、加油站等消费场景设立办卡点。
- **社区推广**：深入社区开展信用卡宣传和办卡活动。
- **线上获客**：手机银行、微信等渠道线上申请，卡片邮寄到家。

### 风控特色
- **本地化风控模型**：利用本地消费数据、社保数据等构建风控模型。
- **交叉验证**：通过代发工资流水、公积金缴存等多维度验证客户还款能力。
- **智能审批**：引入大数据和AI技术，提升自动审批效率和准确性。
- **贷后管理**：依托社区网点进行贷后跟踪和催收。

## 常见问题

### Q1：信用卡和借记卡有什么区别？
最核心的区别是：借记卡需要先存款再消费（不能透支），信用卡可以先消费再还款（有信用额度）。此外，信用卡有免息期（最长50-56天），在免息期内全额还款不产生利息；信用卡有年费（但满足条件可免）；信用卡消费可累积积分兑换权益。从费用角度看，信用卡如果未能全额还款，利息成本较高，需谨慎使用。

### Q2：信用卡年费怎么免？
常见免年费方式包括：一是首年免年费（发卡银行政策）；二是消费达标免次年年费（如年消费满6笔或满一定金额）；三是积分兑换年费；四是特定卡种（如部分白金卡）终身免年费。具体免年费条件因银行和卡种而异，建议开卡前了解清楚。

### Q3：信用卡逾期一天会怎样？
部分银行提供1-3天的还款宽限期（容时服务），在宽限期内还款不算逾期。超过宽限期未还款，则从记账日起按日利率万分之五计算利息和违约金，同时逾期记录上报征信系统，影响个人信用记录。建议设置自动还款或提前还款提醒，避免逾期。

### Q4：信用卡取现和消费有什么区别？
主要区别有：一是免息期不同，取现不享受免息期，从取现当天起按日利率万分之五计息；二是手续费不同，取现手续费一般为取现金额的1%-3%；三是额度不同，取现额度一般为信用额度的50%-100%。建议仅在紧急情况下使用取现功能。

### Q5：信用卡被盗刷了怎么办？
发现盗刷应立即：一是通过手机银行或客服电话进行挂失/锁卡，阻止进一步交易；二是拨打银行客服电话报案并申请争议交易调查；三是到公安机关报案，获取报案回执；四是配合银行完成调查取证。多数银行对确认的盗刷交易提供资金保障（通常72小时内挂失可获全额赔付）。
