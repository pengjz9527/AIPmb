# 联名卡与主题卡

## 产品概述

联名卡（Co-branded Card）是指银行与第三方机构（如航空公司、商超连锁、互联网平台、汽车厂商等）联合发行的信用卡，持卡人可同时享受银行的金融服务和合作方的专属权益。主题卡是围绕特定主题或文化元素（如生肖、城市文化、公益慈善等）设计的信用卡，以主题化权益吸引特定客群。

联名卡和主题卡是城商行和农商行差异化竞争的重要工具。通过与本地知名企业、商业机构、公共服务机构合作，中小银行可以精准触达目标客群，提升品牌影响力，同时为客户提供更具场景化的金融服务体验。

## 产品分类

### 按合作方类型分类
- **航空联名卡**：与航空公司合作，消费积分可兑换航空里程。
- **商超联名卡**：与大型商超合作，提供购物折扣和积分。
- **互联网联名卡**：与互联网平台（如美团、京东、滴滴）合作。
- **汽车联名卡**：与汽车厂商合作，提供购车和养车权益。
- **公共服务联名卡**：与公共交通、社保等合作。
- **酒店联名卡**：与酒店集团合作，提供住宿权益。

### 按主题分类
- **生肖主题卡**：以十二生肖为主题，每年发行对应生肖年。
- **城市文化卡**：以本地文化、地标、特色为主题。
- **公益主题卡**：以环保、公益为主题，部分消费金额捐赠公益。
- **运动主题卡**：以体育运动为主题。
- **青年主题卡**：面向年轻客群的时尚主题卡。

### 按卡等级分类
- **普卡联名卡**：入门级联名卡，门槛低、权益基础。
- **金卡联名卡**：中端联名卡，权益更丰富。
- **白金联名卡**：高端联名卡，权益最优、年费较高。

## 适用对象与准入条件

### 适用对象
- **商旅出行频繁客户**：航空联名卡、酒店联名卡。
- **线上消费活跃客户**：互联网联名卡。
- **本地消费客户**：商超联名卡、公共服务联名卡。
- **年轻时尚群体**：主题卡、青年卡。
- **车主群体**：汽车联名卡。
- **收藏爱好者**：生肖卡、限量版主题卡。

### 准入条件
| 条件 | 具体要求 |
|------|----------|
| 年龄 | 18-65周岁 |
| 身份证明 | 有效身份证件 |
| 收入要求 | 有稳定收入来源 |
| 信用记录 | 征信良好（部分高端卡要求更高） |
| 合作方要求 | 部分联名卡需为合作方会员/客户 |

## 产品核心要素

| 要素 | 内容 |
|------|------|
| 信用额度 | 根据卡等级：普卡5千-10万，金卡1-50万，白金5-100万+ |
| 年费 | 普卡免/100元，金卡200-500元，白金800-3000元（达标可免） |
| 联名权益 | 合作方专属优惠、积分兑换、会员权益等 |
| 积分规则 | 消费1元累积1-5积分（联名消费可加倍） |
| 免息期 | 最长50-56天 |
| 特色卡面 | 合作方品牌元素、主题设计、限量版卡面 |
| 附加权益 | 保险、救援、贵宾厅、代驾等 |

| 会员权益 | 美团会员等级加速 |

#### 商超联名卡
| 权益项目 | 说明 |

## 政策法规依据

1. **《银行卡业务管理办法》**（中国人民银行〔1999〕267号）：银行卡业务基本管理制度。
2. **《商业银行信用卡业务监督管理办法》**（银监会令2011年第2号）：信用卡业务专项监管。
3. **《中国银保监会 中国人民银行关于进一步促进信用卡业务规范健康发展的通知》**（2022年）：联名卡业务规范。
4. **《中国人民银行关于信用卡业务有关事项的通知》**（2016年）：信用卡利率市场化。
5. **《反洗钱法》**：客户身份识别要求。

## 城商行/农商行特色

### 本地化合作
- **本地商超合作**：与本地知名商超、餐饮连锁合作，提供接地气的消费优惠。
- **本地交通联名**：与本地公交、地铁合作，提供出行优惠。
- **本地文化主题**：以本地文化、地标、非遗为主题设计卡面，增强文化认同感。
- **本地景区合作**：与本地旅游景区合作，提供门票折扣和旅游权益。

### 精准客群运营
- **车主卡**：与本地4S店、加油站、停车场合作，为车主提供加油折扣、保养优惠、道路救援等权益。
- **美食卡**：与本地餐饮协会合作，为美食爱好者提供餐饮折扣。
- **校园卡**：与本地高校合作，为学生提供消费优惠和金融服务。
- **农民工卡**：与本地企业客户，如员工群体，提供消费联动等。

## 常见问题

### Q1：联名卡和普通信用卡有什么区别？
联名卡是与第三方机构合作发行的信用卡，普通区别在于：一是合作方不同，联名卡提供特定权益（如航空里程、商超积分、会员权益）；二是权益更丰富，联名卡通常年费较高但权益有限；联名卡权益可能权益的。

 Q2：信用卡年费怎么免？
信用卡免年费条件因：一满足条件免次年年费
- 消费达标免次年年费（如年消费满6笔或满指定金额）  
- 积分兑换年费： 银行积分兑换年费
- 特定卡种终身免年费（如部分白金卡）。

### Q3：联名卡积分怎么兑换？
联名卡积分可通过积分比例兑换（如消费1元累积1-5积分）。兑换规则通常为银行和卡种而异。以 合作方平台查询： 如微信银行、支付宝、美团等平台。兑换比例1 100-500积分
- **特定卡种专属活动**： 节日促销、生日奖励、周年庆典等
