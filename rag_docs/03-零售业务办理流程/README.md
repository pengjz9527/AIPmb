# 零售业务办理流程

## 分类说明

本分类收录银行零售业务各项业务的标准办理流程文档，面向城商行和农商行零售业务条线，提供从客户视角和柜员视角的操作指引。每篇文档均包含流程概述、操作步骤、业务规则、异常处理、风控要点等内容，可作为一线员工操作手册和业务培训材料使用。

## 文档规范

### 统一结构

每篇文档均按照以下结构编写：

- **流程概述**：流程目的、适用业务、办理渠道
- **前置条件**：办理前需要满足的条件和所需材料
- **操作步骤**：编号步骤详细描述各环节
- **各环节详解**：操作细节和注意事项
- **业务规则**：限额、费用、时效、限制条件等
- **异常处理**：常见异常场景及处理方式
- **风控要点**：风险控制点和合规要求
- **城商行/农商行特色**：中小银行的差异化做法
- **关联知识**：相关文档的相对路径链接

### 编写要求

1. 流程步骤清晰、可操作，如同操作手册
2. 内容符合最新监管要求和行业实践（截至2025年）
3. 每篇文档不少于1500字
4. 体现城商行/农商行的业务特色与差异化做法
