# 个人开户流程

## 流程概述

个人开户是银行零售业务的基础入口，指自然人客户在银行开立个人银行结算账户的过程。适用于首次建立业务关系、新增账户、异地开户等场景。目前主要办理渠道包括柜面开户、智能柜员机（STM）开户和线上开户三种方式。

根据中国人民银行《关于落实个人银行账户分类管理制度的通知》（银发〔2016〕302号）及后续改进通知（银发〔2022〕87号），个人银行账户分为Ⅰ类户、Ⅱ类户和Ⅲ类户，实行分类管理。

## 前置条件

### 基本要求

- 年满16周岁（未满16周岁需监护人代理）
- 持有有效身份证件
- 同一客户在同一银行只能开立一个Ⅰ类户

### 所需材料

| 开户类型 | 所需材料 |
|----------|----------|
| 本人开户 | 有效身份证原件（居民身份证） |
| 代理开户 | 双方身份证原件、授权委托书（公证） |
| 未成年人开户 | 监护人身份证、户口簿、出生证明 |
| 境外个人开户 | 护照/港澳台通行证、有效签证/居留许可 |

### 辅助材料（视情况要求提供）

- 实名认证手机号（与身份证一致的运营商号码）
- 居住证明（近3个月水电费账单、租房合同等）
- 工作证明或收入证明

## 操作步骤

1. **客户到店/线上发起申请** → 2. **身份验证与联网核查** → 3. **选择账户类型（Ⅰ/Ⅱ/Ⅲ类户）** → 4. **填写开户信息** → 5. **设置账户密码** → 6. **签署相关协议** → 7. **系统开户与确认** → 8. **发放银行卡/存折或电子账户**

## 各渠道详解

### 一、柜面开户（全流程）

1. **客户取号排队**：大堂经理引导客户取号，了解开户用途
2. **柜员审核身份证件**：使用身份证阅读器读取信息，人工核对照片
3. **联网核查**：通过人民银行联网核查系统验证身份证真实性
4. **人脸识别**：通过人脸识别系统与身份证照片比对
5. **询问开户用途**：了解客户开户目的（工资代发、日常结算、储蓄等）
6. **选择账户类型**：根据客户需求推荐合适的账户类型
7. **录入客户信息**：填写联系电话、地址、职业等基本信息
8. **设置密码**：客户在密码键盘上设置6位数字交易密码
9. **签署协议**：《个人银行结算账户管理协议》《电子银行服务协议》等
10. **系统开户**：提交系统处理，打印开户凭证
11. **发卡/折**：现场制卡并发放，客户签收确认

### 二、智能柜员机（STM）开户

1. **插入身份证**：客户将身份证放入STM读卡区
2. **人脸识别**：系统自动进行活体检测和人脸比对
3. **联网核查**：自动调用核查接口验证身份
4. **选择开户类型**：屏幕显示Ⅰ/Ⅱ/Ⅲ类户选项及说明
5. **补充个人信息**：录入手机号、职业、地址等
6. **设置密码**：在设备密码键盘上设置交易密码
7. **电子签名**：在触屏上阅读协议并签名确认
8. **现场审核**：银行工作人员现场核实客户身份（2022年新规要求）
9. **制卡发卡**：STM自动制卡并弹出卡槽

### 三、线上开户（Ⅱ/Ⅲ类户）

1. **下载手机银行APP**：从官方应用商店下载
2. **注册账号**：使用手机号注册并验证
3. **身份证拍照上传**：正反面照片OCR识别
4. **人脸识别**：活体检测+与公安数据库照片比对
5. **绑定Ⅰ类户**：从他行或本行Ⅰ类户验证身份（仅限Ⅱ/Ⅲ类户）
6. **设置密码**：设置登录密码和交易密码
7. **电子签约**：在线阅读并签署相关协议
8. **开户成功**：系统生成电子账户号

## 账户分类对比

| 对比项 | Ⅰ类户 | Ⅱ类户 | Ⅲ类户 |
|--------|--------|--------|--------|
| 开户渠道 | 柜面、STM | 柜面、STM、线上 | 柜面、STM、线上 |
| 是否实体卡 | 是 | 可以是 | 否（电子账户） |
| 存取现金 | 不限 | 日累计1万/年累计20万 | 不可 |
| 转账限额 | 不限 | 日累计1万/年累计20万 | 日累计2000/年累计5万 |
| 消费缴费 | 不限 | 日累计1万/年累计20万 | 日累计2000/年累计5万 |
| 向非绑定账户转账 | 不限 | 日累计1万/年累计20万 | 日累计2000/年累计5万 |
| 购买理财产品 | 不限 | 可以 | 可以（余额≤2000元） |
| 账户余额上限 | 无 | 无 | 2000元 |

## 业务规则

1. **一人一户原则**：同一客户在同一银行只能开立一个Ⅰ类户（含全国性银行和地方性银行分别计算）
2. **实名制要求**：必须使用本人有效身份证件开户，严禁冒名开户
3. **手机号实名**：绑定手机号须为本人实名认证号码
4. **开户费用**：首次开卡免收工本费、年费和小额账户管理费
5. **激活要求**：线上开立的Ⅱ/Ⅲ类户需绑定本行或他行Ⅰ类户验证身份后方可使用
6. **开户数量限制**：同一客户在同一银行开立的Ⅱ类户和Ⅲ类户数量各有上限（通常各5个）
7. **睡眠账户管理**：连续180天无交易且余额为零的账户可能被转为睡眠户

## 异常处理

### 场景一：联网核查不通过

- 确认身份证是否在有效期内
- 核查系统显示信息与客户是否一致
- 如公安系统数据更新延迟，可要求客户提供辅助身份证明（户口簿、驾驶证等）
- 必要时联系人民银行当地分支机构核实

### 场景二：人脸识别失败

- 检查光线环境，调整拍摄角度
- 确认客户是否整容或容貌变化较大
- 如多次失败，转柜面人工审核
- 整容客户需提供整容前后对比证明

### 场景三：已有Ⅰ类户需再开

- 告知客户已有Ⅰ类户，只能开立Ⅱ/Ⅲ类户
- 如客户希望更换主账户，需先销户再新开
- 可提供升降级服务（Ⅰ类户降为Ⅱ类户，再新开Ⅰ类户）

### 场景四：异常开户行为识别

- 多人同行开户、同一地址大量开户等可疑行为需重点关注
- 开户后立即大额交易、频繁转账等行为触发反洗钱预警
- 拒绝可疑开户并登记上报

## 风控要点

1. **断卡行动要求**：严格落实"断卡"行动要求，防范电信网络诈骗涉案账户
2. **开户意愿核实**：核实客户真实开户意愿，排除被胁迫或被利诱开户
3. **尽职调查**：对高风险客户加强尽职调查，必要时要求提供收入来源证明
4. **账户功能匹配**：根据客户身份特征和风险等级合理设置账户功能和限额
5. **持续监控**：开户后持续监控账户交易，及时发现异常交易行为
6. **合规记录**：完整记录开户过程中的身份核实、意愿确认等环节

## 城商行/农商行特色

1. **本地化服务**：城商行/农商行网点深耕本地社区，对本地客户较为熟悉，可结合社区走访、村委推荐等方式辅助身份核实
2. **助农取款点开户**：农商行可在助农取款服务点设立简易开户渠道，方便农村客户
3. **方言服务**：农村老年客户开户时提供方言服务，耐心讲解开户流程和注意事项
4. **社保卡联动**：与地方社保部门合作，开户同时办理社保卡功能绑定
5. **代发工资批量开户**：为本地中小企业提供批量代发工资开户服务，简化流程
6. **非柜面限额审慎设置**：对新开户客户初始设置较低的非柜面交易限额，根据实际使用情况逐步调整
7. **反诈宣传嵌入**：在开户环节嵌入反诈宣传，签署防范电信诈骗承诺书
