# 定期存款存取流程

## 流程概述

定期存款是银行吸收稳定资金来源的核心存款产品，指客户与银行约定存款期限和利率，到期支取本息的存款业务。适用于有闲置资金、追求稳健收益的个人客户。办理渠道包括柜面、手机银行、网上银行和智能柜员机。

定期存款主要类型包括：整存整取、零存整取、整存零取、存本取息、定活两便、通知存款等，其中整存整取是最常见的品种。

## 前置条件

### 存入条件

- 已开立本行个人结算账户（Ⅰ类户或Ⅱ类户）
- 资金来源合法合规
- 存入金额达到起存金额要求

### 各类型起存金额

| 存款类型 | 起存金额 | 存期选项 |
|----------|----------|----------|
| 整存整取 | 50元 | 3个月/6个月/1年/2年/3年/5年 |
| 零存整取 | 5元 | 1年/3年/5年 |
| 整存零取 | 1000元 | 1年/3年/5年 |
| 存本取息 | 5000元 | 1年/3年/5年 |
| 通知存款 | 5万元 | 1天/7天 |
| 定活两便 | 50元 | 不定存期 |

## 操作步骤

### 定期存款存入流程

1. **选择存款产品** → 2. **确认存款金额与存期** → 3. **选择到期处理方式** → 4. **资金划转（活期转定期）** → 5. **系统确认并生成存单/电子凭证** → 6. **客户确认签收**

### 到期支取流程

1. **到期日系统提醒** → 2. **客户办理支取（或自动转存）** → 3. **利息计算** → 4. **本息划入活期账户** → 5. **存单/凭证回收**

### 提前支取流程

1. **客户申请提前支取** → 2. **身份验证与存单核验** → 3. **利息重新计算（按活期利率）** → 4. **本息划入活期账户** → 5. **存单/凭证回收**

## 各环节详解

### 一、存入环节

#### 柜面存入

1. 客户携带身份证和银行卡/存折到柜台
2. 告知柜员存款金额和存期
3. 柜员在系统中操作活期转定期
4. 选择是否自动转存（到期自动续存同期限）
5. 打印定期存单或在存折上打印记录
6. 客户核对存单信息并签收

#### 电子渠道存入

1. 登录手机银行或网上银行
2. 进入"存款"或"定期存款"模块
3. 选择存款产品和存期
4. 输入存入金额
5. 选择到期处理方式（自动转存/到期转活期）
6. 确认交易并输入交易密码
7. 系统生成电子存单凭证

### 二、到期支取

#### 自动转存设置下的处理

- **设置了自动转存**：到期日系统自动将本息按原存期续存，利率按转存日挂牌利率执行
- **未设置自动转存**：到期日系统自动将本息转入活期账户

#### 到期支取操作

1. 柜面支取：携带身份证、存单/存折到柜台办理
2. 电子渠道：手机银行/网银"我的存款"中选择支取
3. 到期日当天及之后均可支取，利息按约定利率计算
4. 存单回收作废处理

### 三、提前支取

#### 全部提前支取

1. 利息按支取日活期利率重新计算（非原约定利率）
2. 需提供身份证原件
3. 代理支取需双方身份证原件和授权书

#### 部分提前支取

1. 仅限整存整取，部分支取后剩余金额不低于起存金额
2. 已支取部分按活期利率计息
3. 剩余部分仍按原利率和原存期计息
4. 部分提前支取仅限一次（部分银行允许两次）

#### 利息损失示例

假设存入10万元，1年期，年利率1.75%：
- 正常到期利息：100,000 × 1.75% = 1,750元
- 存满10个月提前支取（活期利率0.2%）：100,000 × 0.2% × (10/12) ≈ 166.67元
- 利息损失：1,750 - 166.67 = 1,583.33元

## 业务规则

1. **利率执行**：存入时锁定利率，存期内利率调整不影响已存入的定期存款
2. **逾期支取**：超过存期部分按支取日活期利率计息
3. **提前支取计息**：全部或部分提前支取，提前支取部分按活期利率计息
4. **自动转存利率**：每次转存按转存日当天挂牌利率重新定价
5. **计息方式**：逐笔计息法，按实际天数计算
6. **存单挂失**：定期存单遗失需立即办理挂失，挂失后7个工作日可补办
7. **质押贷款**：定期存单可用于质押贷款，质押率通常不超过存单面额的90%
8. **继承处理**：存款人死亡后，继承人需持公证书或法院判决书办理支取

## 异常处理

### 场景一：存单遗失

- 立即通过客服电话或柜面办理口头挂失
- 5个工作日内到柜面办理书面挂失
- 挂失7个工作日后可申请补发存单或支取
- 如挂失期间存单到期，挂失解除后方可支取

### 场景二：存单到期但无法到柜面

- 设置了自动转存的无需处理
- 未设置自动转存的，到期后自动转入活期
- 可通过手机银行远程操作支取

### 场景三：提前支取时利息争议

- 向客户解释提前支取按活期计息的规则（已在存单上注明）
- 如客户对金额有疑问，提供详细的利息计算清单
- 建议客户考虑部分提前支取以减少利息损失

### 场景四：大额现金存入定期

- 单笔现金存入超过5万元需出示身份证
- 超过20万元需提前预约
- 关注资金来源合理性，必要时进行反洗钱尽调

## 风控要点

1. **大额存款来源审查**：对大额现金存入进行资金来源询问和记录
2. **频繁存取监控**：短期内频繁存入支取定期存款，可能涉及洗钱或资金异常
3. **代理支取核实**：代理支取需严格核实双方身份和授权关系
4. **存单真伪验证**：支取时核验存单防伪特征，防范伪造存单
5. **利率合规**：严格执行人民银行基准利率和自律机制利率上限
6. **反洗钱报告**：单笔或累计大额交易按规定上报反洗钱系统

## 城商行/农商行特色

1. **利率优势**：城商行/农商行定期存款利率通常较国有大行上浮幅度更大，具有吸储优势
2. **特色存款产品**：推出本地特色存款产品（如"丰收存""惠民存"等），利率适当优惠
3. **节日营销**：利用春节、中秋等节日推出限时优惠利率存款
4. **社区化服务**：针对老年客户群体提供上门办理到期支取服务
5. **存单质押便利**：本地客户存单质押贷款审批更快，手续更简便
6. **存款保险宣传**：主动宣传50万元存款保险保障，增强客户信心
7. **跨地区限制**：部分城商行/农商行受经营区域限制，定期存款仅限本地办理
