# 大额存单购买与转让流程

## 流程概述

大额存单（Certificate of Deposit, CD）是银行面向个人和企业发行的大额存款凭证，属于一般性存款产品，受存款保险保障。个人大额存单起存金额通常为20万元，利率较同期限定期存款有上浮优势。适用于资金量较大、追求较高固定收益且有一定期限偏好的客户。办理渠道包括柜面、手机银行和网上银行。大额存单具有可转让功能，增加了流动性。

## 前置条件

### 购买条件

- 已开立本行个人结算账户（Ⅰ类户）
- 账户余额不低于所购大额存单面额
- 大额存单产品处于发行期内
- 购买金额达到起存金额（通常20万元起）

### 转让条件

- 持有大额存单且已过锁定期（通常为持有满1-6个月）
- 大额存单未到期
- 银行已开通大额存单转让功能
- 转让金额不得低于起存金额

### 所需材料

- 本人有效身份证件
- 本行银行卡/存折
- 如委托他人代办：双方身份证原件及公证授权书

## 操作步骤

### 购买流程

1. **查看产品发行信息** → 2. **确认购买金额与期限** → 3. **阅读产品说明书** → 4. **签署购买协议** → 5. **资金扣划** → 6. **生成大额存单凭证** → 7. **确认购买成功**

### 转让流程

1. **发起转让申请** → 2. **设定转让价格/利率** → 3. **挂单发布** → 4. **买方认购匹配** → 5. **资金清算与过户** → 6. **转让完成确认**

### 到期兑付流程

1. **到期日系统自动处理** → 2. **本息计算** → 3. **资金划入活期账户** → 4. **存单凭证注销**

## 各环节详解

### 一、购买环节

#### 柜面购买

1. 客户到柜台提出购买大额存单需求
2. 柜员查询当前在售的大额存单产品列表
3. 向客户介绍各期限产品的利率和要素
4. 确认购买金额（须为起存金额的整数倍，通常以1万元递增）
5. 阅读并签署《大额存单产品协议书》
6. 从客户活期账户扣划相应资金
7. 系统生成大额存单电子凭证或纸质凭证
8. 客户确认凭证信息无误后签收

#### 电子渠道购买

1. 登录手机银行/网上银行
2. 进入"存款"→"大额存单"模块
3. 查看当前可购买的产品列表（期限、利率、剩余额度）
4. 选择目标产品，输入购买金额
5. 在线阅读产品说明书和协议条款
6. 确认购买并输入交易密码/指纹/人脸识别
7. 系统实时扣款并生成电子存单
8. 在"我的存款"中查看持有的大额存单

### 二、转让环节

#### 转让定价规则

- **折价转让**：转让价格低于存单面值+应计利息，买方收益率高于票面利率
- **平价转让**：转让价格等于面值+应计利息，买方收益率等于票面利率
- **溢价转让**：转让价格高于面值+应计利息，买方收益率低于票面利率

#### 转让操作（以手机银行为例）

1. 登录手机银行，进入"大额存单"模块
2. 选择持有的大额存单，点击"转让"
3. 系统显示存单基本信息（面值、利率、到期日、已持有天数）
4. 设定转让价格（系统提供参考价格和最低价格）
5. 确认转让信息，提交挂单
6. 挂单有效期通常为1-3个工作日
7. 买方通过平台浏览并认购
8. 匹配成功后系统自动完成资金清算和存单过户
9. 转让款项实时到账卖方活期账户

### 三、到期兑付

1. **到期自动兑付**：到期日系统自动将本息划入客户活期账户
2. **利息计算**：按存入时约定的利率和实际存期计算利息
3. **遇节假日处理**：到期日遇节假日顺延至下一工作日，顺延期间按活期利率计息
4. **提前支取（无转让需求时）**：按支取日活期利率计息，损失定期利息

## 业务规则

1. **起存金额**：个人大额存单起存金额通常为20万元，部分银行为30万元
2. **购买金额**：以1万元为最小递增单位
3. **存期选择**：1个月、3个月、6个月、9个月、1年、18个月、2年、3年、5年
4. **利率上浮**：大额存单利率较同期限定期存款上浮20-55个基点（BP）
5. **转让限制**：部分产品设有锁定期（持有后1-6个月方可转让），转让后买方需继续持有至到期或满足最低持有期
6. **转让费用**：部分银行对转让收取手续费（通常为转让金额的0.05%-0.1%），部分银行免费
7. **转让次数**：一张大额存单转让次数不限
8. **不可转让情形**：已质押、已冻结的大额存单不可转让
9. **付息方式**：到期一次还本付息或按月/季付息（视产品设计）
10. **存款保险**：大额存单纳入存款保险保障范围，50万元以内全额保障

## 异常处理

### 场景一：大额存单发行额度售罄

- 告知客户当前期已售罄，建议关注下一期发行计划
- 推荐其他期限或替代产品（定期存款、结构性存款等）
- 可提供预约服务，下期发行时优先通知

### 场景二：转让挂单长时间无人认购

- 建议卖方适当降低转让价格（提高买方收益率）
- 说明市场利率变动对转让价格的影响
- 如确实无法转让，可选择持有到期或提前支取（按活期计息）

### 场景三：转让过程中系统故障

- 转让挂单未匹配的，系统自动撤单
- 已匹配但未完成清算的，由系统人工介入处理
- 资金异常时暂停转让功能，待修复后继续

### 场景四：提前支取与转让的选择

- 帮助客户比较提前支取（活期利率）和转让（可能获得更好收益）的差异
- 通常情况下，转让优于提前支取（尤其是持有时间较长时）
- 如市场利率大幅上升，转让可能折价，需综合评估

## 风控要点

1. **资金来源审查**：大额存单购买金额较大，需关注资金来源合法性
2. **反洗钱监控**：大额存单的购买、转让、兑付均纳入大额交易和可疑交易监控
3. **转让合规性**：转让平台需确保交易真实、价格公允，防止洗钱通道
4. **利率合规**：大额存单利率需符合市场利率定价自律机制的要求
5. **信息披露**：充分披露产品利率、期限、提前支取规则、转让规则等关键信息
6. **投资者适当性**：确保客户风险承受能力与产品匹配

## 城商行/农商行特色

1. **利率定价优势**：城商行/农商行大额存单利率通常高于国有大行10-30BP，吸储竞争力强
2. **发行频次**：受规模和监管限制，发行频次可能低于大行，需提前规划额度
3. **本地营销**：通过社区活动、VIP客户专场等方式进行本地化精准营销
4. **转让平台局限**：部分中小银行转让平台用户量较小，转让流动性可能弱于大行
5. **跨区域限制**：城商行大额存单通常仅限本地区客户购买（受经营区域限制）
6. **VIP客户优先**：对高净值客户提供大额存单优先购买权和专属利率
7. **存单质押联动**：大额存单质押贷款作为本行特色服务，审批更快、利率更优
