# 结构性存款购买流程

## 流程概述

结构性存款是在普通存款基础上嵌入金融衍生品（如期权）的存款产品，将存款与衍生品挂钩，收益与标的资产（利率、汇率、指数、黄金等）的表现挂钩。结构性存款纳入存款保险保障（本金部分），但收益具有不确定性。适用于有一定风险认知能力、追求潜在较高收益的个人客户。办理渠道包括柜面和电子渠道（手机银行/网上银行）。

根据银保监会《关于进一步规范商业银行结构性存款业务的通知》（银保监办发〔2019〕204号），结构性存款须纳入银行表内管理，按存款管理并缴纳存款准备金和存款保险保费。

## 前置条件

### 购买条件

- 已开立本行个人结算账户
- 已完成客户风险评估（风险等级为稳健型及以上）
- 风险评估结果在有效期内（通常1年）
- 首次购买需到柜面进行面签（部分银行要求）
- 购买资金为自有资金

### 所需材料

- 本人有效身份证件
- 本行银行卡/存折
- 风险评估问卷（首次购买时）
- 产品协议书和风险揭示书

## 操作步骤

1. **风险评估（首次）** → 2. **产品选择与了解** → 3. **阅读产品说明书与风险揭示** → 4. **签署协议** → 5. **资金扣划** → 6. **产品成立确认** → 7. **存续期跟踪** → 8. **到期兑付**

## 各环节详解

### 一、风险评估

1. 首次购买结构性存款前须完成风险评估问卷
2. 评估内容涵盖投资经验、风险承受能力、投资目标等
3. 评估结果分为5个等级：保守型、稳健型、平衡型、进取型、激进型
4. 结构性存款通常要求稳健型及以上等级
5. 评估结果有效期为1年，过期需重新评估
6. 柜面首次评估需本人亲自完成，不可代办

### 二、产品选择

#### 常见挂钩标的

| 挂钩类型 | 标的资产 | 收益特征 |
|----------|----------|----------|
| 利率挂钩型 | LIBOR/SHIBOR/LPR | 利率区间观察，触碰区间外则获得高收益 |
| 汇率挂钩型 | 美元/欧元/日元汇率 | 汇率波动区间内获得浮动收益 |
| 黄金挂钩型 | 上海金/国际金价 | 金价涨跌方向判断正确获得高收益 |
| 指数挂钩型 | 沪深300/中证500 | 指数涨幅达到触发条件获得高收益 |
| 信用挂钩型 | 特定企业信用事件 | 未发生信用事件获得较高收益 |

#### 收益结构设计

- **保本保最低收益型**：保证本金和最低收益率（如1.5%），最高可达4-5%
- **保本浮动收益型**：保证本金，收益完全取决于挂钩标的表现
- **二元结构（看涨/看跌）**：标的达到触发条件获得高收益，否则获得低收益

### 三、签约购买

#### 柜面购买流程

1. 客户经理介绍当期在售的结构性存款产品
2. 提供产品说明书、风险揭示书、产品协议
3. 重点说明挂钩标的、收益计算规则、最低收益和最高收益
4. 客户确认风险承受能力与产品匹配
5. 阅读并签署《结构性存款产品协议》和《风险揭示书》
6. 双录环节（根据监管要求，部分银行对结构性存款销售实施双录）
7. 柜员操作系统完成购买
8. 从客户活期账户扣划购买资金
9. 打印购买确认凭证，客户签收

#### 电子渠道购买流程

1. 登录手机银行/网上银行
2. 进入"存款"→"结构性存款"模块
3. 查看在售产品列表，筛选挂钩类型和期限
4. 点击产品详情，查看产品说明书和风险揭示
5. 系统自动校验风险评估结果和有效期
6. 输入购买金额（须为起购金额的整数倍）
7. 在线阅读并勾选同意协议条款
8. 通过人脸识别/密码验证完成购买
9. 资金实时扣划，生成电子持有凭证

### 四、到期兑付

1. **观察期结束**：产品到期日或观察期结束日确定挂钩标的最终表现
2. **收益计算**：根据产品说明书约定的收益计算规则确定实际收益
3. **本息划转**：本金和收益在到期日后1-3个工作日内划入活期账户
4. **到账通知**：通过短信或APP推送通知客户到账信息
5. **收益明细**：提供详细的收益计算清单

## 业务规则

1. **起购金额**：个人结构性存款起购金额通常为1万元，部分产品为5万元或20万元
2. **投资期限**：常见期限为7天、14天、1个月、3个月、6个月、1年
3. **不可提前赎回**：大部分结构性存款不可提前赎回（少数产品设有提前赎回条款）
4. **冷静期**：购买后24小时内可撤销（部分银行设有冷静期）
5. **募集期与成立日**：购买后至产品成立日之间为募集期，按活期利率计息
6. **存款保险**：本金部分纳入存款保险保障（50万元以内）
7. **税费**：结构性存款收益暂不征收个人所得税
8. **限额管理**：单只结构性存款产品设有发行总额上限
9. **适合性管理**：不得向风险评估不合格的客户推荐结构性存款

## 异常处理

### 场景一：风险评估过期

- 提示客户风险评估已过期，需重新评估后方可购买
- 电子渠道可在线完成重评，柜面需重新填写问卷
- 重评结果可能与之前不同，如等级下降可能无法购买部分产品

### 场景二：产品未成立

- 募集金额未达到产品成立最低规模要求时，产品不成立
- 已扣划资金在3个工作日内全额退回活期账户
- 募集期间按活期利率计息

### 场景三：到期收益为零或极低

- 根据产品说明书，最低收益可能为零或极低（如0.5%）
- 向客户解释挂钩标的表现未达预期导致低收益
- 本金不受影响（保本型产品）

### 场景四：客户要求提前赎回

- 向客户说明结构性存款通常不支持提前赎回
- 如有提前赎回条款，说明可能的本金损失和手续费
- 建议客户持有到期以保障本金安全

## 风控要点

1. **适当性管理**：严格执行投资者适当性管理，确保产品风险等级不超过客户风险承受能力
2. **信息披露**：充分披露挂钩标的、收益计算规则、风险因素等关键信息
3. **禁止承诺收益**：不得承诺保本保收益（仅保证本金和最低收益）
4. **销售合规**：销售人员需具备相应资质，不得误导销售
5. **双录管理**：按要求对销售过程进行录音录像
6. **反洗钱监控**：大额购买结构性存款纳入反洗钱监控
7. **表内管理**：结构性存款须纳入表内核算，缴纳存款准备金和保费

## 城商行/农商行特色

1. **产品设计能力**：城商行/农商行衍生品设计能力有限，产品种类较国有大行少，以利率和汇率挂钩型为主
2. **合作渠道**：部分中小银行通过与大行或券商合作获取衍生品对冲能力
3. **收益率竞争**：为吸引客户，城商行结构性存款的最高收益率通常较大行高20-50BP
4. **起购金额灵活**：部分城商行推出1万元起购的结构性存款，降低投资门槛
5. **本地化营销**：结合本地客户偏好设计产品（如黄金挂钩型在部分地区更受欢迎）
6. **风险提示加强**：针对中小城市客户金融知识相对薄弱的情况，加强产品风险说明
7. **代销模式**：部分农商行不具备结构性存款发行资质，采用代销大行产品的方式
