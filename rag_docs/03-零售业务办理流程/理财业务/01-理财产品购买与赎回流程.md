# 理财产品购买与赎回流程

## 流程概述

银行理财产品是商业银行或理财子公司发行的资产管理产品，投资者将资金委托给管理人进行投资运作，按约定获取投资收益。自2018年资管新规实施以来，理财产品已全面净值化转型，打破刚性兑付，产品净值随市场波动。

适用于有投资理财需求、追求高于存款收益的个人客户。办理渠道包括柜面、手机银行、网上银行和智能柜员机。全流程涵盖风险评估、产品选择、签约购买、持有管理和赎回到账五个阶段。

## 前置条件

### 购买条件

- 已开立本行个人结算账户
- 已完成投资者风险评估且在有效期内
- 风险评估结果与产品风险等级匹配
- 首次购买理财产品需在柜面完成面签（部分银行已取消此要求）
- 购买资金为合法自有资金

### 所需材料

- 本人有效身份证件
- 本行银行卡/借记卡
- 风险评估问卷（首次购买时）
- 产品风险揭示书确认签署

## 操作步骤

1. **风险评估（首次/重评）** → 2. **产品浏览与筛选** → 3. **阅读产品说明书** → 4. **确认购买** → 5. **资金扣划** → 6. **产品确认** → 7. **持有期管理** → 8. **赎回/到期** → 9. **资金到账**

## 各环节详解

### 一、风险评估

1. 首次购买理财产品前须完成风险评估问卷
2. 评估内容涵盖年龄、收入、投资经验、风险偏好、投资目标等
3. 风险等级分类（投资者）：
   - C1-保守型：只能购买R1级产品
   - C2-稳健型：可购买R1-R2级产品
   - C3-平衡型：可购买R1-R3级产品
   - C4-进取型：可购买R1-R4级产品
   - C5-激进型：可购买R1-R5级产品
4. 评估结果有效期1年，到期需重新评估
5. 不得向客户推荐超出其风险承受能力的产品

### 二、产品选择

#### 理财产品分类

| 分类维度 | 类型 | 特点 |
|----------|------|------|
| 按风险等级 | R1-R5 | R1最低风险，R5最高风险 |
| 按运作形式 | 封闭式/开放式 | 封闭式有固定期限，开放式可申购赎回 |
| 按投资性质 | 固定收益/混合/权益/商品及衍生品 | 投资标的不同，风险收益特征不同 |
| 按募集方式 | 公募/私募 | 公募面向大众，私募面向合格投资者 |

#### 产品要素对比

- **业绩比较基准**：产品预期的年化收益参考值（不代表实际收益）
- **产品期限**：7天、1个月、3个月、6个月、1年、2年、3年等
- **起购金额**：R1-R2通常1元起购，R3以上通常1万-10万元起
- **管理费**：包括管理费、托管费、销售服务费等（已从净值中扣除）
- **申赎规则**：开放日/封闭期、赎回到账时间

### 三、签约购买

#### 柜面购买流程

1. 理财经理介绍适合的产品
2. 提供产品说明书和风险揭示书
3. 重点说明：产品类型、风险等级、业绩比较基准、投资标的、申赎规则
4. 确认客户风险评估结果与产品风险等级匹配
5. 客户阅读并签署《理财产品协议》和《风险揭示书》
6. 双录环节（根据监管要求，柜面销售理财产品需录音录像）
7. 柜员在系统中执行购买交易
8. 资金从活期账户扣划
9. 打印购买确认单，客户签收

#### 电子渠道购买流程

1. 登录手机银行/网上银行
2. 进入"理财"模块
3. 浏览产品列表，使用筛选功能（风险等级、期限、起购金额）
4. 点击产品详情，查看产品说明书、风险揭示书、历史净值
5. 系统自动校验风险评估结果和匹配性
6. 输入购买金额
7. 在线阅读并勾选同意协议和风险揭示书
8. 通过人脸识别/密码验证完成购买
9. 资金实时扣划，生成持有份额

### 四、持有期管理

#### 净值查看

- 开放式产品：每日公布净值（T+1或T+2）
- 定期开放式产品：开放日前公布净值
- 封闭式产品：定期公布净值（通常每周或每月）

#### 分红方式

- **现金分红**：分红金额划入活期账户
- **红利再投**：分红金额自动转为产品份额

### 五、赎回与到期

#### 开放式产品赎回

1. 登录手机银行/网上银行或到柜面
2. 进入"我的理财"→选择持有的产品
3. 点击"赎回"，输入赎回份额/金额
4. 确认赎回信息（预计到账时间）
5. 提交赎回申请
6. T日赎回，T+1确认，T+2或T+3到账（不同产品有差异）

#### 定期开放式产品赎回

1. 在开放期内提交赎回申请
2. 开放期通常为产品到期前5-10个工作日
3. 未在开放期赎回的，自动续期至下一期
4. 到期后T+2或T+3到账

#### 封闭式产品到期

1. 到期日系统自动处理
2. 无需客户操作
3. 本金和收益在到期后1-3个工作日到账
4. 短信/APP通知到账信息

## 业务规则

1. **冷静期**：购买后24小时内可撤销（部分银行对私募产品设有24小时冷静期）
2. **赎回限制**：部分产品设有最低持有期，持有期内不可赎回
3. **巨额赎回**：单日赎回申请超过产品总份额10%时，管理人可延期办理
4. **费用结构**：管理费0.1%-0.5%/年、托管费0.02%-0.05%/年、销售服务费0.1%-0.5%/年
5. **收益税费**：个人投资者理财收益暂不征收个人所得税
6. **信息披露**：定期披露产品净值、投资组合、重大事项等信息
7. **适当性管理**：不得向风险承受能力不足的客户推荐高风险产品
8. **双录要求**：柜面销售理财产品须进行录音录像

## 异常处理

### 场景一：产品净值跌破本金

- 向客户解释净值化转型后产品不保本的规则
- 说明市场波动是正常现象，净值会随市场回升
- 如客户无法承受亏损，建议赎回并转投低风险产品
- 记录客户投诉并按投诉流程处理

### 场景二：赎回延迟到账

- 确认赎回申请是否在开放期内提交
- 检查是否触发巨额赎回限制
- 如遇节假日，到账时间顺延
- 如系统原因导致延迟，及时通知客户并加急处理

### 场景三：风险评估过期无法购买

- 提示客户风险评估已过期，需重新评估
- 手机银行可在线完成重评（5-10分钟）
- 重评结果可能与之前不同，影响可购买产品范围
- 如重评结果降低，建议客户选择低风险产品

### 场景四：产品提前终止

- 管理人因市场原因或产品规模过小可提前终止产品
- 提前10个工作日公告通知
- 本金和截至终止日的收益在5个工作日内到账
- 向客户解释原因并推荐替代产品

## 风控要点

1. **适当性管理**：严格执行投资者适当性管理，产品风险等级不得超出客户承受能力
2. **信息披露**：充分披露产品风险、投资标的、费用结构等关键信息
3. **禁止保本承诺**：不得承诺保本保收益（存款类产品除外）
4. **销售合规**：销售人员须持有相应资质，不得误导销售
5. **双录管理**：柜面销售全程录音录像，保存期限符合监管要求
6. **资金安全**：理财资金独立托管，与管理人自有资产隔离
7. **反洗钱**：大额理财购买纳入反洗钱监控

## 城商行/农商行特色

1. **产品线有限**：城商行/农商行理财产品种类较少，以固收类产品为主
2. **代销模式**：部分中小银行不具备理财发行资质，采用代销理财子公司产品
3. **本地信任优势**：基于本地客户关系和信任，理财销售更依赖面对面服务
4. **收益率竞争**：为吸引客户，城商行理财产品业绩比较基准通常较大行高20-50BP
5. **投资者教育**：中小城市客户金融素养差异较大，需加强投资者教育
6. **专属产品**：针对本地VIP客户推出专属高收益理财产品
7. **社区营销**：利用社区活动、理财沙龙等方式推广理财产品
