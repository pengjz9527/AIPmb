# 风险评估流程

## 流程概述

投资者风险评估是银行在销售理财产品前对投资者风险承受能力进行识别和评价的过程，是投资者适当性管理的基础环节。通过科学的评估方法，了解投资者的风险偏好、财务状况和投资经验，将投资者分为不同风险等级，并据此推荐相匹配的理财产品。

根据《商业银行理财业务监督管理办法》和《银行保险机构销售适当性管理办法》，银行须在销售前完成投资者风险评估，并不得向投资者推荐超出其风险承受能力的产品。

## 前置条件

### 首次评估条件

- 已开立本行个人结算账户
- 投资者本人亲自到场（柜面）或本人操作（电子渠道）
- 具备完全民事行为能力

### 定期重评条件

- 上次风险评估结果已超过有效期（通常1年）
- 投资者主动要求重新评估
- 银行发现投资者风险承受能力发生重大变化

### 动态调整条件

- 投资者年龄、职业、收入等基本信息发生重大变化
- 投资者投资行为出现异常（如频繁大额亏损交易）
- 投资者主动反馈风险偏好发生变化

## 操作步骤

1. **确定评估类型（首次/重评/动态调整）** → 2. **发放风险评估问卷** → 3. **投资者填写问卷** → 4. **系统评分** → 5. **确定风险等级** → 6. **结果告知与确认** → 7. **结果记录与存档**

## 各环节详解

### 一、评估类型

#### 首次评估

- 投资者首次购买理财产品时进行
- 柜面首次评估须本人亲自填写，不得代办
- 电子渠道首次评估须通过人脸识别/密码验证确认身份
- 评估问卷不少于10道题

#### 定期重评

- 评估结果有效期通常为1年
- 到期前系统自动提醒投资者重评
- 重评可通过电子渠道完成
- 重评结果可能与首次不同

#### 动态调整

- 银行系统根据投资者行为数据动态监测
- 发现风险承受能力明显下降时主动提示
- 投资者可随时主动申请调整

### 二、问卷设计

#### 标准问卷维度（不少于10题）

| 评估维度 | 题目数量 | 示例问题 |
|----------|----------|----------|
| 年龄与健康状况 | 1-2题 | 您的年龄段？ |
| 就业与收入状况 | 2-3题 | 您的年收入大约是？您的收入来源稳定性？ |
| 家庭财务状况 | 1-2题 | 您的家庭负债占收入比例？ |
| 投资经验 | 1-2题 | 您有多少年的投资经验？您曾投资过哪些品种？ |
| 风险偏好 | 2-3题 | 如果投资亏损10%，您会？您期望的投资目标是？ |
| 投资目的 | 1-2题 | 您的主要投资目的是？ |

#### 典型问卷题目示例

**Q1：您的年龄段？**
- A. 18-30岁（5分）
- B. 31-45岁（4分）
- C. 46-60岁（3分）
- D. 60岁以上（2分）

**Q2：您的年收入大约是？**
- A. 50万元以上（5分）
- B. 20-50万元（4分）
- C. 10-20万元（3分）
- D. 10万元以下（2分）

**Q3：如果投资亏损10%，您会？**
- A. 继续持有甚至加仓（5分）
- B. 继续持有观望（4分）
- C. 赎回部分降低风险（3分）
- D. 全部赎回避免更大损失（2分）

### 三、评分与分级

#### 评分规则

- 每题选项对应不同分值
- 系统自动计算总分
- 根据总分区间确定风险等级

#### 风险等级对应关系

| 投资者等级 | 等级名称 | 总分区间（参考） | 可购买产品风险等级 |
|-----------|----------|-----------------|-------------------|
| C1 | 保守型 | 10-20分 | R1（低风险） |
| C2 | 稳健型 | 21-30分 | R1-R2（中低风险及以下） |
| C3 | 平衡型 | 31-40分 | R1-R3（中风险及以下） |
| C4 | 进取型 | 41-50分 | R1-R4（中高风险及以下） |
| C5 | 激进型 | 51-60分 | R1-R5（所有等级） |

#### 产品风险等级说明

| 产品等级 | 风险特征 | 典型产品 |
|----------|----------|----------|
| R1 | 极低风险 | 货币基金、保本存款类 |
| R2 | 较低风险 | 固收类理财（纯债型） |
| R3 | 中等风险 | 固收+理财（含少量权益） |
| R4 | 较高风险 | 混合类/权益类理财 |
| R5 | 高风险 | 衍生品/私募类产品 |

### 四、结果确认与存档

1. 向投资者展示评估结果和风险等级
2. 说明不同风险等级可购买的产品范围
3. 投资者确认评估结果
4. 柜面评估：投资者签字确认
5. 电子渠道评估：电子签名/密码确认
6. 评估记录纳入客户档案管理
7. 系统记录评估时间、评估方式、评估结果

## 业务规则

1. **有效期**：风险评估结果有效期通常为1年，到期须重新评估
2. **不得代办**：首次风险评估须投资者本人亲自完成
3. **适当性匹配**：不得向投资者推荐超出其风险承受能力的产品
4. **主动重评**：投资者财务状况发生重大变化时应主动申请重评
5. **结果可逆**：重评结果可能上升也可能下降
6. **多次评估**：以最近一次评估结果为准
7. **风险警示**：如投资者坚持购买超出风险等级的产品，银行应书面警示（部分银行拒绝销售）
8. **信息保密**：评估信息严格保密，仅用于产品销售适当性管理

## 异常处理

### 场景一：投资者不理解问卷内容

- 理财经理耐心解释每道题的含义
- 用通俗语言帮助理解专业术语
- 不得代替投资者填写答案
- 对于老年客户可提供放大版问卷或协助阅读

### 场景二：评估结果与客户自我认知不符

- 向客户解释评分逻辑和依据
- 如客户认为自身承受能力更强，可重新评估一次
- 但不得引导客户选择特定答案以获得更高等级
- 最终以系统评分结果为准

### 场景三：评估结果为保守型但客户坚持购买高风险产品

- 向客户充分说明产品风险
- 出具《超风险购买警示函》
- 要求客户书面确认已知悉风险并自愿承担
- 双录环节增加风险告知内容
- 部分银行规定不得向保守型客户销售中高风险产品

### 场景四：系统评估与实际行为不一致

- 投资者评估为激进型但实际投资行为极为保守
- 银行可通过行为数据动态调整投资者画像
- 在推荐产品时综合考虑评估结果和实际行为偏好
- 如差异较大，建议投资者重新评估

## 风控要点

1. **真实性**：确保投资者真实填写问卷，不受销售人员引导
2. **合规性**：严格按评估结果匹配产品，不得违规销售超风险产品
3. **可追溯性**：评估过程和结果完整记录，可供监管检查
4. **防引导**：禁止销售人员为达成销售目标引导客户填写高分答案
5. **信息保护**：评估涉及的收入、资产等敏感信息严格保密
6. **特殊群体保护**：对65岁以上老年投资者加强适当性保护
7. **定期审计**：定期审计评估流程的合规性和有效性

## 城商行/农商行特色

1. **方言服务**：针对农村老年客户，提供方言讲解问卷内容
2. **简化版本**：对金融知识较薄弱的客户群体，提供简化易懂版问卷
3. **面对面服务**：城商行/农商行更依赖柜面和理财经理面对面完成评估
4. **耐心引导**：理财经理耐心帮助客户理解风险概念，不催促不引导
5. **本地化题目**：问卷中可加入本地化的收入参考标准（如农业收入）
6. **社区教育**：通过社区金融知识讲座帮助居民建立风险意识
7. **系统支持**：部分中小银行评估系统较简单，需加强系统建设
