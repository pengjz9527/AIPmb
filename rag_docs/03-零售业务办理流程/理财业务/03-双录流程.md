# 双录流程

## 流程概述

双录（录音录像）是指银行在销售理财产品、代销产品、结构性存款等过程中，对销售关键环节进行同步录音录像，以记录销售人员的产品推介、风险揭示和客户确认等过程。双录制度旨在保护金融消费者合法权益，规范银行销售行为，防范误导销售和违规销售。

根据原银监会《银行业金融机构销售专区录音录像管理暂行规定》（银监发〔2017〕47号），商业银行在营业网点销售理财产品须在专门的销售专区进行，并对销售过程实施录音录像。

## 前置条件

### 适用业务范围

| 业务类型 | 是否双录 | 说明 |
|----------|----------|------|
| 理财产品销售 | 是 | 所有理财产品柜面销售 |
| 结构性存款销售 | 是 | 柜面销售结构性存款 |
| 代销基金 | 是 | 柜面代销基金产品 |
| 代销保险 | 是 | 柜面代销保险产品 |
| 代销信托 | 是 | 柜面代销信托产品 |
| 贵金属销售 | 部分 | 投资型贵金属柜面销售 |
| 大额存单 | 否 | 普通存款产品不要求双录 |
| 定期存款 | 否 | 普通存款产品不要求双录 |

### 硬件要求

- 销售专区配备固定的录音录像设备
- 设备能清晰录制销售人员和客户的面部特征和对话
- 录像画面能同时捕捉销售人员和客户
- 录音能清晰记录双方对话内容
- 设备运行稳定，存储容量充足

### 人员要求

- 销售人员须持有相应业务资质（基金从业、保险从业等）
- 销售人员须经双录系统操作培训
- 销售人员熟悉产品知识和话术规范

## 操作步骤

1. **引导客户至销售专区** → 2. **开启录音录像设备** → 3. **身份核实** → 4. **产品推介与风险揭示** → 5. **客户确认** → 6. **签署协议** → 7. **关闭录音录像** → 8. **文件保存**

## 各环节详解

### 一、销售专区设置

#### 专区要求

1. 营业网点须设立专门的理财/代销产品销售专区
2. 专区内有明显标识（"销售专区""录音录像中"等提示牌）
3. 专区内配备录音录像设备
4. 专区内展示产品宣传材料和风险提示
5. 非销售区域不得进行理财产品销售

#### 设备配置

- 高清摄像头（至少720P），固定安装在销售区
- 全向麦克风或桌面麦克风
- 双录终端（电脑/平板，控制录制和存储）
- 存储设备（本地存储+云端备份）
- 时间水印功能（记录录制时间）

### 二、录音录像操作流程

#### 录制开始

1. 引导客户进入销售专区就座
2. 告知客户本区域进行录音录像
3. 启动双录设备，确认画面和声音正常
4. 录制销售人员工号/姓名标识

#### 录制中必须包含的内容

**（1）身份确认环节**

```
销售人员：您好，我是XX银行理财经理XXX（展示工牌），请问您是XXX先生/女士吗？
客户：是的。
销售人员：请问您今天是来办理XX产品的购买吗？
客户：是的。
```

**（2）风险评估确认**

```
销售人员：请问您之前做过风险评估吗？评估结果是否还在有效期内？
客户：做过了，结果是XX等级，还在有效期内。
销售人员：好的，根据您的风险评估结果，您适合购买R1-RX级的产品。
```

**（3）产品推介环节**

```
销售人员：现在向您介绍XX理财产品，该产品风险等级为RX，
产品期限为XX天/月，业绩比较基准为X%-X%。
该产品主要投资于XX，可能面临XX风险。
需要特别提醒您，该产品不保本/不保收益，
产品净值可能上升也可能下降。
```

**（4）风险揭示环节**

```
销售人员：现在为您进行风险揭示。该产品的主要风险包括：
1. 市场风险：产品净值可能因市场波动而下降
2. 流动性风险：封闭期内不可赎回
3. 信用风险：投资标的发行人可能出现违约
4. 管理风险：管理人投资决策可能影响收益
请问您是否了解上述风险？
客户：了解了。
```

**（5）客户确认环节**

```
销售人员：请问您是否已阅读产品说明书和风险揭示书？
客户：已经阅读了。
销售人员：您是否了解该产品不保本不保收益？
客户：了解。
销售人员：您确认购买XX万元该理财产品吗？
客户：确认。
```

**（6）签署环节**

```
销售人员：请您在产品协议和风险揭示书上签字确认。
（客户签字过程录制在画面中）
```

#### 录制结束

1. 确认所有必要环节已录制完毕
2. 告知客户录制结束
3. 关闭录音录像设备
4. 保存录制文件

### 三、存储与管理

#### 存储要求

| 存储要素 | 要求 |
|----------|------|
| 存储期限 | 产品到期后至少5年；发生争议的保存至争议解决后5年 |
| 存储格式 | MP4/H.264（视频），MP3/AAC（音频） |
| 画质要求 | 至少720P，能清晰辨识面部特征 |
| 音质要求 | 能清晰辨识对话内容 |
| 文件命名 | 日期+客户编号+产品编号+流水号 |
| 存储方式 | 本地存储+异地备份/云端存储 |

#### 查阅与调取

- 仅限合规部门、内审部门、监管部门和司法机关调取
- 调取须填写申请单并经主管审批
- 调取记录需登记留痕
- 不得私自复制、传播双录资料

## 业务规则

1. **全程录制**：销售关键环节须全程连续录制，不得选择性录制
2. **画面要求**：画面须同时包含销售人员和客户，能清晰辨认双方面部
3. **声音要求**：对话内容清晰可辨，无明显噪音干扰
4. **不可篡改**：录制文件生成后不可编辑、剪辑或篡改
5. **销售资质**：录制中的销售人员须持有相应业务资质
6. **不得规避**：不得以设备故障等理由规避双录
7. **告知义务**：须在录制前告知客户正在进行录音录像
8. **专区销售**：理财产品须在销售专区内销售，不得在专区外销售

## 异常处理

### 场景一：设备故障无法录制

- 暂停销售操作，不得在无录制情况下继续销售
- 联系IT部门维修设备
- 如短期可修复，等待修复后继续
- 如无法修复，引导客户到其他网点或电子渠道购买
- 设备故障记录并报告

### 场景二：录制过程中客户离开

- 保存已录制的内容
- 客户返回后重新开始录制（重新进行身份确认等全部环节）
- 不得拼接前后两段录制

### 场景三：录制文件缺失或损坏

- 检查存储设备和备份系统
- 如可恢复则恢复文件
- 无法恢复的，视为销售过程缺失，需重新进行销售
- 记录事故并报告合规部门
- 评估是否存在销售违规

### 场景四：客户拒绝被录制

- 向客户解释双录是监管要求，旨在保护消费者权益
- 说明录制内容仅限内部保存，不会外泄
- 如客户仍拒绝，告知无法在柜面完成购买
- 建议客户通过电子渠道自行购买（线上购买不要求双录）

## 风控要点

1. **合规底线**：双录是监管硬性要求，不得规避或变通
2. **话术规范**：销售人员须使用标准化话术，不得随意发挥
3. **禁止误导**：录制中不得出现保本保收益等违规表述
4. **质检管理**：定期对双录文件进行质量检查（画面、声音、内容完整性）
5. **信息安全**：双录资料包含客户面部信息和对话信息，严格保密
6. **权限管理**：双录系统操作权限分级管理
7. **留痕完整**：录制文件与交易记录关联，确保可追溯

## 城商行/农商行特色

1. **设备投入**：城商行/农商行网点多但规模小，双录设备投入压力较大
2. **人员培训**：基层员工双录操作培训需加强，特别是话术规范
3. **系统建设**：部分中小银行双录系统较为基础，需持续升级
4. **方言录制**：农村地区方言较重地区，录制时需注意语言清晰度
5. **远程双录**：部分银行探索远程视频双录，方便偏远地区客户
6. **质检资源**：中小银行质检人员有限，可考虑AI质检辅助
7. **合规意识**：加强基层合规意识，杜绝为完成任务而规避双录
