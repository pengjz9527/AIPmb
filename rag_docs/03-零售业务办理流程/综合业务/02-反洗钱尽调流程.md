# 反洗钱尽调流程

## 流程概述

反洗钱（Anti-Money Laundering, AML）尽调是银行在业务经营中识别、监测和报告洗钱风险的系列工作。银行作为金融机构，承担着防范洗钱和恐怖融资活动的法定义务。反洗钱尽调涵盖可疑交易识别、大额交易报告、可疑交易报告、客户风险等级划分等核心内容。

根据《反洗钱法》《金融机构大额交易和可疑交易报告管理办法》（中国人民银行令〔2016〕第3号）等法律法规，银行须建立健全反洗钱内控制度，配备专职人员，及时报告可疑交易和大额交易。

## 前置条件

### 适用业务范围

所有零售业务均纳入反洗钱监控范围，包括但不限于：
- 存款业务（大额现金存取、频繁存取）
- 贷款业务（资金用途异常、提前还款资金来源不明）
- 转账业务（大额转账、频繁跨境汇款）
- 理财业务（大额购买、短期赎回）
- 银行卡业务（频繁开卡销卡、异常交易）
- 外汇业务（频繁结售汇、异常跨境汇款）

### 组织机构要求

- 设立反洗钱工作领导小组
- 配备专职反洗钱人员
- 建立反洗钱内控制度和操作流程
- 部署反洗钱监测系统

## 操作步骤

1. **客户风险等级划分** → 2. **交易监控与预警** → 3. **可疑交易识别** → 4. **尽调调查** → 5. **报告上报** → 6. **后续跟踪** → 7. **档案管理**

## 各环节详解

### 一、客户风险等级划分

#### 风险等级分类标准

| 风险等级 | 评分区间 | 客户特征 | 监控频率 |
|----------|----------|----------|----------|
| 禁止类 | - | 制裁名单、恐怖组织 | 拒绝业务 |
| 高风险 | 80-100分 | PEP、现金密集型、高风险行业 | 每季度复核 |
| 中风险 | 40-79分 | 外地户籍、频繁交易、跨境业务 | 每半年复核 |
| 低风险 | 0-39分 | 本地工薪族、交易简单稳定 | 每年复核 |

#### 风险评分要素

**客户维度（30%）**

| 因素 | 高风险 | 中风险 | 低风险 |
|------|--------|--------|--------|
| 国籍 | 高风险国家 | 港澳台 | 中国大陆 |
| 职业 | 现金密集型（珠宝商、废品回收） | 个体经营者 | 公务员、教师 |
| 年龄 | 18-22岁或65岁以上 | 23-35岁 | 36-64岁 |

**地域维度（20%）**

| 因素 | 高风险 | 中风险 | 低风险 |
|------|--------|--------|--------|
| 户籍地 | 边境地区、洗钱高发区 | 外省 | 本地 |
| 经营地 | 自贸区、保税区 | 外地 | 本地 |

**业务维度（30%）**

| 因素 | 高风险 | 中风险 | 低风险 |
|------|--------|--------|--------|
| 产品类型 | 跨境业务、大额现金 | 网银转账 | 柜面存款 |
| 交易规模 | 与身份不匹配 | 中等规模 | 小额稳定 |
| 交易频率 | 高频 | 中频 | 低频 |

**行为维度（20%）**

| 因素 | 高风险 | 中风险 | 低风险 |
|------|--------|--------|--------|
| 交易模式 | 频繁变化 | 偶尔变化 | 稳定规律 |
| 配合度 | 拒绝提供信息 | 部分配合 | 积极配合 |
| 异常行为 | 多人共用账户 | - | 正常使用 |

### 二、交易监控与预警

#### 大额交易报告标准

| 交易类型 | 报告标准 |
|----------|----------|
| 现金存取 | 单笔或日累计5万元以上 |
| 转账交易 | 单笔或日累计50万元以上（个人） |
| 跨境交易 | 单笔或日累计20万元以上（含外币等值） |

#### 可疑交易预警规则

**系统自动监测规则（示例）**

| 预警规则 | 触发条件 | 风险等级 |
|----------|----------|----------|
| 大额现金快进快出 | 当日存入并转出，金额≥20万 | 高 |
| 频繁小额交易 | 日交易≥20笔，单笔<5万 | 中 |
| 分散转入集中转出 | 多笔转入后一笔转出，金额≥10万 | 高 |
| 集中转入分散转出 | 一笔转入后多笔转出，金额≥10万 | 高 |
| 睡眠户突然活跃 | 180天无交易后突然大额交易 | 中 |
| 夜间异常交易 | 23:00-5:00频繁交易 | 中 |
| 与高风险国家交易 | 与制裁/高风险国家有资金往来 | 高 |
| 交易与身份不符 | 学生/无业人员有大额交易 | 高 |

### 三、可疑交易识别

#### 常见可疑交易特征

**个人客户可疑特征**

1. **开户阶段**：
   - 多人同行开户，由一人指挥
   - 开户后立即大额转入
   - 开户目的不明确或含糊

2. **交易阶段**：
   - 大额现金存取，无合理解释
   - 交易金额刚好低于报告标准（规避报告）
   - 交易对手地域分散且无关联
   - 资金快进快出，账户余额极低
   - 频繁更换绑定手机号或密码

3. **行为特征**：
   - 对交易目的含糊其辞
   - 不愿提供或提供虚假身份信息
   - 对费用/利率不敏感
   - 频繁开卡、销卡

#### 人工识别方法

1. **柜面观察**：柜员在办理业务时观察客户行为是否异常
2. **交易分析**：反洗钱分析师对预警交易进行人工分析
3. **尽职调查**：对高风险客户进行深入调查
4. **信息比对**：将客户信息与外部数据库比对
5. **关联分析**：分析客户与交易对手之间的关联关系

### 四、尽调调查

#### 调查内容

1. **交易背景调查**：了解交易的商业目的和经济合理性
2. **资金来源调查**：核实资金是否来自合法渠道
3. **交易对手调查**：了解交易对手的身份和与客户的关系
4. **客户经营调查**：了解客户的经营状况和收入来源
5. **关联关系调查**：排查客户与其他可疑客户的关联

#### 调查方式

| 方式 | 说明 |
|------|------|
| 客户访谈 | 约谈客户了解交易目的和资金来源 |
| 电话核实 | 电话核实客户信息和交易背景 |
| 实地走访 | 走访客户经营场所或住所 |
| 数据查询 | 查询工商、税务、司法等外部数据 |
| 同业协查 | 通过行业协会向其他银行协查 |
| 公开信息 | 搜索网络公开信息 |

### 五、报告上报

#### 大额交易报告

1. 系统自动识别大额交易
2. 反洗钱系统自动生成大额交易报告
3. 反洗钱人员审核报告准确性
4. 通过反洗钱监测系统上报人民银行
5. 大额交易报告在交易发生后5个工作日内提交

#### 可疑交易报告

1. 反洗钱人员对预警交易进行分析判断
2. 确认可疑的，撰写可疑交易分析报告
3. 报告内容包括：客户基本信息、可疑交易描述、分析判断、风险评价
4. 合规部门/反洗钱负责人审核
5. 通过反洗钱监测系统上报人民银行反洗钱中心
6. 可疑交易报告在发现后10个工作日内提交
7. 涉及恐怖融资的立即报告（24小时内）

#### 报告保密要求

- 不得向客户透露已提交可疑交易报告
- 不得向无关人员泄露反洗钱工作信息
- 报告档案专人管理、加密存储

### 六、后续跟踪与措施

#### 风险管控措施

| 措施 | 适用情形 | 操作 |
|------|----------|------|
| 加强监控 | 中风险客户 | 提高交易监控频率 |
| 限制功能 | 高风险客户 | 限制非柜面交易、降低限额 |
| 暂停交易 | 重大可疑 | 暂停可疑交易，等待调查结论 |
| 终止关系 | 确认洗钱/制裁 | 终止业务关系，冻结资金 |
| 报送公安 | 涉嫌犯罪 | 向公安机关报案 |

## 业务规则

1. **大额交易报告**：现金交易5万元以上、转账50万元以上须在5个工作日内报告
2. **可疑交易报告**：发现可疑交易须在10个工作日内报告
3. **涉恐报告**：涉及恐怖融资的须立即报告（24小时内）
4. **保存期限**：客户身份资料和交易记录保存至业务关系结束后至少5年
5. **保密义务**：反洗钱工作信息严格保密，不得向客户透露
6. **培训要求**：全员每年至少接受一次反洗钱培训
7. **内控审计**：定期对反洗钱内控制度执行情况进行审计
8. **处罚规定**：未按规定报告可处20万-500万元罚款，相关人员可处1万-10万元罚款

## 异常处理

### 场景一：客户拒绝说明资金来源

- 记录客户拒绝的情况
- 将该客户风险等级上调
- 加强后续交易监控
- 如交易可疑性增加，提交可疑交易报告
- 必要时限制账户功能

### 场景二：系统预警但人工排除

- 人工分析后确认非可疑的，记录排除理由
- 排除理由须充分、合理、有据可查
- 排除记录纳入档案管理
- 定期复核排除的准确性

### 场景三：涉及公职人员的可疑交易

- 高度关注公职人员的异常大额交易
- 可能涉及贪污受贿所得的洗钱
- 提交可疑交易报告并向纪检监察部门通报
- 加强保密，防止信息泄露

### 场景四：电信诈骗涉案账户

- 接到公安机关通报后立即冻结涉案账户
- 配合公安机关调查取证
- 提交可疑交易报告
- 对关联账户进行排查和监控

## 风控要点

1. **合规底线**：反洗钱是法定义务，不得因业务发展而放松
2. **风险为本**：将有限的资源集中在高风险客户和高风险业务上
3. **系统支撑**：建设完善的反洗钱监测系统，提高识别效率
4. **人员专业**：配备专业反洗钱人员，定期培训提升能力
5. **信息安全**：反洗钱工作信息严格保密
6. **内控制度**：建立健全反洗钱内控制度并有效执行
7. **监管配合**：积极配合监管检查和调查工作

## 城商行/农商行特色

1. **本地化风险特征**：结合本地经济特点识别洗钱风险（如边贸走私、地下钱庄）
2. **熟人社会优势**：利用本地社区关系辅助识别可疑客户和交易
3. **系统建设差距**：部分中小银行反洗钱系统较为基础，需加大投入
4. **人员配置**：反洗钱专职人员较少，常由合规岗兼任
5. **培训提升**：基层员工反洗钱意识需持续加强培训
6. **监管关注**：中小银行反洗钱工作是监管检查重点，需高度重视
7. **特色场景**：关注本地特色洗钱风险场景（如农产品贸易洗钱、地下赌场等）
