# 个人住房贷款申请流程

## 流程概述

个人住房贷款是银行向借款人发放的用于购买自住住房的贷款，是银行零售贷款的核心产品。适用于购买新房（一手房）、二手房、经济适用房等。全流程涵盖选房、申请、审批、签约、抵押登记和放款六个主要阶段。办理渠道以柜面为主，部分银行支持线上预申请。

2025年最新政策要点：存量房贷利率已进行批量调整，LPR定价机制持续完善，部分城市取消限购政策，首付比例有所下调（首套房最低15%）。

## 前置条件

### 借款人基本条件

- 年满18周岁，具有完全民事行为能力的中国公民
- 借款人年龄+贷款期限不超过70年（部分银行放宽至75年）
- 有稳定的职业和收入来源
- 个人信用记录良好（近两年内无严重逾期记录）
- 具备偿还贷款本息的能力

### 所需材料清单

| 材料类别 | 具体内容 |
|----------|----------|
| 身份证明 | 身份证、户口簿（已婚提供配偶身份证和户口簿） |
| 婚姻证明 | 结婚证/离婚证/单身声明 |
| 收入证明 | 单位开具的收入证明、近6个月银行流水、纳税证明 |
| 购房材料 | 商品房买卖合同/认购协议、首付款收据 |
| 房产材料 | 不动产权证（二手房）、评估报告 |
| 其他材料 | 公积金缴存证明（公积金贷款）、征信授权书 |

### 首付比例要求（2025年最新）

| 房屋类型 | 首付比例（非限购城市） | 首付比例（限购城市） |
|----------|----------------------|---------------------|
| 首套房 | 15% | 15%-20% |
| 二套房 | 15%-20% | 20%-30% |
| 三套及以上 | 视政策而定 | 部分城市限购 |

## 操作步骤

1. **选房与签订购房合同** → 2. **贷款预申请与初审** → 3. **提交贷款申请材料** → 4. **银行贷前调查** → 5. **贷款审查审批** → 6. **签订贷款合同** → 7. **办理抵押登记** → 8. **保险购买（如需）** → 9. **贷款发放** → 10. **按月还款**

## 各环节详解

### 一、选房与签约

1. 选择符合条件的房源（新房/二手房）
2. 与开发商签订《商品房买卖合同》或与卖方签订《房屋买卖合同》
3. 支付首付款并取得首付款收据
4. 确认购房合同中贷款条款的约定（贷款金额、利率类型等）
5. 新房：开发商提供预售许可证等资质文件
6. 二手房：进行房产评估，确定评估价

### 二、贷款申请

1. 选择贷款银行（可比较不同银行的利率和服务）
2. 填写《个人住房贷款申请表》
3. 选择贷款方式：商业贷款、公积金贷款或组合贷款
4. 选择利率类型：固定利率或LPR浮动利率
5. 选择还款方式：等额本息或等额本金
6. 提交全部申请材料
7. 签署《个人征信查询授权书》
8. 银行进行初步审核（材料完整性、基本条件符合性）

### 三、贷前调查与审批

1. **征信查询**：查询借款人及配偶的央行征信报告
2. **收入核实**：核实收入证明真实性（电话回访单位、比对流水）
3. **房产评估**：二手房由评估机构出具评估报告
4. **抵押物审查**：审查抵押房产的产权、共有情况
5. **还款能力评估**：月供/月收入比不超过50%
6. **审查审批**：信贷审查人员独立审查，审批人审批

### 四、签约与抵押

1. 审批通过后，银行通知借款人签订《个人住房借款合同》
2. 合同要素确认：贷款金额、期限、利率、还款方式、违约条款
3. 借款人（及配偶）亲笔签署合同
4. 到不动产登记中心办理抵押登记手续
5. 银行取得《不动产登记证明》（抵押权登记）
6. 如有需要，购买住房贷款保险

### 五、放款

1. 银行确认抵押登记完成
2. 确认购房合同已备案（新房）
3. 放款审核：确认所有放款条件已满足
4. 放款方式：
   - 新房（一手房）：资金直接划入开发商监管账户
   - 二手房：资金划入卖方账户（资金监管模式下先到监管账户）
5. 放款后生成还款计划表
6. 通知借款人放款成功和首次还款日

## 业务规则

1. **贷款期限**：最长30年，借款人年龄+贷款期限≤70年
2. **贷款利率**（2025年）：LPR浮动利率，首套房利率通常为LPR-20BP至LPR+20BP
3. **还款方式**：等额本息（月供固定）或等额本金（月供递减）
4. **月供收入比**：月供/月收入不超过50%（家庭总负债/总收入不超过55%）
5. **提前还款**：放款满1年后可申请提前还款，部分银行收取违约金（通常为1-3个月利息）
6. **贷款额度**：不超过房价的85%（首套）或80%（二套），不超过公积金最高额度（公积金贷款）
7. **利率调整**：LPR浮动利率每年调整一次，调整日通常为每年1月1日或贷款发放日对应月

## 异常处理

### 场景一：征信不良被拒贷

- 告知客户征信查询结果和不良记录情况
- 建议客户先处理不良记录（还清逾期款项），等待6-24个月后重新申请
- 可建议增加共同借款人或提高首付比例

### 场景二：收入证明不足

- 月供收入比超过50%，建议增加共同借款人
- 补充提供其他收入证明（兼职收入、租金收入、理财收入等）
- 降低贷款金额或延长贷款期限以降低月供

### 场景三：房产评估价低于成交价

- 银行按评估价和成交价孰低原则确定贷款基数
- 实际可贷金额减少，借款人需增加首付
- 可更换评估机构重新评估（需银行认可）

### 场景四：放款延迟

- 确认抵押登记是否完成
- 确认放款条件是否全部满足
- 如遇银行额度紧张，可能需排队等待放款
- 与开发商/卖方沟通延迟放款事宜

## 风控要点

1. **真实性审核**：核实购房交易真实性，防范虚假交易骗贷
2. **首付来源审查**：核查首付款来源，禁止使用消费贷、经营贷充当首付
3. **收入真实性**：交叉验证收入证明、银行流水、纳税记录的一致性
4. **抵押物合规**：确保抵押房产产权清晰、评估合理
5. **集中度管理**：遵守房地产贷款集中度管理要求
6. **反洗钱**：大额房产交易纳入反洗钱监控范围
7. **贷款用途管理**：确保贷款资金用于购房，严禁挪用

## 城商行/农商行特色

1. **本地房产优势**：对本地房产市场更了解，评估更准确、更高效
2. **审批效率**：审批链条较短，审批速度通常快于大行（3-5个工作日）
3. **灵活定价**：利率定价更灵活，可根据客户资质差异化定价
4. **二手房贷款特色**：在本地二手房市场有更深的合作关系，流程更顺畅
5. **公积金组合贷**：与地方公积金中心合作紧密，组合贷款流程更便捷
6. **自建房/宅基地贷款**：农商行可提供农村自建房贷款、宅基地抵押贷款等特色产品
7. **楼盘合作**：与本地开发商建立楼盘合作关系，提供集中办理服务
