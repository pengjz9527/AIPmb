# 个人经营贷款申请流程

## 流程概述

个人经营贷款是银行向个体工商户、小微企业主、家庭农场主等经营主体发放的用于生产经营活动的贷款。适用于日常经营周转、设备购置、原材料采购、店面装修等经营资金需求。办理渠道以线下柜面和客户经理走访为主，部分银行也提供线上预申请。

个人经营贷款是城商行和农商行的重要业务增长点，特别是在服务小微企业和"三农"方面具有战略意义。

## 前置条件

### 借款人基本条件

- 年满22周岁，具有完全民事行为能力
- 经营实体合法注册且正常经营满1年以上
- 有固定的经营场所
- 经营状况良好，有稳定的营业收入
- 个人征信和企业信用记录良好
- 具备偿还贷款本息的能力

### 所需材料清单

| 材料类别 | 具体内容 |
|----------|----------|
| 个人身份 | 身份证、户口簿、婚姻证明 |
| 经营资质 | 营业执照、税务登记证、经营许可证（特殊行业） |
| 经营证明 | 经营场所租赁合同/产权证、近1年经营流水 |
| 财务资料 | 近2年纳税申报表、财务报表（如有）、近6个月银行流水 |
| 担保材料 | 抵押物产权证/评估报告、保证人资信证明 |
| 用途证明 | 采购合同、订单、设备报价单等 |

## 操作步骤

1. **经营资质初步审查** → 2. **提交贷款申请** → 3. **贷前调查（实地走访）** → 4. **授信评估** → 5. **担保方式确认** → 6. **审查审批** → 7. **签订合同** → 8. **办理担保手续** → 9. **放款** → 10. **贷后管理**

## 各环节详解

### 一、经营资质审查

1. 核实营业执照真实性和有效性（通过国家企业信用信息公示系统查询）
2. 确认经营范围与申请贷款用途一致
3. 了解经营年限（通常要求经营满1年以上）
4. 核实经营场所实际存在（非虚拟地址）
5. 查询经营者个人征信和企业信用信息
6. 排查是否涉及诉讼、被执行、行政处罚等负面信息

### 二、贷前调查

#### 实地走访调查

1. **经营场所走访**：实地查看经营场所规模、经营环境、员工数量、库存情况
2. **经营面谈**：与借款人面谈了解经营历史、盈利模式、上下游关系、发展规划
3. **流水分析**：分析经营流水的稳定性、季节性规律、大额异常交易
4. **同业调查**：了解同行业平均利润水平，交叉验证经营利润
5. **周边走访**：向物业、邻居、上下游客户了解经营情况

#### 调查报告要素

- 经营实体基本情况（注册信息、经营年限、主营范围）
- 经营状况分析（营业收入、利润、负债率）
- 贷款用途合理性分析
- 还款能力分析
- 担保方式及价值评估
- 风险评价和授信建议

### 三、授信评估

#### 授信额度测算

| 担保方式 | 额度上限参考 | 适用场景 |
|----------|-------------|----------|
| 信用贷款 | 50万元 | 经营稳定、信用优质的优质客户 |
| 保证担保 | 200万元 | 有合格保证人提供连带责任保证 |
| 房产抵押 | 房产评估值的70% | 有可抵押房产的客户 |
| 商铺抵押 | 商铺评估值的60% | 以商铺作为抵押物 |
| 设备抵押 | 设备评估值的50% | 以生产设备作为抵押物 |
| 存单质押 | 存单面额的90% | 以定期存单/大额存单质押 |

#### 还款能力评估

- 经营净利润覆盖贷款利息（利息保障倍数≥2倍）
- 经营现金流覆盖贷款本息（偿债覆盖率≥1.2倍）
- 家庭总收入扣除生活支出后可用于还款的部分

### 四、担保方式

#### 抵押担保

1. 抵押物评估：由银行认可的评估机构进行评估
2. 抵押登记：到不动产登记中心办理抵押登记
3. 抵押保险：部分银行要求购买财产保险
4. 抵押物管理：贷后定期检查抵押物状态和价值

#### 保证担保

1. 保证人资格审查：有稳定收入、信用良好、无重大负债
2. 保证方式：连带责任保证
3. 保证人面签：保证人需亲自到场签署保证合同

#### 信用贷款

1. 适用于经营稳定、信用优质的客户
2. 额度通常较低（50万元以下）
3. 利率相对较高
4. 审批条件更严格

### 五、签约与放款

1. 签订《个人经营贷款借款合同》和担保合同
2. 确认贷款要素（金额、期限、利率、还款方式、资金用途）
3. 办理抵押登记或保证担保手续
4. 放款审核：确认所有放款条件已满足
5. 放款方式：
   - 受托支付：直接划入供应商/合作方账户（单笔>30万元）
   - 自主支付：划入借款人经营账户（单笔≤30万元）

## 业务规则

1. **贷款期限**：流动资金贷款通常1-3年，固定资产贷款最长5年
2. **贷款利率**：年化利率3.8%-12%，根据担保方式和客户资质定价
3. **还款方式**：等额本息、等额本金、按月付息到期还本、按季结息
4. **贷款额度**：信用类最高50万，抵押类最高1000万（视抵押物价值）
5. **续贷政策**：符合条件的优质客户可申请无还本续贷（续贷宝等产品）
6. **用途限制**：贷款资金只能用于经营用途，严禁用于购房、投资等
7. **额度循环**：部分产品支持授信额度循环使用（授信有效期内随借随还）
8. **贴息政策**：部分地方政府对小微经营贷款提供贴息补助

## 异常处理

### 场景一：经营流水不规范

- 很多个体户经营流水不完整，需辅助其他材料佐证
- 参考POS机流水、微信/支付宝流水、进销存记录
- 实地考察库存和营业额，交叉验证
- 建议客户规范财务管理，开立经营专用账户

### 场景二：抵押物不足值

- 评估值低于预期，可贷金额减少
- 建议增加补充抵押物或增加保证人
- 部分差额可转为信用贷款或保证贷款
- 可考虑政府性融资担保机构提供担保

### 场景三：经营困难申请展期

- 了解经营困难的具体原因（暂时性还是持续性）
- 评估展期后还款能力的恢复可能性
- 展期期限通常不超过原贷款期限的一半
- 展期利率可能上浮

### 场景四：行业风险集中

- 借款人所在行业出现系统性风险（如教培行业政策变化）
- 审慎评估行业前景和借款人转型能力
- 可能收紧该行业贷款审批标准
- 对存量贷款加强贷后监控

## 风控要点

1. **经营真实性**：核实经营活动真实存在，防范空壳公司骗贷
2. **资金用途管控**：严格监控贷款资金用途，防止流入房地产和资本市场
3. **多头授信**：查询借款人在其他银行的授信情况，防范过度授信
4. **关联交易**：识别借款人与上下游的关联关系，防范虚假交易
5. **行业风险**：关注借款人所在行业的周期性风险和政策风险
6. **贷后管理**：定期回访检查经营状况、资金使用情况、抵押物状态
7. **预警管理**：建立贷后预警指标体系（经营流水下降、逾期、诉讼等）

## 城商行/农商行特色

1. **小微服务优势**：城商行/农商行服务小微企业和"三农"是核心定位，政策支持力度大
2. **走访式风控**：依靠客户经理实地走访和"软信息"进行风控，弥补硬数据不足
3. **灵活担保**：接受农村宅基地、大棚、农机具等特色抵押物
4. **政府合作**：与地方政府合作推出贴息贷款、风险补偿基金贷款等
5. **产业链金融**：围绕本地核心产业链（农产品、制造业等）开展上下游融资
6. **快速审批**：审批流程短，小微贷款3-7个工作日放款
7. **整村授信**：农商行推行整村授信模式，批量服务农村经营客户
8. **续贷便利**：推出"无还本续贷"产品，减轻小微客户资金周转压力
