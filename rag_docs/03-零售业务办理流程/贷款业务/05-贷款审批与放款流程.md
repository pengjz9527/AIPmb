# 贷款审批与放款流程

## 流程概述

贷款审批与放款是贷款业务的核心中后台环节，连接贷款申请与贷后管理。审批流程涵盖贷前调查、审查、审批三个层级（简称"三查"中的贷前调查和审查审批），放款环节则涵盖合同签订、担保落实和资金发放。适用于个人住房贷款、消费贷款、经营贷款等各类零售贷款业务。

本流程旨在规范审批与放款各环节的操作标准，确保贷款质量，防范信贷风险。

## 前置条件

### 审批前置条件

- 贷款申请材料齐全、真实、有效
- 借款人已通过基本资格审查
- 征信查询授权已签署
- 贷前调查报告已完成

### 放款前置条件

- 贷款审批已通过
- 借款合同及担保合同已签订
- 担保手续已落实（抵押登记、质押交付、保证签署）
- 放款条件已全部满足
- 资金用途证明材料已提交（如需）

## 操作步骤

### 审批流程

1. **受理与初审** → 2. **征信查询与分析** → 3. **贷前调查** → 4. **审查** → 5. **审批决策** → 6. **审批结果通知**

### 放款流程

1. **合同签订** → 2. **担保手续落实** → 3. **放款审核** → 4. **资金发放** → 5. **放款确认与通知**

## 各环节详解

### 一、受理与初审

1. **材料完整性检查**：核对申请材料清单，确保所有必需材料齐全
2. **材料真实性初判**：检查材料原件与复印件一致性，识别明显伪造
3. **基本资格审查**：
   - 年龄是否符合要求
   - 征信是否有严重不良记录（近两年内连续逾期3次或累计6次以上）
   - 收入是否满足最低要求
   - 贷款用途是否合法合规
4. **系统录入**：将申请信息录入信贷管理系统，生成贷款编号
5. **分配调查人员**：指定贷前调查责任人

### 二、征信查询与分析

#### 征信查询

1. 取得借款人签署的《征信查询授权书》
2. 通过征信系统查询个人信用报告
3. 有共同借款人的（如房贷配偶），同时查询共同借款人征信
4. 有保证人的，查询保证人征信
5. 征信查询记录纳入档案管理

#### 征信分析要点

| 分析维度 | 关注要点 |
|----------|----------|
| 信用记录 | 近两年逾期次数、金额、原因；历史最高逾期期数 |
| 负债情况 | 现有贷款余额、月还款额、信用卡使用率 |
| 查询记录 | 近3个月贷款审批查询次数（多头借贷指标） |
| 公共信息 | 是否有法院判决、强制执行、欠税等记录 |
| 基本信息 | 职业信息、居住信息是否一致 |

### 三、贷前调查

#### 调查内容

1. **借款人面谈**：了解借款目的、还款来源、家庭情况等
2. **收入核实**：核实收入证明真实性
   - 工资收入：电话回访单位HR/财务，核对银行流水
   - 经营收入：实地走访经营场所，分析经营流水
   - 其他收入：核实租金、投资收益等
3. **资产评估**：抵押物评估、质押物验证
4. **用途核实**：验证贷款用途的真实性和合理性
5. **担保评估**：保证人资格和担保能力评估

#### 调查报告要求

- 借款人基本情况
- 借款用途和还款来源分析
- 担保方式及价值评估
- 风险因素分析
- 授信建议（额度、期限、利率、担保方式）

### 四、审查

1. **合规性审查**：确认贷款申请符合监管规定和银行内部制度
2. **风险审查**：评估贷款风险等级，复核授信额度合理性
3. **材料审查**：核实所有材料的真实性和一致性
4. **财务分析**：复核借款人的财务数据和还款能力
5. **担保审查**：评估担保方式的充分性和有效性
6. **出具审查意见**：
   - 同意（按原方案）
   - 有条件同意（调整额度/期限/利率/增加担保）
   - 不同意（拒绝）

### 五、审批决策

#### 审批权限分级

| 贷款金额 | 审批权限 |
|----------|----------|
| 50万元以下 | 支行/网点审批人 |
| 50-200万元 | 分行审批人 |
| 200-500万元 | 总行审批人 |
| 500万元以上 | 总行贷审会 |

#### 审批结果

- **批准**：确认贷款要素（金额、期限、利率、担保方式、还款方式）
- **有条件批准**：需满足特定条件后方可放款（增加担保、降低额度等）
- **拒绝**：不符合贷款条件，通知客户

### 六、合同签订

1. 通知借款人审批结果
2. 预约签约时间
3. 合同文本准备（借款合同、担保合同、抵押合同等）
4. 借款人（及配偶/共有人）亲笔签署全部合同
5. 核实签署人身份（面签面核）
6. 合同要素与审批结果一致性确认
7. 合同盖章和归档

### 七、担保落实与放款

1. **抵押登记**：到不动产登记中心办理抵押权登记
2. **质押交付**：质物移交银行保管
3. **保证签署**：保证人签署保证合同
4. **放款审核**：逐笔审核放款条件是否满足
5. **放款操作**：
   - 确认放款金额与审批一致
   - 选择支付方式（受托支付/自主支付）
   - 系统执行放款交易
6. **放款确认**：打印放款凭证，通知客户

## 业务规则

1. **审贷分离**：调查岗和审查审批岗严格分离，不得兼任
2. **双人调查**：金额较大或风险较高的贷款须双人调查
3. **独立审批**：审批人独立决策，不受行政干预
4. **审批时效**：从受理到审批结果，个人贷款通常3-10个工作日
5. **放款条件**：严格按照审批批复的条件执行，不得放松条件放款
6. **额度管控**：放款金额不得超过审批批准金额
7. **受托支付**：单笔金额超过30万元或监管要求的须受托支付
8. **档案管理**：贷款档案须完整保存，保存期限不低于贷款结清后5年

## 异常处理

### 场景一：审批中客户信息变化

- 如借款人收入/工作/征信发生重大变化，需更新信息重新评估
- 如抵押物评估报告过期（通常有效期6个月），需重新评估
- 如审批超过有效期（通常3-6个月），需重新审批

### 场景二：放款条件无法满足

- 抵押登记办理遇阻（产权争议、共有人不配合等）
- 保证人资信下降无法提供保证
- 建议更换担保方式或补充其他担保措施
- 无法解决的，贷款审批失效

### 场景三：审批人否决

- 审查人或审批人对贷款有不同意见
- 可申请提交上一级审批人或贷审会复议
- 复议需提供补充材料或调整方案
- 最终否决的，通知客户

### 场景四：系统故障影响放款

- 系统故障时暂停放款操作
- 记录待放款笔数，系统恢复后优先处理
- 涉及客户投诉的及时解释和安抚

## 风控要点

1. **三道防线**：贷前调查（第一道）、审查审批（第二道）、贷后管理（第三道）
2. **利益回避**：审批人与借款人有利害关系的须回避
3. **合规底线**：不得向不符合条件的借款人发放贷款
4. **集中度管理**：控制单一客户、行业、区域的贷款集中度
5. **关联交易审查**：识别并审查关联交易，防止利益输送
6. **审批纪律**：严禁越权审批、逆程序审批
7. **信息安全**：审批过程中涉及的征信信息和客户资料严格保密

## 城商行/农商行特色

1. **审批效率**：审批链条短，个人贷款审批通常3-5个工作日，快于大行
2. **灵活审批**：对本地优质客户（公务员、教师、医生）有更灵活的审批政策
3. **软信息运用**：客户经理通过实地走访获取的"软信息"在审批中起重要作用
4. **关系型信贷**：与客户的长期关系作为审批参考因素
5. **分层授权**：支行/网点有较大的审批权限（100万元以下）
6. **贷审会简化**：中小额贷款贷审会流程简化，提高效率
7. **远程审批支持**：部分银行推行远程视频审批，方便偏远地区客户
