# 贷款展期与逾期处理流程

## 流程概述

贷款展期是指借款人在贷款到期前因暂时性还款困难，向银行申请延长贷款期限的行为。逾期处理是指借款人未按合同约定按时还款，银行采取的催收、追偿和不良资产处置等一系列措施。本流程涵盖展期申请审批和逾期管理两大板块，是贷后管理的重要组成部分。

## 前置条件

### 展期申请条件

- 贷款尚未到期（须在到期前30天以上提出申请）
- 借款人有暂时性还款困难但具备未来还款能力
- 无恶意逾期记录（过往还款基本正常）
- 担保仍然有效（抵押物价值未大幅下降、保证人资信未恶化）
- 展期后累计期限不超过监管规定的最长期限

### 逾期触发条件

- 超过还款日（含宽限期）未足额还款
- 分期还款的当期还款金额不足
- 到期一次性还款的到期未还

## 操作步骤

### 展期申请流程

1. **借款人提出展期申请** → 2. **银行受理与调查** → 3. **展期审查** → 4. **审批决策** → 5. **签订展期协议** → 6. **系统调整** → 7. **通知相关方**

### 逾期处理流程

1. **系统标记逾期** → 2. **短信/电话提醒（M1）** → 3. **电话催收（M1-M2）** → 4. **上门催收（M2-M3）** → 5. **委外催收/法律催收（M3+）** → 6. **不良认定（M3+/90天）** → 7. **不良资产处置**

## 各环节详解

### 一、展期申请与审批

#### 展期申请材料

- 《贷款展期申请书》
- 展期原因说明及证明材料（如失业证明、经营困难证明、医疗证明等）
- 未来还款能力说明（新工作录用通知、经营恢复计划等）
- 更新的收入证明和银行流水
- 担保方同意展期的书面意见

#### 展期审查要点

1. **展期原因真实性**：核实借款人暂时性困难的原因是否真实
2. **还款能力恢复预期**：评估展期后借款人还款能力的恢复可能性
3. **担保有效性**：确认抵押物价值、保证人资信是否仍满足要求
4. **展期期限合理性**：展期期限与借款人恢复还款能力的时间匹配
5. **历史还款记录**：借款人过往还款是否基本正常

#### 展期规则

| 原贷款期限 | 展期最长期限 | 累计最长期限 |
|-----------|-------------|-------------|
| 1年以内（含） | 原期限 | 不超过原期限 |
| 1-5年 | 原期限的一半 | 不超过原期限×1.5 |
| 5年以上 | 3年 | 不超过原期限+3年 |

#### 展期利率

- 展期后的利率通常不低于原合同利率
- 根据展期时的市场利率和借款人风险状况重新定价
- 展期利率通常上浮10%-30%

### 二、逾期管理

#### 逾期分级管理

| 逾期等级 | 逾期天数 | 管理措施 | 不良分类 |
|----------|----------|----------|----------|
| M1 | 1-30天 | 短信提醒+电话催收 | 正常/关注 |
| M2 | 31-60天 | 加强电话催收+上门催收 | 关注 |
| M3 | 61-90天 | 委外催收/法律催收 | 次级（不良） |
| M4+ | 91天以上 | 诉讼/仲裁/核销 | 可疑/损失 |

#### 催收流程详解

**M1阶段（1-30天）——温馨提醒**

1. 逾期1-3天：系统自动发送短信提醒
2. 逾期3-7天：AI语音电话或客服中心电话提醒
3. 逾期7-15天：客户经理电话联系借款人
4. 逾期15-30天：发送书面催收函，电话了解逾期原因

**M2阶段（31-60天）——重点催收**

1. 安排专人电话催收，每日至少1次
2. 上门走访借款人住所/经营场所
3. 联系保证人，督促代为还款
4. 发送《催收通知书》（挂号信）
5. 评估抵押物状态和价值

**M3+阶段（61天以上）——强力催收**

1. 委托第三方催收机构催收
2. 发送律师函
3. 提起诉讼或仲裁
4. 申请财产保全
5. 执行抵押物拍卖/变卖
6. 对保证人追偿

### 三、不良资产处置

#### 不良认定标准

- **逾期90天以上**：一般认定为不良贷款（次级/可疑/损失）
- **五级分类**：正常 → 关注 → 次级 → 可疑 → 损失
- 次级及以下为不良贷款

#### 处置方式

| 处置方式 | 适用情形 | 处置周期 |
|----------|----------|----------|
| 催收化解 | 借款人恢复还款能力 | 1-6个月 |
| 重组 | 借款人有能力部分还款 | 3-12个月 |
| 以物抵债 | 抵押物可变现 | 6-12个月 |
| 诉讼执行 | 有可执行财产 | 6-24个月 |
| 债权转让 | 批量处置不良 | 3-6个月 |
| 核销 | 已穷尽一切手段 | 12个月以上 |

#### 贷款重组

1. 与借款人协商制定新的还款方案
2. 可能的调整：降低利率、延长期限、减免部分利息
3. 签订重组协议
4. 重组后的贷款至少归为关注类
5. 重组后至少正常还款6个月后方可上调分类

## 业务规则

1. **展期次数**：一笔贷款通常只允许展期1次，特殊情况最多2次
2. **展期利率**：展期后利率通常上浮，不低于原利率
3. **逾期罚息**：逾期部分按合同约定的罚息利率计息（通常为正常利率的1.3-1.5倍）
4. **复利计算**：未按时支付的利息按罚息利率计收复利
5. **征信报送**：逾期信息按月报送央行征信系统
6. **催收合规**：催收须依法合规，禁止暴力催收、骚扰、泄露隐私
7. **诉讼时效**：贷款债权诉讼时效为3年（自逾期之日起算）
8. **核销条件**：符合财政部规定的呆账核销条件方可核销
9. **展期对征信影响**：展期信息会报送征信系统，影响信用记录

## 异常处理

### 场景一：借款人失联

- 通过预留的紧急联系人联系借款人
- 上门走访借款人住所和工作单位
- 通过公安系统查询借款人最新联系方式
- 通过法院公告送达催收文书
- 必要时申请法院宣告失踪

### 场景二：抵押物价值下降

- 重新评估抵押物当前市场价值
- 抵押率超过警戒线（通常70%），要求借款人追加担保
- 借款人无法追加担保的，可要求提前部分还款降低贷款余额
- 必要时采取财产保全措施

### 场景三：保证人拒绝代偿

- 发送《保证人代偿通知书》
- 说明连带保证责任的法律义务
- 如保证人仍拒绝，提起诉讼将保证人列为共同被告
- 申请保证人财产保全

### 场景四：借款人恶意逃废债

- 收集借款人转移资产、恶意逃避债务的证据
- 向法院申请撤销借款人恶意转移资产的行为
- 申请列入失信被执行人名单
- 追究借款人刑事责任（涉嫌贷款诈骗罪的）

## 风控要点

1. **展期审慎性**：展期不等于延期风险，需评估展期后是否真正能化解风险
2. **五级分类准确**：严格按照标准进行贷款五级分类，不得人为调节
3. **催收合规**：严禁暴力催收、骚扰式催收、向无关第三方泄露信息
4. **诉讼时效管理**：及时中断诉讼时效，防止债权失效
5. **抵押物管理**：定期监控抵押物价值变化，及时采取保全措施
6. **呆账核销合规**：严格按照财政部和监管部门规定进行核销
7. **信息报送准确**：征信信息报送准确、及时，不得漏报、错报

## 城商行/农商行特色

1. **人情化管理**：对本地熟悉的客户，展期审批更灵活，考虑客户实际困难
2. **走访式催收**：利用本地网点和人员优势，上门催收更方便
3. **村委协助**：农村地区可借助村委会、社区力量协助催收
4. **重组灵活性**：对确有困难的本地客户，重组条件更灵活（减免利息、延长期限）
5. **风险补偿基金**：部分地方政府设立小微企业贷款风险补偿基金，可分担不良损失
6. **批量处置**：城商行/农商行可与地方AMC合作批量处置不良资产
7. **法律资源**：与本地律所有稳定合作关系，法律催收效率较高
