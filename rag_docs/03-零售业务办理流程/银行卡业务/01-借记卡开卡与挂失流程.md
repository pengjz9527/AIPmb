# 借记卡开卡与挂失流程

## 流程概述

借记卡是银行向客户发行的不具有透支功能的银行卡，客户须先存款后消费/取现。借记卡开卡是客户建立银行账户和使用银行卡的基础环节。挂失是在银行卡遗失、被盗或存在安全风险时采取的紧急保护措施。办理渠道包括柜面、智能柜员机（STM）和手机银行（挂失）。

借记卡类型包括：标准借记卡（银联）、联名卡（与商户合作）、主题卡（特色设计）、社保金融卡（社保+金融功能）等。

## 前置条件

### 开卡条件

- 年满16周岁（未满16周岁由监护人代办）
- 持有有效居民身份证
- 在同一银行持有的借记卡数量未超限（通常不超过4张）
- 未被列入银行黑名单

### 挂失条件

- 持卡人为本行客户
- 卡状态为正常或受限（非已销户/已挂失）
- 能提供有效的身份验证信息

### 开卡所需材料

| 开卡类型 | 所需材料 |
|----------|----------|
| 本人开卡 | 有效身份证原件 |
| 代办开卡 | 双方身份证原件、授权委托书（公证） |
| 未成年人开卡 | 监护人身份证、户口簿、出生证明 |
| 批量开卡 | 单位证明、员工身份证复印件、批量开卡申请表 |

## 操作步骤

### 开卡流程

1. **客户申请** → 2. **身份验证** → 3. **联网核查** → 4. **人脸识别** → 5. **信息录入** → 6. **设置密码** → 7. **签约电子银行** → 8. **制卡发卡** → 9. **客户签收**

### 挂失流程

1. **口头挂失（紧急）** → 2. **书面挂失（正式）** → 3. **制发新卡** → 4. **新卡领取/激活**

## 各环节详解

### 一、开卡环节

#### 柜面开卡

1. 客户到柜台提出开卡申请
2. 柜员使用身份证阅读器读取身份信息
3. 联网核查：通过人民银行系统验证身份证真实性
4. 人脸识别：将客户面部与身份证照片比对
5. 询问开卡用途（日常结算、工资代发、社保等）
6. 录入客户信息：手机号、职业、地址等
7. 选择卡种（标准卡/联名卡/主题卡）
8. 设置6位交易密码
9. 签约电子银行服务（手机银行、网上银行、短信通知）
10. 设置非柜面交易限额
11. 柜员在系统中提交开卡交易
12. 实时制卡：柜面制卡机制作银行卡（含芯片）
13. 客户签收并领取银行卡
14. 告知客户用卡安全须知

#### STM（智能柜员机）开卡

1. 客户在STM上选择"开卡"功能
2. 将身份证放入读卡区
3. 系统自动进行联网核查和人脸识别
4. 现场工作人员确认客户身份
5. 客户在触摸屏上补充个人信息
6. 选择卡种和电子银行服务
7. 设置交易密码
8. 电子签名确认
9. STM自动制卡并弹出
10. 客户取卡并设置手机银行

#### 批量开卡（代发工资场景）

1. 企业/单位提交批量开卡申请和员工名单
2. 提供员工身份证复印件和开卡授权书
3. 银行批量制作银行卡
4. 银行工作人员上门或通知员工到网点激活
5. 激活时须本人设置密码并签约电子银行
6. 未激活的卡不能进行任何交易

### 二、挂失环节

#### 口头挂失（临时挂失）

| 项目 | 说明 |
|------|------|
| 办理渠道 | 客服电话、手机银行、网上银行 |
| 办理方式 | 拨打客服热线或登录APP操作 |
| 所需信息 | 卡号、姓名、身份证号、密码等 |
| 有效期 | 5天（部分银行为7天或15天） |
| 功能 | 冻结卡片交易功能，防止资金损失 |
| 费用 | 免费 |
| 自动解除 | 到期未办理书面挂失的自动解除 |

#### 口头挂失操作

1. **电话挂失**：
   - 拨打银行客服热线（如96XXX）
   - 按语音提示选择"挂失"
   - 输入卡号、身份证号、密码验证身份
   - 确认挂失信息
   - 记录挂失编号

2. **手机银行挂失**：
   - 登录手机银行APP
   - 进入"我的账户"→"挂失"
   - 选择需要挂失的银行卡
   - 确认挂失
   - 系统即时冻结卡片

#### 书面挂失（正式挂失）

1. 客户携带身份证原件到柜面办理
2. 填写《挂失申请书》
3. 柜员核实客户身份
4. 核实原卡信息和挂失状态
5. 缴纳挂失手续费（通常10元）
6. 系统办理正式挂失
7. 打印挂失凭证，客户签收

#### 补发新卡

1. 书面挂失7个工作日后可申请补发新卡
2. 部分银行支持即时补发（当场领卡）
3. 补发新卡的方式：
   - **同号补卡**：保留原卡号（制卡周期较长，约7-15个工作日）
   - **异号补卡**：新卡号（即时或3个工作日）
4. 新卡激活后，原卡自动作废
5. 原卡关联的代发、代扣业务：
   - 同号补卡：无需变更
   - 异号补卡：需到相关单位变更卡号

### 三、其他卡业务

#### 密码挂失与重置

1. 忘记密码：到柜面办理密码挂失
2. 携带身份证和银行卡
3. 填写《密码挂失申请书》
4. 7个工作日后到柜面重置密码
5. 部分银行支持STM即时重置

#### 换卡（卡损坏/到期换卡）

1. 卡面损坏或芯片失效：到柜面申请换卡
2. 卡到期（卡面印有有效期）：到期前银行主动通知换卡
3. 换卡手续费：损坏换卡收取工本费（约5-10元），到期换卡免费
4. 换卡周期：同号换卡7-15个工作日，异号换卡即时

## 业务规则

1. **开卡数量限制**：同一客户在同一银行开立的借记卡不超过4张
2. **开卡费用**：首次开卡免收工本费和年费
3. **激活要求**：未激活的卡不能进行任何交易
4. **密码规则**：6位数字密码，不得使用连续数字或重复数字
5. **挂失费用**：口头挂失免费，书面挂失10元/次
6. **挂失有效期**：口头挂失5天，书面挂失7个工作日后补卡
7. **账户保护**：挂失后如因挂失前交易导致的损失，按银行规定处理
8. **安全提示**：银行不对因客户泄露密码导致的损失承担责任

## 异常处理

### 场景一：身份证过期开卡

- 告知客户身份证已过期，无法开卡
- 建议客户先到公安机关更新身份证
- 更新后可持新身份证到银行开卡
- 如已有银行卡，也需更新身份证信息

### 场景二：挂失后找回原卡

- 口头挂失：到柜面办理解除口头挂失
- 书面挂失：已办理书面挂失的不能解除，须等待补发新卡
- 补发新卡后找回的原卡自动作废，不可使用

### 场景三：代理挂失

- 代理挂失需提供双方身份证原件和授权书
- 代理人只能办理口头挂失
- 书面挂失和补发新卡须持卡人本人办理
- 特殊情况下（住院、行动不便）可申请上门服务

### 场景四：挂失期间资金争议

- 口头挂失前的交易损失由客户自行承担
- 口头挂失后的交易（如有）由银行负责
- 如存在争议，银行配合客户报警处理
- 保留相关证据（监控录像、交易记录等）

## 风控要点

1. **断卡行动**：严格落实"断卡"行动要求，防范涉案账户
2. **身份核实**：开卡环节严格核实客户身份，防止冒名开卡
3. **限额管理**：新开卡初始设置较低的非柜面交易限额
4. **反诈提醒**：开卡时进行反诈宣传，签署防诈骗承诺书
5. **挂失安全**：挂失环节严格核实来电人身份，防止恶意挂失
6. **异常监控**：监控频繁挂失补卡行为，防范欺诈
7. **信息保护**：客户信息和密码严格保密

## 城商行/农商行特色

1. **社保卡联动**：城商行/农商行普遍承担地方社保卡发卡任务，社保金融卡是重要产品
2. **市民卡/公交卡**：与地方政府合作发行市民卡或公交联名卡
3. **助农取款卡**：农商行发行面向农户的专属借记卡，在助农取款点使用
4. **开卡便利**：社区网点多，开卡更便利，STM覆盖率高
5. **挂失上门**：对行动不便的老年客户或偏远农村客户提供上门挂失服务
6. **方言服务**：挂失客服电话提供方言选项，方便农村客户
7. **即时补卡**：网点配备即时制卡设备，补卡无需等待
