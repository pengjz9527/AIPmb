## 文档结构

每篇文档统一采用以下结构：

| 章节 | 内容 |
|------|------|
| 法规概述 | 法规全称、发布机构、发布时间、文号、效力层级 |
| 核心内容摘要 | 主要条款和核心要求 |
| 零售业务相关条款解读 | 与零售银行业务直接相关的重点条款逐条解读 |
| 对城商行/农商行的影响 | 对中小银行零售业务的具体影响和应对策略 |
| 合规实施要点 | 落地实施的关键步骤、制度建设要求 |
| 监管处罚案例 | 近年来因违反该法规被处罚的典型案例 |
| 关联知识 | 相关文档链接 |

## 更新维护说明

- **更新频率**：建议每季度更新一次，或在重大法规修订时即时更新
- **信息来源**：以中国银保监会/国家金融监督管理总局官方网站发布的正式文本为准
- **注意事项**：部分法规可能已被新规替代或修订，使用前请以最新官方文本为准
