# 商业银行法（零售相关条款）

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《中华人民共和国商业银行法》 |
| 发布机构 | 全国人民代表大会常务委员会 |
| 首次发布 | 1995年5月10日（第八届全国人大常委会第十三次会议） |
| 最新修订 | 2015年8月29日（第十二届全国人大常委会第十六次会议修正） |
| 文号 | 主席令第十三号（2015年修正版） |
| 效力层级 | 法律（仅次于宪法的最高效力层级） |

《商业银行法》是我国银行业的基本法之一，与《银行业监督管理法》共同构成银行业监管的法律基础。该法全面规定了商业银行的设立、业务范围、经营规则、财务会计、监督管理及法律责任等内容。其中，大量条款与零售银行业务直接相关，是开展个人存贷款、银行卡、消费者保护等业务的根本法律依据。

> 注：截至2024年，商业银行法正在推进新一轮大修，2020年曾发布修订建议稿（征求意见稿），新增多项与消费者保护、金融科技创新相关的内容，正式修订版尚未出台。

## 核心内容摘要

### 一、商业银行基本制度

- **第2条**：明确商业银行可以经营的业务范围，共14项，其中与零售业务直接相关的包括：吸收公众存款、发放短期/中期/长期贷款、办理国内外结算、办理票据承兑与贴现、发行金融债券、代理发行/兑付政府债券、买卖政府债券/金融债券、从事同业拆借、买卖/代理买卖外汇、从事银行卡业务、提供信用证服务及担保、代理收付款项及代理保险业务、提供保管箱服务等。
- **第3条**：商业银行经中国人民银行批准，可以经营结汇、售汇业务。
- **第4条**：商业银行以安全性、流动性、效益性为经营原则，实行自主经营，自担风险，自负盈亏，自我约束。

### 二、存款业务规则

- **第29条**：商业银行办理个人储蓄存款业务，应当遵循存款自愿、取款自由、存款有息、为存款人保密的原则。对个人储蓄存款，商业银行有权拒绝任何单位或者个人查询、冻结、扣划，但法律另有规定的除外。
- **第30条**：对单位存款，商业银行有权拒绝任何单位或者个人查询，但法律、行政法规另有规定的除外；有权拒绝任何单位或者个人冻结、扣划，但法律另有规定的除外。
- **第33条**：商业银行应当保证存款本金和利息的支付，不得拖延、拒绝支付存款本金和利息。

### 三、贷款业务规则

- **第35条**：商业银行贷款，应当对借款人的借款用途、偿还能力、还款方式等情况进行严格审查。商业银行贷款，应当实行审贷分离、分级审批的制度。
- **第36条**：商业银行贷款，借款人应当提供担保。经商业银行审查、评估，确认借款人资信良好，确能偿还贷款的，可以不提供担保。
- **第37条**：商业银行贷款，应当与借款人订立书面合同。
- **第38条**：商业银行应当按照中国人民银行规定的贷款利率的上下限，确定贷款利率。
- **第39条**：商业银行贷款，应当遵守资产负债比例管理的规定（资本充足率不低于8%、贷款余额与存款余额比例不超过75%、流动性资产余额与流动性负债余额的比例不低于25%、对同一借款人的贷款余额与商业银行资本余额的比例不超过10%等）。
- **第40条**：商业银行不得向关系人发放信用贷款；向关系人发放担保贷款的条件不得优于其他借款人同类贷款的条件。

### 四、其他零售业务规则

- **第43条**：商业银行在中华人民共和国境内不得从事信托投资和证券经营业务，不得向非自用不动产投资或者向非银行金融机构和企业投资，但国家另有规定的除外。
- **第50条**：商业银行办理业务，提供服务，按照规定收取手续费。收费项目和标准由国务院银行业监督管理机构、中国人民银行根据职责分工，分别会同国务院价格主管部门制定。
- **第52条**：商业银行的工作人员不得利用职务上的便利，贪污、挪用、侵占本行或者客户的资金。

## 零售业务相关条款解读

### （一）储蓄存款四项原则（第29条）

第29条确立了个人储蓄存款的四项基本原则，这是银行零售存款业务的法律基石：

1. **存款自愿**：个人有权自主决定是否存款、选择哪家银行存款、存多少、存什么种类。银行不得强制或变相强制个人存款，不得设置不合理条件限制存款。
2. **取款自由**：存款人有权随时或在约定期限内支取存款。银行不得以任何理由拒绝或拖延支付（除法律明确规定的冻结、扣划情形）。活期存款可随时支取，定期存款提前支取按活期利率计息。
3. **存款有息**：银行应当按照中国人民银行规定的利率政策，向存款人支付存款利息。存款利率应当公开透明，不得随意变更已约定的利率。
4. **为存款人保密**：银行及其工作人员对存款人的存款信息负有保密义务，除法律明确授权的国家机关（如法院、检察院、公安机关、税务机关、海关等）依法查询外，不得向任何单位和个人泄露存款人的存款信息。

**实务要点**：该条同时赋予商业银行拒绝权，即有权拒绝任何单位或个人对个人储蓄存款的查询、冻结、扣划，除非法律另有规定。这一条款是银行保护客户隐私的直接法律依据。

### （二）贷款审贷分离制度（第35条）

第35条确立了贷款业务的三项核心制度：

1. **严格审查义务**：银行必须对借款人的借款用途、偿还能力、还款方式进行全面审查。这一义务在个人贷款领域尤为重要，2024年《个人贷款管理办法》在此基础上进一步细化了面谈面签、贷款用途监控等要求。
2. **审贷分离**：贷款的审查和贷款的发放应当由不同的部门或人员负责，形成内部制衡机制。在零售业务中，这一原则体现在客户经理（贷前调查）与审批人员（贷款审批）的岗位分离。
3. **分级审批**：根据贷款金额、风险程度等因素，建立分级审批制度。零售贷款中，小额消费贷款可以授权支行审批，大额贷款需上报分行或总行审批。

### （三）关系人贷款限制（第40条）

第40条规定商业银行不得向关系人发放信用贷款，向关系人发放担保贷款的条件不得优于其他借款人同类贷款的条件。关系人包括：商业银行的董事、监事、管理人员、信贷业务人员及其近亲属，以及上述人员投资或者担任高级管理职务的公司、企业和其他经济组织。

在零售业务中的适用：银行员工的个人消费贷款、信用卡授信等，需严格遵守关系人贷款限制，不得享受优于普通客户的条件。

### （四）资产负债比例管理（第39条）

第39条规定的资产负债比例管理要求对零售业务有深远影响：

- **存贷比**：虽然2015年修正已将存贷比从75%的硬性约束调整为流动性监测指标，但仍需关注。
- **单一客户集中度**：对同一借款人的贷款余额不超过银行资本余额的10%。在个人贷款中一般不会触及，但对于大额个人经营贷款需要注意。
- **资本充足率**：不低于8%的要求约束了银行零售贷款业务的整体扩张空间。

### （五）存款支付保障（第33条）

第33条要求商业银行保证存款本金和利息的支付，不得拖延、拒绝支付。这一条款在流动性危机场景下尤为重要，是存款人权益保护的核心条款。2015年《存款保险条例》的实施为这一条款提供了制度保障。

## 对城商行/农商行的影响

### 1. 存款业务合规压力

城商行和农商行在存款竞争中处于相对劣势地位，可能面临通过高息揽储、变相提高利率等方式争揽存款的诱惑。《商业银行法》第31条明确规定不得违反规定提高或者降低利率以及采取其他不正当手段吸收存款，违规将面临行政处罚。城商行/农商行应：
- 严格执行存款利率自律机制要求
- 不得通过返现、送礼等方式变相突破利率上限
- 创新存款产品需在合规框架内进行

### 2. 贷款业务审慎经营

城商行/农商行的零售贷款业务需严格遵守审贷分离、分级审批的要求。由于机构规模较小，人员配置有限，在审贷分离的执行上可能面临实际困难。建议：
- 建立适合自身规模的审贷分离制度
- 充分利用总行集中审批、远程审批等方式实现审贷分离
- 加强信贷人员培训，提升风险识别能力

### 3. 关系人贷款管理

城商行/农商行由于扎根地方，与地方政府、企业之间的关系较为紧密，关系人贷款的识别和管理难度较大。应当：
- 建立关系人名单管理制度，定期更新
- 在信贷系统中设置关系人标识和审批拦截
- 加强内部审计，定期排查关系人贷款合规情况

### 4. 消费者保护义务

随着金融消费者权益保护力度的加大，城商行/农商行需要在产品销售、信息披露、投诉处理等方面投入更多资源。《商业银行法》修订建议稿中已增加了消费者保护的专章内容，银行应提前做好准备。

## 合规实施要点

### 1. 制度建设

| 制度名称 | 对应法条 | 关键内容 |
|----------|----------|----------|
| 个人储蓄存款管理办法 | 第29、30、33条 | 存款四项原则、保密义务、支付保障 |
| 个人贷款管理办法 | 第35、36、37条 | 审贷分离、分级审批、书面合同 |
| 关系人贷款管理制度 | 第40条 | 关系人识别、审批限制、条件管控 |
| 服务收费管理制度 | 第50条 | 收费项目公示、禁止乱收费 |
| 资产负债管理办法 | 第39条 | 资本充足率、集中度管理 |

### 2. 系统建设

- 在核心系统中设置关系人标识和审批流程控制
- 建立贷款利率定价系统，确保利率在合规范围内
- 完善客户信息保密技术措施，防止信息泄露
- 建立存款支付保障的流动性应急预案

### 3. 培训与监督

- 定期对一线柜员、客户经理进行《商业银行法》培训
- 将法律合规要求纳入绩效考核指标
- 建立内部审计机制，定期检查各项业务的合规情况
- 设立合规举报渠道，鼓励员工举报违规行为

## 监管处罚案例

### 案例一：某城商行违规高息揽储案（2022年）

- **违规事实**：该城商行通过在存款产品中嵌入"积分兑换""现金红包"等方式，变相突破存款利率自律上限，吸收个人存款约12亿元。
- **处罚结果**：被银保监局罚款300万元，责令限期整改，对相关责任人给予警告并处以罚款。
- **涉及法条**：第31条（不得采取不正当手段吸收存款）、第47条（不得违反规定提高或降低利率）。

### 案例二：某农商行向关系人违规发放贷款案（2023年）

- **违规事实**：该农商行信贷部门负责人向其近亲属发放信用贷款200万元，未按照关系人贷款限制规定执行；同时向多家关联企业发放贷款，利率低于同类贷款基准利率10%。
- **处罚结果**：被罚款150万元，信贷部门负责人被取消任职资格5年。
- **涉及法条**：第40条（关系人贷款限制）。

### 案例三：某银行泄露客户存款信息案（2021年）

- **违规事实**：某股份制银行员工利用职务便利，查询并向第三方提供客户个人储蓄存款信息，涉及客户200余人。
- **处罚结果**：涉事员工被开除并移送司法机关，银行被罚款200万元。
- **涉及法条**：第29条（为存款人保密）、第52条（禁止利用职务便利侵占客户资金）。

### 案例四：某城商行违规收费案（2023年）

- **违规事实**：该城商行在办理个人贷款时，强制搭售保险产品并收取手续费，同时以"贷款管理费""信息咨询费"等名义向借款人收取额外费用，合计违规收费约800万元。
- **处罚结果**：被罚款400万元，责令退还全部违规收费。
- **涉及法条**：第50条（服务收费应当按照规定收取）。
