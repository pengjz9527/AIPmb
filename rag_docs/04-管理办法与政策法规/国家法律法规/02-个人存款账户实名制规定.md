# 个人存款账户实名制规定

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《个人存款账户实名制规定》 |
| 发布机构 | 国务院 |
| 发布时间 | 2000年3月20日（国务院令第285号） |
| 最新修订 | 2011年1月8日修订（国务院令第588号） |
| 文号 | 国务院令第285号（原版）/ 国务院令第588号（修订版） |
| 效力层级 | 行政法规 |

《个人存款账户实名制规定》是我国个人存款账户管理的基础性法规，要求个人在金融机构开立存款账户时必须使用真实身份信息。该规定的实施对于防范洗钱、打击腐败、维护金融秩序具有重要意义。同时，中国人民银行后续出台的多个配套文件进一步完善了实名制的执行细则，包括《金融机构客户身份识别和客户身份资料及交易记录保存管理办法》等。

## 核心内容摘要

### 一、实名制的核心要求

- **第2条**：个人在金融机构开立个人存款账户时，必须出示本人法定身份证件，使用身份证件上的姓名。
- **第3条**：金融机构为个人开立存款账户时，应当核对并登记其身份证件上的姓名和号码。
- **第4条**：个人在金融机构开立存款账户时，应当出示本人身份证件，不得使用假名或他人名义。

### 二、有效身份证件范围

根据规定及中国人民银行后续通知，有效身份证件包括：

| 类别 | 证件类型 | 适用人群 |
|------|----------|----------|
| 首选证件 | 居民身份证、临时居民身份证 | 中国大陆公民 |
| 补充证件 | 户口簿（限16周岁以下） | 未成年人 |
| 军人证件 | 军人身份证件、武装警察身份证件 | 现役军人、武警 |
| 港澳居民 | 港澳居民来往内地通行证、港澳居民居住证 | 港澳居民 |
| 台湾居民 | 台湾居民来往大陆通行证、台湾居民居住证 | 台湾居民 |
| 外国公民 | 护照、外国人永久居留身份证 | 外国公民 |

### 三、金融机构的审查义务

- **第5条**：金融机构应当按照规定核对并登记个人身份证件上的姓名和号码，不得为身份不明确的个人开立存款账户。
- **第6条**：个人代理他人开立存款账户的，金融机构应当对代理人的身份证件进行核对并登记。
- **第7条**：金融机构及其工作人员对个人的存款信息负有保密义务。

### 四、违规处罚

- **第8条**：个人不使用真实姓名开立存款账户的，由金融机构责令改正，并可处以罚款。
- **第9条**：金融机构违反实名制规定的，由中国人民银行给予警告，并处以罚款；情节严重的，对直接负责的主管人员和其他直接责任人员给予纪律处分。

## 零售业务相关条款解读

### （一）开户环节的实名制执行

实名制在零售业务中的核心应用场景是开户环节。根据规定及中国人民银行《关于落实个人银行账户分类管理制度的通知》（银发〔2016〕302号）：

**1. I类账户开户**

I类账户（全功能账户）必须通过银行柜面或自助机具（经银行工作人员现场核验身份）开立，开户时须：
- 出示有效身份证件的原件（不接受复印件或照片）
- 银行工作人员对证件进行真实性核验（人证一致性比对）
- 在系统中登记完整的身份信息（姓名、证件号码、证件有效期等）
- 留存身份证件复印件或影印件

**2. II类和III类账户开户**

II类和III类账户可通过电子渠道开立，但需绑定I类账户或信用卡账户进行身份验证，本质上仍然遵循实名制要求，只是验证方式从"面对面"转变为"绑定验证"。

**3. 代理开户**

代理他人开户时，银行须同时核验代理人和被代理人的身份证件，并要求代理人签署代理关系声明。对于无民事行为能力人或限制民事行为能力人（如未成年人），由法定代理人代理开户，须提供监护关系证明。

### （二）身份证件核验要求

银行在开户环节应当对身份证件进行多维度核验：

1. **证件真伪核验**：通过联网核查系统（公安部公民身份信息联网核查系统）核验身份证件的真实性。
2. **人证一致性核验**：通过人脸识别、指纹比对等技术手段确认开户人与证件持有人为同一人。
3. **证件有效期核验**：确保身份证件在有效期内。对于证件已过期的存量客户，应通知客户及时更新，逾期未更新的应限制账户功能。
4. **联网核查异常处理**：当联网核查结果不一致时，应要求客户提供辅助身份证明材料（如户口簿、驾驶证、社保卡等）。

### （三）存量账户实名制清理

根据中国人民银行的工作部署，银行需要对存量非实名账户进行清理：
- 对2000年4月1日（实名制实施日）之前开立的未实名账户，要求客户在规定期限内到银行办理实名确认手续。
- 对身份证件已过期的账户，限制其交易功能直至更新证件信息。
- 对同一客户在同一银行开立多个账户的情况，进行身份信息的统一归集和核实。

### （四）特殊情形处理

**1. 未成年人开户**
- 16周岁以下：由法定监护人代理开户，提供监护人身份证件、未成年人户口簿及监护关系证明。
- 16-18周岁：以自己的劳动收入为主要生活来源的，可视为完全民事行为能力人，可自行开户。

**2. 老年人开户**
- 对于行动不便的老年人，银行可提供上门服务完成开户。
- 对于不会签名的老年人，可以按手印或盖章代替签名。

**3. 外籍人士开户**
- 需提供有效护照或外国人永久居留身份证。
- 需核实签证有效期和居留许可。

## 对城商行/农商行的影响

### 1. 农村地区的特殊挑战

农商行服务的农村地区客户群体中，部分老年客户和偏远地区客户可能没有居民身份证或证件过期。农商行需要：
- 加大身份证办理的宣传和协助力度
- 与公安部门协作，为偏远地区客户提供便捷的证件办理渠道
- 在合规前提下，为特殊客户提供替代性身份验证方案

### 2. 系统改造投入

城商行/农商行需要投入资源进行系统改造：
- 升级核心系统，支持身份证件联网核查功能
- 部署人脸识别设备，实现人证一致性核验
- 建立证件到期预警和自动提醒机制
- 完善代理开户的身份核验流程

### 3. 人员培训

- 加强一线柜员对各类有效身份证件的识别能力培训
- 定期更新身份证件防伪特征知识
- 培训柜员处理联网核查异常情况的规范流程
- 强化对客户身份信息保密的意识

### 4. 数据治理

- 对存量客户数据进行全面清理，补全缺失的身份信息
- 建立客户身份信息的持续更新机制
- 与公安部门的数据共享机制对接，实现身份信息的自动核验

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 个人存款账户实名制实施细则 | 开户流程、证件核验标准、异常处理流程 |
| 客户身份识别管理办法 | 身份识别程序、辅助证件认定、代理开户管理 |
| 存量账户实名制清理方案 | 清理范围、时间安排、处置措施 |
| 客户信息保密管理制度 | 信息使用范围、访问权限、保密措施 |
| 身份证件过期管理办法 | 到期提醒、功能限制、恢复流程 |

### 2. 技术保障措施

- **联网核查系统对接**：确保开户、大额交易等关键环节均能实时调用公安部联网核查系统。
- **人脸识别技术应用**：在柜面、自助设备、手机银行等渠道部署人脸识别功能。
- **OCR证件识别**：利用光学字符识别技术自动读取证件信息，减少人工录入错误。
- **证件有效期管理系统**：建立证件到期自动预警和客户通知机制。

### 3. 内部控制措施

- 将实名制执行情况纳入网点和柜员的绩效考核
- 定期对开户环节进行合规检查和暗访
- 建立实名制违规的责任追究机制
- 设置客户投诉和举报渠道

### 4. 与监管的沟通

- 定期向当地人民银行报送实名制执行情况报告
- 配合监管部门开展实名制专项检查
- 及时反馈实名制执行中遇到的困难和问题

## 监管处罚案例

### 案例一：某农商行未严格执行实名制案（2022年）

- **违规事实**：该农商行辖内多个网点在办理个人存款账户开户时，未对客户身份证件进行联网核查，仅凭柜员目视核验即办理开户。经查，共有326个账户存在实名制执行不到位的情况。
- **处罚结果**：被人民银行当地支行罚款50万元，责令限期整改。
- **涉及条款**：第5条（金融机构应当核对并登记身份证件）。

### 案例二：某城商行代理开户未核验代理人身份案（2021年）

- **违规事实**：该城商行在办理代理开户业务时，仅核验了被代理人的身份证件复印件，未核验代理人的身份证件原件，导致多个账户被他人冒名开立。
- **处罚结果**：被罚款30万元，相关责任人被给予警告处分。
- **涉及条款**：第6条（代理开户应核验代理人身份证件）。

### 案例三：某银行未及时更新客户证件信息案（2023年）

- **违规事实**：某银行对身份证件已过期的客户未及时进行信息更新，也未限制其账户功能。经查，共有1.2万个账户存在证件过期未更新的情况。
- **处罚结果**：被罚款80万元。
- **涉及条款**：实名制持续管理要求。

### 案例四：某农商行虚假开户案（2022年）

- **违规事实**：某农商行内部员工与外部人员勾结，利用虚假身份证件开立个人存款账户200余个，用于非法资金转移。
- **处罚结果**：银行被罚款200万元，涉事员工被移送司法机关。
- **涉及条款**：第5条（不得为身份不明确的个人开立存款账户）。
