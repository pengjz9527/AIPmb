# 存款保险条例

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《存款保险条例》 |
| 发布机构 | 国务院 |
| 发布时间 | 2015年2月17日（国务院令第660号） |
| 施行时间 | 2015年5月1日 |
| 文号 | 国务院令第660号 |
| 效力层级 | 行政法规 |

《存款保险条例》标志着我国正式建立存款保险制度，是金融安全网的重要组成部分。该条例与中央银行的最后贷款人职能、金融监管部门的审慎监管共同构成金融安全网的三大支柱。存款保险制度的建立，对于保护存款人利益、维护金融稳定、促进银行业公平竞争具有重大意义，特别是对中小银行（城商行、农商行）而言，存款保险制度既是保障也是约束。

## 核心内容摘要

### 一、适用范围

- **第2条**：在中华人民共和国境内设立的商业银行、农村合作银行、农村信用合作社等吸收存款的银行业金融机构，应当依照本条例的规定投保存款保险。投保机构包括：商业银行（含外资法人银行）、农村合作银行、农村信用合作社、村镇银行等。

### 二、保费缴纳

- **第9条**：存款保险费率由基准费率和风险差别费率构成。费率水平由存款保险基金管理机构根据经济金融发展状况、存款结构情况以及存款保险基金的累积水平等因素制定和调整。
- **第10条**：投保机构应当按照存款保险基金管理机构的要求定期报送被保险存款余额、存款结构情况以及与保费计算有关的其他资料。
- **第11条**：投保机构应当按照确定的费率按时足额缴纳保费。

### 三、最高偿付限额

- **第5条**：存款保险实行限额偿付，最高偿付限额为人民币**50万元**。中国人民银行会同国务院有关部门可以根据经济发展、存款结构变化、金融风险状况等因素调整最高偿付限额，报国务院批准后公布执行。
- **同一存款人在同一家投保机构所有被保险存款**的本金和利息合并计算，在最高偿付限额以内的，实行全额偿付；超出最高偿付限额的部分，依法从投保机构清算财产中受偿。

### 四、偿付情形

- **第4条**：有下列情形之一的，存款人有权要求存款保险基金管理机构在规定的限额内偿付被保险存款：
  - （一）存款保险基金管理机构担任投保机构的接管组织；
  - （二）存款保险基金管理机构实施被撤销投保机构的清算；
  - （三）人民法院裁定受理对投保机构的破产申请；
  - （四）经国务院批准的其他情形。

### 五、偿付时限

- **第7条**：存款保险基金管理机构应当在本条例第四条规定的情形发生之日起**7个工作日内**足额偿付存款。

### 六、被保险存款范围

- 被保险存款包括投保机构吸收的人民币存款和外币存款。
- **不包括**：金融机构同业存款、投保机构的高级管理人员在本机构的存款、存款保险基金管理机构规定不予保险的其他存款。

## 零售业务相关条款解读

### （一）50万元最高偿付限额的实务影响

50万元的最高偿付限额是根据我国存款结构测算确定的，据测算该限额能够覆盖**99.63%** 的存款人。这一限额对零售业务有以下影响：

**1. 对大额存款客户的影响**

个人存款超过50万元的客户，在中小银行（尤其是城商行、农商行）存款时可能产生安全顾虑。银行需要：
- 向客户做好存款保险制度的宣传解释工作
- 引导客户合理配置存款，分散存放于不同银行
- 开发大额存单等创新产品，在合规范围内增强客户信心

**2. 对存款结构的影响**

存款保险制度实施后，部分大额存款人可能将存款从中小银行转移至大型银行，这对城商行、农商行的存款来源产生一定压力。银行应：
- 加强客户关系管理，通过优质服务留住客户
- 提升银行自身品牌形象和市场信誉
- 优化存款产品结构，增强吸引力

**3. 同一存款人在同一银行的合并计算**

50万元限额是按同一存款人在同一投保机构的所有存款合并计算的，包括活期、定期、通知存款等各类存款的本金和利息。这意味着：
- 客户在同一家银行的不同网点开立的存款账户，均合并计算
- 夫妻双方的存款分别计算（各自50万元限额）
- 以不同身份证件开立的账户，合并计算

### （二）存款保险标识的展示要求

根据中国人民银行的规定，投保机构应当在营业场所显著位置展示存款保险标识，并在网站、宣传材料中说明本机构已参加存款保险。具体要求：

1. **营业网点展示**：在营业场所入口处、现金柜台、自助设备区域等位置张贴存款保险标识。
2. **线上渠道展示**：在手机银行、网上银行等电子渠道展示存款保险标识。
3. **产品宣传说明**：在存款产品宣传材料中明确说明该产品属于存款保险保障范围。
4. **新开户告知**：在新开存款账户时，向客户告知存款保险制度的基本内容。

### （三）保费缴纳对银行成本的影响

存款保险费率由基准费率和风险差别费率构成：
- **基准费率**：所有投保机构统一适用的基础费率。
- **风险差别费率**：根据投保机构的风险状况确定的差异化费率，风险越高的机构缴纳的保费越多。

对于城商行、农商行而言：
- 由于风险评级相对较低，可能面临较高的差别费率
- 保费支出增加了经营成本，直接影响利润水平
- 需要加强风险管理，争取较低的风险差别费率

### （四）不被保险存款的处理

下列存款不在存款保险保障范围内：
- **金融机构同业存款**：包括银行间存放的资金
- **高级管理人员在本机构的存款**：投保机构的董事、监事、高级管理人员在本行的存款
- **理财产品**：银行理财产品不属于存款，不在保障范围内

零售业务中需要注意的是，结构性存款虽然在法律性质上属于存款，但其中的衍生品部分可能涉及特殊处理。银行应当明确告知客户哪些产品受存款保险保障，哪些不受保障。

## 对城商行/农商行的影响

### 1. 存款竞争的"双刃剑"效应

**积极方面**：
- 存款保险制度为中小银行提供了与大型银行同等的信用背书，有助于增强客户信心
- 消除了"银行太大不会倒"的心理，一定程度上拉平了竞争起跑线

**消极方面**：
- 部分客户可能认为需要存款保险保障的银行"不够安全"
- 在极端情况下，可能引发中小银行的存款流失
- 大型银行可能利用其"系统重要性"地位，对中小银行形成隐性竞争压力

### 2. 流动性管理的挑战

存款保险制度下，存款人在银行出现风险时有权在7个工作日内获得偿付。这意味着：
- 银行需要保持充足的流动性储备
- 建立存款流失的应急预案
- 加强舆情监控，防止不实信息引发挤兑

### 3. 早期纠正机制的约束

《存款保险条例》第15条规定，存款保险基金管理机构发现投保机构存在可能严重危及存款安全的情形时，可以要求投保机构及时采取补充资本、控制资产增长、控制重大交易授信等措施。这对城商行、农商行的经营自主权形成一定约束。

### 4. 差别费率的激励效应

风险差别费率机制对银行形成了正向激励：
- 风险管理能力强的银行缴纳较少的保费
- 促使银行加强内部控制和风险管理
- 推动银行优化资产质量，降低不良率

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 存款保险管理制度 | 保费计算、缴纳流程、数据报送 |
| 存款保险标识管理办法 | 标识展示位置、规格标准、维护更新 |
| 存款保险信息报送制度 | 报送内容、报送频率、数据质量管控 |
| 存款保险应急预案 | 风险事件处置流程、存款偿付配合 |
| 存款保险宣传制度 | 宣传内容、宣传方式、客户教育 |

### 2. 数据统计与报送

- 建立被保险存款余额的准确统计系统
- 确保按存款人维度合并计算存款余额
- 按时向存款保险基金管理机构报送相关数据
- 定期核对数据质量，确保保费计算准确

### 3. 客户沟通与教育

- 制作通俗易懂的存款保险宣传材料
- 培训一线员工掌握存款保险制度的核心内容
- 在网点显著位置展示存款保险标识
- 对新开账户客户进行存款保险制度的告知

### 4. 内部管理与考核

- 将存款保险相关工作纳入部门考核指标
- 定期对存款保险制度执行情况进行自查
- 配合存款保险基金管理机构的检查和评估
- 建立保费缴纳的台账管理

## 监管处罚案例

### 案例一：某村镇银行未按时缴纳保费案（2021年）

- **违规事实**：某村镇银行连续两个季度未按时缴纳存款保险保费，累计欠缴保费约120万元。
- **处罚结果**：被存款保险基金管理机构（中国人民银行）责令限期补缴，并处以欠缴金额1倍的罚款。
- **涉及条款**：第11条（按时足额缴纳保费）。

### 案例二：某农商行存款保险标识展示不规范案（2022年）

- **违规事实**：某农商行辖内15个网点中，有8个网点未在营业场所显著位置展示存款保险标识，部分网点虽展示但标识已破损模糊。
- **处罚结果**：被当地人民银行责令限期整改，给予通报批评。
- **涉及条款**：存款保险标识管理相关规定。

### 案例三：某城商行存款保险数据报送不准确案（2023年）

- **违规事实**：某城商行在向存款保险基金管理机构报送被保险存款数据时，因系统缺陷导致部分存款未纳入统计，少报被保险存款余额约3亿元，导致少缴保费。
- **处罚结果**：被罚款60万元，要求限期修复系统并重新报送数据。
- **涉及条款**：第10条（如实报送相关资料）。

### 案例四：包商银行被接管事件中的存款保险实践（2019年）

- **事件背景**：2019年5月，包商银行因严重信用危机被中国人民银行、银保监会联合接管。
- **存款保险的作用**：存款保险基金通过收购与承接方式，保障了包商银行个人存款和5000万元以下对公存款的全额偿付，有效维护了存款人利益和金融稳定。
- **启示**：该案例充分展示了存款保险制度在化解银行风险、保护存款人利益方面的重要作用，也为中小银行敲响了风险管理的警钟。
