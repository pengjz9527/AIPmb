# 中小银行普惠金融监管考核

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《关于2023年加力提升小微企业金融服务质量的通知》等系列文件 |
| 发布机构 | 原中国银行保险监督管理委员会 / 国家金融监督管理总局 |
| 核心文件 | 《关于2021年进一步推动小微企业金融服务高质量发展的通知》《商业银行小微企业金融服务监管评价办法》（银保监发〔2020〕29号） |
| 效力层级 | 规范性文件 + 监管评价体系 |

中小银行普惠金融监管考核是监管部门推动银行业金融机构加大普惠金融力度的重要抓手。考核体系以"两增两控"为核心目标，通过差异化监管政策引导城商行、农商行加大对小微企业、"三农"和社区居民的金融服务力度。普惠金融既是城商行、农商行的使命定位，也是重要的业务增长极。

## 核心内容摘要

### 一、"两增两控"目标

| 目标 | 具体内容 |
|------|----------|
| **两增** | 普惠型小微企业贷款增速不低于各项贷款增速；有贷款余额的户数不低于上年水平 |
| **两控** | 合理控制普惠型小微企业贷款资产质量（不良率控制在合理范围）；合理控制贷款综合成本（包括利率和各项费用） |

### 二、普惠型小微企业贷款定义

- **单户授信总额1000万元及以下**的小微企业贷款
- 包括：小微企业法人贷款、个体工商户贷款、小微企业主贷款
- 以单户授信总额作为认定标准，而非贷款余额

### 三、差异化监管政策

| 政策 | 内容 |
|------|------|
| 不良容忍度 | 普惠型小微企业贷款不良率不高于各项贷款不良率**3个百分点**以内可不作为监管考核扣分因素 |
| 存贷比 | 对涉农贷款和小微企业贷款较多的农商行、村镇银行，存贷比不作为硬性监管指标 |
| 资本补充 | 支持中小银行通过发行永续债、二级资本债等方式补充资本 |
| 准备金 | 对普惠型小微企业贷款达到一定比例的银行实行较低的存款准备金率 |

### 四、监管评价体系

《商业银行小微企业金融服务监管评价办法》从五个维度对银行进行评价：

| 评价维度 | 权重 | 评价要点 |
|----------|------|----------|
| 信贷投放 | 25% | 贷款增量、增速、户数 |
| 产品服务 | 15% | 产品创新、服务效率、定价管理 |
| 风险管理 | 25% | 不良率、拨备覆盖率、风险缓释 |
| 内部管理 | 20% | 组织架构、绩效考核、尽职免责 |
| 监管配合 | 15% | 数据报送、政策落实、整改情况 |

评价结果分为一级、二级A、二级B、三级、四级共五个等级。

## 零售业务相关条款解读

### （一）普惠金融与零售业务的融合

普惠金融与零售业务高度重叠，主要体现在：

**1. 个体工商户贷款**

个体工商户是普惠金融的重要服务对象，同时属于零售业务范畴：
- 个体工商户贷款纳入普惠型小微企业贷款统计
- 贷款用途可以是经营周转、设备采购、店面装修等
- 还款方式灵活，可以是按月付息到期还本、等额本息等
- 担保方式可以是信用、保证、抵押、质押等

**2. 小微企业主贷款**

小微企业主以个人名义申请的经营贷款：
- 用于企业生产经营的个人贷款纳入普惠金融统计
- 需要核实借款人的小微企业主身份
- 贷款资金须用于企业经营，不得用于个人消费
- 授信额度根据企业经营规模和还款能力确定

**3. 农户贷款**

农商行发放的涉农个人贷款：
- 农户小额贷款是普惠金融的重要组成部分
- 可以是信用贷款或担保贷款
- 利率可享受支农再贷款等政策优惠
- 纳入涉农贷款统计和考核

### （二）"两增两控"在零售业务中的落实

**1. 贷款增量目标**

- 每年年初制定普惠型小微企业贷款增量计划
- 将增量目标分解至各分支机构和业务部门
- 建立增量目标的月度监测和通报机制
- 对增量目标完成不力的机构进行约谈和督导

**2. 贷款户数目标**

- 积极拓展首贷户（首次获得银行贷款的客户）
- 扩大贷款覆盖面，增加有贷款余额的客户数量
- 优化客户结构，避免过度依赖少数大客户
- 建立户数目标的监测和考核机制

**3. 资产质量控制**

- 在扩大贷款规模的同时关注资产质量
- 普惠型小微企业贷款不良率不超过各项贷款不良率3个百分点
- 建立普惠贷款的贷前调查、贷中审查、贷后管理全流程管控
- 对不良贷款及时进行清收和处置

**4. 成本控制**

- 合理确定普惠型小微企业贷款利率
- 不得在贷款利率之外另行收取不合理费用
- 积极落实无还本续贷政策，降低客户融资成本
- 充分利用再贷款、再贴现等政策工具降低资金成本

### （三）尽职免责制度

尽职免责制度是激励基层信贷人员积极发放普惠贷款的重要保障：

**1. 适用条件**

符合以下条件的，可以对信贷人员进行尽职免责：
- 信贷人员已按照制度规定履行了贷前调查、贷中审查、贷后管理等职责
- 贷款形成不良不是由于信贷人员的故意或重大过失导致
- 信贷人员不存在与借款人串通骗取贷款等违规行为

**2. 不适用条件**

以下情形不适用尽职免责：
- 信贷人员未按照规定履行职责
- 信贷人员存在故意或重大过失
- 信贷人员与借款人串通骗取贷款
- 信贷人员收受借款人好处

**3. 实施要求**

- 建立明确的尽职免责认定标准和程序
- 设立独立的尽职免责认定委员会
- 对每笔不良贷款进行尽职调查和认定
- 认定结果作为人员考核和问责的依据

### （四）首贷户拓展

"首贷户"拓展是普惠金融的重要工作方向：

**1. 首贷户认定**

- 从未在任何银行获得过贷款的客户
- 或虽然获得过贷款但已全部还清且超过1年未再获得贷款的客户

**2. 拓展策略**

- 与当地工商、税务部门合作，获取新注册企业信息
- 利用大数据风控技术，为无信用记录客户进行信用评估
- 开发针对首贷户的专属信贷产品
- 降低首贷户的准入门槛（如减免担保要求）

**3. 激励措施**

- 对首贷户贷款给予内部资金转移定价（FTP）优惠
- 将首贷户拓展情况纳入客户经理的绩效考核
- 对首贷户不良贷款给予更高的尽职免责容忍度

## 对城商行/农商行的影响

### 1. 战略定位的重要支撑

普惠金融是城商行、农商行的核心战略定位之一：
- 监管部门要求中小银行"回归本源、聚焦主业"
- 普惠金融贷款是中小银行差异化竞争的重要领域
- 普惠金融考核结果直接影响监管评级
- 普惠金融表现优秀的银行可获得更多监管支持

### 2. 经营效益的双重影响

**积极方面**：
- 普惠贷款可获得央行再贷款等低成本资金支持
- 普惠贷款可享受定向降准等优惠政策
- 普惠贷款有助于拓展基础客户群
- 普惠贷款的综合收益（含中间业务收入）可能较高

**消极方面**：
- 普惠贷款的单笔金额小，运营成本相对较高
- 普惠贷款的风险相对较高，不良率可能上升
- 利率下行压力压缩了利差空间
- 需要投入更多人力和系统资源

### 3. 考核压力的持续传导

- "两增两控"目标每年都需要完成
- 监管评价结果直接影响监管评级和市场准入
- 普惠贷款增速要求可能高于贷款整体增速
- 首贷户拓展和信用贷款占比提升等指标持续加压

### 4. 数字化转型的推动

普惠金融的规模扩张离不开数字化支撑：
- 利用大数据风控技术提升审批效率
- 开发线上申贷渠道，降低获客成本
- 建立智能化的贷后监控系统
- 利用移动互联网技术拓展服务覆盖范围

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 普惠金融发展战略规划 | 发展目标、重点领域、资源配置 |
| 普惠型贷款管理办法 | 产品体系、审批流程、风控标准 |
| 尽职免责管理办法 | 认定标准、认定程序、结果运用 |
| 普惠贷款定价管理办法 | 利率定价、费用管理、成本控制 |
| 首贷户拓展管理办法 | 目标客群、产品适配、考核激励 |
| 普惠金融数据报送管理制度 | 报送内容、报送频率、数据质量 |

### 2. 组织架构

- 设立普惠金融事业部或专职部门
- 在分支机构配备普惠金融专职团队
- 建立普惠金融的独立核算和考核体系
- 配置普惠金融专项信贷额度

### 3. 产品创新

- 开发适合小微企业和个体工商户的标准化信贷产品
- 推出线上申请、快速审批的"秒批"贷款产品
- 开发无还本续贷产品，降低客户资金压力
- 开发基于纳税、水电费等数据的信用贷款产品

### 4. 数据管理

- 建立普惠金融数据的专项统计系统
- 确保数据报送的准确性和及时性
- 建立普惠金融考核的数据支持平台
- 定期开展普惠金融业务的数据质量检查

## 监管处罚案例

### 案例一：某城商行普惠贷款数据造假案（2023年）

- **违规事实**：该城商行通过将大中型企业贷款"化整为零"、将非经营性个人贷款纳入普惠贷款统计等方式，虚增普惠型小微企业贷款规模约8亿元。
- **处罚结果**：被罚款300万元，监管评价等级下调。
- **涉及条款**：普惠金融数据报送要求。

### 案例二：某农商行普惠贷款违规收费案（2022年）

- **违规事实**：该农商行在发放普惠型小微企业贷款时，强制要求借款人购买人身意外保险，并以"贷款承诺费""账户管理费"等名义收取额外费用，导致客户实际融资成本远高于合同约定利率。
- **处罚结果**：被罚款200万元，责令退还全部违规收费。
- **涉及条款**：普惠贷款成本控制要求。

### 案例三：某城商行未完成普惠贷款增量目标案（2023年）

- **违规事实**：该城商行连续两年未完成普惠型小微企业贷款的增量目标和户数目标，且未采取有效措施改善。
- **处罚结果**：被监管机构约谈高管，限制新业务准入。
- **涉及条款**："两增"目标要求。

### 案例四：某农商行尽职免责落实不到位案（2022年）

- **违规事实**：该农商行未建立有效的尽职免责制度，多名信贷人员因普惠贷款形成不良被扣减绩效和降级处理，严重挫伤了基层员工发放普惠贷款的积极性。
- **处罚结果**：被监管机构下发监管意见书，要求限期整改。
- **涉及条款**：尽职免责制度要求。
