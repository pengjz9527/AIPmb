# 农村商业银行支农支小政策

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《关于推进农村商业银行坚守定位 强化治理 提升金融服务能力的意见》等系列文件 |
| 发布机构 | 原中国银行保险监督管理委员会 |
| 核心文件 | 银保监办发〔2019〕5号《关于推进农村商业银行坚守定位 强化治理 提升金融服务能力的意见》；《农村中小银行机构行政许可事项实施办法》 |
| 效力层级 | 规范性文件 + 行政许可规章 |

农村商业银行（农商行）是由农村信用社改制而来的地方性商业银行，其核心使命是"支农支小"——服务"三农"（农业、农村、农民）和小微企业。监管部门通过一系列政策文件明确了农商行的经营定位、业务范围和监管要求，要求农商行坚守"支农支小"本源，不得偏离主业盲目扩张。

## 核心内容摘要

### 一、支农支小定位

- **第1条**：农村商业银行应当始终坚持服务县域、支农支小的市场定位，将信贷资源主要用于满足当地农户和小微企业的金融需求。
- **核心要求**：
  - 涉农贷款和小微企业贷款占比不低于全部贷款的**70%**
  - 农户贷款和小微企业贷款的户数占比不低于**80%**
  - 新增可贷资金主要用于当地

### 二、经营区域限制

- 农商行原则上应当在注册地所在的县（市）域范围内开展业务
- 不得跨区域发放贷款（线上贷款参照互联网贷款管理办法执行）
- 不得在注册地辖区以外设立营业网点（除省联社统一组织外）

### 三、涉农贷款考核指标

| 指标 | 要求 |
|------|------|
| 涉农贷款占比 | 不低于全部贷款的60%（部分省份可适度放宽至50%） |
| 农户贷款占比 | 不低于全部贷款的30% |
| 涉农贷款增速 | 不低于各项贷款平均增速 |
| 新型农业经营主体贷款 | 鼓励加大对家庭农场、农民专业合作社等新型农业经营主体的信贷支持 |

### 四、差异化监管政策

| 政策 | 具体内容 |
|------|----------|
| 存款准备金率 | 对涉农贷款占比高的农商行实行较低的存款准备金率 |
| 再贷款支持 | 通过支农再贷款为农商行提供低成本资金来源 |
| 不良容忍度 | 涉农贷款不良率容忍度高于各项贷款不良率3个百分点 |
| 税收优惠 | 对农户小额贷款利息收入免征增值税 |
| 风险补偿 | 地方政府设立涉农贷款风险补偿基金 |

### 五、省联社改革

- 省联社应当转变职能，从行政管理向服务支持转变
- 省联社不得干预农商行的独立经营决策
- 推进省联社改革（多种模式：金融服务公司、联合银行、统一法人等）

## 零售业务相关条款解读

### （一）农户贷款业务

**1. 农户贷款的定义与分类**

农户贷款是指农商行向农户（以家庭为单位）发放的用于农业生产经营和个人消费的贷款：

| 类型 | 用途 | 特点 |
|------|------|------|
| 农户生产经营贷款 | 种植、养殖、农产品加工等 | 季节性强、金额较小、期限灵活 |
| 农户消费贷款 | 住房、教育、医疗等消费 | 金额较小、期限较短 |
| 农户创业贷款 | 农村创业、返乡创业 | 政策支持力度大、利率优惠 |

**2. 农户贷款的担保方式**

- **信用贷款**：对信用良好、还款能力充足的农户可发放信用贷款
- **保证贷款**：由自然人或法人提供担保
- **抵押贷款**：以农村房屋、土地经营权、农机具等作为抵押
- **质押贷款**：以存单、保单等作为质押

**3. 农户贷款的创新模式**

- **整村授信**：以行政村为单位，对全村农户进行批量授信
- **阳光信贷**：公开贷款条件、流程和结果，接受群众监督
- **联保贷款**：若干农户自愿组成联保小组，相互承担连带保证责任
- **产业链贷款**：以农业龙头企业为核心，对上下游农户进行授信

### （二）小微企业贷款业务

**1. 县域小微企业的特点**

- 规模较小，经营不够规范
- 财务报表不完整，信息透明度低
- 抵押物不足，融资担保困难
- 经营周期性强，资金需求波动大
- 与当地农业产业链紧密关联

**2. 小微企业贷款的评估方法**

- **"三看三不看"原则**：看人品、看产品、看抵押；不看报表看流水、不看抵押看信用、不看规模看潜力
- **软信息收集**：通过走访了解借款人的家庭情况、社会关系、经营口碑
- **现金流分析**：通过银行流水、水电费、纳税记录等侧面数据验证经营状况
- **交叉验证**：通过多个信息源对借款人信息进行交叉验证

**3. 小微企业贷款的创新产品**

- **随借随还**：一次授信、循环使用，提高资金使用效率
- **无还本续贷**：在贷款到期前，符合条件的客户可直接续贷，无需先还后贷
- **订单融资**：以小微企业的订单或应收账款作为还款来源
- **银税互动**：与税务部门合作，以纳税信用替代抵押担保

### （三）涉农贷款的特殊政策

**1. 农村承包土地经营权抵押贷款**

- 试点地区可以农村承包土地的经营权作为抵押物
- 需要对土地经营权进行评估和确权
- 建立土地经营权的流转和处置机制
- 政府提供风险补偿和担保支持

**2. 农民住房财产权抵押贷款**

- 试点地区可以农民住房财产权（宅基地使用权）作为抵押物
- 需要完善住房财产权的登记和确权
- 建立住房财产权的评估和处置机制
- 确保借款人有基本居住保障

**3. 农机具抵押贷款**

- 以大型农机具作为抵押物发放贷款
- 需要农机具的购置发票和产权登记
- 建立农机具的评估和处置机制

### （四）农村信用体系建设

**1. 信用户、信用村、信用乡镇评定**

- 对农户进行信用等级评定（如AAA、AA、A级信用户）
- 对行政村进行信用村评定
- 对乡镇进行信用乡镇评定
- 信用等级与授信额度和利率挂钩

**2. 信用信息共享**

- 与当地政府部门共享农户信用信息
- 建立农村信用信息数据库
- 与央行征信系统对接
- 探索利用替代数据（如水电费缴纳、农产品交易记录）进行信用评估

### （五）农村支付服务

**1. 助农取款服务**

- 在农村便利店、村委会等场所布放助农取款终端
- 提供小额取款、转账、查询等基本金融服务
- 单笔取款限额一般为1000-2000元
- 弥补农村网点覆盖不足的短板

**2. 移动支付推广**

- 推广手机银行在农村地区的使用
- 与电商平台合作，拓展农村支付场景
- 推广二维码支付在农贸市场、乡村旅游等场景的应用
- 开展农村金融知识普及活动

## 对城商行/农商行的影响

### 1. 业务定位的坚守与聚焦

- 农商行必须将支农支小作为核心战略方向
- 不得偏离主业盲目追求规模扩张和跨区域经营
- 涉农贷款和小微企业贷款占比须达到监管要求
- 新增可贷资金应主要用于当地

### 2. 产品创新的推动

- 需要持续开发适合农村和小微企业的金融产品
- 利用金融科技提升服务效率和覆盖面
- 开发基于大数据的线上贷款产品
- 探索供应链金融、产业链金融等创新模式

### 3. 风险管理的挑战

- 农村贷款信息不对称问题突出
- 农业生产受自然灾害和市场价格波动影响大
- 农村抵押物不足，担保方式有限
- 需要建立适合农村特点的风险评估模型

### 4. 盈利能力的平衡

- 支农支小业务的社会效益与经济效益需要平衡
- 涉农贷款利率受政策约束，利差空间有限
- 需要通过政策补贴和规模效应提升盈利能力
- 运营成本相对较高，需要通过数字化手段降本增效

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 支农支小发展战略规划 | 发展定位、目标分解、资源配置 |
| 农户贷款管理办法 | 授信标准、审批流程、贷后管理 |
| 小微企业贷款管理办法 | 客户准入、风险评估、产品体系 |
| 涉农贷款统计和考核办法 | 统计口径、考核指标、激励约束 |
| 农村信用体系建设管理办法 | 信用评定标准、信息共享机制 |
| 助农取款服务管理办法 | 服务点管理、风险控制、费用管理 |

### 2. 服务网络建设

- 优化农村网点布局，提升网点服务能力
- 布放助农取款终端，延伸服务触角
- 建设流动银行服务车，服务偏远地区
- 推广手机银行和网上银行，提升线上服务能力

### 3. 人才队伍建设

- 招聘和培养懂农业、爱农村的金融人才
- 加强农村金融专业知识和技能的培训
- 建立扎根基层的激励机制
- 完善客户经理的考核和激励体系

### 4. 政策争取与利用

- 积极争取支农再贷款等政策支持
- 充分利用涉农贷款税收优惠政策
- 参与政府风险补偿和担保机制
- 配合监管部门开展农村金融改革创新试点

## 监管处罚案例

### 案例一：某农商行偏离支农支小定位案（2022年）

- **违规事实**：该农商行涉农贷款占比仅为35%，远低于60%的监管要求；大量信贷资源投向房地产和地方政府融资平台，偏离了支农支小的经营定位。
- **处罚结果**：被监管机构约谈高管，限制新业务准入和市场准入，要求制定回归本源整改方案。
- **涉及条款**：支农支小定位要求。

### 案例二：某农商行跨区域发放贷款案（2023年）

- **违规事实**：该农商行在注册地辖区以外发放了大量个人经营贷款和消费贷款，跨区域贷款余额达到20亿元，违反了属地经营的要求。
- **处罚结果**：被罚款300万元，要求限期清退跨区域贷款。
- **涉及条款**：经营区域限制。

### 案例三：某农商行涉农贷款数据不实案（2022年）

- **违规事实**：该农商行将多笔非涉农贷款（如个人住房贷款、消费贷款）纳入涉农贷款统计，虚增涉农贷款规模约5亿元。
- **处罚结果**：被罚款150万元，监管评价等级下调。
- **涉及条款**：涉农贷款统计要求。

### 案例四：某农商行支农再贷款资金使用违规案（2023年）

- **违规事实**：该农商行将人民银行发放的支农再贷款资金（利率优惠）用于发放非涉农贷款，套取利差收益约500万元。
- **处罚结果**：被人民银行收回再贷款资金，并处以罚款。
- **涉及条款**：支农再贷款资金使用要求。
