# 城商行异地经营监管要求

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《关于加强城市商业银行异地分支机构管理的通知》及相关监管文件 |
| 发布机构 | 原中国银行业监督管理委员会 / 国家金融监督管理总局 |
| 核心文件 | 银监发〔2006〕89号《关于加强城市商业银行异地分支机构管理的通知》；银保监办发〔2021〕相关窗口指导 |
| 关联文件 | 《商业银行互联网贷款管理暂行办法》（跨区域经营限制） |
| 效力层级 | 规范性文件 + 监管窗口指导 |

城商行异地经营监管是近年来中小银行监管的重点领域。监管部门明确要求城商行坚持"属地经营"原则，原则上不得跨区域开展业务。这一政策导向旨在引导城商行回归服务地方经济的本源，防范因盲目扩张导致的风险失控。对于零售业务而言，异地经营限制直接影响了城商行的获客范围、产品设计和渠道布局。

## 核心内容摘要

### 一、属地经营原则

- 城商行应当坚持服务地方经济、服务中小企业、服务城市居民的市场定位
- 原则上应当在注册地所在的城市（地级市或直辖市）范围内开展业务
- 不得在注册地辖区以外的地区设立分支机构（除经监管批准外）

### 二、异地分支机构管理

- **设立条件**：城商行设立异地分支机构须满足严格条件，包括资本充足率、资产质量、公司治理等方面的要求
- **审批程序**：异地分支机构的设立须经银行业监督管理机构审批
- **管理要求**：异地分支机构应纳入总行统一管理体系，接受总行和所在地监管机构的双重监管

### 三、跨区域业务限制

| 限制领域 | 具体要求 |
|----------|----------|
| 线下贷款 | 不得向注册地辖区以外的客户发放贷款（含个人贷款） |
| 互联网贷款 | 不得通过互联网渠道向注册地辖区以外的客户发放贷款 |
| 存款业务 | 不得在异地通过不正当手段吸收存款 |
| 理财销售 | 不得在异地开展理财产品销售活动 |

### 四、存量异地业务整改

- 对于已存在的异地业务，应逐步压缩规模
- 不得新增异地贷款业务
- 制定存量异地业务的清收和退出计划
- 在监管规定的期限内完成整改

### 五、联合贷款的跨区域限制

- 城商行通过互联网联合贷款方式发放贷款的，不得跨区域经营
- 合作方（如互联网银行、消费金融公司）可以帮助获客，但最终放款须符合区域限制
- 跨区域联合贷款的存量业务须在过渡期内完成整改

## 零售业务相关条款解读

### （一）"当地客户"的认定标准

在属地经营原则下，"当地客户"的认定标准直接关系到零售业务的开展范围：

**1. 个人客户的认定**

符合以下条件之一的个人，可认定为"当地客户"：
- 户籍所在地在银行注册地辖区内
- 常住地址在银行注册地辖区内（需提供居住证明）
- 工作单位在银行注册地辖区内（需提供工作证明）
- 经营场所（个体工商户）在银行注册地辖区内

**2. 认定方式**

- 通过身份证件地址判断户籍所在地
- 通过水电费账单、房屋租赁合同等判断常住地址
- 通过劳动合同、社保缴纳记录等判断工作单位所在地
- 通过营业执照判断个体工商户的经营场所

**3. 系统管控**

- 在信贷系统中设置客户属地校验规则
- 对非当地客户的贷款申请进行系统拦截
- 建立客户属地认定的动态更新机制
- 对跨区域的贷款申请进行人工审核和判断

### （二）线下零售业务的属地化经营

**1. 存款业务**

- 在注册地辖区内的网点开展存款业务不受限制
- 不得通过互联网渠道在异地吸收存款
- 不得通过第三方渠道在异地开展存款营销活动
- 不得以高息、返现等方式吸引异地客户存款

**2. 个人贷款业务**

- 个人贷款的客户须为"当地客户"
- 贷前调查须在辖区内进行（实地走访或线上核实）
- 抵押物原则上应在辖区内
- 贷后管理须能够在辖区内有效开展

**3. 信用卡业务**

- 信用卡申请客户的常住地须在银行注册地辖区内
- 不得在全国范围内进行信用卡营销推广
- 信用卡的消费和取现不受区域限制（这是银行卡的通用功能）
- 信用卡的授信审批须由银行自主完成

**4. 理财业务**

- 理财产品的销售对象须为"当地客户"
- 不得通过互联网渠道向异地客户销售理财产品
- 可通过代销方式将产品销售给异地客户（由代销机构负责客户管理）

### （三）线上业务的区域限制

互联网渠道的普及使跨区域经营变得更加容易，因此监管对线上业务的区域限制更加关注：

**1. 互联网贷款**

- 城商行通过互联网渠道发放个人贷款的，客户须为当地客户
- 通过地理围栏技术、IP地址识别等手段确保客户属地
- 在贷款申请环节增加客户属地的系统校验
- 对存量跨区域互联网贷款制定清退计划

**2. 线上存款**

- 通过手机银行、网上银行吸收的存款不受区域限制（客户已开户）
- 但不得通过互联网渠道主动在异地拓展存款客户
- 不得与第三方平台合作在异地开展存款营销活动

**3. 线上理财**

- 已开户客户通过线上渠道购买理财产品不受限制
- 但新客户开户须符合属地要求
- 不得通过互联网渠道向异地客户推荐理财产品

### （四）存量异地业务的处置

**1. 清退策略**

- 对存量异地贷款进行全面梳理和分类
- 对到期贷款自然收回，不再续贷
- 对未到期贷款，制定分期清退计划
- 对不良贷款，加大清收处置力度

**2. 客户沟通**

- 向异地客户说明业务调整的必要性
- 为异地客户推荐当地银行的服务
- 对未到期贷款做好客户服务，确保平稳退出
- 妥善处理异地客户的投诉和诉求

**3. 数据管理**

- 建立存量异地业务的专项台账
- 定期统计异地业务的压缩进度
- 按时向监管机构报告整改情况
- 确保异地业务数据的准确性和完整性

### （五）合规边界的把握

**1. 边界情形分析**

| 情形 | 是否合规 | 说明 |
|------|----------|------|
| 户籍在当地但长期在外地工作的客户 | 原则上合规 | 以户籍为准，但贷后管理需考虑 |
| 外地客户到当地网点主动办理业务 | 原则上合规 | 需确认客户有合理的属地关联 |
| 通过互联网渠道获客的异地客户 | 不合规 | 违反属地经营原则 |
| 在当地开户后搬到外地的客户 | 需具体分析 | 存量客户关系存续，但新增业务需谨慎 |
| 客户在当地有房产但户籍不在 | 一般合规 | 有属地关联，但需佐证 |

**2. 合规底线**

- 不得主动在异地开展营销推广活动
- 不得通过互联网渠道向异地客户发放贷款
- 不得与互联网平台合作突破区域限制
- 不得以各种方式变相开展异地业务

## 对城商行/农商行的影响

### 1. 业务增长空间的收窄

跨区域经营限制对城商行的业务增长产生了显著影响：
- 零售贷款的增长受限于当地市场容量
- 线上贷款业务无法实现全国扩张
- 部分以互联网贷款为主要增长动力的城商行面临转型压力
- 存款增长也受限于当地客户的存款能力

### 2. 竞争策略的调整

- 从"跑马圈地"式的外延扩张转向"精耕细作"的内涵式发展
- 加大对当地市场的深入挖掘
- 提升在本地市场的市场份额和客户粘性
- 通过差异化服务和产品创新增强竞争力

### 3. 联合贷款业务的调整

- 需要重新审视与互联网平台的合作模式
- 联合贷款的规模可能面临压缩
- 需要提升自主获客和自主风控能力
- 探索与本地场景结合的线上贷款产品

### 4. 数字化转型的方向调整

- 数字化转型从"获客扩张"转向"效率提升"
- 利用数字化手段深耕本地市场
- 建设本地化的数据能力和风控模型
- 提升线上渠道的客户体验和服务能力

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 属地经营管理办法 | 属地认定标准、业务边界、系统管控 |
| 互联网贷款跨区域经营管理办法 | 地理围栏、客户校验、存量整改 |
| 异地业务存量清退管理办法 | 清退计划、进度监控、客户沟通 |
| 客户属地认定管理办法 | 认定标准、认定流程、动态更新 |
| 合作方管理制度 | 合作方准入、业务边界、区域管控 |

### 2. 系统管控

- 在核心系统中设置客户属地的校验规则
- 部署地理围栏技术，识别客户的地理位置
- 对互联网渠道的贷款申请增加区域校验
- 建立异地业务的系统监控和预警机制

### 3. 人员培训

- 对全行员工进行属地经营政策的培训
- 对一线人员进行客户属地识别的培训
- 对互联网业务团队进行跨区域经营限制的培训
- 定期开展属地经营合规检查

### 4. 监管沟通

- 定期向监管机构报告属地经营执行情况
- 对存量异地业务的整改进度进行定期报告
- 对边界情形及时与监管机构沟通确认
- 积极配合监管机构开展属地经营专项检查

## 监管处罚案例

### 案例一：某城商行违规跨区域发放贷款案（2023年）

- **违规事实**：该城商行通过互联网渠道向全国多个省份的客户发放个人消费贷款，跨区域贷款余额达到35亿元。客户属地校验系统未有效运行，大量非当地客户的贷款申请被审批通过。
- **处罚结果**：被罚款500万元，暂停互联网贷款新业务6个月。
- **涉及条款**：属地经营原则、互联网贷款跨区域限制。

### 案例二：某城商行异地存款违规案（2022年）

- **违规事实**：该城商行通过与第三方互联网平台合作，以高利率和现金红包等方式在全国范围内吸收个人存款约50亿元，严重违反了属地经营和存款管理的要求。
- **处罚结果**：被罚款400万元，高管被监管谈话，要求限期清退异地存款。
- **涉及条款**：属地经营原则、存款管理要求。

### 案例三：某城商行联合贷款跨区域经营案（2022年）

- **违规事实**：该城商行与某互联网银行合作开展联合贷款业务，实质上由互联网银行负责获客和风控，城商行仅负责资金供给，大量贷款客户不在城商行注册地辖区内。
- **处罚结果**：被罚款300万元，要求限期整改。
- **涉及条款**：联合贷款跨区域限制。

### 案例四：某城商行存量异地业务整改不力案（2023年）

- **违规事实**：该城商行在监管要求存量异地业务整改后，仅形式上停止新增异地业务，但通过"续贷""展期"等方式实质性保留了大量异地贷款，整改进度严重滞后。
- **处罚结果**：被监管机构通报批评，限制市场准入。
- **涉及条款**：存量异地业务整改要求。
