# 村镇银行与农信社改制政策

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《村镇银行管理暂行规定》《农村中小银行机构行政许可事项实施办法》等系列文件 |
| 发布机构 | 原中国银行业监督管理委员会 / 国家金融监督管理总局 |
| 核心文件 | 银监发〔2007〕1号《村镇银行管理暂行规定》；《关于进一步推动村镇银行化险改革工作的通知》（2023年） |
| 最新政策 | 2023-2024年推进村镇银行改革化险，推动主发起行增持、吸收合并、重组等 |
| 效力层级 | 部门规章 + 规范性文件 |

村镇银行和农村信用社（农信社）是我国农村金融体系的重要组成部分。近年来，监管部门持续推进村镇银行和农信社的改革化险工作，通过主发起行增持、吸收合并、重组等方式推动机构改革，强化公司治理，提升风险防控能力。这一系列政策对城商行、农商行的发展战略和业务布局产生了深远影响。

## 核心内容摘要

### 一、村镇银行基本制度

**1. 设立条件**

- 发起人中须有至少一家银行业金融机构作为主发起人
- 主发起行持股比例不低于**15%**（2014年后调高要求）
- 注册资本最低限额为300万元（县市级）或100万元（乡镇级）
- 有符合规定的章程、组织架构和管理制度

**2. 经营定位**

- 村镇银行应当立足县域、服务社区、支农支小
- 信贷资金主要用于当地农户和小微企业
- 不得跨区域经营
- 不得发放与定位不符的大额贷款

**3. 公司治理**

- 设立股东大会、董事会、监事会和高级管理层
- 主发起行在公司治理中发挥重要作用
- 独立董事制度（条件允许的应当设立）
- 关联交易管理

### 二、农信社改制

**1. 改制路径**

| 路径 | 内容 | 适用情形 |
|------|------|----------|
| 改制为农商行 | 将农信社整体改制为农村商业银行 | 条件成熟的地区 |
| 合并组建农商行 | 多个农信社合并组建一家农商行 | 单个农信社规模较小 |
| 省联社改革 | 省联社转型为金融服务公司或联合银行 | 省级层面 |

**2. 改制条件**

- 不良贷款率降至合理水平
- 资本充足率达到监管要求
- 公司治理架构完善
- 股权结构清晰，股东资质合规
- 获得地方政府的支持

**3. 改制流程**

- 制定改制方案（包括股权设置、资产处置、人员安排等）
- 清产核资和资产评估
- 处置不良资产
- 募集股本，引入合格股东
- 完善公司治理架构
- 向监管机构申请改制审批
- 完成工商登记和牌照变更

### 三、村镇银行改革化险

**1. 主发起行增持**

- 鼓励主发起行增持村镇银行股权，提升持股比例
- 主发起行持股比例可以提升至**51%以上**甚至**100%**
- 增持后主发起行对村镇银行的风险管控能力增强
- 2023年以来，多家大型银行和股份制银行增持了旗下村镇银行

**2. 吸收合并**

- 主发起行将村镇银行吸收合并为分支机构
- 村镇银行的独立法人资格注销
- 有利于降低管理成本和提升风控能力
- 典型案例：多家国有大行已将旗下村镇银行吸收合并为支行

**3. 重组**

- 引入新的战略投资者进行重组
- 多家村镇银行合并为一家规模更大的机构
- 通过资产重组化解存量风险
- 优化股权结构和公司治理

### 四、监管要求

**1. 资本充足率要求**

| 指标 | 最低要求 |
|------|----------|
| 核心一级资本充足率 | 7.5% |
| 一级资本充足率 | 8.5% |
| 资本充足率 | 10.5% |

**2. 贷款集中度**

- 对单一借款人的贷款余额不超过资本净额的**10%**
- 对单一集团客户的贷款余额不超过资本净额的**15%**

**3. 流动性要求**

- 流动性比例不低于**25%**
- 流动性匹配率不低于**100%**

## 零售业务相关条款解读

### （一）村镇银行零售业务定位

**1. 服务当地农户**

村镇银行的核心使命是服务当地农户：
- 发放农户小额贷款，支持农业生产经营
- 提供存取款、转账、代理等基础金融服务
- 参与农村信用体系建设
- 开展金融知识普及和教育

**2. 服务小微企业**

- 为当地小微企业和个体工商户提供信贷支持
- 开发适合当地产业特点的金融产品
- 提供结算、代理等综合金融服务
- 参与当地政府的普惠金融工程

**3. 服务社区居民**

- 为县城和乡镇居民提供消费贷款、住房贷款等服务
- 代理销售保险、基金等金融产品
- 提供便民缴费、社保代发等服务
- 开展适老化服务和无障碍服务

### （二）改制对零售业务的影响

**1. 吸收合并后的业务整合**

主发起行将村镇银行吸收合并后：
- 原有村镇银行的客户自动成为主发起行的客户
- 零售产品体系需要整合和统一
- 核心系统需要迁移和对接
- 员工需要重新安排和培训
- 品牌标识需要更换

**2. 产品和服务的提升**

- 改制后可以借助主发起行的产品和技术优势
- 零售产品种类更加丰富
- 线上服务能力得到增强
- 风控能力显著提升
- 客户体验得到改善

**3. 品牌和信誉的提升**

- 从"村镇银行"变为大行的分支机构，客户信任度提升
- 存款保险保障不变（50万元最高偿付限额）
- 客户服务标准与大行接轨
- 有利于吸引更多客户和存款

### （三）农信社改制为农商行的零售业务影响

**1. 业务范围的扩大**

- 改制为农商行后，业务范围更加广泛
- 可以开展理财、信用卡、外汇等更多零售业务
- 但核心定位仍然是支农支小
- 可以在更大范围内设立分支机构

**2. 治理结构的完善**

- 公司治理更加规范
- 风险管理能力增强
- 内部控制更加完善
- 信息披露更加透明

**3. 竞争力的提升**

- 品牌形象和市场信誉提升
- 人才引进和培养能力增强
- 科技投入和创新能力提升
- 融资渠道更加多元化

### （四）改制过程中的客户权益保护

**1. 存款客户权益**

- 改制不影响存款客户的权益
- 存款本金和利息按照原有约定执行
- 存款保险保障继续有效
- 客户可以选择继续使用或转存

**2. 贷款客户权益**

- 改制不影响贷款合同的效力
- 贷款本金和利息按照原有合同约定执行
- 还款方式和期限不变
- 客户的信用记录不受影响

**3. 理财产品客户权益**

- 改制不影响理财产品的存续
- 产品到期后按照合同约定兑付
- 客户可以选择继续持有或提前赎回（如有此选项）
- 新的理财产品由改制后的机构发行

### （五）改制后的零售业务发展战略

**1. 客户基础整合**

- 对原村镇银行/农信社的客户进行全面梳理
- 建立统一的客户信息管理系统
- 识别和挖掘客户的潜在需求
- 开展客户维护和拓展活动

**2. 产品体系优化**

- 整合和优化零售产品体系
- 保留受欢迎的原有产品
- 引入主发起行的优质产品
- 开发适合当地市场的特色产品

**3. 渠道建设**

- 优化网点布局和服务功能
- 对接主发起行的电子渠道
- 推广手机银行和网上银行
- 完善助农取款服务网络

**4. 队伍建设**

- 对原有员工进行业务培训
- 引入主发起行的专业人才
- 建立统一的绩效考核体系
- 加强企业文化融合

## 对城商行/农商行的影响

### 1. 作为主发起行的责任与挑战

对于作为村镇银行主发起行的城商行、农商行：
- 需要承担村镇银行的风险管控责任
- 面临增持或吸收合并的决策压力
- 需要投入更多资源支持村镇银行发展
- 对村镇银行的并表管理要求更高

### 2. 战略机遇

- 通过吸收合并村镇银行扩大业务覆盖范围
- 通过增持村镇银行增强控制力和协同效应
- 通过参与农信社改制拓展业务版图
- 利用政策支持获取低成本资金和税收优惠

### 3. 风险管理

- 需要关注村镇银行的存量风险
- 在改制过程中做好风险评估和尽职调查
- 建立有效的并表风险管理体系
- 防范改制过程中的操作风险和声誉风险

### 4. 合规管理

- 严格遵守改制审批的程序要求
- 确保改制方案的合规性
- 做好改制过程中的信息披露
- 妥善处理客户权益保护问题

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 村镇银行管理办法 | 主发起行职责、公司治理、风险管理 |
| 村镇银行并表管理办法 | 并表范围、并表方法、内部交易消除 |
| 改制实施方案 | 改制路径、时间安排、资源投入 |
| 客户权益保护方案 | 存款保障、贷款衔接、产品过渡 |
| 员工安置方案 | 岗位安排、薪酬待遇、培训计划 |
| 品牌整合方案 | 标识更换、宣传策略、客户沟通 |

### 2. 改制工作流程

- 成立改制工作领导小组和工作专班
- 制定详细的改制实施方案和时间表
- 开展清产核资和资产评估
- 制定不良资产处置方案
- 设计股权结构和募集方案
- 完善公司治理架构和管理制度
- 向监管机构申请改制审批
- 完成工商登记、牌照变更等法律手续
- 开展系统迁移和业务切换
- 做好员工培训和客户沟通

### 3. 风险管理

- 对村镇银行进行全面的尽职调查
- 评估存量资产质量和潜在风险
- 制定风险化解和处置方案
- 建立并表风险管理的系统和流程
- 加强改制期间的流动性风险管理

### 4. 沟通与报告

- 定期向监管机构报告改制进展
- 与地方政府保持密切沟通
- 做好员工的思想工作和利益保障
- 向客户做好改制信息的传达和服务保障

## 监管处罚案例

### 案例一：某村镇银行严重违规经营案（2022年河南村镇银行事件）

- **事件背景**：2022年4月，河南多家村镇银行出现存款提取困难，引发社会广泛关注。
- **违规事实**：相关村镇银行的股东与外部人员勾结，通过互联网平台违规吸收公众存款，资金被大股东挪用，公司治理形同虚设。
- **处置结果**：监管部门依法对相关村镇银行实施接管，垫付存款本金（50万元以内全额垫付），多名涉案人员被追究刑事责任。
- **教训**：村镇银行公司治理失效、股东行为失控可能导致严重风险，主发起行必须切实履行管理责任。

### 案例二：某村镇银行贷款集中度过高案（2023年）

- **违规事实**：某村镇银行对单一客户的贷款余额超过资本净额的30%，严重超出10%的监管上限；多笔大额贷款集中于少数关联企业。
- **处罚结果**：被罚款100万元，要求限期压降集中度。
- **涉及条款**：贷款集中度监管要求。

### 案例三：某农信社改制过程中违规操作案（2022年）

- **违规事实**：某农信社在改制为农商行过程中，资产评估不实、股权转让不规范、不良贷款处置不透明，导致改制后的农商行资产质量存在较大隐患。
- **处罚结果**：改制方案被监管机构退回，要求重新进行资产评估和不良处置。
- **涉及条款**：改制审批程序要求。

### 案例四：某村镇银行偏离定位案（2023年）

- **违规事实**：某村镇银行偏离支农支小定位，大量资金投向房地产和政府融资平台，涉农贷款和小微企业贷款占比仅为25%，远低于监管要求。
- **处罚结果**：被监管机构通报批评，限制市场准入，要求限期整改。
- **涉及条款**：村镇银行经营定位要求。
