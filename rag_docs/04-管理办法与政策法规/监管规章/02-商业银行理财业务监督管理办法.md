# 商业银行理财业务监督管理办法

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《商业银行理财业务监督管理办法》 |
| 发布机构 | 原中国银行保险监督管理委员会 |
| 发布时间 | 2018年9月26日（银保监会令2018年第6号） |
| 配套文件 | 《关于规范金融机构资产管理业务的指导意见》（资管新规，2018年4月） |
| 文号 | 银保监会令2018年第6号 |
| 效力层级 | 部门规章 |

《商业银行理财业务监督管理办法》（以下简称"理财新规"）是资管新规的重要配套文件，全面重塑了银行理财业务的监管框架。该办法确立了理财业务"卖者尽责、买者自负"的基本原则，推动理财产品从预期收益型向净值化转型，强化投资者适当性管理和信息披露要求。对于零售业务而言，理财新规的实施深刻改变了银行理财产品的销售方式、产品形态和客户沟通模式。

## 核心内容摘要

### 一、理财产品分类

| 类别 | 分类标准 | 风险等级 |
|------|----------|----------|
| 固定收益类 | 投资于存款、债券等债权类资产的比例不低于80% | 相对较低 |
| 权益类 | 投资于权益类资产的比例不低于80% | 相对较高 |
| 商品及金融衍生品类 | 投资于商品及金融衍生品的比例不低于80% | 较高 |
| 混合类 | 投资于债权类、权益类、商品及衍生品类资产且任一未达上述标准 | 中等 |

### 二、净值化管理要求

- **第43条**：商业银行应当对理财产品实行净值化管理，净值生成应当符合企业会计准则的规定，及时反映基础资产的收益和风险。
- 过渡期内（至2021年底，后延长至2022年底）允许部分存量产品按原有方式计量，新发产品必须符合净值化要求。

### 三、投资者适当性管理

- **第26条**：商业银行应当对投资者进行风险承受能力评估，不得向风险承受能力低于理财产品风险等级的投资者销售理财产品。
- **第27条**：投资者风险承受能力评估结果有效期不超过一年，超过一年未重新评估的，不得购买新的理财产品。

### 四、信息披露要求

- **第44-50条**：商业银行应当在理财产品销售前、存续期间、到期后分别进行信息披露，包括产品说明书、定期报告、重大事项临时报告等。
- 信息披露应当通过银行官方网站、手机银行等投资者可便捷获取的渠道进行。

### 五、禁止刚性兑付

- **第43条**：商业银行开展理财业务，应当确保理财业务与其他业务相分离，理财产品与其代销的理财产品相分离，理财产品之间相分离，理财业务操作与其他业务操作相分离。
- 不得以任何形式承诺保本保收益。

### 六、理财产品销售管理

- **第31条**：商业银行只能通过本行渠道（含营业网点和电子渠道）销售理财产品，或者通过其他商业银行、农村合作银行、村镇银行、农村信用合作社等吸收公众存款的银行业金融机构代理销售。
- **第32条**：理财产品销售前，应进行"双录"（录音录像），确保销售过程的合规性。

## 零售业务相关条款解读

### （一）净值化转型对零售客户的影响

净值化管理是理财新规的核心变革，对零售客户的影响深远：

**1. 打破刚兑预期**

过去，银行理财产品以"预期收益率"作为宣传卖点，客户形成了"保本保收益"的心理预期。净值化转型后：
- 理财产品不再承诺固定收益，净值可能波动甚至亏损
- 客户需要承担投资风险，"卖者尽责、买者自负"
- 银行需要加强客户教育，帮助客户理解净值化产品

**2. 产品形态变化**

- 从"预期收益型"转变为"净值型"
- 产品说明书中不再出现"预期收益率"，改为展示"业绩比较基准"
- 净值定期更新（至少每周更新一次），客户可随时查看产品净值

**3. 客户分层管理**

净值化转型后，不同风险偏好的客户需要差异化的产品匹配：
- 保守型客户：推荐现金管理类、短期固收类产品
- 稳健型客户：推荐中长期固收+产品
- 积极型客户：可推荐混合类、权益类产品

### （二）投资者适当性管理的实务操作

投资者适当性管理是理财销售的核心合规要求：

**1. 风险评估**

- 首次购买理财产品的客户必须完成风险承受能力评估问卷
- 评估维度包括：财务状况、投资经验、投资目的、风险偏好、年龄、健康状况等
- 评估结果分为五个等级：保守型、稳健型、平衡型、进取型、激进型
- 评估结果有效期为一年，到期需重新评估

**2. 产品风险等级**

理财产品风险等级分为五级：
- PR1（低风险）：如现金管理类理财产品
- PR2（中低风险）：如纯债类理财产品
- PR3（中风险）：如固收+理财产品
- PR4（中高风险）：如权益类理财产品
- PR5（高风险）：如金融衍生品类理财产品

**3. 适配原则**

- 客户风险等级 ≥ 产品风险等级方可购买
- 风险等级不匹配的，不得销售
- 客户坚持购买超出其风险承受能力产品的，需签署风险揭示书并进行特别提示

**4. 双录要求**

- 在营业网点销售理财产品时，须在销售专区进行录音录像
- 双录内容应包括：客户身份确认、风险评估结果确认、产品信息告知、客户确认购买意愿等
- 双录资料保存期限不少于产品到期后6个月

### （三）理财产品销售的禁止行为

理财新规明确了多项销售禁止行为：

1. **禁止误导销售**：不得将理财产品与存款或其他低风险产品进行混淆宣传
2. **禁止承诺保本保收益**：不得以任何形式向投资者承诺本金不受损失或承诺最低收益
3. **禁止强制搭售**：不得将理财产品与其他产品捆绑销售
4. **禁止私售"飞单"**：不得销售未经总行审批的第三方理财产品
5. **禁止虚假宣传**：不得夸大理财产品收益或隐瞒风险
6. **禁止代客操作**：不得代替客户在销售文件上签字或代替客户操作电子渠道

### （四）公募与私募理财产品的区别

| 对比维度 | 公募理财产品 | 私募理财产品 |
|----------|-------------|-------------|
| 销售对象 | 不特定社会公众 | 合格投资者（不超过200人） |
| 投资门槛 | 1元起购 | 固定收益类30万元、混合类40万元、权益类/衍生品类100万元 |
| 信息披露 | 公开披露 | 向投资者单独披露 |
| 投资范围 | 受较严格限制 | 相对灵活 |
| 适用客户 | 普通零售客户 | 高净值客户 |

### （五）理财子公司模式的过渡

理财新规允许商业银行设立理财子公司开展理财业务。对于城商行、农商行而言：
- 暂不具备设立理财子公司条件的，可以继续以银行名义开展理财业务，但应当实行专营部门管理
- 可通过代销其他银行理财子公司产品丰富产品线
- 需要逐步提升自身投资管理能力

## 对城商行/农商行的影响

### 1. 产品转型压力

城商行、农商行的理财业务长期以预期收益型产品为主，净值化转型面临重大挑战：
- 投研能力不足，难以支撑净值化产品的投资管理
- 系统建设滞后，难以满足净值计算和信息披露要求
- 客户基础以中低风险偏好为主，净值波动可能导致客户流失

**应对策略**：
- 优先发展现金管理类和短期固收类净值产品
- 通过代销大行理财子公司产品丰富产品线
- 加强与外部投研机构的合作

### 2. 销售合规要求提升

理财新规对销售环节提出了更加严格的合规要求：
- 投资者适当性管理需要完善
- 双录系统需要部署和维护
- 销售人员需要培训和持证上岗
- 销售行为规范需要持续监督

### 3. 信息披露工作量增加

- 每只理财产品都需要单独编制和发布定期报告
- 重大事项需要及时发布临时公告
- 信息披露内容需要法律合规审核
- 对IT系统和人员配置提出了更高要求

### 4. 与理财子公司的竞合关系

- 城商行、农商行可以成为理财子公司产品的代销渠道
- 但同时也面临理财子公司产品对自营产品的替代压力
- 需要在代销与自营之间找到平衡点
- 代销能力（渠道、客户、服务）成为核心竞争力

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 理财产品管理办法 | 产品设计、审批、发行、兑付 |
| 理财产品销售管理办法 | 销售流程、适当性管理、双录管理 |
| 投资者适当性管理办法 | 风险评估、产品匹配、持续管理 |
| 理财产品信息披露管理办法 | 披露内容、披露频率、披露渠道 |
| 理财产品投资管理制度 | 投资决策、风险控制、合规管理 |
| 理财产品风险管理制度 | 风险评估、压力测试、应急处置 |

### 2. 系统建设

- **理财产品管理系统**：产品创设、存续管理、到期清算
- **净值计算系统**：估值核算、净值生成、审计确认
- **投资者适当性管理系统**：风险评估、自动匹配、到期提醒
- **双录系统**：录音录像、存储管理、调阅查询
- **信息披露系统**：自动生成报告、多渠道发布

### 3. 人员管理

- 理财销售人员须具备相关从业资格
- 定期组织理财产品知识和合规培训
- 建立销售人员的行为监督和违规问责机制
- 配备专业的投资管理人员和风险管理人员

## 监管处罚案例

### 案例一：某城商行理财产品违规销售案（2022年）

- **违规事实**：该城商行在销售理财产品时，未对投资者进行风险承受能力评估，直接向客户销售高风险等级理财产品；部分销售人员将理财产品宣传为"保本保收益"产品，误导投资者。
- **处罚结果**：被罚款200万元，相关销售人员被给予警告处分。
- **涉及条款**：第26条（投资者适当性管理）、第32条（销售行为规范）。

### 案例二：某农商行理财产品刚兑案（2023年）

- **违规事实**：该农商行在理财产品到期时，对于净值出现亏损的产品，通过向产品注入自有资金的方式实现"刚性兑付"，未让客户承担投资损失。
- **处罚结果**：被罚款300万元，理财业务部门暂停新业务6个月。
- **涉及条款**：第43条（禁止刚性兑付）。

### 案例三：某银行理财"飞单"案（2021年）

- **违规事实**：某银行支行客户经理在银行营业场所内，私自向客户销售非银行审批的第三方理财产品（私募基金），累计销售金额约2000万元。该私募基金最终爆雷，客户损失惨重。
- **处罚结果**：银行被罚款150万元，涉事客户经理被取消任职资格终身并移送司法机关。
- **涉及条款**：第31条（销售渠道管理）。

### 案例四：某城商行理财产品信息披露违规案（2022年）

- **违规事实**：该城商行未按照要求对理财产品进行定期信息披露，多只理财产品连续6个月未发布净值公告；部分理财产品的基础资产发生重大风险事件，未及时发布临时公告。
- **处罚结果**：被罚款100万元。
- **涉及条款**：第44-50条（信息披露要求）。
