# 商业银行信用卡业务监督管理办法

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《商业银行信用卡业务监督管理办法》 |
| 发布机构 | 原中国银行业监督管理委员会 |
| 发布时间 | 2011年1月13日（银监会令2011年第2号） |
| 最新修订 | 2023年发布《关于进一步促进信用卡业务规范健康发展的通知》（银保监规〔2022〕13号）作为补充 |
| 文号 | 银监会令2011年第2号 |
| 效力层级 | 部门规章 |

《商业银行信用卡业务监督管理办法》是信用卡业务的基本监管规章，全面规范了信用卡的发卡管理、授信审批、收单管理、催收管理、息费披露等各环节。近年来，监管部门陆续发布多项补充文件，进一步规范信用卡业务，重点关注息费透明、过度授信、催收规范等热点问题。对于零售业务而言，信用卡是获客、活客和增收的重要产品线。

## 核心内容摘要

### 一、发卡管理

- **第37条**：发卡银行应当建立信用卡申请材料的亲访亲签制度。信用卡申请人必须本人填写申请表并签名，发卡银行工作人员应当亲访申请人工作单位或住所并签名确认（首次申请信用卡的客户）。
- **第39条**：发卡银行不得向未满18周岁的自然人发行信用卡（附属卡除外）；向经查已满18周岁但无固定工作、无稳定收入来源的自然人发行信用卡时，须落实第二还款来源。
- **第41条**：发卡银行应当对信用卡申请人开展资信调查，充分核实申请人的身份、收入、信用状况等信息。

### 二、授信管理

- **第42条**：发卡银行应当建立信用卡授信管理制度，根据申请人的资信状况确定授信额度。
- **第44条**：发卡银行不得将信用卡发卡营销业务外包。
- **第50条**：发卡银行应当建立信用卡授信额度动态管理机制，定期对持卡人信用状况进行评估，根据评估结果调整授信额度。

### 三、收单管理

- **第56条**：收单银行应当建立特约商户管理制度，对特约商户进行实名制管理，审核商户的经营资质。
- **第62条**：收单银行应当对特约商户的交易进行监控，发现异常交易及时采取管控措施。

### 四、催收管理

- **第68条**：发卡银行应当建立信用卡催收管理制度，催收人员不得对与债务无关的第三人进行催收，不得采用暴力、胁迫、恐吓或辱骂等不当催收行为。
- **第70条**：在特殊情况下，发卡银行可以与持卡人协商达成个性化分期还款协议，最长不超过5年（60期）。

### 五、息费披露

- **第37条（2023年补充通知）**：发卡银行应当在信用卡合同中明确披露年化利率，包括透支利率、分期手续费年化利率等。不得仅展示日利率或月利率。

## 零售业务相关条款解读

### （一）发卡营销管理规范

**1. 亲访亲签制度**

信用卡首次申请要求亲访亲签，实务中的执行方式：
- 银行工作人员到申请人工作单位或住所进行面访
- 申请人在银行营业网点当面办理
- 通过远程视频方式进行面谈面签（需满足特定技术条件）
- 通过银行自助终端（如VTM机）办理，经银行工作人员远程审核

**2. 禁止营销外包**

- 发卡银行不得将信用卡营销业务外包给第三方机构
- 但可以与商业机构合作开展联名卡业务，联名方可以协助推荐客户
- 银行员工不得委托他人代为办理信用卡申请受理业务

**3. 学生卡管理**

- 不得向未满18周岁的学生发行信用卡（附属卡除外）
- 向已满18周岁的大学生发行信用卡，须落实第二还款来源（父母或监护人书面确认）
- 学生卡授信额度应从严控制

### （二）授信审批核心要求

**1. 授信审批原则**

- 授信审批应当独立进行，不受营销部门的干预
- 审批人员应当基于申请人的真实资信状况做出审批决定
- 禁止"重发卡、轻授信"的粗放经营模式

**2. 过度授信防控**

2023年补充通知明确要求：
- 银行应当建立客户总体授信额度管控机制，将客户在本行的所有信用卡授信额度纳入统一管理
- 不得向客户发放超出其还款能力的信用卡额度
- 对连续6个月无交易的长期睡眠卡进行额度调降
- 刚性扣减要求：在核定授信额度时，应扣减客户在其他银行的已有授信额度

**3. 睡眠卡管理**

- 连续18个月以上无客户主动交易且当前透支余额、溢缴款为零的长期睡眠卡，应纳入清理范围
- 银行应主动联系持卡人确认是否继续使用该卡
- 对于确认不再使用的，应及时注销
- 监管要求睡眠卡占比不得超过本行发卡总量的20%

### （三）催收管理规范

**1. 催收行为规范**

信用卡催收是近年来社会关注的热点问题，监管要求：
- 催收时间应在每日8:00-21:00之间，不得在深夜或凌晨进行催收
- 不得对与债务无关的第三人进行催收（紧急联系人除外，且仅限通知债务人联系银行）
- 不得采用暴力、威胁、恐吓、侮辱、诽谤等方式催收
- 不得泄露债务人的个人信息
- 催收人员应当佩戴工作证件，表明身份
- 催收过程应当进行录音

**2. 催收外包管理**

- 发卡银行可以将催收业务外包给专业催收机构
- 但须对外包机构进行严格筛选和管理
- 银行对催收外包机构的行为承担最终责任
- 应定期对外包机构的催收行为进行检查和评估

**3. 个性化分期还款**

- 在特殊情况下（如持卡人因病、因灾导致还款困难），银行可与持卡人协商个性化分期还款方案
- 分期还款期限最长不超过5年（60期）
- 个性化分期还款方案期间，银行可根据实际情况减免部分利息和费用
- 协商过程中不得向持卡人收取额外的协商费用

### （四）息费透明披露

**1. 年化利率披露**

- 透支利息：须披露年化利率（APR），不得仅展示日利率（万分之五/日）
- 分期手续费：须将手续费折算为年化利率后披露
- 违约金：须明确违约金的计算标准和收取方式
- 取现手续费：须明确取现手续费率和取现利息的计算方式

**2. 账单信息披露**

- 每月账单应清晰展示：本期应还金额、最低还款金额、还款到期日、透支利率、年化利率
- 账单中应包含还款提示信息
- 分期业务的账单应展示：剩余本金、剩余期数、每期应还金额

### （五）信用卡业务收费管理

近年来监管对信用卡收费管理趋严：
- 不得以"手续费""服务费"等名义变相收取高息
- 分期手续费率应当在合理范围内
- 银行应当对信用卡各项收费项目进行公示
- 不得在客户不知情的情况下收取额外费用

## 对城商行/农商行的影响

### 1. 信用卡业务定位与发展

城商行、农商行的信用卡业务面临以下挑战：
- 与大行在品牌、渠道、产品方面的差距较大
- 发卡规模和交易量较小，规模效应不明显
- 风控能力相对薄弱
- 系统建设和运维成本较高

**发展建议**：
- 立足本地市场，发挥属地优势
- 发行特色联名卡，与本地商户合作
- 聚焦特定客群（如公务员、教师、医护等）
- 与信用卡组织合作，借助其平台资源

### 2. 催收管理的合规压力

城商行、农商行的催收管理需要重点关注：
- 建立完善的催收管理制度和操作规范
- 对催收人员进行系统培训和资质管理
- 对催收外包机构进行严格筛选和持续监控
- 建立催收投诉的处理和反馈机制

### 3. 息费管理合规

- 系统需要支持年化利率的自动计算和展示
- 需要在各个渠道（APP、网站、纸质账单）同步更新利率信息
- 需要对现有收费项目进行全面梳理和公示

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 信用卡发卡管理办法 | 申请受理、资信调查、亲访亲签 |
| 信用卡授信审批制度 | 审批流程、额度管理、动态调额 |
| 信用卡催收管理办法 | 催收规范、外包管理、投诉处理 |
| 信用卡息费管理制度 | 利率定价、费用收取、信息披露 |
| 信用卡风险管理制度 | 欺诈防控、反洗钱、风险预警 |
| 信用卡客户投诉管理办法 | 投诉受理、处理流程、反馈机制 |

### 2. 系统建设

- **信用卡核心系统**：发卡、账务、清算、催收全流程管理
- **风控决策引擎**：申请审批、欺诈检测、行为评分
- **催收管理系统**：案件分配、催收记录、外包管理
- **双录系统**：信用卡销售过程的录音录像
- **息费展示系统**：年化利率自动计算、多渠道信息同步

### 3. 消费者权益保护

- 建立信用卡客户投诉的快速响应机制
- 对投诉集中问题进行专项分析和整改
- 定期开展信用卡业务合规自查
- 将消费者保护纳入信用卡业务考核指标

## 监管处罚案例

### 案例一：某城商行信用卡过度授信案（2023年）

- **违规事实**：该城商行在信用卡授信审批中未执行刚性扣减要求，对已在多家银行持有高额信用卡的客户继续发放大额信用卡，导致部分客户总体授信额度远超其收入水平。
- **处罚结果**：被罚款200万元，暂停信用卡新发卡业务3个月。
- **涉及条款**：第42条（授信管理）、2023年补充通知。

### 案例二：某银行信用卡催收违规案（2022年）

- **违规事实**：某银行信用卡中心的外包催收机构在催收过程中，频繁对持卡人及其家属进行电话骚扰，使用威胁性语言，并向持卡人单位领导透露债务信息。
- **处罚结果**：银行被罚款150万元，终止与该催收机构的合作关系。
- **涉及条款**：第68条（催收管理规范）。

### 案例三：某农商行信用卡息费披露违规案（2023年）

- **违规事实**：该农商行在信用卡分期业务宣传中，仅展示月手续费率（0.6%/月），未披露年化利率（实际年化约13%），误导消费者认为分期手续费较低。
- **处罚结果**：被罚款80万元，责令整改并退还多收手续费。
- **涉及条款**：第37条（息费披露）。

### 案例四：某银行学生卡违规发卡案（2021年）

- **违规事实**：某银行在大学校园内开展信用卡营销活动，向多名未满18周岁的学生发放信用卡，且未落实第二还款来源。
- **处罚结果**：被罚款50万元，相关责任人被给予警告处分。
- **涉及条款**：第39条（学生卡管理）。
