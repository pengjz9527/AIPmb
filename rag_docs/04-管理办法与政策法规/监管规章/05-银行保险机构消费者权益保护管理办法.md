# 银行保险机构消费者权益保护管理办法

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《银行保险机构消费者权益保护管理办法》 |
| 发布机构 | 原中国银行保险监督管理委员会 |
| 发布时间 | 2022年12月26日（银保监会令2022年第9号） |
| 施行时间 | 2023年3月1日 |
| 文号 | 银保监会令2022年第9号 |
| 效力层级 | 部门规章 |

《银行保险机构消费者权益保护管理办法》（以下简称"消保办法"）是我国银行保险行业消费者权益保护领域的专项规章，系统性地规定了金融消费者的八项基本权利，明确了银行保险机构在产品设计、销售、服务等各环节的消费者保护义务。该办法的出台标志着金融消费者权益保护进入了一个新的阶段，对银行零售业务的影响尤为深远。

## 核心内容摘要

### 一、金融消费者的八项基本权利

| 序号 | 权利名称 | 核心要求 |
|------|----------|----------|
| 1 | 知情权 | 真实、准确、完整、及时地披露产品和服务信息 |
| 2 | 自主选择权 | 消费者有权自主选择金融产品和服务 |
| 3 | 公平交易权 | 不得设置不合理的交易条件或搭售产品 |
| 4 | 财产安全权 | 保障消费者资金安全，防范资金损失 |
| 5 | 依法求偿权 | 消费者因购买金融产品受到损害时有权获得赔偿 |
| 6 | 受教育权 | 开展金融知识普及和消费者教育活动 |
| 7 | 受尊重权 | 尊重消费者的人格尊严和民族风俗习惯 |
| 8 | 信息安全权 | 保护消费者个人信息安全，防止泄露 |

### 二、适当性管理

- **第19条**：银行保险机构应当建立金融消费者适当性管理制度，对金融产品和服务进行风险评级，对金融消费者进行风险承受能力评估，将适当的产品和服务提供给适当的消费者。
- **第20条**：不得向金融消费者推荐与其风险承受能力不匹配的金融产品或服务。

### 三、信息披露

- **第14-18条**：银行保险机构应当以显著方式向消费者披露产品或服务的关键信息，包括但不限于：利率、费用、收益风险、合同期限、违约责任等。
- 信息披露应当使用通俗易懂的语言，避免使用过于专业的术语。

### 四、投诉处理

- **第35-40条**：银行保险机构应当建立健全消费者投诉处理工作机制，明确投诉处理流程和时限。
- 一般投诉应当在15个工作日内做出处理决定并告知消费者；复杂投诉可延长至30个工作日。
- 银行保险机构应当定期分析投诉数据，及时发现和解决系统性问题。

### 五、禁止行为

- **第22条**：不得进行虚假或引人误解的宣传
- **第23条**：不得强制搭售或捆绑销售
- **第24条**：不得利用格式条款免除自身责任、加重消费者责任
- **第25条**：不得泄露或非法使用消费者个人信息

## 零售业务相关条款解读

### （一）知情权保障

知情权是金融消费者最基本的权利，在零售业务中的具体体现：

**1. 存款产品**
- 明确告知存款利率、计息方式、存期、提前支取规则
- 对于结构性存款等特殊产品，需要详细说明收益结构和风险
- 不得将存款产品与理财产品混淆宣传

**2. 贷款产品**
- 明确告知贷款年化利率（APR）、还款方式、总还款金额
- 披露可能产生的各项费用（如提前还款手续费、违约金等）
- 说明贷款逾期后的利息计算方式和催收措施

**3. 理财产品**
- 披露产品风险等级、投资范围、业绩比较基准（非承诺收益）
- 说明产品净值波动的原因和影响
- 告知投资者赎回规则和可能的赎回限制

**4. 信用卡产品**
- 披露透支利率（年化）、分期手续费率（年化）、违约金标准
- 说明最低还款额的计算方式
- 告知免息期规则和逾期后果

### （二）自主选择权保护

自主选择权要求银行尊重消费者的选择自由：

**1. 禁止强制搭售**
- 不得在发放贷款时强制要求购买保险、基金等产品
- 不得将开通手机银行、短信通知等服务作为办理业务的前提条件
- 不得强制客户办理不必要的增值服务

**2. 禁止不当诱导**
- 不得通过夸大收益、隐瞒风险等方式诱导消费者购买不适合的产品
- 不得利用信息不对称优势误导消费者做出选择
- 不得在消费者明确拒绝后继续推销

**3. 保障退出权利**
- 消费者有权提前终止金融产品或服务合同
- 银行应当明确提前终止的条件和费用
- 不得设置不合理的提前终止障碍

### （三）公平交易权保障

**1. 格式合同管理**
- 格式合同条款应当公平合理，不得免除银行自身责任
- 不得加重消费者的责任或义务
- 不得排除消费者的主要权利
- 对格式合同中的免责条款，应以显著方式提示消费者注意

**2. 收费管理**
- 收费项目和标准应当公示
- 不得收取未予告知的费用
- 不得以"手续费""服务费"等名义变相提高利率
- 服务价格应当与提供的服务相匹配

**3. 服务标准**
- 不得对不同消费者实行不合理的差别对待
- 不得因消费者的年龄、性别、民族、宗教信仰等因素而歧视
- 为老年人、残障人士等特殊群体提供便利服务

### （四）信息安全权保障

**1. 信息收集**
- 收集个人信息应当遵循合法、正当、必要原则
- 应当明确告知消费者信息收集的目的、范围和方式
- 应当取得消费者的明确同意

**2. 信息使用**
- 个人信息仅用于约定的目的，不得擅自扩大使用范围
- 不得将消费者信息用于未经授权的营销活动
- 不得将消费者信息出售或非法提供给第三方

**3. 信息安全**
- 采取必要的技术措施和管理措施保护信息安全
- 建立信息安全事件应急预案
- 发生信息泄露事件时及时通知消费者并采取补救措施

### （五）投诉处理机制

**1. 投诉渠道**

银行应建立多元化的投诉渠道：
- 营业网点现场投诉
- 客服电话投诉
- 网上银行/手机银行在线投诉
- 信函投诉
- 监管部门投诉转办

**2. 投诉处理流程**

- **受理**：收到投诉后2个工作日内确认受理
- **调查**：对投诉事项进行调查核实
- **处理**：一般投诉15个工作日内处理完毕，复杂投诉可延长至30个工作日
- **反馈**：将处理结果以书面或电话方式告知消费者
- **回访**：对投诉处理结果进行满意度回访

**3. 投诉数据分析**

- 定期统计和分析投诉数据
- 识别投诉热点和系统性问题
- 制定改进措施并向管理层报告
- 按监管要求报送投诉数据

## 对城商行/农商行的影响

### 1. 消费者保护体系建设的投入加大

城商行、农商行需要加大消费者保护方面的投入：
- 建立专门的消费者权益保护部门或指定专人负责
- 完善投诉处理系统，实现投诉全流程线上化管理
- 建立消费者保护审查机制，在产品设计、营销推广前进行消保审查
- 加强员工培训，提升全员消费者保护意识

### 2. 产品和服务的合规审查

- 对现有零售产品进行全面消保合规审查
- 修订不符合消保要求的产品说明书和合同文本
- 完善产品销售的适当性管理流程
- 建立新产品消保审查的前置程序

### 3. 信息披露和透明度提升

- 全面梳理各项零售产品的收费项目
- 在各渠道统一披露产品年化利率和费用信息
- 使用通俗易懂的语言编制产品说明
- 对重要条款以加粗、标注等方式突出显示

### 4. 投诉管理能力建设

- 建立7×24小时投诉受理机制
- 提升投诉处理的专业化水平
- 建立投诉溯源整改机制
- 将投诉处理效果纳入绩效考核

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 消费者权益保护管理办法 | 组织架构、职责分工、考核评价 |
| 消费者适当性管理办法 | 风险评估、产品匹配、持续管理 |
| 消费者投诉管理办法 | 投诉受理、处理流程、反馈机制 |
| 消费者信息安全管理办法 | 信息收集、使用、存储、共享 |
| 产品和服务消保审查办法 | 审查范围、审查标准、审查流程 |
| 消费者金融知识教育制度 | 教育内容、教育方式、效果评估 |

### 2. 组织架构

- 设立消费者权益保护委员会（由高级管理层牵头）
- 设立消费者权益保护专职部门
- 在各业务条线配备消保联络员
- 建立消保工作与绩效考核挂钩机制

### 3. 消保审查机制

- 对新产品上市前进行消保审查
- 对营销宣传材料进行消保合规审查
- 对合同文本进行公平性审查
- 对收费项目进行合理性审查

### 4. 培训与宣传

- 每年至少开展一次全员消保培训
- 对一线销售人员进行专项消保培训
- 开展消费者金融知识普及活动
- 建立消保案例分享和学习机制

## 监管处罚案例

### 案例一：某城商行侵害消费者知情权案（2023年）

- **违规事实**：该城商行在销售个人贷款产品时，未在合同中明确披露年化利率，仅以"日利率万分之三"的方式展示利率信息，导致消费者无法准确了解贷款成本。同时，在贷款发放时扣收"贷款服务费"，未事先告知消费者。
- **处罚结果**：被罚款200万元，责令退还违规收取的服务费。
- **涉及条款**：第14-16条（信息披露）、第22条（禁止虚假宣传）。

### 案例二：某农商行强制搭售案（2022年）

- **违规事实**：该农商行在办理农户贷款时，强制要求借款人购买人身意外保险和手机银行短信通知服务，将上述服务作为贷款审批的前置条件。
- **处罚结果**：被罚款150万元，责令退还全部搭售费用。
- **涉及条款**：第23条（禁止强制搭售）。

### 案例三：某银行客户信息泄露案（2023年）

- **违规事实**：某银行将信用卡客户信息提供给合作的保险公司用于营销推广，未获得客户授权同意，涉及客户信息约50万条。
- **处罚结果**：被罚款500万元，相关责任人分别被罚款10万元。
- **涉及条款**：第25条（信息安全保护）。

### 案例四：某城商行投诉处理违规案（2023年）

- **违规事实**：该城商行未建立有效的投诉处理机制，消费者投诉受理率不足60%，投诉处理超时率超过40%，未按监管要求报送投诉数据。
- **处罚结果**：被罚款80万元，责令限期整改。
- **涉及条款**：第35-40条（投诉处理管理）。
