# 商业银行金融资产风险分类办法

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《商业银行金融资产风险分类办法》 |
| 发布机构 | 中国银行保险监督管理委员会、中国人民银行 |
| 发布时间 | 2023年2月11日（银保监会、人民银行令2023年第1号） |
| 施行时间 | 2023年7月1日 |
| 过渡期 | 2023年7月1日至2025年12月31日 |
| 文号 | 银保监会、人民银行令2023年第1号 |
| 效力层级 | 部门规章 |

《商业银行金融资产风险分类办法》（以下简称"风险分类新规"）是对原银监会2007年《贷款风险分类指引》的重大升级。新规将风险分类对象从贷款扩展至承担信用风险的全部金融资产，进一步明确了五级分类标准，强化了逾期天数与分类结果的对应关系，提高了风险分类的准确性和审慎性。该办法的实施对银行零售贷款的资产质量管理提出了更高要求。

## 核心内容摘要

### 一、适用范围扩展

- **第2条**：本办法适用于商业银行承担信用风险的全部金融资产，包括贷款、债券投资、同业资产、表外业务等。
- 风险分类从"贷款五级分类"扩展为"金融资产五级分类"。

### 二、五级分类标准

| 分类 | 定义 | 核心特征 |
|------|------|----------|
| 正常 | 债务人能够履行合同，没有足够理由怀疑贷款本息不能按时足额偿还 | 履约能力良好 |
| 关注 | 尽管债务人目前有能力偿还贷款本息，但存在一些可能对偿还产生不利影响的因素 | 存在不利因素 |
| 次级 | 债务人的偿还能力出现明显问题，完全依靠其正常营业收入无法足额偿还贷款本息，即使执行担保，也可能会造成一定损失 | 偿还能力明显问题 |
| 可疑 | 债务人无法足额偿还贷款本息，即使执行担保，也肯定要造成较大损失 | 肯定造成较大损失 |
| 损失 | 在采取所有可能的措施或一切必要的法律程序之后，贷款本息仍然无法收回，或只能收回极少部分 | 基本无法收回 |

### 三、逾期天数与分类对应关系（新规核心变化）

新规明确了逾期天数与风险分类之间的硬性对应关系：

| 逾期天数 | 最低分类 |
|----------|----------|
| 逾期超过**90天** | 至少归为**次级** |
| 逾期超过**270天** | 至少归为**可疑** |
| 逾期超过**360天** | 至少归为**损失** |

**重要变化**：
- 原规定中逾期90天以上贷款应归为次级及以下，新规更加明确
- 新增逾期270天以上归为可疑的硬性要求
- 新增逾期360天以上归为损失的硬性要求
- 不允许以"重组""续贷"等方式规避逾期天数与分类的对应关系

### 四、重组资产的特殊规定

- **第18条**：重组资产是指因债务人发生财务困难，银行对其合同条款做出调整（包括展期、降息、减免本息等）的资产。
- 重组资产应至少归为**次级**类。
- 重组后的观察期不少于**1年**，观察期内不得上调分类。
- 观察期内再次重组的，应至少归为**可疑**类，并重新计算观察期。

### 五、交叉违约认定

- **第16条**：债务人在本行的任一债权被分类为次级或以下的，该债务人在本行的所有债权均应被分类为次级或以下（交叉传染）。
- 这一规定防止银行通过对同一债务人的不同贷款进行差异化分类来掩盖真实风险。

### 六、零售贷款的特殊规定

- **第14条**：对于零售贷款，可以采用脱手法（即按账户层面分类）或资产池法（即按资产组合层面分类）进行风险分类。
- 采用资产池法的，应当确保资产池内各贷款的风险特征同质。

## 零售业务相关条款解读

### （一）个人贷款的风险分类

**1. 分类标准的具体应用**

个人贷款的风险分类应综合考虑以下因素：
- **逾期天数**：最直接的分类依据，严格执行新规的硬性对应关系
- **借款人还款能力**：收入变化、就业状况、负债情况
- **担保情况**：抵押物价值变化、担保人担保能力
- **还款意愿**：借款人主动沟通情况、还款计划调整
- **外部环境**：经济形势、行业风险、区域风险

**2. 个人住房贷款的特殊考虑**

- 个人住房贷款通常有抵押物担保，在分类时需综合考虑抵押物价值
- 抵押物价值低于贷款余额的，应适当下调分类
- 对于房价大幅下跌区域的贷款，应加强风险评估
- 逾期超过90天的个人住房贷款，即使有充足的抵押物，也应至少归为次级

**3. 个人消费贷款的分类**

- 个人消费贷款通常为信用贷款，无抵押物担保
- 逾期天数是分类的主要依据
- 需要关注借款人的多头借贷情况
- 对于信用卡透支，应按照新规要求严格分类

### （二）逾期天数计算规则

**1. 逾期天数的起算**

- 从合同约定的还款日次日开始计算
- 分期还款的，以每期应还金额为基数分别计算逾期天数
- 到期一次还本付息的，以到期日次日为起算日

**2. 逾期天数的终止**

- 借款人足额偿还逾期本息后，逾期天数清零
- 部分偿还的，按照先息后本的原则确定逾期天数的终止
- 重组后的贷款，逾期天数重新计算

**3. 实务中的注意事项**

- 不得通过"借新还旧"等方式人为缩短逾期天数
- 不得通过"还旧借新"等方式规避逾期认定
- 系统应自动计算逾期天数并触发分类调整

### （三）重组贷款的管理

**1. 个人贷款重组的认定标准**

符合以下情形之一的，应认定为重组贷款：
- 因借款人财务困难而同意展期
- 因借款人财务困难而同意降低利率
- 因借款人财务困难而同意减免部分本金或利息
- 因借款人财务困难而同意调整还款方式（如从等额本息改为先息后本）

**2. 重组贷款的分类规则**

- 重组前：根据逾期天数和借款人实际状况确定分类
- 重组后：至少归为次级类
- 观察期内（至少1年）：不得上调分类
- 观察期满后：根据借款人还款表现和还款能力重新评估分类
- 再次重组：至少归为可疑类，重新计算观察期

**3. 个人贷款重组的实务操作**

- 建立个人贷款重组的审批流程和权限管理
- 做好重组贷款的台账管理和跟踪监控
- 对重组后的贷款保持密切的贷后关注
- 在系统中设置重组贷款的标识和分类限制

### （四）不良贷款的认定与管理

**1. 不良贷款的范围**

- 次级、可疑、损失三类统称为不良贷款
- 不良贷款率 = 不良贷款余额 / 全部贷款余额 × 100%

**2. 拨备计提要求**

| 分类 | 拨备覆盖率最低要求 |
|------|-------------------|
| 正常 | 1.5% |
| 关注 | 3% |
| 次级 | 25% |
| 可疑 | 50% |
| 损失 | 100% |

**3. 不良贷款的处置**

- 催收清收：通过自主催收或委托催收回收贷款
- 诉讼追偿：通过司法途径追偿贷款
- 资产转让：将不良资产打包转让给资产管理公司
- 核销：符合条件的不良贷款可以进行账务核销（账销案存）

### （五）零售贷款的资产池分类法

**1. 适用条件**

- 适用于风险特征同质的零售贷款组合
- 如信用卡透支、小额消费贷款等
- 需要建立科学的资产池分类模型

**2. 模型要求**

- 模型应基于历史违约数据进行校准
- 模型应能够预测贷款组合的预期损失率
- 模型应定期进行回测和验证
- 模型参数应根据经济环境变化及时调整

**3. 资产池分类法的监管要求**

- 采用资产池分类法须经监管机构认可
- 应建立完善的数据支持和模型文档
- 应定期向监管机构报告资产池分类的结果

## 对城商行/农商行的影响

### 1. 不良贷款率可能上升

新规实施后，城商行、农商行的不良贷款率可能面临上升压力：
- 逾期天数与分类的硬性对应关系更加严格
- 逾期270天以上必须归为可疑，逾期360天以上必须归为损失
- 交叉违约认定可能使更多贷款被下调分类
- 重组贷款的分类限制使不良化解更加困难

### 2. 拨备计提压力加大

不良贷款增加意味着需要计提更多的拨备：
- 拨备覆盖率要求不变，但不良贷款基数扩大
- 直接影响利润水平
- 需要优化资产结构，降低风险加权资产

### 3. 系统改造需求

- 核心系统需要支持逾期天数的自动计算和分类触发
- 需要建立重组贷款的标识和跟踪管理功能
- 需要实现交叉违约的自动识别和分类传染
- 需要完善拨备计提的自动化计算功能

### 4. 资产质量管理的精细化

- 需要建立更加精细化的贷后管理体系
- 需要加强对逾期贷款的早期干预
- 需要优化贷款重组的策略和流程
- 需要加强不良贷款的清收处置力度

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 金融资产风险分类管理办法 | 分类标准、分类流程、分类权限 |
| 贷款逾期管理办法 | 逾期认定、催收启动、分类调整 |
| 贷款重组管理办法 | 重组认定、审批流程、观察期管理 |
| 不良贷款管理办法 | 不良认定、清收处置、核销管理 |
| 拨备计提管理办法 | 计提标准、计算方法、审批流程 |
| 风险分类内部审计制度 | 审计范围、审计频率、结果报告 |

### 2. 系统建设

- **风险分类系统**：自动分类、逾期天数计算、交叉违约识别
- **拨备计提系统**：自动计算、情景模拟、监管报告
- **重组贷款管理系统**：重组标识、观察期跟踪、分类管控
- **不良资产管理系统**：案件管理、催收跟踪、核销审批

### 3. 数据治理

- 确保贷款基础数据的准确性和完整性
- 建立逾期天数的精确计算机制
- 对历史数据进行全面清理和核对
- 建立数据质量的定期检查和考核机制

### 4. 人员培训

- 对信贷人员进行风险分类新规的系统培训
- 对审批人员进行不良贷款识别能力培训
- 对贷后管理人员进行催收和重组操作培训
- 定期组织风险分类准确性评估

## 监管处罚案例

### 案例一：某城商行贷款风险分类不准确案（2023年）

- **违规事实**：该城商行有多笔贷款逾期超过90天但未被分类为次级及以下；对同一客户的多笔贷款进行了差异化分类（交叉违约未执行）；部分重组贷款在观察期内被上调分类。
- **处罚结果**：被罚款300万元，相关责任人分别被罚款5-10万元。
- **涉及条款**：逾期天数与分类对应关系、交叉违约认定、重组贷款管理。

### 案例二：某农商行不良贷款隐瞒案（2022年）

- **违规事实**：该农商行通过"借新还旧""展期""重组"等方式将大量不良贷款维持在"正常"或"关注"类别，实际不良贷款率远高于报表数据。经监管检查，真实不良贷款率比报表高出约3个百分点。
- **处罚结果**：被罚款500万元，行长被免职。
- **涉及条款**：五级分类标准、逾期天数对应关系。

### 案例三：某银行拨备计提不足案（2023年）

- **违规事实**：某银行在风险分类不准确的基础上，拨备计提也存在不足，拨备覆盖率仅为120%，低于150%的监管要求。
- **处罚结果**：被罚款200万元，要求限期补充拨备。
- **涉及条款**：拨备计提相关规定。

### 案例四：某城商行重组贷款违规分类案（2023年）

- **违规事实**：该城商行对多笔个人经营贷款进行重组后，未按新规要求将其至少归为次级类，而是在重组后仍维持"关注"类分类，且观察期设置不足1年。
- **处罚结果**：被罚款150万元。
- **涉及条款**：第18条（重组资产分类规则）。
