# 银行业金融机构从业人员行为管理指引

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《银行业金融机构从业人员行为管理指引》 |
| 发布机构 | 原中国银行业监督管理委员会 |
| 发布时间 | 2018年3月（银监发〔2018〕9号） |
| 配套文件 | 《关于预防银行业保险业从业人员金融违法犯罪的指导意见》（2020年） |
| 文号 | 银监发〔2018〕9号 |
| 效力层级 | 规范性文件 |

《银行业金融机构从业人员行为管理指引》是银行业从业人员行为规范的基础性文件，旨在加强银行业金融机构对从业人员行为的管理，防范从业人员违法违规行为，维护银行业的声誉和金融秩序。该指引对零售业务中的柜面操作、产品销售、客户服务等环节具有重要的规范作用。

## 核心内容摘要

### 一、从业人员行为管理的基本要求

- **第5条**：银行业金融机构应当建立从业人员行为管理体系，明确行为管理的组织架构、制度建设、信息系统、监督评价等内容。
- **第6条**：从业人员应当遵守法律法规、监管规定、行业自律规则以及本机构的内部规章制度。

### 二、禁止行为

| 类别 | 禁止行为 |
|------|----------|
| 违法违规 | 参与非法集资、洗钱、内幕交易等违法犯罪活动 |
| 利益冲突 | 利用职务便利谋取不正当利益、收受回扣 |
| 信息安全 | 泄露客户信息、商业秘密、内部文件 |
| 职业操守 | 弄虚作假、伪造文件、虚假报告 |
| 市场秩序 | 恶意竞争、诋毁同业、虚假宣传 |
| 个人品行 | 参与赌博、吸毒、嫖娼等违法活动 |
| 投资行为 | 违规经商办企业、违规持有非上市公司股份 |

### 三、内部管控措施

- **第10条**：银行业金融机构应当建立从业人员行为监测制度，对从业人员的行为进行日常监测和异常排查。
- **第11条**：银行业金融机构应当建立从业人员行为评估制度，定期对从业人员的行为进行评估。
- **第12条**：银行业金融机构应当建立从业人员行为投诉和举报制度，鼓励内部举报违法违规行为。

### 四、问责机制

- **第15条**：银行业金融机构应当建立从业人员违规行为的责任追究制度，明确违规行为的认定标准和处罚措施。
- **第16条**：对于涉嫌违法犯罪的从业人员，应当及时移送司法机关处理。

### 五、行为排查重点

根据监管要求，从业人员行为排查应重点关注：
- 账户异常交易（大额资金进出、频繁转账等）
- 异常消费行为（消费水平与收入明显不匹配）
- 异常社交行为（与不良中介、高利贷人员往来密切）
- 异常工作行为（非工作时间频繁进入办公区域、越权操作等）
- 个人征信异常（多头借贷、大量信用卡透支等）

## 零售业务相关条款解读

### （一）柜面人员行为规范

**1. 现金操作规范**

柜面人员在处理现金业务时应遵守：
- 严格执行"一笔一清"原则，不得将不同客户的业务混淆
- 现金收付须当面点清，不得离开客户视线
- 大额现金收付须复核
- 不得私自挪用、侵占客户资金
- 不得利用柜面操作便利进行"空存实取"或"实存空取"

**2. 客户信息保护**

- 不得向任何无关人员透露客户的存款、贷款、投资等信息
- 不得利用工作便利查询与自己业务无关的客户信息
- 不得将客户信息拍照、截屏或带出工作场所
- 不得在公共场合讨论客户信息
- 离职后仍负有客户信息保密义务

**3. 销售行为规范**

- 不得强制或诱导客户购买不适合的产品
- 不得代替客户签名或操作
- 不得在销售过程中承诺保本保收益
- 不得销售未经批准的第三方产品（"飞单"）
- 不得在营业场所从事与银行业务无关的商业活动

### （二）客户经理行为规范

**1. 贷前调查规范**

- 实地调查须两人同行
- 不得接受客户的宴请、礼品、礼金
- 不得编造或篡改借款人的调查资料
- 不得与借款人串通骗取贷款
- 不得隐瞒借款人的重大风险事项

**2. 贷后管理规范**

- 贷后检查须真实、有效，不得走过场
- 不得帮助借款人掩盖贷款挪用行为
- 不得与借款人串通逃避还款责任
- 不得私自向借款人收取费用
- 不得泄露贷后检查中获取的信息

**3. 催收行为规范**

- 催收应当依法进行，不得采用暴力、威胁等非法手段
- 不得向无关第三人透露借款人的债务信息
- 不得在非工作时间进行骚扰性催收
- 不得私自减免借款人的利息或本金
- 不得与借款人达成私下的还款协议

### （三）理财销售人员行为规范

**1. 适当性管理**

- 必须对客户进行风险承受能力评估
- 不得向风险等级不匹配的客户推荐产品
- 不得以"内部产品""特供产品"等名义误导客户
- 不得代替客户完成风险评估

**2. 信息披露**

- 必须向客户充分披露产品风险
- 不得夸大产品收益或隐瞒产品风险
- 不得将理财产品与存款混淆宣传
- 不得使用"保本""零风险"等误导性用语

**3. 售后服务**

- 定期向客户通报产品净值变化
- 产品出现重大风险时及时通知客户
- 不得在客户不知情的情况下进行产品转换
- 不得诱导客户频繁赎回和购买产品

### （四）管理人员行为规范

**1. 授信管理**

- 不得违规干预贷款审批
- 不得强迫下属发放不符合条件的贷款
- 不得越权审批贷款
- 不得为关系人提供不当便利

**2. 费用管理**

- 不得私设"小金库"
- 不得违规发放奖金或补贴
- 不得将业务费用转嫁给客户
- 不得虚报业务数据套取费用

**3. 人事管理**

- 不得任人唯亲、打击报复
- 不得违规安排亲属在本行任职
- 不得在员工考核中弄虚作假
- 不得违反回避制度

### （五）从业人员行为监测

**1. 账户监测**

银行应当对本行从业人员的银行账户进行监测：
- 关注异常大额资金进出
- 关注与客户的资金往来
- 关注与不良中介的资金往来
- 关注多头借贷和过度负债

**2. 行为排查**

定期开展从业人员行为排查：
- 每半年至少开展一次全面排查
- 对关键岗位人员开展专项排查
- 对异常行为及时进行调查核实
- 对查实的违规行为严肃处理

**3. 八小时以外行为管理**

- 关注从业人员的社交圈和生活圈
- 了解从业人员的兴趣爱好和消费习惯
- 对参与高风险投资、赌博等行为的从业人员进行重点关注
- 建立从业人员行为异常的预警机制

## 对城商行/农商行的影响

### 1. 人员管理的特殊挑战

城商行、农商行在从业人员行为管理方面面临特殊挑战：
- 员工队伍素质参差不齐
- 地方人情社会特征明显，关系人管理难度大
- 基层网点分散，行为监督覆盖难度大
- 员工流动性大，行为管理连续性不足

### 2. 行为管理体系建设

- 建立适合自身规模的行为管理组织架构
- 制定覆盖全员的行为管理制度和操作规范
- 部署从业人员行为监测信息系统
- 建立行为管理的培训和宣传机制

### 3. 重点风险防控

- 防范柜面操作风险（挪用资金、空存实取等）
- 防范信贷领域道德风险（内外勾结、虚假贷款等）
- 防范销售领域违规行为（误导销售、飞单等）
- 防范信息安全风险（信息泄露、倒卖客户信息等）

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 从业人员行为管理办法 | 管理框架、职责分工、工作流程 |
| 从业人员禁止行为清单 | 明确列举各类禁止行为 |
| 从业人员行为排查办法 | 排查范围、排查方式、处理流程 |
| 从业人员违规行为问责办法 | 违规认定、处罚标准、申诉程序 |
| 从业人员举报和投诉管理办法 | 举报渠道、保护措施、奖励机制 |
| 关键岗位人员管理制度 | 轮岗制度、强制休假、行为评估 |

### 2. 系统建设

- **行为监测系统**：员工账户监测、异常行为预警、行为评分
- **征信监测系统**：员工征信定期查询、负债监控
- **操作风险监控系统**：操作行为审计、越权操作预警
- **举报管理系统**：举报受理、调查处理、结果反馈

### 3. 培训教育

- 新员工入职时进行职业道德和行为规范培训
- 每年至少开展一次全员行为管理培训
- 定期通报违规行为典型案例
- 建立行为管理的文化宣传机制

### 4. 监督评价

- 内部审计部门定期对行为管理进行审计
- 行为管理工作纳入各部门和网点的绩效考核
- 建立行为管理的定期报告制度
- 配合监管机构开展从业人员行为检查

## 监管处罚案例

### 案例一：某农商行员工挪用客户存款案（2022年）

- **违规事实**：该农商行某支行柜员利用职务便利，在办理客户存取款业务时挪用客户存款约300万元，用于网络赌博。该柜员的异常行为长期未被发现。
- **处罚结果**：银行被罚款200万元，涉事柜员被判处有期徒刑10年，支行行长和分管副行长被取消任职资格。
- **涉及条款**：从业人员禁止行为、行为监测制度。

### 案例二：某城商行客户经理"飞单"案（2023年）

- **违规事实**：该城商行某客户经理在银行营业场所内向客户推荐和销售第三方私募基金产品，累计销售金额约1500万元。基金爆雷后客户损失严重，银行声誉受损。
- **处罚结果**：银行被罚款300万元，涉事客户经理被开除并移送司法机关。
- **涉及条款**：从业人员禁止行为（飞单）。

### 案例三：某银行员工泄露客户信息案（2021年）

- **违规事实**：某银行信息科技部门员工利用系统管理员权限，批量导出客户个人信息约10万条，出售给第三方营销公司。
- **处罚结果**：银行被罚款150万元，涉事员工以侵犯公民个人信息罪被判处有期徒刑2年。
- **涉及条款**：信息安全禁止行为。

### 案例四：某农商行员工行为排查失职案（2023年）

- **违规事实**：该农商行连续三年未开展从业人员行为排查，多名员工存在参与民间借贷、经商办企业等违规行为未被发现。
- **处罚结果**：被罚款100万元，分管人力资源的副行长被给予警告处分。
- **涉及条款**：行为监测和排查制度。
