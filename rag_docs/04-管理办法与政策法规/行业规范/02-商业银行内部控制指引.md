# 商业银行内部控制指引

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《商业银行内部控制指引》 |
| 发布机构 | 原中国银行业监督管理委员会 |
| 发布时间 | 2014年9月12日（银监发〔2014〕40号） |
| 文号 | 银监发〔2014〕40号 |
| 效力层级 | 规范性文件 |

《商业银行内部控制指引》是银行内控体系建设的基本规范，确立了内部控制的五要素框架（内部环境、风险评估、控制活动、信息与沟通、内部监督），明确了商业银行各业务条线的关键控制要求。对于零售业务而言，内部控制的有效性直接关系到操作风险、合规风险和声誉风险的防控水平。

## 核心内容摘要

### 一、内部控制五要素

| 要素 | 内容 | 在零售业务中的体现 |
|------|------|-------------------|
| 内部环境 | 公司治理、组织架构、企业文化、人力资源 | 零售业务组织架构、岗位职责、人员管理 |
| 风险评估 | 风险识别、风险分析、风险应对 | 零售业务风险识别和评估机制 |
| 控制活动 | 授权审批、职责分离、实物控制 | 业务授权、审贷分离、印章管理 |
| 信息与沟通 | 信息系统、报告机制、沟通渠道 | 业务系统、报表体系、上下沟通 |
| 内部监督 | 持续监督、专项监督、缺陷报告 | 内部审计、合规检查、问题整改 |

### 二、内部控制目标

- **第4条**：商业银行内部控制的目标包括：
  - （一）保证国家有关法律规定及规章的贯彻执行
  - （二）保证商业银行发展战略和经营目标的实现
  - （三）保证商业银行风险管理的有效性
  - （四）保证商业银行业务记录、会计信息、财务信息和其他管理信息的真实、准确、完整和及时

### 三、关键控制措施

- **第14条**：商业银行应当建立健全内部控制制度体系，对各项业务活动和管理活动制定全面、系统、规范的业务制度和管理制度。
- **第15条**：商业银行应当合理确定各项业务活动和管理活动的风险控制点，采取适当的控制措施，执行标准化的业务操作流程。

### 四、职责分离要求

- **第18条**：商业银行应当根据各分支机构和各部门的经营能力、管理水平、风险状况和业务发展需要，建立相应的授权体系。
- **第19条**：商业银行应当明确关键岗位及其控制要求，关键岗位应当实行定期或不定期的人员轮换和强制休假制度。

## 零售业务相关条款解读

### （一）零售业务的内部环境

**1. 组织架构**

零售业务的组织架构应当遵循"职责分离、相互制衡"的原则：
- 前台（营销和客户服务）与中台（风险管理和合规）分离
- 业务发起与业务审批分离
- 资金管理与业务操作分离
- 信息系统开发与业务操作分离

**2. 岗位职责**

零售业务关键岗位的职责应当清晰明确：
- 柜员岗位：现金收付、账户管理、客户服务的操作职责
- 客户经理岗位：贷前调查、客户关系维护、产品推介的营销职责
- 审批人员岗位：贷款审查审批的独立审批职责
- 合规人员岗位：业务合规审查、制度执行监督的合规职责
- 审计人员岗位：业务审计、风险评估、整改跟踪的审计职责

**3. 人力资源**

- 关键岗位人员应当具备相应的专业能力和从业资格
- 建立关键岗位人员的定期轮岗制度（一般不超过3年）
- 实行强制休假制度（每年不少于5个工作日）
- 建立关键岗位人员的离任审计制度

### （二）零售业务的风险评估

**1. 风险识别**

零售业务的主要风险类型：

| 风险类型 | 主要表现 | 控制重点 |
|----------|----------|----------|
| 操作风险 | 柜面差错、系统故障、内部欺诈 | 流程规范、系统控制、人员管理 |
| 信用风险 | 贷款违约、信用卡坏账 | 授信审批、贷后管理、催收 |
| 合规风险 | 违规销售、信息披露不充分 | 合规审查、培训教育、监督检查 |
| 声誉风险 | 客户投诉、媒体曝光 | 投诉处理、舆情监控、危机公关 |
| 信息安全风险 | 客户信息泄露、网络攻击 | 信息安全制度、技术防护、应急响应 |

**2. 风险评估方法**

- 定性与定量相结合
- 建立零售业务风险指标体系
- 定期开展风险评估（至少每年一次）
- 对新业务、新产品进行专项风险评估

### （三）零售业务的关键控制活动

**1. 存款业务控制**

- 开户审批控制：实名制核验、授权审批
- 现金管理控制：大额现金审批、现金盘点、库款限额
- 账户管理控制：账户冻结/解冻审批、销户审批
- 利率控制：利率定价授权、异常利率监控

**2. 贷款业务控制**

- 授信审批控制：审贷分离、分级审批、独立审批
- 担保管理控制：抵押物评估、担保合同签署、抵质押登记
- 放款控制：放款条件审核、资金划转审批
- 贷后管理控制：定期检查、风险预警、逾期催收

**3. 理财销售控制**

- 适当性控制：客户风险评估、产品风险匹配
- 双录控制：销售过程录音录像、资料保存
- 信息披露控制：产品说明书签署、风险揭示书确认
- 交易控制：交易金额限制、交易确认机制

**4. 银行卡业务控制**

- 发卡控制：申请审核、身份核实、卡片制发
- 授信控制：额度审批、调额审批、降额处理
- 交易控制：交易限额、异常交易监控、欺诈检测
- 催收控制：催收授权、催收行为规范、诉讼管理

### （四）信息与沟通

**1. 业务系统控制**

- 系统权限管理：基于角色的访问控制（RBAC）
- 操作日志管理：完整记录系统操作行为
- 参数管理：关键参数的变更须经审批
- 系统变更管理：变更申请、测试、上线的全流程管控

**2. 管理报告**

- 建立零售业务经营分析的定期报告制度
- 建立风险事件的及时报告机制
- 建立合规问题的专项报告制度
- 建立客户投诉的统计分析报告制度

**3. 沟通机制**

- 建立总行-分行-支行的纵向沟通机制
- 建立零售业务与风险管理、合规、审计等部门的横向沟通机制
- 建立与监管机构的定期沟通机制
- 建立客户反馈和投诉的接收处理机制

### （五）内部监督

**1. 持续监督**

- 各级管理人员对日常业务操作的监督
- 合规部门对业务合规性的持续监控
- 风险管理部门对风险指标的持续监控
- 运营管理部门对操作规范的持续检查

**2. 专项监督**

- 内部审计部门定期对零售业务进行专项审计
- 合规部门定期开展合规检查
- 对重点业务和风险高发领域进行专项检查
- 对关键岗位人员的操作行为进行抽查

**3. 内部控制评价**

- 每年至少开展一次全面的内部控制评价
- 评价范围应覆盖零售业务的所有关键环节
- 评价结果应向董事会和高级管理层报告
- 对发现的内控缺陷制定整改计划并跟踪落实

## 对城商行/农商行的影响

### 1. 内控体系建设的差距

城商行、农商行在内控体系建设方面存在常见差距：
- 内控制度不完善，部分业务环节缺少控制措施
- 制度执行不到位，"写在纸上、挂在墙上"的现象
- 职责分离不充分，一人多岗、交叉操作的情况
- 信息系统控制薄弱，系统权限管理不规范

### 2. 关键控制点的强化

- 加强柜面操作的标准化和系统化控制
- 完善信贷业务的审贷分离和分级审批
- 强化理财销售的适当性管理和双录管理
- 提升信息安全的技术防护措施

### 3. 内审能力的提升

- 建设专业化的内部审计团队
- 提升内部审计的独立性和权威性
- 加强审计方法和技术的创新
- 建立审计发现问题的整改跟踪机制

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 内部控制管理办法 | 内控框架、职责分工、评价机制 |
| 零售业务操作手册 | 各业务条线的标准操作流程 |
| 授权管理办法 | 授权体系、权限设置、动态调整 |
| 关键岗位管理办法 | 轮岗制度、强制休假、离任审计 |
| 内部控制评价办法 | 评价标准、评价方法、结果运用 |
| 内部控制缺陷管理办法 | 缺陷认定、整改计划、跟踪考核 |

### 2. 系统支持

- 在业务系统中嵌入关键控制点（审批流、限额控制、授权校验）
- 建立操作行为的系统日志记录和审计功能
- 部署异常操作的自动检测和预警功能
- 建设内部控制的信息化管理平台

### 3. 持续改进

- 定期评估内部控制的有效性
- 根据业务发展和风险变化及时更新控制措施
- 对内部控制缺陷进行根因分析和系统性整改
- 建立内部控制的最佳实践分享机制

## 监管处罚案例

### 案例一：某城商行内部控制失效导致重大操作风险案（2022年）

- **违规事实**：该城商行某支行多名员工串通，利用内部控制漏洞，通过伪造存款凭证、虚构贷款等方式挪用资金约5000万元。审计发现，该支行存在柜员兼任授权员、印章管理混乱、重要空白凭证管理失控等内控缺陷。
- **处罚结果**：银行被罚款600万元，多名涉事员工被移送司法机关。
- **涉及条款**：第19条（职责分离）、第20条（实物控制）。

### 案例二：某农商行内部审计缺失案（2023年）

- **违规事实**：该农商行连续3年未对零售业务进行内部审计，内审部门仅有1名兼职人员，无法有效履行监督职责。
- **处罚结果**：被罚款80万元，责令限期整改。
- **涉及条款**：第25条（内部监督）。

### 案例三：某银行系统权限管理违规案（2022年）

- **违规事实**：某银行的信贷管理系统存在严重权限管理问题，多名已离职或调岗人员的系统权限未及时回收，导致部分人员可以越权操作。
- **处罚结果**：被罚款120万元。
- **涉及条款**：第18条（授权管理）、第22条（信息系统控制）。

### 案例四：某城商行关键岗位未轮岗案（2023年）

- **违规事实**：该城商行多名关键岗位人员（包括信贷审批人员、资金管理人员）在同一岗位工作超过5年未轮岗，违反了关键岗位定期轮岗的要求。
- **处罚结果**：被罚款50万元。
- **涉及条款**：第19条（关键岗位管理）。
