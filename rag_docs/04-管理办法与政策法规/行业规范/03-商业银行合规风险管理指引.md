# 商业银行合规风险管理指引

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《商业银行合规风险管理指引》 |
| 发布机构 | 原中国银行业监督管理委员会 |
| 发布时间 | 2006年10月25日（银监发〔2006〕91号） |
| 文号 | 银监发〔2006〕91号 |
| 效力层级 | 规范性文件 |

《商业银行合规风险管理指引》是我国银行业合规管理体系建设的基础性文件，明确了合规风险管理的组织架构、制度体系、运行机制和文化建设要求。该指引要求商业银行建立"三道防线"的合规管理架构，将合规管理融入各项业务的全流程。对于零售业务而言，合规管理的有效性直接关系到业务的可持续发展和风险防控水平。

## 核心内容摘要

### 一、合规风险的定义

- **第3条**：合规风险是指商业银行因没有遵循法律、规则和准则可能遭受法律制裁、监管处罚、重大财务损失和声誉损失的风险。

### 二、合规管理组织架构

- **第8条**：商业银行应当建立合规管理组织架构，包括董事会、高级管理层、合规管理部门、业务部门等各层级的合规管理职责。
- **第9条**：董事会应当对商业银行的合规管理承担最终责任。
- **第10条**：高级管理层应当有效管理商业银行的合规风险。

### 三、"三道防线"模型

| 防线 | 责任主体 | 核心职能 |
|------|----------|----------|
| 第一道防线 | 业务部门 | 业务操作中的合规执行和自我管理 |
| 第二道防线 | 合规管理部门、风险管理部门 | 合规制度建设、监督审查、风险评估 |
| 第三道防线 | 内部审计部门 | 合规审计、独立评价、缺陷跟踪 |

### 四、合规管理制度体系

- **第14条**：商业银行应当建立合规管理的基本制度体系，包括合规政策、合规手册、合规审查程序、合规培训制度、合规举报制度等。
- **第15条**：商业银行应当建立合规风险的识别、评估、监测和报告程序。

### 五、合规文化建设

- **第6条**：商业银行应当树立"合规从高层做起""合规创造价值"的合规文化理念。
- **第7条**：商业银行应当将合规文化建设纳入企业文化建设体系。

## 零售业务相关条款解读

### （一）零售业务合规管理框架

**1. 合规管理职责分工**

零售业务合规管理应当明确三道防线的职责：

**第一道防线（零售业务部门）**：
- 在业务操作中严格执行合规要求
- 对本部门业务的合规性进行自我检查
- 及时报告合规问题和风险事件
- 配合合规部门开展合规审查

**第二道防线（合规管理部门）**：
- 制定零售业务的合规管理制度和操作规范
- 对零售新产品和新业务进行合规审查
- 对零售业务的合规执行情况进行监督检查
- 跟踪监管法规变化，及时更新合规要求

**第三道防线（内部审计部门）**：
- 对零售业务的合规管理体系进行独立评价
- 对零售业务的关键环节进行合规审计
- 对合规问题的整改情况进行跟踪审计
- 向董事会和高级管理层报告审计结果

**2. 合规管理的关键环节**

零售业务合规管理的重点环节包括：

| 环节 | 合规要点 | 常见风险 |
|------|----------|----------|
| 产品设计 | 产品合规审查、消费者保护评估 | 产品条款不合规、侵害消费者权益 |
| 营销推广 | 宣传材料审核、销售行为规范 | 虚假宣传、误导销售 |
| 业务办理 | 操作规程执行、身份核验 | 操作违规、身份造假 |
| 收费管理 | 收费公示、禁止乱收费 | 违规收费、搭售产品 |
| 信息披露 | 信息完整准确、披露及时 | 信息不完整、披露不及时 |
| 投诉处理 | 受理及时、处理公正 | 投诉不受理、处理不公平 |

### （二）新产品新业务合规审查

**1. 合规审查的范围**

零售业务中的新产品和新业务上线前须进行合规审查：
- 新开发的存款、贷款、理财、信用卡等产品
- 新的业务渠道（如新的互联网渠道、第三方合作渠道）
- 新的业务模式（如联合贷款、助贷等）
- 对现有产品的重大修改（如调整利率、修改合同条款）

**2. 合规审查的内容**

- 产品或业务是否符合现行法律法规和监管要求
- 产品条款是否公平合理，是否侵害消费者权益
- 业务流程是否设置了必要的合规控制点
- 信息披露是否充分、准确
- 风险揭示是否到位
- 反洗钱和反恐融资措施是否落实

**3. 合规审查的流程**

- 业务部门提交合规审查申请及相关材料
- 合规部门进行合规审查并出具审查意见
- 对存在合规问题的，提出修改意见或否决建议
- 合规审查意见作为产品上线的必要条件

### （三）合规监测与报告

**1. 合规监测指标**

零售业务的合规监测指标包括：
- 合规检查发现问题的数量和分布
- 监管处罚的次数和金额
- 客户投诉的数量和类型
- 合规培训的覆盖率和效果
- 制度更新的及时性和完整性

**2. 合规报告机制**

- **定期报告**：按月或按季度向管理层报告合规状况
- **专项报告**：对重大合规事件进行专项报告
- **监管报告**：按监管要求向监管机构报送合规信息
- **即时报告**：对突发合规事件进行即时上报

**3. 合规事件处理**

- 建立合规事件的分级分类管理
- 对合规事件进行调查、分析和处理
- 制定合规事件的整改和预防措施
- 建立合规事件的问责机制

### （四）合规培训与教育

**1. 培训体系**

- **新员工培训**：入职时进行合规基础知识和职业道德培训
- **岗位培训**：针对不同岗位的合规专项培训
- **专项培训**：针对新法规、新业务的专项合规培训
- **高管培训**：面向高级管理层的合规管理培训

**2. 培训内容**

- 法律法规和监管政策解读
- 内部规章制度学习
- 典型案例分析
- 合规操作实务

**3. 培训考核**

- 每年至少开展一次合规知识考核
- 考核结果与员工绩效挂钩
- 对考核不合格的进行补训和补考
- 建立合规培训的档案记录

### （五）合规文化建设

**1. 合规文化理念**

- "合规从高层做起"：管理层率先垂范
- "合规人人有责"：全员参与合规管理
- "合规创造价值"：合规经营是可持续发展的基础
- "主动合规"：从被动应付到主动预防

**2. 合规文化建设措施**

- 定期开展合规文化宣传活动
- 评选和表彰合规标兵
- 建立合规案例分享机制
- 将合规文化融入企业文化建设

## 对城商行/农商行的影响

### 1. 合规管理体系建设的挑战

城商行、农商行在合规管理体系建设方面面临的主要挑战：
- 合规管理人才匮乏，专业水平有待提升
- 合规管理制度不完善，覆盖面不足
- 合规管理手段落后，信息化水平不高
- 合规文化建设薄弱，"重业务、轻合规"的思想仍有市场

### 2. 合规管理资源的配置

- 合理配置合规管理人员（不低于全行员工总数的2%）
- 在零售业务部门配备专职或兼职合规联络员
- 加大合规管理信息系统的建设投入
- 建立合规管理的外脑资源（律师事务所、咨询公司）

### 3. 监管沟通与应对

- 建立与当地监管机构的常态化沟通机制
- 及时了解监管政策动态和监管重点
- 积极配合监管检查和调查
- 对监管意见和监管要求认真整改落实

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 合规管理政策 | 合规管理框架、职责分工、基本原则 |
| 合规审查管理办法 | 审查范围、审查流程、审查标准 |
| 合规风险监测和报告制度 | 监测指标、报告频率、报告路径 |
| 合规培训管理制度 | 培训对象、培训内容、考核标准 |
| 合规举报和问责制度 | 举报渠道、保护机制、问责标准 |
| 合规文化建设方案 | 建设目标、实施措施、效果评估 |

### 2. 系统建设

- **合规管理信息系统**：制度管理、合规检查、问题整改、培训管理
- **法规库系统**：法规收录、变化追踪、影响分析
- **合规审查系统**：审查流程管理、审查意见跟踪
- **合规监测系统**：指标监测、预警管理、报告生成

### 3. 评价与改进

- 每年开展一次合规管理体系有效性评价
- 评价结果纳入管理层绩效考核
- 对合规管理体系的不足制定改进计划
- 持续跟踪合规管理体系的改进效果

## 监管处罚案例

### 案例一：某城商行合规管理失效案（2023年）

- **违规事实**：该城商行合规管理部门形同虚设，合规审查流于形式；多项新产品未经合规审查即上线销售；合规部门人员严重不足（仅2人），无法有效履行职责。
- **处罚结果**：被罚款200万元，合规总监被免职。
- **涉及条款**：第8-10条（合规管理组织架构）。

### 案例二：某农商行合规培训缺失案（2022年）

- **违规事实**：该农商行连续两年未组织全员合规培训，新员工入职未进行合规培训，一线销售人员对消费者权益保护的基本要求不了解。
- **处罚结果**：被罚款50万元。
- **涉及条款**：第14条（合规培训制度）。

### 案例三：某银行"三道防线"失效案（2023年）

- **违规事实**：某银行的"三道防线"在零售业务管理中全部失效：第一道防线（业务部门）违规操作未被发现，第二道防线（合规部门）未开展有效检查，第三道防线（内审部门）连续三年未审计零售业务。
- **处罚结果**：被罚款400万元，多名高管被问责。
- **涉及条款**："三道防线"合规管理架构。

### 案例四：某城商行合规文化缺失案（2022年）

- **违规事实**：该城商行管理层重业务轻合规，将合规管理视为"阻碍业务发展"的因素；业务部门为完成考核指标，大量存在违规销售、违规收费等行为。
- **处罚结果**：被罚款250万元，多名高管被监管谈话。
- **涉及条款**：第6-7条（合规文化建设）。
