# 银行营业网点服务规范

## 法规概述

| 项目 | 内容 |
|------|------|
| 法规全称 | 《银行业营业网点文明规范服务评价指标体系和评分标准》及《银行营业网点服务基本要求》（GB/T 32320-2015） |
| 发布机构 | 中国银行业协会（行业标准）、国家标准化管理委员会（国家标准） |
| 发布时间 | 国家标准2015年12月；中银协评价标准2019年修订 |
| 配套文件 | 《关于银行保险机构切实解决老年人运用智能技术困难的通知》（2021年） |
| 效力层级 | 国家标准（推荐性）+ 行业自律规范 |

银行营业网点服务规范是银行业服务质量管理的基本框架，涵盖了网点环境、服务流程、人员素质、设施设备、投诉处理等多个方面。近年来，随着适老化服务和无障碍设施建设的推进，银行网点服务规范又增加了新的内涵。对于城商行、农商行而言，网点是零售业务的核心渠道，服务规范的执行水平直接影响客户体验和市场竞争力。

## 核心内容摘要

### 一、网点服务基本要求

| 维度 | 具体要求 |
|------|----------|
| 网点环境 | 整洁明亮、布局合理、标识清晰、安全舒适 |
| 服务设施 | 功能齐全、运行正常、便于使用 |
| 服务流程 | 规范高效、公开透明、便捷利民 |
| 人员素质 | 持证上岗、礼貌待客、专业规范 |
| 信息管理 | 信息公开、保护隐私、安全合规 |
| 投诉处理 | 渠道畅通、处理及时、反馈有效 |

### 二、网点功能分区

- **现金服务区**：办理现金存取、兑换等业务
- **非现金服务区**：办理开户、转账、理财等非现金业务
- **自助服务区**：ATM、自助终端等自助设备
- **客户等候区**：客户等候休息区域
- **贵宾服务区**：为高端客户提供的专属服务区域
- **理财专区**：理财产品咨询和销售（须配备双录设备）

### 三、服务时间管理

- 营业网点应当在显著位置公示营业时间
- 营业时间内不得提前停止办理业务
- 对营业结束前到达的客户应当提供服务
- 临时调整营业时间应提前公告

### 四、适老化服务要求

根据《关于银行保险机构切实解决老年人运用智能技术困难的通知》：
- 保留传统柜面服务渠道，不得强制老年人使用自助设备
- 为老年人提供大字版、语音版等适老化智能终端产品
- 配备老花镜、放大镜等辅助工具
- 设置老年人优先服务窗口
- 提供上门服务渠道

### 五、无障碍设施要求

- 设置无障碍通道或坡道
- 配备盲文键盘或语音提示设备
- 提供轮椅等辅助器具
- 设置低位服务柜台
- 配备手语服务或翻译服务

## 零售业务相关条款解读

### （一）柜面服务规范

**1. 服务礼仪**

- 迎接客户：主动起身迎接、微笑问候、询问需求
- 业务办理：双手接递、快速操作、准确无误
- 业务完成：双手递交回单、提醒注意事项、致谢送别
- 称呼规范：使用"先生""女士"等尊称，避免直呼姓名

**2. 业务操作规范**

- 现金收付：当面点清、唱收唱付
- 凭证填写：指导客户正确填写，不得代替客户填写
- 密码设置：由客户自行设置，银行人员不得询问或记录
- 签名确认：确认客户身份后由客户本人签名

**3. 特殊情况处理**

- 客户投诉：先安抚客户情绪，认真倾听客户诉求，记录投诉内容
- 系统故障：向客户说明情况，预估恢复时间，提供替代方案
- 业务纠纷：及时报告上级管理人员，按流程妥善处理
- 突发事件：启动应急预案，保障客户人身和财产安全

### （二）自助设备服务规范

**1. ATM服务**

- 保持ATM设备正常运行，故障率不超过2%
- 设备故障时及时张贴提示并报告维修
- 保持ATM区域清洁、安全
- 在ATM区域设置安全提示和防诈骗提示
- 24小时提供ATM服务（条件允许的情况下）

**2. 智能终端服务**

- 安排专人引导客户使用智能终端
- 不得强制客户使用智能终端替代柜面服务
- 为不熟悉操作的客户提供全程指导
- 保护客户在操作过程中的隐私

**3. 自助设备安全管理**

- 安装监控设备，覆盖自助设备使用区域
- 定期检查设备是否被安装侧录器或针孔摄像头
- 在设备旁设置安全使用提示
- 建立设备异常交易的监测和预警机制

### （三）理财销售专区规范

**1. 专区设置**

- 在营业网点设置独立的理财销售专区
- 专区内配备双录设备（录音录像）
- 专区显著位置展示"理财销售专区"标识
- 专区内展示理财产品的风险提示信息

**2. 双录管理**

- 理财产品销售全过程须在双录专区内进行
- 录音录像应清晰记录销售人员的风险揭示和客户的确认
- 双录资料保存期限不少于产品到期后6个月
- 定期对双录质量进行检查和评估

**3. 信息展示**

- 在专区内展示在售理财产品的信息
- 展示投资者风险提示和教育材料
- 展示银行理财业务资质信息
- 展示投诉渠道和联系方式

### （四）适老化服务详解

**1. 物理环境适老化**

- 网点入口设置无障碍坡道和扶手
- 地面防滑处理，设置防滑垫
- 等候区配备舒适的座椅（有扶手和靠背）
- 柜台设置低位服务窗口
- 提供老花镜、放大镜、轮椅等辅助工具
- 卫生间设置无障碍设施

**2. 服务流程适老化**

- 设置老年人优先服务窗口或绿色通道
- 为行动不便的老年人提供引导和陪同服务
- 保留传统纸质业务办理方式
- 对老年人进行耐心细致的业务指导
- 提供大字版业务凭证和宣传资料
- 为不会签名的老年人提供按手印等替代方式

**3. 数字服务适老化**

- 手机银行推出"长辈版"或"关怀版"
- 界面字体放大、功能简化、操作便捷
- 提供语音导航和语音输入功能
- 在网点配备专人协助老年人使用智能设备
- 不得强制老年人通过线上渠道办理业务

**4. 上门服务**

- 对行动不便的老年客户提供上门服务
- 上门服务须两人同行
- 上门服务须进行身份核实和录音录像
- 上门服务的内容和流程须符合内控要求

### （五）无障碍服务详解

**1. 视障人士服务**

- 配备盲文业务指南和表单
- ATM配备语音导航功能
- 提供工作人员全程引导和协助
- 允许导盲犬进入营业场所

**2. 听障人士服务**

- 配备手语服务人员或翻译服务
- 提供文字交流工具
- 柜台配备书写板或电子设备
- 叫号系统配备屏幕显示功能

**3. 肢体残障人士服务**

- 设置无障碍通道和坡道
- 设置低位柜台和服务设施
- 提供轮椅等辅助设备
- 工作人员主动提供帮助

### （六）投诉处理规范

**1. 现场投诉处理**

- 第一时间安抚客户情绪
- 将客户引导至相对私密的区域
- 认真倾听客户诉求并做好记录
- 能够当场解决的问题立即解决
- 不能当场解决的，承诺处理时限

**2. 投诉跟踪**

- 建立投诉台账，记录投诉内容和处理进度
- 在承诺时限内完成投诉处理
- 处理完成后及时回复客户
- 对投诉处理结果进行客户满意度回访

**3. 投诉分析**

- 定期统计分析投诉数据
- 识别投诉热点和共性问题
- 分析投诉产生的根本原因
- 制定改进措施并向管理层报告

## 对城商行/农商行的影响

### 1. 网点服务标准化建设

城商行、农商行的网点服务标准化水平参差不齐：
- 部分老旧网点硬件设施不达标
- 服务意识和专业技能有待提升
- 服务流程不够规范和标准化
- 适老化和无障碍设施投入不足

### 2. 差异化竞争策略

网点服务是城商行、农商行与大行竞争的重要差异化手段：
- 发挥"地缘优势"，提供更贴近社区的服务
- 建立"有温度"的服务品牌
- 在网点布局上贴近目标客户群体
- 提供大行难以覆盖的个性化服务

### 3. 适老化服务的重点投入

城商行、农商行的客户群体中老年人占比较高：
- 适老化服务是必答题而非选择题
- 需要在硬件设施和软件服务两方面加大投入
- 适老化服务水平直接影响老年客户的留存
- 适老化服务是社会责任的重要体现

### 4. 农村网点的特殊要求

农商行在农村地区的网点需要：
- 考虑农村客户的特殊需求和习惯
- 提供更加通俗易懂的业务说明
- 配备能够使用方言服务的工作人员
- 在赶集日等特殊时段提供延时服务

## 合规实施要点

### 1. 制度建设

| 制度名称 | 关键内容 |
|----------|----------|
| 营业网点服务管理办法 | 服务标准、服务流程、服务考核 |
| 营业网点环境管理规范 | 环境卫生、功能分区、设施维护 |
| 客户投诉管理办法 | 投诉受理、处理流程、跟踪反馈 |
| 适老化服务管理办法 | 适老化设施、服务流程、人员培训 |
| 无障碍服务管理规范 | 无障碍设施、特殊服务、人员培训 |
| 营业网点应急管理制度 | 应急预案、演练安排、处置流程 |

### 2. 硬件建设

- 按照标准对现有网点进行功能分区改造
- 配备和完善自助设备、智能终端
- 建设和完善双录系统
- 配置适老化和无障碍设施
- 安装安防监控设备

### 3. 服务培训

- 新入职员工的服务礼仪和规范培训
- 全员服务意识和服务技能定期培训
- 适老化服务和无障碍服务专项培训
- 投诉处理技巧和情景模拟训练
- 服务明星评选和技能竞赛

### 4. 监督检查

- 建立网点服务质量定期检查制度
- 开展神秘客户检查（暗访）
- 实施客户满意度调查
- 将服务质量纳入网点和员工的绩效考核
- 对服务标兵进行表彰和奖励

## 监管处罚案例

### 案例一：某城商行网点服务态度恶劣案（2022年）

- **违规事实**：该城商行某网点柜员在办理业务时对客户态度恶劣、语言粗暴，被客户拍摄视频上传至社交媒体，引发广泛关注，严重损害银行声誉。
- **处罚结果**：涉事柜员被解除劳动合同，网点负责人被免职，银行被当地银行业协会通报批评。
- **涉及规范**：服务礼仪和服务态度要求。

### 案例二：某银行拒绝老年人柜面服务案（2023年）

- **违规事实**：某银行网点要求所有客户（包括老年人）必须通过智能终端办理业务，拒绝为老年人提供柜面服务。多名老年客户投诉后仍未得到解决。
- **处罚结果**：被监管机构通报批评，要求限期整改。
- **涉及规范**：适老化服务要求、不得强制使用自助设备。

### 案例三：某农商行网点无障碍设施缺失案（2022年）

- **违规事实**：某农商行新建网点未按照要求设置无障碍通道和低位柜台，导致轮椅使用者无法正常进入网点办理业务。
- **处罚结果**：被当地残联投诉并要求整改。
- **涉及规范**：无障碍设施建设要求。

### 案例四：某银行网点投诉处理不当案（2023年）

- **违规事实**：某银行网点在收到客户投诉后未在规定时限内处理，且对投诉客户进行"拉黑"处理，限制其在该网点办理业务。
- **处罚结果**：被罚款50万元，网点负责人被给予警告处分。
- **涉及规范**：投诉处理时限和公平处理要求。
