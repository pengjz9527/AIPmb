# 注册登录与账户管理 - 业务规则

## 一、注册业务规则

### 1.1 注册方式与权限

| 注册方式 | 身份核验方式 | 可获权限 | 适用客户群 |
|---------|------------|---------|-----------|
| 柜面签约 | 面对面身份证核验 | 全功能（转账、理财、贷款等） | 所有客户 |
| 手机号自助注册 | 手机号验证+身份信息验证 | 仅查询（余额、明细） | 本行存量客户 |
| 远程视频开户 | 视频面签+身份证OCR+人脸比对 | Ⅱ类户全功能 | 新客户 |
| STM自助注册 | 身份证读取+人脸比对 | 全功能 | 网点到店客户 |

### 1.2 注册条件

1. **年龄要求**：16周岁及以上（16-18周岁需提供收入证明或由监护人陪同）
2. **证件要求**：有效居民身份证（外籍人士持护照，港澳台居民持对应通行证）
3. **手机号要求**：本人实名手机号，且与银行预留手机号一致（柜面签约时设置）
4. **一人一号**：同一手机号仅可注册一个手机银行用户
5. **设备要求**：支持iOS 13+或Android 8.0+的智能手机

### 1.3 注册限额规则

| 注册方式 | 日转账限额 | 单笔转账限额 | 理财购买限额 |
|---------|-----------|------------|------------|
| 柜面签约（默认） | 50万元 | 20万元 | 不限 |
| 柜面签约（VIP） | 200万元 | 100万元 | 不限 |
| 自助注册 | 0（仅查询） | 0 | 0 |
| 远程开户（Ⅱ类户） | 1万元 | 5000元 | 1万元/日 |

## 二、登录业务规则

### 2.1 登录安全规则

| 规则项 | 规则内容 | 触发条件 |
|-------|---------|---------|
| 密码错误锁定 | 连续5次输错，锁定30分钟 | 同一用户连续密码错误 |
| 累计错误锁定 | 当日累计10次错误，锁定至次日 | 同一用户当日累计错误 |
| 异地登录提醒 | 弹窗提示"检测到异地登录" | 登录IP归属地与常用地址偏差>200km |
| 新设备登录 | 需短信验证+人脸识别 | 首次在新设备上登录 |
| 会话超时 | 15分钟无操作自动退出 | 登录态下无交互操作 |
| 多端互斥 | 踢出旧设备登录态 | 同一账号在多设备登录 |

### 2.2 生物识别登录规则

| 规则项 | 指纹登录 | 面容登录 |
|-------|---------|---------|
| 开启前提 | 已设置密码登录 | 已设置密码登录 |
| 首次开启验证 | 登录密码验证 | 登录密码验证 |
| 失败回退 | 3次失败→密码登录 | 3次失败→密码登录 |
| 强制密码周期 | 每72小时需密码登录一次 | 每72小时需密码登录一次 |
| 设备重启后 | 必须密码登录 | 必须密码登录 |
| 安全限制 | 设备新增指纹需重新验证密码 | 面部变化过大需重新验证 |

## 三、密码管理规则

### 3.1 登录密码规则

| 规则项 | 规则内容 |
|-------|---------|
| 长度要求 | 8-20位字符 |
| 复杂度要求 | 必须包含大小写字母和数字，可选特殊字符 |
| 禁止规则 | 不得为连续数字、重复数字、生日等弱密码 |
| 历史密码 | 不得与前5次使用的密码相同 |
| 有效期 | 90天未修改强制提醒（可配置是否强制修改） |
| 修改方式 | 旧密码验证通过 → 设置新密码 → 二次确认 |
| 重置方式 | 线上：人脸识别+身份信息核验；线下：柜面持身份证办理 |

### 3.2 交易密码规则

| 规则项 | 规则内容 |
|-------|---------|
| 长度要求 | 6位数字 |
| 禁止规则 | 不得为连续数字（123456）、重复数字（111111）、生日 |
| 修改方式 | 旧交易密码+登录密码验证 → 设置新密码 |
| 重置方式 | 仅支持柜面办理，持有效身份证件 |
| 使用场景 | 转账、理财购买、大额缴费等资金类交易 |
| 错误锁定 | 连续3次输错，当日交易功能锁定 |

## 四、账户绑定规则

### 4.1 本行账户绑定

| 规则项 | 规则内容 |
|-------|---------|
| 绑定数量 | 最多绑定20个本行账户（含银行卡和存折） |
| 绑定验证 | 输入卡号+交易密码+短信验证码 |
| 自动绑定 | 柜面签约注册时自动绑定签约卡 |
| 卡片状态 | 仅支持绑定正常状态卡片（非冻结、非挂失） |
| 绑定生效 | 验证通过后即时生效 |

### 4.2 他行账户绑定

| 规则项 | 规则内容 |
|-------|---------|
| 绑定渠道 | 通过超级网银（网上支付跨行清算系统） |
| 前提条件 | 他行卡已开通他行网银/手机银行 |
| 绑定验证 | 他行网银授权验证 |
| 功能限制 | 仅支持余额查询和转入资金，不支持从他行转出 |
| 绑定数量 | 最多绑定10个他行账户 |

### 4.3 账户解绑规则

| 规则项 | 规则内容 |
|-------|---------|
| 解绑方式 | 手机银行App自助解绑 |
| 验证要求 | 登录密码+短信验证码 |
| 解绑影响 | 解绑后该账户的自动转账、定投等签约关系同时失效 |
| 签约卡解绑 | 手机银行签约卡不可解绑，需到柜面注销手机银行 |

## 六、合规要求

### 6.1 实名制合规

1. **柜面签约**：面对面核验身份证原件，联网核查身份信息
2. **线上注册**：通过公安部身份核验系统验证身份信息
3. **远程开户**：视频面签+身份证OCR+活体检测三重验证
4. **持续识别**：身份证件过期前30天提醒更新，过期后限制非查询功能

### 6.2 反洗钱合规

1. **客户身份识别**：注册时完成客户身份识别和风险等级评定
2. **可疑交易监测**：注册后立即纳入反洗钱监测系统
3. **信息更新**：客户基本信息变更时需重新进行身份核实

### 6.3 个人信息保护

1. **最小必要原则**：仅采集业务所需的个人信息
2. **授权同意**：首次注册时展示隐私政策，获得用户明确同意
3. **信息脱敏**：卡号、身份证号等敏感信息在界面展示时脱敏处理
4. **数据删除**：注销手机银行后，个人信息按规定期限保留后删除

## 七、异常处理

| 异常场景 | 处理方式 | 客户指引 |
|---------|---------|---------|
| 注册失败（身份核验不通过） | 提示前往柜面办理 | "身份信息核验未通过，请携带身份证到网点办理" |
| 登录锁定 | 30分钟后自动解锁或柜面解锁 | "密码错误次数过多，请30分钟后重试或前往柜台解锁" |
| 验证码收不到 | 检查手机号、信号、短信拦截 | "请检查手机信号和短信拦截设置，或联系客服" |
| 绑定账户失败 | 核实卡号和密码，检查卡片状态 | "请确认卡号正确且卡片状态正常" |
| 人脸识别失败 | 调整光线和角度，最多3次 | "请确保光线充足，正对摄像头，去除面部遮挡" |
| 设备异常 | 新设备需增强认证 | "检测到新设备登录，请完成身份验证" |
