# 注册登录与账户管理 - 用户操作流程

## 一、注册操作流程

### 1.1 柜面签约注册流程

**前置条件**：
- 持有本行有效银行卡（借记卡或存折）
- 持有有效居民身份证原件
- 本人实名手机号

**操作步骤**：

1. 客户携带身份证和银行卡前往本行任一网点
2. 向柜员提出开通手机银行申请
3. 柜员核验客户身份证原件，进行联网核查
4. 柜员在系统中开通手机银行服务，设置初始登录密码
5. 客户确认并签署《手机银行服务协议》
6. 柜员完成签约，将签约回执交付客户
7. 客户下载手机银行App，使用手机号+初始密码首次登录
8. 首次登录后系统提示修改初始密码，客户设置新密码
9. 注册完成，可正常使用全部功能

**注意事项**：
- 初始密码需在首次登录后立即修改
- 预留手机号必须为客户本人实名手机号
- 柜面签约可设置个性化转账限额

### 1.2 手机号自助注册流程（查询版）

**前置条件**：
- 为本行存量客户（已在本行开立账户）
- 持有本人实名手机号
- 已下载手机银行App

**操作步骤**：

1. 打开手机银行App，点击首页"注册"按钮
2. 阅读并同意《手机银行服务协议》和《隐私政策》
3. 输入姓名、身份证号码、银行卡号
4. 系统发送短信验证码至预留手机号
5. 输入短信验证码完成验证
6. 设置登录密码（8-20位，含字母和数字）
7. 确认登录密码
8. 注册成功，获得查询权限

**注意事项**：
- 自助注册仅获得查询权限，无法进行转账交易
- 如需转账功能，需到柜面升级签约
- 身份证号和银行卡号需与银行留存信息完全一致

### 1.3 远程开户注册流程

**前置条件**：
- 年满18周岁的中国公民
- 持有有效居民身份证
- 手机支持视频通话功能
- 处于视频银行服务时间内（通常8:00-20:00）

**操作步骤**：

1. 打开手机银行App，选择"在线开户"
2. 拍摄身份证正反面照片，系统自动OCR识别
3. 确认识别信息，补充个人信息（职业、地址等）
4. 进入视频面签环节，与银行工作人员视频通话
5. 工作人员核验身份信息，确认开户意愿
6. 完成活体检测（摇头、眨眼等动作）
7. 设置交易密码（6位数字）
8. 设置登录密码
9. 绑定同名他行Ⅰ类户（作为资金来源验证）
10. 系统创建Ⅱ类电子账户
11. 开户成功，自动注册手机银行

**注意事项**：
- 远程开户仅开立Ⅱ类电子账户，日限额1万元
- 视频面签需在服务时间内进行
- 环境光线充足，背景简洁，便于人脸采集

## 二、登录操作流程

### 2.1 密码登录流程

**前置条件**：已完成手机银行注册

**操作步骤**：

1. 打开手机银行App
2. 在登录页面输入手机号（已自动填充上次登录号码）
3. 输入登录密码（使用安全键盘）
4. 系统验证通过后进入首页
5. 首次在新设备登录，需额外完成短信验证

**异常处理**：
- 密码错误：显示"密码错误，还剩X次机会"
- 账户锁定：显示"账户已锁定，请30分钟后重试"
- 忘记密码：点击"忘记密码"跳转密码重置流程

### 2.2 指纹登录流程

**前置条件**：已开启指纹登录功能，当前设备支持指纹识别

**操作步骤**：

1. 打开手机银行App
2. 系统自动弹出指纹验证窗口
3. 将手指放在指纹传感器上
4. 指纹验证通过，自动进入首页
5. 如验证失败，可重试或选择"使用密码登录"

### 2.3 面容登录流程

**前置条件**：已开启面容登录功能，当前设备支持面容识别

**操作步骤**：

1. 打开手机银行App
2. 系统自动调起面容识别
3. 正对手机前置摄像头
4. 面容验证通过，自动进入首页
5. 如验证失败，调整光线和角度重试，或选择"使用密码登录"

## 三、密码管理操作流程

### 3.1 修改登录密码

**前置条件**：已登录手机银行，已知旧登录密码

**操作步骤**：

1. 进入"我的" → "安全设置" → "修改登录密码"
2. 输入当前登录密码
3. 输入新登录密码（8-20位，含大小写字母和数字）
4. 再次输入新登录密码确认
5. 系统提示"密码修改成功，请重新登录"
6. 自动退出登录，使用新密码重新登录

### 3.2 重置登录密码（忘记密码）

**前置条件**：已注册手机银行，忘记密码

**操作步骤**：

1. 在登录页面点击"忘记密码"
2. 输入注册手机号和身份证号码
3. 系统发送短信验证码
4. 输入短信验证码
5. 进入人脸识别环节
6. 正对摄像头完成活体检测（按提示做动作）
7. 人脸比对通过
8. 设置新的登录密码
9. 确认新密码
10. 密码重置成功，使用新密码登录

### 3.3 设置/修改交易密码

**前置条件**：已登录手机银行

**操作步骤（首次设置）**：

1. 进入"我的" → "安全设置" → "设置交易密码"
2. 输入登录密码进行验证
3. 系统发送短信验证码
4. 输入短信验证码
5. 设置6位交易密码
6. 再次确认交易密码
7. 设置成功

**操作步骤（修改交易密码）**：

1. 进入"我的" → "安全设置" → "修改交易密码"
2. 输入旧交易密码
3. 设置新的6位交易密码
4. 再次确认新密码
5. 修改成功

## 四、账户绑定操作流程

### 4.1 绑定本行账户

**前置条件**：已登录手机银行，持有本行银行卡

**操作步骤**：

1. 进入"我的" → "账户管理" → "添加账户"
2. 选择"添加本行账户"
3. 输入银行卡号（支持扫码识别卡号）
4. 输入交易密码
5. 系统发送短信验证码至预留手机号
6. 输入短信验证码
7. 验证通过，账户添加成功
8. 可设置账户别名（如"工资卡""储蓄卡"）

### 4.2 账户解绑操作

**前置条件**：已登录手机银行

**操作步骤**：

1. 进入"我的" → "账户管理"
2. 选择需要解绑的账户
3. 点击"解除绑定"
4. 输入登录密码验证
5. 系统发送短信验证码
6. 输入验证码确认
7. 弹出确认提示："解绑后该账户的自动转账等签约关系将同时失效，确认解绑？"
8. 点击"确认解绑"
9. 解绑成功

## 五、账户查询操作流程

### 5.1 余额查询

**操作步骤**：

1. 登录后在首页查看"我的账户"卡片
2. 账户列表显示各账户余额（默认脱敏显示）
3. 点击"小眼睛"图标切换余额显示/隐藏
4. 点击具体账户进入详情页查看完整信息

### 5.2 交易明细查询

**操作步骤**：

1. 在首页点击需要查询的账户
2. 进入账户详情页，点击"交易明细"
3. 默认显示最近1个月交易记录
4. 可通过筛选条件缩小范围：
   - 时间范围：最近7天/1个月/3个月/6个月/1年/自定义
   - 交易类型：收入/支出/全部
   - 金额范围：自定义金额区间
5. 点击具体交易查看详情（交易时间、对方信息、交易类型、流水号等）
6. 可点击"申请电子回单"生成回单

### 5.3 电子回单下载

**操作步骤**：

1. 进入"我的" → "电子回单"
2. 选择账户
3. 设置查询时间范围
4. 系统展示符合条件的交易列表
5. 勾选需要下载回单的交易（支持全选）
6. 点击"生成回单"
7. 系统生成PDF格式回单文件
8. 可选择"保存到手机"或"分享至微信/邮件"

## 六、退出登录操作

### 6.1 主动退出

**操作步骤**：

1. 进入"我的"页面
2. 点击"安全退出"按钮
3. 弹出确认提示
4. 点击"确认退出"
5. 返回登录页面

### 6.2 超时自动退出

- 登录态下15分钟无任何操作（点击、滑动等），系统自动退出登录
- 弹出提示："您已长时间未操作，为安全起见已自动退出"
- 点击"重新登录"返回登录页面
