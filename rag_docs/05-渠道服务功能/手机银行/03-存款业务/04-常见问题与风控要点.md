# 手机银行存款业务常见问题与风控要点

## 一、常见问题（FAQ）

### Q1：定期存款提前支取会损失多少利息？

**答：** 提前支取定期存款，支取部分将按照支取日当天的活期存款利率计息，而非原定期利率。这意味着您将损失定期利率与活期利率之间的利息差额。

举例说明：您存入10万元1年期整存整取，年利率1.95%。持有6个月后提前支取，活期利率0.20%。
- 到期可获利息：100,000 × 1.95% = 1,950元
- 提前支取利息：100,000 × 0.20% / 360 × 180 = 100元
- 利息损失：1,950 - 100 = 1,850元

**建议：** 如有资金周转需求，优先考虑部分提前支取（仅整存整取支持一次），剩余部分仍按原利率计息。也可考虑通过大额存单转让功能获得更好的流动性。

### Q2：自动转存后的利率和原利率不一样怎么办？

**答：** 自动转存时，新存期的利率按照转存日当天的挂牌利率执行，而非您原存款时的利率。这意味着：

- 如果转存日利率高于原利率，您将获得更高的收益。
- 如果转存日利率低于原利率，收益会降低。
- 转存后如需调整，可在到期日前取消自动转存，到期后手动操作转存或支取。

建议在央行降息周期中关闭自动转存，到期后比较各银行利率后再做决定。

### Q3：大额存单转让价格怎么定？能赚到利息吗？

**答：** 大额存单转让价格由卖方在系统限定的区间内自主设定：

- **价格下限**：本金 × 97%（即可能小幅亏损本金）
- **价格上限**：本金 + 持有期间应计利息

卖方定价策略建议：
- 如果急需资金，可接近下限定价，容易快速成交，但可能损失少量本金。
- 如果不急，可接近上限定价，等待更长时间匹配，可锁定大部分利息收益。
- 系统会自动计算不同转让价格对应的实际收益率，供双方参考。

买方购买转让大额存单后，按票面利率继续计息，到期按票面金额兑付，收益为票面利息与转让价差之和。

### Q4：通知存款忘记发出支取通知，直接支取可以吗？

**答：** 可以支取，但利率会受影响：

- **未通知直接支取**：按活期利率计息，损失通知存款利率与活期利率之间的利息差。
- **正确做法**：先发出支取通知，一天通知存款等待1天、七天通知存款等待7天后再支取，可享受通知存款利率。
- **智能通知存款**：系统自动管理通知天数，支取时自动按实际通知天数计算利率，更为便捷。

建议大额资金且短期可能使用时选择智能通知存款，兼顾收益与灵活性。

### Q5：存款证明开立后，存款还能用吗？

**答：** 这取决于存款证明的类型：

- **时点证明**：仅证明申请时点的存款余额，不冻结存款，开立后存款可正常使用。
- **时段证明**：会冻结对应金额的存款，冻结期限与证明有效期一致。冻结期间该部分存款不可支取、转账或用于其他交易。
- **提前解冻**：如需提前解冻，需将存款证明原件全部交回银行（纸质证明需交回原件，电子证明需在手机银行申请撤销），解冻后证明自动作废。

建议根据实际需求选择证明类型，如仅用于展示资金实力可选时点证明，如需证明存款历史则需时段证明。

### Q6：手机银行的存款和柜面存款一样安全吗？

**答：** 完全一样安全。手机银行办理的存款业务与柜面办理的具有同等法律效力：

1. 存款资金存入同一账户体系，受存款保险保障（50万元以内全额赔付）。
2. 手机银行开立的定期存款同样生成电子存单，与纸质存单具有同等法律效力。
3. 所有存款操作均有完整的交易记录和流水，可随时查询和打印。
4. 如需纸质凭证，可至柜面补打或通过手机银行申请邮寄。

### Q7：零存整取连续几个月没存怎么办？

**答：** 零存整取有严格的续存规则：

- 每月应在约定日期前存入约定金额。
- 如果当月漏存，次月可补存（补存上月+存入当月，共计两笔）。
- 如果连续2个月未存（含补存），该零存整取账户将停止续存，进入冻结状态。
- 冻结后的账户可选择：(1) 提前支取（按活期利率计息）；(2) 至柜面申请恢复续存。
- 系统会在扣款失败后通过推送消息提醒客户。

### Q8：利率计算器算出来的利息和实际到手利息不一样？

**答：** 利率计算器提供的是估算结果，实际利息可能因以下原因略有差异：

1. **计息天数差异**：实际计息按"算头不算尾"原则，存款天数可能与预估有1天差异。
2. **利率调整**：活期存款利率可能在存款期间发生调整，计算器使用当前利率而非实际执行利率。
3. **利息税**：目前存款利息暂免征收个人所得税，如政策变化将影响到手利息。
4. **闰年因素**：闰年2月有29天，会影响实际计息天数。

差异通常在几角到几元范围内，如对实际利息有疑问，可在存款详情中查看精确的计息明细。

## 二、风险控制要点

### 2.1 存款业务操作风险

| 风险类型 | 风险描述 | 控制措施 |
|----------|----------|----------|
| 误操作风险 | 客户错误选择存期或存款类型 | 提交前展示关键信息确认页，展示预期利息对比 |
| 提前支取损失 | 客户不清楚提前支取按活期计息的规则 | 提前支取前强制展示利息损失试算结果 |
| 自动转存利率风险 | 转存利率可能低于客户预期 | 自动转存前推送通知，告知转存利率 |
| 大额存单转让亏损 | 卖方定价过低导致本金亏损 | 转让定价设置下限（本金×97%），并展示损益测算 |
| 存款证明冻结纠纷 | 客户不清楚时段证明会冻结存款 | 开立前明确告知冻结规则和期限 |

### 2.2 存款业务合规风险

| 风险类型 | 风险描述 | 控制措施 |
|----------|----------|----------|
| 利率违规 | 存款利率超过央行规定上限 | 系统自动校验利率合规性，禁止人工干预利率 |
| 存款保险超限额 | 客户存款超过50万元保险限额不知情 | 存款页面展示存款保险标识和说明 |
| 反洗钱 | 大额现金存款来源不明 | 单笔5万元以上现金存款需登记资金来源 |
| 产品适当性 | 向不适当客户推荐存款产品 | 老年客户购买复杂存款产品前进行适当性评估 |

### 2.3 存款业务系统风险

| 风险类型 | 风险描述 | 控制措施 |
|----------|----------|----------|
| 重复交易 | 网络延迟导致重复提交存款操作 | 系统防重机制，同一请求60秒内不重复处理 |
| 资金不一致 | 扣款成功但开户失败 | 冲正机制自动退回扣款资金 |
| 利率同步延迟 | 利率调整后系统未及时更新 | 每日日终批量同步，实时生效前展示提示 |
| 大额存单超售 | 额度用尽后仍接受购买 | 实时额度管控，售罄自动关闭购买通道 |

### 2.4 重点风控场景

**存款资金安全保护：**

- 定期存款提前支取需进行交易密码验证 + 短信验证码双重认证。
- 大额存单转让需进行人脸识别验证。
- 存款证明开立涉及资金冻结的，需进行交易密码验证。
- 所有存款变动实时推送短信通知。

**异常交易监控：**

- 短时间内频繁开立和支取定期存款（当日开当日取），触发可疑交易预警。
- 大额存单转让价格异常偏离市场水平，系统自动标记并人工复核。
- 同一客户短期内多次申请存款证明，触发反洗钱关注。
- 通知存款频繁存取（疑似利用通知存款替代活期），触发合规审查。

**客户权益保护：**

- 存款产品页面明确展示利率、期限、提前支取规则等关键信息。
- 大额存单购买前强制阅读产品说明书和风险揭示书。
- 自动转存功能默认关闭，需客户主动开启（避免未预期的资金锁定）。
- 提前支取前展示利息损失测算，确保客户充分知情。
