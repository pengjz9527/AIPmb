# 手机银行贷款业务常见问题与风控要点

## 一、常见问题（FAQ）

### Q1：贷款申请被拒绝了，多久可以重新申请？

**答：** 贷款申请被拒绝后的重新申请规则如下：

1. **因征信原因被拒**：建议改善信用记录后6个月再重新申请。连续逾期记录需满2年才会从征信报告中消除。
2. **因负债过高被拒**：待已有贷款偿还一定比例（负债率降至70%以下）后即可重新申请。
3. **因基本信息不满足被拒**（如工作年限不足）：待条件满足后随时可重新申请。
4. **系统原因被拒**：可立即重新申请或联系客服处理。

每次申请拒绝后系统会展示具体原因，建议针对性改善后再申请。短期内频繁申请（如一个月内超过3次）会影响信用评估。

### Q2：贷款审批通过但还没放款，可以取消吗？

**答：** 可以取消，但需要在放款前操作：

1. 贷款审批通过后、合同签署前：可在"贷款进度"页面直接点击"取消申请"，无任何费用。
2. 合同已签署但尚未放款：需联系客服申请取消，可能需支付合同工本费（通常50元）。
3. 已放款：无法取消，只能通过提前还款方式结清（需满足提前还款条件）。

建议审批通过后仔细确认贷款条款再签署合同，避免不必要的麻烦。

### Q3：还款日当天忘记还款了怎么办？会怎样？

**答：** 还款日当天忘记还款的处理方式取决于时间：

1. **还款日当天15:00前**：立即通过手机银行手动还款，实时到账，不算逾期。
2. **还款日当天15:00后但仍在宽限期内（消费贷/信用贷有3天宽限期）**：在宽限期内还款不计罚息、不上报征信。
3. **超过宽限期**：开始计算罚息（合同利率×150%），并上报征信。

**补救措施：**

- 尽快还款，减少逾期天数和罚息金额。
- 若因特殊原因（如生病住院、自然灾害等）导致逾期，可提交证明材料申请减免罚息。
- 保持良好的后续还款记录，单次短期逾期对征信影响有限。

### Q4：提前还款后月供/期限什么时候变化？

**答：** 提前部分还款后的调整时间如下：

1. **申请提交**：提交提前还款申请。
2. **T+1日生效**：系统处理提前还款扣款。
3. **当期或次期生效**：
   - 若提前还款在当期还款日之前完成，当期即按新计划执行。
   - 若提前还款在当期还款日之后完成，从下一期开始按新计划执行。
4. **还款计划更新**：生效后可在"还款计划"中查看更新后的安排。

建议在还款日前5个工作日提交提前还款申请，确保当期即可按新计划执行。

### Q5：经营贷需要提供哪些材料？线上能全部完成吗？

**答：** 经营贷所需材料和线上化程度如下：

**线上提交材料：**
- 身份证正反面照片
- 营业执照照片
- 近6个月经营流水（可截图或拍照）
- 经营场所照片（部分产品需要）

**抵押类额外需要：**
- 房产证/不动产权证照片
- 抵押物现状照片
- 需线下办理抵押登记（银行工作人员协助）

**线上化程度：**
- 信用类经营贷：全流程线上化，从申请到放款均可在手机银行完成。
- 抵押类经营贷：申请和审批线上化，但抵押登记需线下办理（银行安排专人对接）。

### Q6：贷款利率可以调整吗？怎么申请利率优惠？

**答：** 贷款利率调整规则如下：

1. **已发放贷款**：固定利率贷款在合同期内利率不变；LPR浮动利率贷款每年调整一次（以最近一次LPR为准）。
2. **申请时获取优惠利率的方式**：
   - 提升在本行的金融资产（AUM越高利率越低）
   - 保持良好征信记录
   - 参加本行利率优惠活动（如节假日促销利率）
   - VIP客户自动享受利率折扣
3. **利率协商**：大额贷款（50万元以上）可通过客户经理协商利率。

### Q7：贷款可以更换还款账户吗？

**答：** 可以更换还款账户，但需满足以下条件：

1. 新还款账户必须是本行同名借记卡（户名与贷款人一致）。
2. 需在还款日前3个工作日操作更换（当期仍从原账户扣款）。
3. 更换路径："我的贷款"→选择贷款→"还款管理"→"更换还款账户"。
4. 更换成功后系统推送确认通知。
5. 新账户需确保在还款日前有足够余额。

### Q8：贷款结清后征信什么时候更新？

**答：** 贷款结清后的征信更新时间：

1. **银行报送时间**：贷款结清后，本行在次月15日前向人民银行征信中心报送结清信息。
2. **征信更新时间**：征信中心收到信息后1-2个工作日内更新。
3. **实际影响**：结清后约30-45天内征信报告中该笔贷款状态更新为"结清"。
4. **如需立即证明**：可在手机银行下载《贷款结清证明》，该证明可作为结清的即时凭证。

## 二、风险控制要点

### 2.1 贷款申请风控

| 风控环节 | 具体措施 | 触发条件 |
|----------|----------|----------|
| 身份核验 | 人脸识别活体检测+身份证OCR识别 | 每次贷款申请 |
| 反欺诈检测 | 设备指纹、地理位置、行为模式分析 | 每次申请自动执行 |
| 征信查询 | 查询央行征信报告（需客户授权） | 每次贷款申请 |
| 多头借贷检测 | 检测近期征信查询次数和已有贷款笔数 | 近1个月征信查询≥6次触发预警 |
| 黑名单校验 | 匹配反欺诈黑名单和法院失信名单 | 每次申请自动执行 |
| 收入核实 | 代发工资流水或纳税记录交叉验证 | 申请金额≥10万元 |

### 2.2 贷中风控

| 风控环节 | 具体措施 | 说明 |
|----------|----------|------|
| 资金用途监控 | 对贷款资金流向进行跟踪分析 | 发现流入房地产/股市等违规领域立即预警 |
| 还款能力跟踪 | 定期评估借款人还款能力变化 | 收入大幅下降或负债激增触发预警 |
| 逾期早期预警 | 还款日前短信提醒+逾期后分级催收 | 降低逾期发生率和损失 |
| 提前还款监控 | 分析提前还款行为是否异常 | 短期内借入即还可能存在套利行为 |

### 2.3 贷后风控

| 风控环节 | 具体措施 | 说明 |
|----------|----------|------|
| 逾期催收管理 | 按逾期天数分级催收 | 短信→电话→上门→法律诉讼 |
| 不良贷款处置 | 对长期逾期贷款进行核销或转让 | 符合监管要求的处置方式 |
| 征信报送 | 按时准确向征信中心报送贷款信息 | 每月15日前完成报送 |
| 贷款用途检查 | 对大额贷款进行用途合规检查 | 要求提供消费凭证 |

### 2.4 重点风控场景

**防范过度借贷：**

- 系统自动计算客户总负债率（所有贷款月供/月收入），超过70%拒绝审批。
- 近1个月在其他银行申请贷款超过3笔的，系统提示多头借贷风险。
- 消费贷+信用贷总授信额度不超过客户年收入的2倍。
- 对年轻客户（18-25岁）设置更审慎的额度上限。

**防范贷款资金违规使用：**

- 放款后对资金流向进行T+1监控。
- 贷款资金转入房地产相关账户（开发商、中介等）立即预警。
- 贷款资金转入证券账户或基金购买平台立即预警。
- 大额提现（单笔超过贷款金额30%）触发用途核实。

**防范身份冒用：**

- 贷款申请全流程人脸识别验证（申请、签约、放款三个环节）。
- 申请设备需为已认证的常用设备，新设备申请需额外验证。
- 申请IP地址与客户常住地址不匹配时触发预警。
- 借款人与还款账户非同一人的申请自动拒绝。

**保护消费者权益：**

- 贷款页面明确展示年化利率（IRR口径）和贷款总成本。
- 不向无还款能力的客户（如无收入来源的在校学生）发放贷款。
- 催收行为规范：不在非正常时间（晚10点至早8点）催收，不向无关第三方催收。
- 客户有权在签约前了解全部费用和利率信息。
