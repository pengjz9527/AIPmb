# 手机银行理财业务规则

## 一、投资者适当性管理规则

### 1.1 客户风险等级分类

| 客户风险等级 | 等级代码 | 评估标准 | 可购买产品范围 |
|-------------|----------|----------|---------------|
| 保守型 | C1 | 无法承受任何投资损失，追求资金绝对安全 | R1（低风险）产品 |
| 稳健型 | C2 | 可承受较小损失（5%以内），追求稳健收益 | R1-R2产品 |
| 平衡型 | C3 | 可承受一定损失（10%以内），追求资产保值增值 | R1-R3产品 |
| 成长型 | C4 | 可承受较大损失（20%以内），追求较高收益 | R1-R4产品 |
| 进取型 | C5 | 可承受重大损失（20%以上），追求高收益 | R1-R5产品 |

### 1.2 产品风险等级分类

| 产品风险等级 | 等级代码 | 产品类型 | 风险特征 |
|-------------|----------|----------|----------|
| 低风险 | R1 | 货币基金、国债、大额存单 | 本金几乎无损失风险 |
| 中低风险 | R2 | 纯债基金、固定收益类理财 | 本金损失概率较低 |
| 中风险 | R3 | 混合型基金、混合类理财 | 有一定本金损失风险 |
| 中高风险 | R4 | 偏股混合基金、权益类理财 | 本金损失概率较高 |
| 高风险 | R5 | 股票型基金、衍生品、私募产品 | 可能损失全部本金 |

### 1.3 适当性匹配规则

- **严格匹配原则**：客户只能购买风险等级不超过自身风险承受能力的产品。
- **超额购买限制**：若客户坚持购买超出风险等级的产品，需签署《风险不匹配购买确认书》，并经过二次风险提示（R4及以上产品需进行双录）。
- **65岁以上客户**：购买R3及以上产品需由客户经理面谈确认或进行双录。
- **评估结果过期**：风险评估结果有效期1年，过期后无法购买新产品（赎回不受影响），需重新评估。

### 1.4 风险评估问卷规则

| 评估要素 | 题目数量 | 评分权重 | 说明 |
|----------|----------|----------|------|
| 年龄与收入 | 2题 | 15% | 年龄越大、收入越稳定，风险承受能力越低 |
| 投资经验 | 2题 | 20% | 投资经验越丰富，可承受更高风险 |
| 风险偏好 | 3题 | 35% | 核心评估维度，直接反映风险态度 |
| 投资期限 | 1题 | 15% | 投资期限越长，可承受更高风险 |
| 损失承受 | 2题 | 15% | 最大可承受损失比例 |

## 二、冷静期规则

### 2.1 理财产品冷静期

| 产品类型 | 冷静期时长 | 起算时间 | 冷静期内权益 |
|----------|-----------|----------|-------------|
| 公募理财产品 | 24小时 | 签署购买确认后 | 可无条件撤销购买，全额退款 |
| 私募理财产品 | 24小时 | 签署购买确认后 | 可无条件撤销购买，全额退款 |
| R4/R5级产品 | 48小时 | 签署购买确认后 | 可无条件撤销，全额退款 |

**冷静期规则说明：**

- 冷静期内客户可无条件撤销购买，已扣款项在撤销后T+1日退回。
- 冷静期结束后不可撤销，需通过赎回方式退出（受赎回规则约束）。
- 冷静期仅适用于首次购买的产品，追加购买不设冷静期。
- 定期开放型产品仅在开放日购买时享有冷静期。

### 2.2 保险产品犹豫期

| 产品类型 | 犹豫期时长 | 起算时间 | 犹豫期内权益 |
|----------|-----------|----------|-------------|
| 长期人身保险 | 15天 | 签收保单次日 | 全额退保（扣除工本费10元） |
| 短期保险（1年以下） | 无犹豫期 | — | — |
| 互联网保险产品 | 15天 | 电子保单签收次日 | 全额退保 |

## 三、赎回确认规则

### 3.1 理财产品赎回确认时间

| 产品类型 | 赎回确认日 | 资金到账日 | 说明 |
|----------|-----------|-----------|------|
| 现金管理类（T+0） | 当日确认 | 当日或T+1日到账 | 工作日15:00前提交当日确认 |
| 开放式固收类（T+1） | T+1日确认 | T+2日到账 | T日15:00前提交T+1日确认 |
| 定期开放类 | 开放日确认 | 开放日后T+1至T+3日到账 | 需在开放日提交 |
| 封闭式 | 到期自动兑付 | 到期日后T+1至T+3日到账 | 到期前不可赎回 |

**特别说明：**

- T日15:00后提交的赎回申请视同T+1日提交。
- 非工作日提交的赎回申请顺延至下一个工作日处理。
- 巨额赎回（单日赎回超过产品总份额10%）时，管理人有权延期办理。

### 3.2 基金赎回确认规则

| 基金类型 | 赎回确认日 | 资金到账日 |
|----------|-----------|-----------|
| 货币基金 | T日确认 | T+1日到账（部分支持T+0快速赎回，单日限额1万元） |
| 债券基金 | T+1日确认 | T+2至T+3日到账 |
| 股票/混合基金 | T+1日确认 | T+3至T+5日到账 |
| QDII基金 | T+2日确认 | T+7至T+10日到账 |

### 3.3 赎回费用规则

| 持有时间 | 赎回费率（基金） | 说明 |
|----------|-----------------|------|
| 7天以内 | 1.5%（惩罚性费率） | 短期交易惩罚，全部计入基金资产 |
| 7天-30天 | 0.75% | — |
| 30天-1年 | 0.5% | — |
| 1年-2年 | 0.25% | — |
| 2年以上 | 0% | 免赎回费 |

## 四、双录要求

### 4.1 双录适用范围

"双录"即录音录像，是理财产品销售过程的合规要求：

| 适用场景 | 双录要求 | 说明 |
|----------|----------|------|
| 单笔购买金额≥20万元的理财产品 | 必须双录 | 大额购买需记录销售过程 |
| 购买R4及以上风险等级产品 | 必须双录 | 高风险产品销售需全程记录 |
| 65岁以上客户购买R3及以上产品 | 必须双录 | 老年客户保护 |
| 客户主动要求双录 | 按需双录 | 客户有权要求对销售过程录音录像 |

### 4.2 线上双录流程

1. 系统提示需要进行录音录像。
2. 客户确认开启摄像头和麦克风权限。
3. 系统播放产品风险提示（语音+字幕）。
4. 客户逐条确认关键风险事项（语音确认或点击确认）。
5. 客户朗读风险确认声明。
6. 双录文件自动保存，保存期限不少于产品到期后5年。
7. 双录完成后方可继续购买。

## 五、销售误导禁令

### 5.1 禁止行为清单

| 禁止行为 | 具体表现 | 违规后果 |
|----------|----------|----------|
| 承诺保本保收益 | "保证不亏""收益一定超过X%" | 严重违规，可处以罚款和吊销从业资格 |
| 夸大收益 | 展示历史最高收益作为预期收益 | 违规，责令改正 |
| 隐瞒风险 | 不告知客户可能亏损的情况 | 严重违规 |
| 强制搭售 | 购买贷款产品强制购买保险/理财 | 严重违规 |
| 代客操作 | 代替客户进行风险评估或购买操作 | 严重违规 |
| 不当比较 | 将理财产品与存款进行收益比较 | 违规，责令改正 |
| 虚假宣传 | 使用不实数据或案例进行宣传 | 严重违规 |

### 5.2 合规展示要求

- 理财产品页面必须标注"理财非存款，产品有风险，投资需谨慎"。
- 必须展示产品风险等级和客户风险等级匹配关系。
- 历史业绩不代表未来收益，需明确标注。
- 预期收益率/业绩比较基准需标注计算口径和数据区间。
- 费率结构需完整展示（管理费、托管费、销售服务费、超额业绩报酬等）。

## 六、基金定投规则

### 6.1 定投执行规则

| 定投要素 | 规则说明 |
|----------|----------|
| 扣款时间 | 指定日期上午10:00自动扣款 |
| 扣款失败 | 账户余额不足时扣款失败，当日15:00二次尝试；仍失败则该期定投跳过 |
| 连续失败 | 连续3期扣款失败，定投计划自动暂停 |
| 节假日处理 | 定投日遇节假日，顺延至下一个工作日执行 |
| 确认份额 | 按扣款日当日基金净值确认申购份额 |
| 最低金额 | 每期最低100元 |

### 6.2 智能定投规则

- 智能定投根据参考指数的估值水平自动调整定投金额。
- 估值低于历史30%分位时：定投金额上浮至基准的150%。
- 估值处于30%-70%分位时：按基准金额定投。
- 估值高于70%分位时：定投金额下调至基准的50%。
- 客户可自定义估值参考指标和调整比例。

## 七、贵金属交易规则

### 7.1 账户贵金属交易规则

| 交易要素 | 规则说明 |
|----------|----------|
| 交易时间 | 工作日9:00-15:00（部分品种延长至次日凌晨2:00） |
| 最小交易单位 | 纸黄金：1克；纸白银：1克 |
| 买卖价差 | 纸黄金：0.8元/克；纸白银：0.04元/克 |
| 持仓限制 | 单一品种最大持仓：纸黄金10000克，纸白银50000克 |
| 止损/止盈 | 支持设置止损和止盈价格 |
| 不计利息 | 账户贵金属不计付利息 |

## 八、费率规则

### 8.1 理财产品费率

| 费用项目 | 费率标准 | 收取方式 |
|----------|----------|----------|
| 销售服务费 | 0.1%-0.5%/年 | 从产品净值中逐日计提 |
| 管理费 | 0.1%-0.8%/年 | 从产品净值中逐日计提 |
| 托管费 | 0.02%-0.05%/年 | 从产品净值中逐日计提 |
| 超额业绩报酬 | 超出业绩基准部分的50% | 到期或赎回时收取 |

### 8.2 基金费率

| 费用项目 | 费率标准 | 说明 |
|----------|----------|------|
| 申购费 | 0.1%-1.5% | 手机银行渠道通常享受4折优惠 |
| 赎回费 | 0%-1.5% | 持有时间越长费率越低 |
| 管理费 | 0.5%-1.5%/年 | 从基金净值中逐日计提 |
| 托管费 | 0.1%-0.25%/年 | 从基金净值中逐日计提 |
