# 手机银行理财业务用户操作流程

## 一、风险评估问卷操作流程

### 首次风险评估

1. 登录手机银行APP，进入"理财"板块。
2. 首次进入时系统自动弹出风险提示："首次购买理财产品前需完成风险评估"。
3. 点击"开始评估"进入问卷页面。
4. 逐题作答（共10题）：
   - 第1题：您的年龄段是？（18-30岁/31-45岁/46-60岁/60岁以上）
   - 第2题：您的家庭年收入大约是？（选择区间）
   - 第3题：您有多少年的投资经验？（无经验/1-3年/3-5年/5年以上）
   - 第4题：您投资的主要目的是？（保值/稳健增值/追求高收益）
   - 第5题：如果投资亏损10%，您会？（立即卖出/观望/继续持有/加仓）
   - 第6-10题：风险偏好和投资期限相关问题
5. 完成全部题目后点击"提交"。
6. 系统即时显示评估结果：
   - 风险承受等级（如"稳健型C2"）
   - 可购买产品范围（如"R1-R2风险等级产品"）
   - 评估有效期（1年）
7. 点击"确认"保存评估结果。

### 风险评估到期重新评估

1. 评估结果到期前30天、前7天系统推送提醒。
2. 到期后进入理财板块时提示重新评估。
3. 重新评估流程与首次评估一致。
4. 评估期间持仓产品的赎回不受影响。

## 二、理财产品浏览与搜索操作流程

1. 进入手机银行"理财"页面。
2. 顶部展示理财产品分类导航：
   - 活期理财（随时申赎）
   - 短期理财（1-3个月）
   - 中期理财（3-12个月）
   - 长期理财（1年以上）
   - 新发产品
3. 点击任一分类进入对应产品列表。
4. 使用筛选功能进一步缩小范围：
   - 风险等级筛选（R1/R2/R3/R4/R5）
   - 起购金额筛选
   - 发行方筛选（本行/理财子公司）
   - 排序方式（收益率/销量/新品）
5. 使用搜索功能：在搜索框输入产品名称或代码。
6. 点击具体产品进入详情页，查看：
   - 产品基本信息（名称、代码、风险等级、起购金额）
   - 业绩比较基准/历史收益率
   - 产品期限和开放日安排
   - 投资范围和策略
   - 费率结构
   - 历史净值走势图
   - 产品说明书和风险揭示书
7. 如需对比多只产品：勾选"加入对比"（最多3只），在"对比"页面查看横向对比表。

## 三、理财产品购买操作流程

### 3.1 首次购买

1. 在产品详情页点击"立即购买"按钮。
2. 系统校验风险评估状态：
   - 已评估且等级匹配：继续购买流程。
   - 未评估或已过期：提示先完成风险评估。
   - 等级不匹配：弹出风险不匹配提示，需签署确认书后方可继续。
3. 输入购买金额（系统显示起购金额和递增单位）。
4. 系统展示购买试算：
   - 购买金额
   - 预估手续费（如有）
   - 确认日（T日/T+1日）
   - 预计起息日
5. 阅读并勾选《理财产品购买协议》《风险揭示书》《投资者权益须知》。
6. 若产品需要双录（金额≥20万元或R4及以上）：
   - 系统提示开启摄像头和麦克风。
   - 播放风险提示视频/语音。
   - 客户逐条确认风险事项。
   - 双录完成后继续购买流程。
7. 核对购买信息（产品名称、金额、风险等级、确认日）。
8. 输入交易密码确认。
9. 进入冷静期倒计时（24小时或48小时，视产品风险等级）。
10. 冷静期内可在"我的理财"中撤销购买。
11. 冷静期结束后正式确认份额，开始计息。

### 3.2 追加购买

1. 在"我的理财"→持仓产品→点击"追加购买"。
2. 输入追加金额。
3. 确认购买信息。
4. 输入交易密码确认。
5. 追加购买不设冷静期，按开放日净值确认。

## 四、理财产品赎回操作流程

1. 进入"我的理财"→"持仓"→选择需要赎回的产品。
2. 点击"赎回"按钮。
3. 系统校验赎回条件：
   - 封闭式产品：提示"该产品为封闭式，到期前不可赎回"。
   - 开放期产品：校验是否在交易时间内（工作日9:00-15:00）。
   - 非交易时间：提示"当前非交易时间，赎回申请将在下一个工作日提交"。
4. 选择赎回方式：
   - 部分赎回：输入赎回份额（剩余份额须≥产品最低持有量）。
   - 全部赎回：赎回全部持有份额。
5. 系统展示赎回试算：
   - 赎回份额
   - 最新净值
   - 预估赎回金额（份额×净值-赎回费用）
   - 预计确认日和到账日
6. 确认赎回信息。
7. 输入交易密码确认。
8. 系统提示"赎回申请已提交"。
9. 确认日后可在"交易记录"中查看赎回确认结果。
10. 到账日收到资金入账通知。

## 五、基金定投设置操作流程

### 5.1 创建定投计划

1. 进入"理财"→"基金"→选择目标基金。
2. 在基金详情页点击"定投"按钮。
3. 设置定投参数：
   - 定投金额：输入每期定投金额（最低100元，100元递增）。
   - 定投周期：选择"每周"或"每月"。
   - 定投日：每周选择星期几（一至五）；每月选择几号（1-28日）。
   - 扣款账户：选择用于扣款的活期账户。
4. 选择定投方式：
   - 普通定投：每期按固定金额投资。
   - 智能定投：根据市场估值自动调整金额（展示调整规则说明）。
5. 阅读并同意《基金定投协议》。
6. 确认定投计划信息。
7. 输入交易密码确认。
8. 系统提示"定投计划设置成功"，展示首期扣款日期。

### 5.2 管理定投计划

1. 进入"我的理财"→"定投管理"。
2. 查看所有定投计划列表（状态：正常/暂停/已终止）。
3. 点击具体计划进行管理：
   - **修改**：调整定投金额、定投日、定投方式。
   - **暂停**：暂停下一期及后续的定投扣款（可随时恢复）。
   - **终止**：永久终止定投计划（已持有的基金份额不受影响）。
4. 修改定投参数后需重新输入交易密码确认。

## 六、保险产品购买操作流程

1. 进入"理财"→"保险"板块。
2. 浏览保险产品分类（寿险/重疾/医疗/意外/财产）。
3. 选择感兴趣的保险产品，进入详情页。
4. 仔细阅读产品说明、保险条款、免责条款。
5. 点击"立即投保"。
6. 填写投保信息：
   - 投保人信息（姓名、身份证号、手机号、地址）
   - 被保险人信息（可与投保人不同，需填写关系）
   - 受益人信息（法定/指定，指定需填写姓名和身份证号）
7. 选择保险方案（保额、保障期限、缴费期限）。
8. 完成健康告知（人身险产品）：
   - 逐条回答健康问题（是/否）
   - 存在异常情况的需上传相关材料
   - 健康告知不通过的可能被拒保或加费
9. 确认投保信息、保费金额和缴费方式。
10. 阅读并勾选《保险条款》《投保须知》《免责条款》《客户信息使用授权》。
11. 输入交易密码完成支付。
12. 投保成功后生成电子保单，推送至手机银行"我的保险"。
13. 人身险产品开始计算15天犹豫期。

## 七、贵金属买卖操作流程

### 7.1 账户贵金属买入

1. 进入"理财"→"贵金属"→"账户贵金属"。
2. 选择交易品种（纸黄金/纸白银）。
3. 查看实时行情（买入价/卖出价/K线图）。
4. 点击"买入"。
5. 输入买入数量（克数）。
6. 系统展示买入试算（买入金额 = 数量 × 银行卖出价）。
7. 确认买入信息。
8. 输入交易密码确认。
9. 买入成功，持仓实时更新。

### 7.2 账户贵金属卖出

1. 在贵金属交易页面点击"卖出"。
2. 输入卖出数量（不超过当前持仓）。
3. 系统展示卖出试算（卖出金额 = 数量 × 银行买入价）。
4. 确认卖出信息。
5. 输入交易密码确认。
6. 卖出成功，资金实时回到活期账户。

### 7.3 实物贵金属购买

1. 进入"贵金属"→"实物贵金属"。
2. 浏览实物产品列表（金条、金币等，展示规格、价格、库存）。
3. 选择产品，进入详情页。
4. 选择规格和数量。
5. 选择配送方式：快递配送（免费）或网点自取。
6. 若选择配送，填写收货地址。
7. 确认订单信息和金额。
8. 输入交易密码完成支付。
9. 等待配送（3-5个工作日）或按通知前往网点自取。

## 八、持仓查询与收益查看操作流程

### 8.1 持仓查询

1. 进入"理财"→"我的持仓"。
2. 顶部展示投资类资产总市值和总收益。
3. 按产品分类展示持仓列表：
   - 理财产品：名称、持有金额、持有收益、持有天数
   - 基金：名称、持有份额、持有金额、持有收益、持有收益率
   - 保险：产品名称、保单号、保费、保单状态
   - 贵金属：品种、持仓数量、持仓市值、浮动盈亏
4. 点击具体产品查看持仓详情（买入记录、净值走势、交易历史）。

### 8.2 收益查看

1. 进入"我的持仓"→点击"收益总览"。
2. 查看收益汇总：
   - 累计总收益（所有产品的已实现+未实现收益）
   - 本月收益
   - 本年收益
   - 当日收益
3. 查看收益趋势图（按日/周/月/年切换时间维度）。
4. 点击具体产品查看该产品的收益明细。
5. 使用"收益计算器"功能：输入投资金额和预期收益率，计算预期收益。
