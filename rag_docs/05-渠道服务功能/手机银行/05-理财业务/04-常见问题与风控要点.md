# 手机银行理财业务常见问题与风控要点

## 一、常见问题（FAQ）

### Q1：理财产品亏损了怎么办？能保本吗？

**答：** 自资管新规实施以来，理财产品已打破刚性兑付，不再承诺保本保收益：

1. **理财产品净值化**：所有理财产品按照市值法估值，产品净值随市场波动，可能出现亏损。
2. **不同风险等级产品的亏损概率不同**：
   - R1/R2产品亏损概率很低，但理论上仍有可能。
   - R3产品可能出现短期净值回撤（通常5%以内）。
   - R4/R5产品可能出现较大亏损。
3. **应对建议**：
   - 短期波动是正常现象，不建议恐慌性赎回。
   - 关注产品的长期业绩表现而非短期波动。
   - 如确实无法承受波动，可考虑将资金转入更低风险等级的产品。
4. **银行责任**：银行不对投资损失承担赔偿责任，但需履行诚实信用、谨慎勤勉的管理义务。

### Q2：风险评估结果过期了，持仓的理财产品还能赎回吗？

**答：** 可以赎回。风险评估过期仅影响新产品的购买，不影响已持有产品的赎回操作。

- 持仓产品的赎回、收益分配等操作不受风险评估状态影响。
- 如需购买新产品，需先完成重新风险评估。
- 风险评估到期前30天和前7天系统会推送提醒，建议及时更新。
- 重新评估流程简便，通常5分钟内即可完成。

### Q3：基金定投连续扣款失败了怎么办？

**答：** 基金定投扣款失败的处理规则如下：

1. **首次扣款失败**：当日15:00系统进行二次扣款尝试。
2. **二次仍失败**：该期定投跳过，不补扣，等待下一期。
3. **连续3期失败**：定投计划自动暂停，系统推送通知。
4. **恢复定投**：可在"定投管理"中手动恢复暂停的定投计划。
5. **扣款失败原因**：
   - 账户余额不足（最常见原因）
   - 扣款账户已冻结或挂失
   - 基金暂停申购

**建议**：确保扣款账户余额充足，或设置账户余额提醒功能。如资金周转紧张，可先暂停定投再恢复。

### Q4：保险产品的犹豫期是怎么算的？退保会损失多少？

**答：** 保险产品犹豫期规则如下：

1. **犹豫期时长**：长期人身保险产品为15天（自签收保单次日起算）。
2. **犹豫期内退保**：全额退还已交保费（扣除工本费10元），基本无损失。
3. **犹豫期后退保**：退还保单现金价值，通常低于已交保费，可能产生较大损失。
   - 缴费前几年现金价值很低，退保损失最大。
   - 缴费后期现金价值逐渐接近已交保费。
4. **犹豫期计算**：从签收电子保单的次日起算15个自然日，最后一天若为节假日则顺延。
5. **退保路径**："我的保险"→选择保单→"申请退保"。

**特别提醒**：犹豫期后退保可能损失较大，建议在犹豫期内仔细审阅保险条款，如不满意尽早退保。

### Q5：理财产品的收益和宣传的不一样，正常吗？

**答：** 理财产品实际收益与宣传数据可能存在差异，这通常是正常的：

1. **业绩比较基准不等于实际收益**：业绩比较基准仅为参考，不代表承诺收益。
2. **历史业绩不代表未来**：产品页面展示的历史收益率仅为过往表现，不代表未来收益。
3. **净值波动**：产品净值随底层资产市值波动，每日收益可能为正也可能为负。
4. **费用扣除**：页面展示的收益通常为扣除管理费和托管费后的净收益，但申购/赎回费需另行计算。

**异常情形**：若实际收益与业绩比较基准确实存在较大差异（如超过2个百分点），可联系银行客服了解原因。银行有义务对投资运作情况进行说明。

### Q6：账户贵金属交易时间与股票市场不同，为什么？

**答：** 账户贵金属交易时间有其特殊性：

1. **交易时间**：工作日9:00-15:00为常规交易时间，部分品种延长至次日凌晨2:00（覆盖欧美交易时段）。
2. **原因说明**：账户贵金属价格挂钩国际贵金属市场价格，国际市场的交易时段覆盖更长。
3. **非交易时间**：不可进行买卖操作，但可以查看持仓和行情。
4. **节假日**：国家法定节假日暂停交易。
5. **与实物贵金属的区别**：实物贵金属购买不受交易时间限制，7×24小时可下单。

### Q7：购买理财产品时需要"双录"是什么意思？

**答：** "双录"即录音录像，是监管部门要求的理财产品销售合规措施：

1. **目的**：记录销售过程，确保客户充分了解产品风险，保护投资者权益。
2. **适用范围**：大额购买（≥20万元）、高风险产品（R4及以上）、老年客户购买R3及以上产品。
3. **线上双录流程**：
   - 开启手机摄像头和麦克风。
   - 系统播放风险提示（需完整观看）。
   - 逐条确认风险事项。
   - 朗读风险确认声明。
   - 全程录音录像保存。
4. **隐私保护**：双录文件由银行安全存储，仅用于合规审查和纠纷处理，不会对外泄露。
5. **双录耗时**：通常5-10分钟完成。

### Q8：赎回理财产品后资金什么时候到账？

**答：** 赎回资金到账时间因产品类型而异：

| 产品类型 | 赎回确认日 | 资金到账时间 |
|----------|-----------|-------------|
| 现金管理类（T+0） | 当日 | 当日或T+1日到账 |
| 开放式固收类 | T+1日 | T+2日到账 |
| 定期开放类 | 开放日 | 开放日后1-3个工作日 |
| 封闭式（到期） | 到期日 | 到期后1-3个工作日 |

**注意事项：**
- 工作日15:00后提交的赎回申请顺延至下一个工作日确认。
- 遇节假日到账时间顺延。
- 巨额赎回可能导致延期到账，银行会提前通知。

## 二、风险控制要点

### 2.1 投资者适当性风控

| 风控环节 | 具体措施 | 触发条件 |
|----------|----------|----------|
| 风险评估强制 | 首次购买前必须完成风险评估 | 无有效评估结果时拦截购买 |
| 等级匹配校验 | 系统自动校验客户等级与产品风险匹配 | 每次购买时自动校验 |
| 超风险购买拦截 | 超出风险等级需二次确认+签署确认书 | 客户等级低于产品风险等级 |
| 老年客户保护 | 65岁以上客户购买高风险产品需双录 | 客户年龄≥65且购买R3及以上 |
| 评估到期提醒 | 到期前30天/7天推送提醒 | 评估结果即将过期 |

### 2.2 交易风控

| 风控环节 | 具体措施 | 说明 |
|----------|----------|------|
| 冷静期管控 | 购买后设置24/48小时冷静期 | 冷静期内可无条件撤销 |
| 双录合规 | 大额/高风险产品销售全程录音录像 | 确保销售过程合规 |
| 交易限额 | 贵金属交易设置持仓上限 | 控制单一品种风险敞口 |
| 反洗钱监控 | 大额申购/赎回触发反洗钱审查 | 单笔50万元以上 |
| 异常交易检测 | 频繁申赎、短期大额交易监控 | 防范洗钱和市场操纵 |

### 2.3 产品端风控

| 风控环节 | 具体措施 | 说明 |
|----------|----------|------|
| 产品准入 | 严格审查代销产品资质和风险评级 | 确保产品质量和风险等级准确 |
| 信息披露 | 定期披露产品净值和投资报告 | 确保客户知情权 |
| 巨额赎回管控 | 单日赎回超10%可延期处理 | 防范流动性风险 |
| 产品风险评级 | 定期对在售产品重新评估风险等级 | 确保风险评级时效性 |

### 2.4 重点风控场景

**防范销售误导：**

- 产品页面强制展示风险提示语"理财非存款，产品有风险，投资需谨慎"。
- 禁止使用"保本""零风险""稳赚"等误导性用语。
- 历史业绩展示必须标注"历史业绩不代表未来收益"。
- 购买流程中设置多次确认环节，确保客户充分知情。
- 客户投诉渠道畅通，对销售误导投诉快速响应处理。

**防范洗钱风险：**

- 单笔申购金额超过50万元触发大额交易报告。
- 短期内频繁申赎同一产品（如7天内申赎超过3次）触发可疑交易预警。
- 多个账户短期内向同一理财账户转入资金触发关联分析。
- 保险产品大额趸交（一次性缴清）触发增强尽职调查。

**保护投资者权益：**

- 冷静期内无条件撤销购买，保障客户"后悔权"。
- 保险犹豫期内全额退保，降低客户决策风险。
- 定期向客户推送持仓报告和收益情况，保障知情权。
- 产品到期前提前通知客户做好资金安排。
- 客户投诉24小时内首次响应，15个工作日内给出处理结果。

**信息安全保护：**

- 客户风险评估结果严格保密，仅用于产品销售匹配。
- 双录文件安全存储，保存期限不少于产品到期后5年。
- 客户持仓信息加密存储，防止数据泄露。
- 贵金属交易信息实时加密传输，防止行情数据篡改。
