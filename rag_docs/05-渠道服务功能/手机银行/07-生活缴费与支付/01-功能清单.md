# 手机银行 — 生活缴费与支付功能清单

## 一、功能模块概述

生活缴费与支付板块是手机银行为客户提供日常民生场景的线上化支付服务入口，涵盖水、电、燃气等基础公用事业缴费，通信充值，交通罚款缴纳，社保公积金查询，以及扫码支付、收付款码、NFC支付等多种支付方式。该板块以"让数据多跑路，让客户少跑腿"为设计理念，致力于为客户提供一站式、全场景的生活缴费与支付体验。

|----------|----------|-------------|
| 账户余额支付 | 缴费、扫码 | 交易密码 / 生物识别 |

## 五、功能可用性说明

1. **地区差异**：部分缴费项目（如暖气费、有线电视费）仅在特定城市开放，系统根据客户定位或选择的城市自动展示可用项目。
2. **机构合作**：缴费项目依赖于我行与各公用事业单位/机构的合作协议，覆盖范围可能因地区而异。
3. **维护窗口**：缴费渠道可能因第三方系统维护在特定时段不可用，通常在每月账单结算期（每月1-5日）或系统升级期间。
4. **版本要求**：NFC支付功能需手机支持NFC且系统版本满足要求（iOS 13.0+ / Android 8.0+）。
