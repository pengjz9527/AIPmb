# 手机银行 — 生活缴费与支付用户操作流程

## 一、生活缴费操作流程

### 1.1 水费/电费/燃气费缴纳

1. 登录手机银行APP，在首页点击"生活缴费"入口，或在底部导航栏选择"生活"频道。
2. 系统自动定位当前城市，如需切换城市，点击页面顶部城市名称进行选择。
3. 选择缴费类别："水费"、"电费"或"燃气费"。
4. 选择对应的缴费单位（如所在城市有多家供水/供电公司，系统列出可选机构）。
5. 输入用户编号（户号）。如已保存常用户号，可直接从列表中选择。
6. 系统自动查询欠费金额或待缴账单，页面展示户名（部分隐藏）、户号、欠费金额等信息。
7. 确认缴费信息无误后，点击"立即缴费"。
8. 选择付款账户（从已绑定账户列表中选择）。
9. 输入交易密码或通过指纹/面容识别完成身份验证。
10. 系统提交缴费请求，页面展示"缴费处理中"。
11. 缴费成功后，页面跳转至缴费成功页面，展示缴费金额、户号、交易时间、交易流水号等信息。
12. 可选择"查看电子回单"、"继续缴费"或"返回首页"。

**温馨提示**：
- 首次缴费前建议核对户名，确认为本人或家人的缴费户号。
- 可点击"保存户号"将该户号添加至常用列表，下次缴费无需重复输入。

### 1.2 话费/流量充值

1. 进入"生活缴费"页面，选择"话费充值"。
2. 输入需要充值的手机号码（默认为当前登录手机号，可修改）。
3. 选择运营商（系统根据手机号自动识别运营商）。
4. 选择充值金额（快捷金额：30/50/100/200/300/500元，或自定义金额）。
5. 如选择流量充值，切换至"流量"标签页，选择流量包类型和有效期。
6. 确认充值信息（手机号、金额、运营商），点击"立即充值"。
7. 选择付款账户，输入交易密码或生物识别验证。
8. 充值成功后页面展示充值结果，通常1-5分钟内到账。

### 1.3 交通罚款缴纳

1. 进入"生活缴费"页面，选择"交通罚款"。
2. 首次使用需进行授权验证，同意查询交通违法信息。
3. 输入车牌号码、发动机号后6位（或车辆识别代号后6位）。
4. 系统查询并展示该车辆未处理的交通违法记录列表。
5. 选择需要缴纳的罚款记录（支持多选）。
6. 确认罚款信息（违法时间、地点、违法行为、罚款金额、扣分情况）。
7. 点击"确认缴纳"，选择付款账户并完成身份验证。
8. 缴纳成功后，系统提示"罚款已入库"，可保存电子缴款凭证。

### 1.4 学费缴纳

1. 进入"生活缴费"页面，选择"学费缴纳"。
2. 选择学校名称（可通过搜索功能快速定位）。
3. 输入学生学号或身份证号。
4. 系统查询并展示应缴费用明细（学费、住宿费、教材费等）。
5. 确认缴费项目和金额，点击"确认缴费"。
6. 选择付款账户，完成身份验证。
7. 缴费成功后，保存电子缴费凭证，可转发或下载PDF格式。

## 二、社保公积金查询操作流程

### 2.1 社保查询

1. 进入"生活缴费"页面，选择"社保查询"。
2. 首次查询需进行身份认证：输入姓名、身份证号，完成人脸识别验证。
3. 认证通过后，页面展示社保账户概览：养老保险个人账户余额、医疗保险个人账户余额等。
4. 点击具体险种（养老/医疗/失业/工伤/生育）查看详细缴纳明细。
5. 可按年度、月份筛选查询缴费记录。
6. 点击"缴费证明下载"可生成并下载社保缴费证明PDF文件。

### 2.2 公积金查询

1. 进入"生活缴费"页面，选择"公积金查询"。
2. 首次查询需进行身份认证并授权查询公积金信息。
3. 认证通过后，展示公积金账户信息：账户余额、月缴存额、缴存基数、缴存比例、缴存单位等。
4. 点击"缴存明细"查看按月缴存的详细记录。
5. 点击"贷款信息"可查看公积金贷款余额、还款计划（如有）。
6. 支持"公积金提取"线上申请（部分城市支持）。

## 三、扫码支付操作流程

### 3.1 主扫支付（我扫商户）

1. 登录手机银行APP，点击首页"扫一扫"图标或"扫码付"入口。
2. 打开手机摄像头，对准商户二维码进行扫描。
3. 扫描成功后，页面展示商户名称和商户信息。
4. 输入支付金额（部分商户预设金额则自动显示）。
5. 确认商户名称和支付金额无误。
6. 根据金额大小选择验证方式：
   - 1000元以下：可选择免密支付（需已开启）。
   - 1000元以上：输入交易密码或指纹/面容识别。
   - 大额支付（超5000元）：交易密码+短信验证码双重验证。
7. 验证通过后，系统完成扣款，页面展示"支付成功"。
8. 可查看交易详情，包括商户名称、交易金额、交易时间、订单号等。

### 3.2 被扫支付（商户扫我）

1. 点击首页"收付款"或"付款码"入口。
2. 页面展示当前付款码（条形码+二维码）。
3. 如未开启免密支付，需先输入交易密码后展示付款码。
4. 将付款码对准商户扫码设备。
5. 商户扫码成功后，系统自动完成扣款。
6. 页面展示"支付成功"及交易详情。
7. 如需刷新付款码，点击"刷新"按钮或等待60秒自动刷新。

### 3.3 收付款码管理

1. 进入"收付款"页面，可切换"付款码"和"收款码"。
2. **付款码**：展示给商户/他人扫描，向我方付款。
3. **收款码**：展示给他人扫描，他人向我方付款。
4. 收款码支持设置收款金额（可选），设置后他人扫码时自动填入金额。
5. 可切换收款账户（从已绑定账户中选择默认收款账户）。
6. 收款码支持保存为图片（含水印），用于线上收款。

## 四、NFC支付操作流程

### 4.1 NFC手机闪付设置与使用

1. 确认手机支持NFC功能，并在系统设置中开启NFC。
2. 打开手机银行APP，进入"我的"→"安全设置"→"NFC支付管理"。
3. 点击"开通手机闪付"，阅读并同意《手机闪付服务协议》。
4. 选择绑定银行卡（从已绑定账户中选择），完成短信验证码+交易密码验证。
5. 开通成功后，设置默认支付卡和免密限额。
6. **使用方式**：无需打开手机银行APP，解锁手机后将手机靠近POS机感应区域（通常为手机背面上半部分），即可完成支付。
7. 1000元以下交易无需验证，1000元以上需指纹/面容/密码验证。

### 4.2 NFC交通卡充值

1. 将实体交通卡贴附在手机背面NFC感应区域。
2. 打开手机银行APP，系统自动识别交通卡信息（卡号、余额）。
3. 选择充值金额（50/100/200元或自定义金额）。
4. 选择付款账户，完成身份验证。
5. 保持交通卡贴附在手机背面，等待充值写入完成（约3-5秒）。
6. 充值成功后，页面展示充值金额和卡内新余额。

## 五、缴费管理操作流程

### 5.1 缴费户号管理

1. 进入"生活缴费"页面，点击右上角"户号管理"或"管理"。
2. 查看已保存的缴费户号列表，按缴费类型分类展示。
3. **添加户号**：点击"添加户号"→选择缴费类型→选择缴费单位→输入户号→系统验证户号有效性→保存并设置别名（如"我家水费"）。
4. **编辑户号**：点击已有户号→修改别名或删除户号。
5. **设置缴费提醒**：在户号详情页开启"到期提醒"，设置提醒日期（通常为账单日前3天）。

### 5.2 自动缴费设置

1. 进入"生活缴费"页面，点击"自动缴费"或"代扣管理"。
2. 点击"新增代扣"，选择缴费项目和户号。
3. 选择代扣账户（用于自动扣款的银行卡）。
4. 阅读并同意《自动代扣授权协议》。
5. 输入交易密码，完成代扣签约。
6. 签约成功后，系统每月在账单生成后自动从指定账户扣缴。
7. 如需撤销代扣，在"代扣管理"中选择对应项目，点击"撤销代扣"并确认。

### 5.3 缴费记录查询与凭证下载

1. 进入"生活缴费"页面，点击"缴费记录"。
2. 展示近12个月的缴费记录列表，按时间倒序排列。
3. 支持按缴费类型（水/电/气/话费/罚款等）筛选。
4. 支持按时间范围筛选（近1个月/3个月/6个月/12个月/自定义）。
5. 点击具体缴费记录查看详情：缴费类型、缴费金额、缴费户号、交易时间、交易流水号。
6. 点击"下载电子回单"，生成PDF格式的电子回单文件。
7. 电子回单支持通过微信、邮件等方式分享。

## 六、常见问题处理

| 问题场景 | 处理步骤 |
|----------|----------|
| 缴费后未到账 | 1. 等待5-10分钟确认缴费状态 → 2. 在"缴费记录"查看交易状态 → 3. 若显示"缴费失败"，系统将在1-5个工作日自动退款 → 4. 若长时间未退款，联系客服 |
| 缴费户号无法识别 | 1. 核对户号是否正确 → 2. 确认所选缴费单位是否正确 → 3. 尝试联系对应机构核实户号 → 4. 如仍无法解决，前往柜面办理 |
| 扫码支付失败 | 1. 检查网络连接 → 2. 确认账户余额是否充足 → 3. 确认是否超过支付限额 → 4. 尝试更换支付方式 → 5. 如持续失败，联系客服 |
| NFC支付无响应 | 1. 确认手机NFC功能已开启 → 2. 确认手机闪付已开通 → 3. 调整手机与POS机的感应位置 → 4. 重启NFC功能后重试 |
