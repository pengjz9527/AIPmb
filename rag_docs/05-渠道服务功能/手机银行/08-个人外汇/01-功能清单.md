# 手机银行 — 个人外汇功能清单

## 一、功能模块概述

个人外汇板块是手机银行为客户提供个人外汇业务服务的功能模块，涵盖结汇（外币兑换人民币）、购汇（人民币兑换外币）、跨境汇款、外汇存款查询、汇率查询、年度便利化额度查询等核心功能。该板块严格遵守国家外汇管理局相关法规和政策，在年度便利化额度内为客户提供便捷的线上外汇服务，满足客户因私出境旅游、留学、商务等合理用汇需求。

## 四、功能可用性说明

1. **服务时间**：结汇、购汇交易时间为工作日9:30-22:00（部分银行延长至23:30），非交易时间仅可查询汇率。周末及法定节假日不办理结汇购汇业务。
2. **账户要求**：办理结汇购汇需持有我行I类借记卡并开通外汇功能。外汇账户须为一类账户或已开通外汇子账户的二类账户。
3. **实名认证**：所有外汇业务均需通过实名认证，首次使用需完成身份信息核实。
4. **地区限制**：部分城市分行的手机银行可能因当地外管政策差异，在功能范围或额度管理上有所不同。
