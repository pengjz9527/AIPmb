# 手机银行 — 个人外汇业务规则

## 一、年度便利化额度规则

### 1.1 额度基本规定

| 规则项 | 规则说明 |
|--------|----------|
| 年度便利化额度 | 每人每年等值5万美元（含结汇和购汇各自5万美元额度） |
| 额度计算周期 | 自然年（每年1月1日至12月31日） |
| 额度重置 | 每年1月1日自动重置，上年度未使用额度不结转 |
| 结汇额度 | 独立计算，年度累计结汇金额不超过等值5万美元 |
| 购汇额度 | 独立计算，年度累计购汇金额不超过等值5万美元 |
| 额度占用 | 每笔结汇/购汇交易按交易当日汇率折算为美元计入额度 |
| 额度查询 | 可通过手机银行、网上银行、柜面等渠道实时查询已用额度和剩余额度 |

### 1.2 超额度业务规则

| 规则项 | 规则说明 |
|--------|----------|
| 超额度结汇 | 需携带有效身份证件及真实性证明材料（如合同、发票、税单等）至柜面办理 |
| 超额度购汇 | 需携带有效身份证件及真实性证明材料至柜面办理，审核材料包括但不限于：留学需提供录取通知书及学费证明，旅游需提供行程单等 |
| 审核时效 | 柜面提交材料后，一般3-5个工作日完成审核 |
| 分拆规避 | 禁止通过分拆方式规避额度管理，系统将监测并预警异常分拆行为 |

### 1.3 分拆购汇监测规则

| 监测场景 | 触发条件 | 处理方式 |
|----------|----------|----------|
| 多人同日购汇汇出 | 5人以上同日向同一境外收款人购汇汇款 | 系统预警，纳入分拆监控名单 |
| 短期内频繁购汇 | 同一客户7天内购汇超过3次 | 系统预警，触发人工审核 |
| 蚂蚁搬家式购汇 | 多人（5人以上）在短期内分别购汇后汇至同一境外账户 | 系统预警，上报外汇管理局 |
| 接近额度上限购汇 | 单笔购汇金额接近等值5万美元（如4.8万-4.99万美元） | 系统标记，纳入重点关注 |

## 二、购汇业务规则

### 2.1 购汇用途申报

| 用途类别 | 具体用途 | 需提供材料（线上简化） | 单笔限额 |
|----------|----------|----------------------|----------|
| 因私旅游 | 出境旅游消费 | 线上申报即可 | 年度额度内 |
| 境外留学 | 学费、生活费 | 线上申报；超额度需提供录取通知书/学费单 | 年度额度内 |
| 公务及商务出行 | 差旅、商务活动 | 线上申报 | 年度额度内 |
| 探亲 | 探望境外亲属 | 线上申报 | 年度额度内 |
| 境外就医 | 境外医疗费用 | 线上申报；超额度需提供医疗证明 | 年度额度内 |
| 境外咨询 | 法律/会计/税务等专业咨询 | 线上申报 | 年度额度内 |
| 其他服务贸易 | 其他合法服务贸易项下 | 视具体情况 | 需柜面审核 |
| 境外投资 | 境外证券投资/房产投资 | **不支持** — 个人外汇不得用于境外证券投资及购买境外房产 | — |

### 2.2 购汇合规要求

1. **真实性声明**：客户购汇时需签署《个人购汇申请书》，声明购汇用途真实合法。
2. **禁止用途**：个人购汇不得用于境外买房、证券投资、购买人寿保险和投资性返还分红类保险等尚未开放的资本项目。
3. **不得出借额度**：个人不得出借年度便利化额度协助他人购汇。
4. **不得分拆**：不得通过多人次分拆方式规避额度及真实性管理。
5. **违规处罚**：违反外汇管理规定的，外汇管理局可处以罚款，并将违规信息纳入个人征信记录。

### 2.3 购汇交易规则

| 规则项 | 规则说明 |
|--------|----------|
| 交易时间 | 工作日9:30-22:00（具体以各分行公告为准） |
| 汇率基准 | 按交易提交时我行实时公布的外汇卖出价执行 |
| 汇率锁定 | 提交交易后汇率即时锁定，不接受事后汇率变动申诉 |
| 最低金额 | 购汇最低金额为等值100元人民币 |
| 资金到账 | 购汇成功后外币实时存入客户外汇账户 |
| 交易撤销 | 交易确认后不可撤销；交易提交后未确认的可在15秒内取消 |

## 三、结汇业务规则

### 3.1 结汇交易规则

| 规则项 | 规则说明 |
|--------|----------|
| 结汇资金来源 | 客户外汇账户内的现汇或现钞存款 |
| 现汇结汇汇率 | 按实时现汇买入价执行（优于现钞买入价） |
| 现钞结汇汇率 | 按实时现钞买入价执行 |
| 最低金额 | 结汇最低金额为等值1美元 |
| 资金到账 | 结汇成功后人民币实时存入客户人民币账户 |
| 交易时间 | 工作日9:30-22:00 |

### 3.2 结汇特殊规则

1. **现钞与现汇差异**：现钞结汇汇率低于现汇结汇汇率，因现钞存在运输和保管成本。
2. **外币现钞存入**：客户将外币现钞存入外汇账户后再结汇，存入时需至柜面办理。
3. **大额结汇**：单笔结汇金额超过等值1万美元，需进行交易用途说明。
4. **来源合规**：结汇资金来源须合法合规，大额结汇可能触发反洗钱审查。

## 四、跨境汇款业务规则

### 4.1 汇出汇款规则

| 规则项 | 规则说明 |
|--------|----------|
| 汇款币种 | 支持美元、欧元、英镑、日元、港币、澳元、加元、瑞士法郎等主要币种 |
| 汇款金额 | 单笔不超过等值5万美元（便利化额度内），超额度需柜面办理 |
| 汇款用途 | 须与购汇申报用途一致，不得用于禁止用途 |
| 收款信息要求 | 需提供收款人姓名、收款银行名称、银行SWIFT代码、收款人账号/IBAN |
| 汇款手续费 | 手续费+电报费（按我行收费标准执行，VIP客户可享优惠） |
| 中间行费用 | 汇款可能经过中间行转汇，中间行费用从汇款金额中扣除（OUR/SHA/BEN模式可选） |
| 汇款时效 | 一般1-3个工作日到账，具体视收款行所在国家/地区及汇款路径而定 |

### 4.2 跨境汇款审核要求

| 审核场景 | 审核内容 | 审核时效 |
|----------|----------|----------|
| 便利化额度内汇款 | 系统自动审核：身份验证、额度校验、黑名单筛查 | 实时或当日完成 |
| 超额度汇款 | 人工审核：需提供真实性证明材料 | 3-5个工作日 |
| 敏感国家/地区 | 加强审核：对汇款至高风险国家/地区的汇款进行强化审查 | 5-10个工作日 |
| 大额汇款 | 单笔超过等值1万美元需额外信息确认 | 当日或T+1完成 |
| 反洗钱筛查 | 所有跨境汇款均需通过OFAC/联合国/中国反洗钱名单筛查 | 实时筛查，命中则转入人工处理 |

### 4.3 汇入汇款规则

| 规则项 | 规则说明 |
|--------|----------|
| 入账方式 | 小额汇入（等值5000美元以下）自动入账；大额汇入通知客户确认后入账 |
| 申报要求 | 汇入金额超过等值5000美元，客户需进行国际收支申报 |
| 申报时效 | 收到通知后5个工作日内完成申报，逾期未申报可能影响后续入账 |
| 结汇处理 | 汇入外币可选择保留外币入账或结汇为人民币入账 |

## 五、外汇管制规定

### 5.1 外汇管理核心规定

1. **经常项目可兑换**：个人因私旅游、留学、商务等经常项目下的外汇需求，在便利化额度内凭有效身份证件即可办理。
2. **资本项目管制**：个人资本项目（境外投资、境外购房、境外证券投资等）尚未开放，个人不得通过分拆等方式规避管制。
3. **个人外汇账户管理**：个人外汇账户分为外汇储蓄账户和外汇结算账户，个人客户使用外汇储蓄账户。
4. **外币现钞管理**：个人提取外币现钞，当日累计等值1万美元以下直接在柜面办理；超过等值1万美元需提前预约并提供用途说明。

### 5.2 违规处理

| 违规行为 | 处理措施 |
|----------|----------|
| 虚假申报购汇用途 | 纳入个人外汇"关注名单"，2年内取消便利化额度 |
| 分拆购汇/结汇 | 纳入"关注名单"，2年内取消便利化额度，并可能处以罚款 |
| 出借额度 | 双方均纳入"关注名单"，2年内取消便利化额度 |
| 非法套汇 | 由外汇管理机关给予警告，强制兑回，并处以罚款 |
| 洗钱行为 | 移交司法机关处理，依法追究刑事责任 |

## 六、汇率管理规则

| 规则项 | 规则说明 |
|--------|----------|
| 汇率来源 | 以我行外汇牌价为准，参考银行间外汇市场汇率 |
| 汇率更新频率 | 交易时间内每30秒-1分钟更新一次 |
| 汇率类型 | 现汇买入价、现汇卖出价、现钞买入价、现钞卖出价、中间价 |
| 点差说明 | 买入价与卖出价之间的差额为银行点差收益 |
| 汇率锁定 | 交易提交后按当时汇率执行，不支持汇率回溯 |
| 大额汇率优惠 | 单笔结汇/购汇超过等值10万美元，可申请优惠汇率（需柜面或客户经理办理） |
