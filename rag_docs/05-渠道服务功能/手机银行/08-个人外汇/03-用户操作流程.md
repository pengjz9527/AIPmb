# 手机银行 — 个人外汇用户操作流程

## 一、购汇操作流程

### 1.1 人民币购汇（便利化额度内）

1. 登录手机银行APP，在首页点击"外汇"或在底部导航栏找到"外汇"频道入口。
2. 进入外汇主页，查看实时汇率行情和各币种汇率概览。
3. 点击"购汇"按钮，进入购汇页面。
4. 选择购汇币种（美元、欧元、英镑、日元等）。
5. 系统展示当前该币种的实时卖出价（即购汇价格）。
6. 输入购汇金额（外币金额或人民币金额，系统自动按汇率换算）。
7. 选择购汇用途：
   - 因私旅游（选择目的地国家/地区、预计出行日期）
   - 境外留学（选择学校所在国家、学费/生活费）
   - 公务及商务出行（选择目的地、出差事由）
   - 探亲（选择亲属所在国家）
   - 其他（选择具体用途类别）
8. 阅读并确认《个人购汇申请书》，声明购汇用途真实合法，不涉及境外买房、证券投资等禁止用途。
9. 选择购汇资金存入的外汇账户（如有多个外汇账户可选择）。
10. 确认购汇信息：币种、金额、汇率、折合人民币金额、购汇用途。
11. 输入交易密码或通过指纹/面容识别完成身份验证。
12. 购汇成功后，系统展示交易确认页：交易币种、金额、成交汇率、交易时间、剩余额度。
13. 可点击"查看交易记录"或"查看额度"进行后续操作。

**温馨提示**：
- 购汇汇率以提交时的实时汇率为准，页面展示的参考汇率可能因市场波动与实际成交汇率存在差异。
- 请如实申报购汇用途，虚假申报将影响个人外汇信用记录。

### 1.2 超额度购汇申请

1. 在手机银行外汇页面点击"购汇"，输入金额后如系统提示已超额度，点击"超额度购汇申请"。
2. 选择购汇用途类别，系统提示需要提供的证明材料清单。
3. 根据提示上传证明材料照片（如录取通知书、学费缴费单、医疗证明等）。
4. 填写补充说明（如有特殊需求）。
5. 提交申请后，系统提示"您的超额度购汇申请已提交，预计3-5个工作日完成审核"。
6. 可在"我的申请"中查看审核进度。
7. 审核通过后，需至柜面完成购汇操作（目前超额度购汇暂不支持全线上完成）。

## 二、结汇操作流程

### 2.1 外币结汇（便利化额度内）

1. 进入手机银行"外汇"页面，点击"结汇"按钮。
2. 选择结汇币种，系统自动展示该客户账户下各币种外币余额。
3. 输入结汇金额（外币金额），系统按实时买入价折算人民币金额。
4. 选择结汇资金类型：现汇结汇或现钞结汇（页面展示不同的买入价）。
5. 确认结汇信息：币种、结汇金额、成交汇率、折合人民币金额。
6. 选择结汇后人民币入账账户。
7. 输入交易密码或通过指纹/面容识别完成身份验证。
8. 结汇成功后，人民币实时存入指定账户。
9. 查看交易确认页，可保存或分享交易凭证。

### 2.2 汇率提醒设置

1. 在"外汇"页面点击"汇率提醒"入口。
2. 选择关注币种（如USD/CNY）。
3. 设置提醒条件：
   - 当卖出价低于某一值时提醒（适合计划购汇的客户）
   - 当买入价高于某一值时提醒（适合计划结汇的客户）
4. 设置提醒方式：APP推送/短信（可同时选择）。
5. 设置提醒有效期（如30天内有效）。
6. 保存提醒设置，系统将在汇率达到条件时自动推送通知。
7. 可在"汇率提醒管理"中查看、修改或删除已设置的提醒。

## 三、跨境汇款操作流程

### 3.1 汇出汇款

1. 进入"外汇"页面，点击"跨境汇款"或"汇出汇款"。
2. 首次使用需阅读并签署《跨境汇款服务协议》。
3. 选择汇款币种和汇出账户。
4. 填写收款人信息：
   - 收款人姓名（英文或当地语言）
   - 收款人地址（国家/地区、城市）
   - 收款银行名称
   - 收款银行SWIFT代码（8位或11位）
   - 收款银行地址
   - 收款人账号或IBAN号码
   - 收款人联系电话（部分国家要求）
5. 如已保存常用收款人，可从列表中选择，无需重复填写。
6. 输入汇款金额（外币金额）。
7. 选择费用承担方式：
   - OUR（汇款人承担全部费用）：收款人收到全额
   - SHA（共同承担）：汇款人付本行费用，收款人付中间行费用
   - BEN（收款人承担）：全部费用从汇款金额中扣除
8. 填写汇款附言/用途说明。
9. 系统展示汇款信息确认页：汇款金额、手续费、电报费、预计到账时间、收款人信息。
10. 确认信息无误后，输入交易密码+短信验证码完成身份验证。
11. 大额汇款（超过等值1万美元）可能需要额外的人脸识别验证。
12. 提交成功后，系统展示汇款回执：汇款编号、汇款金额、手续费、预计到账日期。
13. 可在"汇款记录"中查看汇款状态：处理中→已汇出→已到账。

### 3.2 汇入汇款确认

1. 当有境外汇款入账时，手机银行推送通知"您有一笔境外汇入款项"。
2. 点击通知或进入"外汇"→"汇入汇款"查看详情。
3. 展示汇入信息：汇款金额、币种、汇款国家、汇款人信息。
4. 如金额超过等值5000美元，需填写国际收支申报信息：
   - 交易性质（如服务贸易/货物贸易/个人间转移等）
   - 交易编码（根据申报指引选择）
   - 交易附言
5. 选择入账方式：
   - 保留外币入账（存入外汇账户）
   - 结汇入人民币账户（按当时买入价结汇）
6. 确认入账后，资金进入指定账户。

### 3.3 常用收款人管理

1. 在"跨境汇款"页面，点击"收款人管理"。
2. 点击"添加收款人"，填写收款人姓名、银行信息、SWIFT代码、账号等。
3. 保存收款人信息，可设置别名（如"美国学校学费"）。
4. 下次汇款时，在收款人列表中选择即可快速填入信息。
5. 支持编辑和删除已保存的收款人。
6. 建议为常用收款人设置别名，方便识别和管理。

## 四、外汇账户查询操作流程

### 4.1 外汇存款查询

1. 进入"外汇"页面，点击"我的外汇账户"或"外汇资产"。
2. 页面展示外汇账户概览：
   - 各币种活期存款余额
   - 各币种定期存款余额
   - 折合人民币总资产（按当日中间价折算）
3. 点击具体币种查看该币种的账户详情：
   - 现汇余额/现钞余额
   - 可用余额/冻结金额
   - 近期交易明细
4. 交易明细支持按时间、交易类型（存入/支出/结汇/购汇/汇入/汇出）筛选。

### 4.2 外币定期存款

1. 在"外汇账户"页面，点击"外币定期存款"。
2. 选择存款币种和存款期限（1个月/3个月/6个月/1年）。
3. 系统展示各期限的存款利率。
4. 输入存款金额（从活期账户转入）。
5. 确认存款信息（币种、金额、期限、利率、到期日）。
6. 输入交易密码完成身份验证。
7. 存款成功后，资金从活期转入定期，到期后自动转回活期（可设置自动续存）。

## 五、额度查询操作流程

### 5.1 年度便利化额度查询

1. 进入"外汇"页面，点击"额度查询"。
2. 页面展示年度便利化额度概览：
   - **购汇额度**：年度总额度5万美元，已使用X美元，剩余额度X美元。
   - **结汇额度**：年度总额度5万美元，已使用X美元，剩余额度X美元。
   - 进度条直观展示额度使用情况。
3. 点击"查看明细"查看本年度各笔购汇/结汇记录：
   - 交易日期
   - 交易类型（购汇/结汇）
   - 币种及金额
   - 折合美元金额
   - 交易用途
4. 页面底部展示下一年度额度恢复日期：1月1日。
5. 如额度已用尽，系统提示"您本年度便利化额度已用尽，如需继续办理外汇业务，请携带相关材料至柜面办理"。

## 六、汇率查询与换算操作流程

### 6.1 汇率查询

1. 进入"外汇"页面，点击"汇率查询"或首页汇率牌价入口。
2. 查看外汇牌价表：列出所有支持币种的买入价、卖出价、中间价。
3. 点击某一币种查看详细信息：
   - 现汇买入价（银行买入客户现汇的价格）
   - 现汇卖出价（银行卖出外汇给客户的价格，即客户购汇价格）
   - 现钞买入价（银行买入客户现钞的价格）
   - 现钞卖出价（银行卖出外币现钞的价格）
   - 中间价
4. 点击"走势图"查看该币种汇率的历史变化曲线。
5. 可选择时间区间：日/周/月/3个月/半年/1年。

### 6.2 汇率换算器

1. 在"汇率查询"页面点击"汇率换算"。
2. 选择源币种和目标币种。
3. 输入金额。
4. 系统实时计算并展示换算结果（按当前中间价/买入价/卖出价分别展示）。
5. 可切换源币种和目标币种方向。
6. 支持常用货币对的快速切换（USD/CNY、EUR/CNY、GBP/CNY、JPY/CNY等）。

## 七、常见问题与处理

| 问题场景 | 处理步骤 |
|----------|----------|
| 购汇提示额度不足 | 1. 在"额度查询"确认剩余额度 → 2. 减少购汇金额至额度内 → 3. 如需超额度购汇，提交超额度申请至柜面办理 |
| 汇率页面无法加载 | 1. 检查是否在交易时间内（工作日9:30-22:00） → 2. 检查网络连接 → 3. 尝试退出重进 → 4. 如仍异常，联系客服 |
| 跨境汇款状态长时间"处理中" | 1. 在"汇款进度查询"查看最新状态 → 2. 一般1-3个工作日到账，部分国家需5个工作日 → 3. 超过5个工作日未到账，联系客服查询 |
| 汇入款项未收到通知 | 1. 检查消息通知设置是否开启 → 2. 在"外汇账户"查看是否有未入账汇入款 → 3. 联系汇款人确认汇款是否成功发出 → 4. 提供SWIFT MT103报文号联系客服追踪 |
