# 手机银行 — 安全设置与个人信息功能清单

## 一、功能模块概述

安全设置与个人信息板块是手机银行的核心安全管理模块，为客户提供全面的账户安全防护能力。涵盖登录密码管理、交易密码管理、生物识别认证设置、设备管理、手机号修改、个人信息维护、隐私管理及消息通知设置等功能。该板块以"安全、便捷、合规"为原则，在保障账户安全的同时，提供流畅的用户体验，确保客户在享受便捷金融服务的同时，个人信息和资金安全得到有效保障。

## 三、安全认证方式对照

| 认证方式 | 安全等级 | 适用场景 | 组合使用 |
|----------|----------|----------|----------|
| 登录密码 | 中 | 登录手机银行 | 可与短信验证组合 |
| 交易密码（6位数字） | 中高 | 转账、支付、缴费等交易 | 可与短信验证组合 |
| 短信验证码 | 中 | 辅助验证 | 与密码组合使用 |
| 指纹识别 | 中高 | 登录、小额交易 | 可替代密码 |
| 面容识别 | 高 | 登录、交易验证 | 可替代密码 |
| 人脸识别（活体检测） | 高 | 密码重置、大额交易、身份验证 | 与短信验证组合 |
| U盾/数字证书 | 最高 | 大额转账（50万以上） | 独立使用 |

## 四、功能可用性说明

1. **设备兼容性**：生物识别功能依赖设备硬件支持，部分老旧设备可能不支持指纹或面容识别。
2. **系统版本要求**：安全设置功能需手机银行APP 5.0及以上版本，建议客户保持APP最新版本。
3. **网络要求**：部分安全设置操作（如密码修改、设备绑定）需在联网环境下完成。
4. **柜面业务**：手机号修改、交易密码重置、姓名变更、证件号码变更等关键信息修改需至柜面办理，不可通过线上完成。
