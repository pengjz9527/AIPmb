# 手机银行 — 适老化与无障碍用户操作流程

## 一、版本切换操作流程

### 1.1 首次登录切换至大字版

1. 打开或更新手机银行APP至最新版本。
2. 登录手机银行账号。
3. 系统检测到设备系统字体较大或用户年龄为60岁以上时，自动弹出推荐提示："为您推荐大字版（关爱版），字体更大、操作更简单，是否切换？"
4. 点击"切换大字版"，系统立即切换至大字版界面。
5. 如不需要切换，点击"使用标准版"继续使用默认界面。
6. 切换成功后，首页展示大字版界面：超大字体显示账户余额，大图标展示常用功能。
7. 版本选择自动保存，下次登录直接进入大字版。

### 1.2 手动切换版本

1. 在手机银行任意界面，点击右上角"设置"图标（大字版中设置图标较大，易于点击）。
2. 在设置页面中找到"版本切换"或"大字版/标准版切换"选项。
3. 点击"大字版"或"标准版"进行切换。
4. 系统提示"即将切换至[大字版/标准版]，是否确认？"
5. 点击"确认切换"，页面刷新后进入目标版本。
6. 如在标准版中，也可通过"我的"→"设置"→"大字版"进行切换。

### 1.3 子女远程协助切换

1. 子女在自己的手机银行中进入"亲情账户"功能。
2. 选择已绑定的父母账户。
3. 点击"远程协助"→"切换大字版"。
4. 系统推送通知至父母的手机银行："您的家人[姓名]建议将手机银行切换为大字版"。
5. 父母在通知中点击"同意切换"，手机银行自动切换至大字版。
6. 如父母未在APP中操作，也可在下次打开APP时看到切换提示。

## 二、大字版常用功能操作流程

### 2.1 查看账户余额

1. 打开大字版手机银行，首页顶部以超大字体显示已绑定银行卡的账户余额。
2. 如绑定多张卡，左右滑动切换不同银行卡。
3. 点击"查看明细"按钮（大按钮，易于点击），进入交易明细页面。
4. 交易明细以大字体列表展示：每笔交易的日期、金额、交易类型、对方户名。
5. 点击某笔交易可查看交易详情（交易流水号、交易时间、交易类型等详细信息）。
6. 如开启了语音播报功能，系统自动朗读账户余额信息。
7. 点击页面左上角"返回"按钮返回首页。

### 2.2 大字版转账

1. 在首页点击"转账"大图标按钮。
2. 进入转账页面，页面布局简洁清晰：
   - 收款人信息输入区（大输入框）
   - 金额输入区（超大数字键盘）
   - 确认按钮区（大按钮）
3. 选择收款人：
   - **常用收款人**：点击"常用收款人"按钮，从列表中选择（大字体展示姓名和卡号后四位）。
   - **新收款人**：点击"输入新收款人"，输入收款人姓名和银行卡号（大输入框，数字键盘自动弹出）。
4. 输入转账金额：
   - 使用超大数字键盘输入金额。
   - 金额以特大字体显示在输入框中。
   - 系统实时显示账户可用余额供参考。
5. 如开启了语音功能，可点击麦克风图标，语音说出"给[姓名]转[金额]元"。
6. 点击"下一步"按钮进入确认页面。
7. 确认页面以超大字体展示：收款人姓名、收款卡号、转账金额、手续费。
8. 系统语音播报转账信息："即将向[姓名]转账[金额]元，请确认"。
9. 确认无误后，点击"确认转账"大按钮。
10. 输入交易密码（大键盘）或使用指纹/面容验证。
11. 转账成功后，页面以绿色大字显示"转账成功"，并语音播报结果。
12. 可点击"继续转账"或"返回首页"。

### 2.3 大字版生活缴费

1. 在首页点击"生活缴费"大图标按钮。
2. 进入缴费页面，展示常用缴费类型大图标：
   - 水费（蓝色水滴图标+文字"水费"）
   - 电费（黄色闪电图标+文字"电费"）
   - 燃气费（橙色火焰图标+文字"燃气费"）
   - 话费（绿色电话图标+文字"话费"）
3. 点击需要缴费的类型。
4. 如已保存常用户号，系统展示户号列表，点击选择即可。
5. 如未保存户号，进入输入页面，输入缴费户号。
6. 系统查询并展示缴费信息（户名、欠费金额），以大字体显示。
7. 确认缴费信息无误后，点击"立即缴费"大按钮。
8. 输入交易密码或使用生物识别验证。
9. 缴费成功后，页面以大字体显示"缴费成功"，语音播报"水费/电费/燃气费缴费成功，金额[XX]元"。
10. 系统提示"是否保存此户号方便下次使用？"点击"保存"或"不保存"。

### 2.4 大字版话费充值

1. 在首页点击"话费"大图标按钮。
2. 系统自动填入当前登录手机号（大字体显示），如需为他人充值可修改号码。
3. 展示常用充值金额选项（大按钮）：30元、50元、100元、200元。
4. 点击选择充值金额。
5. 确认充值信息（手机号、金额），点击"立即充值"。
6. 输入交易密码或生物识别验证。
7. 充值成功后，语音播报"话费充值成功，金额[XX]元"。

## 三、语音辅助使用操作流程

### 3.1 语音搜索功能

1. 在手机银行任意页面，点击搜索框旁的麦克风图标。
2. 系统提示"请说出您想要的功能"，手机进入语音识别状态。
3. 说出功能需求，例如：
   - "我要查余额" → 自动跳转至账户余额页面
   - "我要转账" → 自动跳转至转账页面
   - "交电费" → 自动跳转至电费缴费页面
   - "查信用卡账单" → 自动跳转至信用卡账单页面
   - "买理财" → 自动跳转至理财产品页面
4. 系统识别语音指令后，自动导航至对应功能页面。
5. 如识别不准确，系统展示"您是不是想找："并列出相似功能供选择。

### 3.2 语音转账

1. 进入转账页面，点击页面底部的麦克风图标（语音转账入口）。
2. 系统提示"请说出收款人和金额"。
3. 说出转账指令，如："给张三的工商银行卡转2000元"。
4. 系统解析语音指令，自动填入收款人（从常用收款人中匹配）、金额等信息。
5. 页面展示解析结果，语音播报："将为收款人张三转账2000元，请确认"。
6. 如信息正确，说"确认"或点击"确认转账"按钮。
7. 如信息有误，说"取消"或点击"取消"按钮，重新输入。
8. 确认后输入交易密码或使用生物识别验证。
9. 转账成功后语音播报结果。

### 3.3 语音播报功能

1. 进入"设置"→"语音设置"→"语音播报"。
2. 开启"页面内容朗读"开关。
3. 选择朗读范围：
   - 全部朗读：朗读所有页面文字内容
   - 仅关键信息：仅朗读金额、操作结果等关键信息
   - 仅操作提示：仅朗读操作引导和提示信息
4. 开启后，在浏览各页面时，系统将自动朗读页面关键内容。
5. 也可在任意页面点击"朗读本页"悬浮按钮，手动触发当前页面朗读。
6. 朗读时点击页面任意位置可暂停朗读。

### 3.4 语音客服

1. 在首页或"客户服务"页面，点击"语音客服"入口。
2. 系统接通智能语音客服，提示"您好，我是智能客服小助手，请问有什么可以帮您？"
3. 说出您的问题或需求，例如：
   - "我的卡被吞了怎么办"
   - "怎么修改密码"
   - "最近的理财产品有哪些"
   - "帮我查一下上个月的交易记录"
4. 智能客服理解问题后，语音回答并同时在页面展示文字答案。
5. 如需转人工客服，说"转人工"或点击"转人工客服"按钮。
6. 等待人工客服接通后继续沟通。

## 四、无障碍功能使用流程

### 4.1 读屏软件使用（Android TalkBack）

1. 在手机"设置"→"无障碍"→"TalkBack"中开启TalkBack功能。
2. 打开手机银行APP，TalkBack自动朗读欢迎语和登录页面元素。
3. **基本操作**：
   - 单指左右滑动：依次切换焦点到不同UI元素，TalkBack朗读当前元素名称和状态。
   - 单指双击：激活当前焦点元素（等同于点击）。
   - 双指上下滑动：滚动页面内容。
   - 双指向右滑动：返回首页。
4. 登录流程：TalkBack朗读"登录密码输入框"→双击激活→输入密码→朗读"登录按钮"→双击登录。
5. 查看余额：TalkBack朗读"账户余额"及金额数字。
6. 所有操作元素均有完整的无障碍标签，TalkBack可准确朗读。

### 4.2 读屏软件使用（iOS VoiceOver）

1. 在iPhone"设置"→"辅助功能"→"VoiceOver"中开启VoiceOver。
2. 打开手机银行APP，VoiceOver自动朗读页面内容。
3. **基本操作**：
   - 单指左右滑动：切换焦点元素。
   - 单指双击：激活元素。
   - 三指上下滑动：滚动页面。
   - 双指搓动（转子操作）：切换朗读模式（按标题/按链接/按按钮等）。
4. 使用转子功能快速定位到"按钮"或"表单"元素，提高操作效率。

### 4.3 色盲友好模式

1. 进入"设置"→"显示设置"→"色盲友好模式"。
2. 开启后，页面中所有依赖颜色传达的信息增加替代标识：
   - 涨跌信息：增加文字"涨"/"跌"和箭头符号。
   - 状态信息：成功（绿色+勾号）、失败（红色+叉号）、警告（黄色+叹号）。
   - 图表数据：不同数据系列使用不同线型和图案区分。

### 4.4 动画减弱设置

1. 进入"设置"→"显示设置"→"减少动画效果"。
2. 开启后，页面切换动画、过渡效果、加载动画均被减弱或关闭。
3. 页面切换变为即时跳转，减少视觉干扰。
4. 适用于对动画效果敏感的用户（如前庭功能障碍患者）。

## 五、亲情账户操作流程

### 5.1 子女绑定父母账户

1. 子女在自己的手机银行中，进入"我的"→"亲情账户"。
2. 点击"添加亲情账户"→"添加父母"。
3. 输入父母的银行卡号或手机号。
4. 系统验证账户信息后，发送绑定邀请至父母的手机银行。
5. 父母在自己的手机银行中收到邀请通知："您的子女[姓名]邀请您开通亲情账户"。
6. 父母阅读《亲情账户服务协议》，确认授权范围（余额查看、交易明细查看、动账通知等）。
7. 父母点击"同意绑定"，输入交易密码确认。
8. 绑定成功后，子女可在"亲情账户"中查看父母的账户概览。

### 5.2 子女远程协助父母设置

1. 子女在"亲情账户"中选择父母的账户。
2. 点击"远程协助"。
3. 选择协助项目：
   - **切换大字版**：推送切换建议至父母手机。
   - **设置常用户号**：为父母添加水电燃气缴费户号。
   - **设置常用收款人**：添加父母常用的转账收款人。
   - **设置交易限额**：建议降低交易限额。
4. 操作完成后，父母端收到确认通知。
5. 父母确认后，设置即时生效。

### 5.3 子女查看父母账户动账通知

1. 在亲情账户中，进入"通知设置"。
2. 开启"同步动账通知"（需父母授权）。
3. 开启后，父母账户发生资金变动时，子女手机银行同步收到通知。
4. 通知内容包含：交易类型、交易金额、交易时间、对方户名。
5. 如发现异常交易，可点击通知中的"异常报告"按钮联系客服。

## 六、常见问题处理

| 问题场景 | 处理步骤 |
|----------|----------|
| 大字版界面显示不完整 | 1. 确认APP已更新至最新版本 → 2. 检查系统字体大小设置是否超出支持范围 → 3. 尝试重启APP → 4. 如仍有问题，联系客服 |
| 语音识别不准确 | 1. 确保环境安静，减少背景噪音 → 2. 将手机靠近嘴边说话 → 3. 说话速度适中、吐字清晰 → 4. 检查手机麦克风是否正常 → 5. 尝试切换至手动输入 |
| 读屏软件朗读异常 | 1. 确认APP已更新至最新版本 → 2. 检查TalkBack/VoiceOver是否为最新版本 → 3. 尝试重启读屏软件和APP → 4. 如仍有问题，联系客服反馈 |
| 子女无法绑定父母账户 | 1. 确认输入的父母银行卡号/手机号正确 → 2. 确认父母账户状态正常 → 3. 确认父母手机银行已安装并登录 → 4. 如仍无法绑定，双方携带证件至柜面办理 |
