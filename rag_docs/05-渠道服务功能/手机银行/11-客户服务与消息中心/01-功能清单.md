# 手机银行 — 客户服务与消息中心功能清单

## 一、功能模块概述

客户服务与消息中心板块是手机银行连接客户与银行的服务枢纽，提供多渠道客服接入、智能化问题解答、工单管理、消息推送、电子回单等综合服务功能。该板块以"客户为中心"为服务理念，通过智能客服机器人实现7×24小时即时响应，人工客服提供专业深度服务，消息中心确保客户及时获取账户变动、业务提醒、营销活动等重要信息，构建全方位、多层次的客户服务体系。

## 六、功能可用性说明

1. **智能客服**：7×24小时全天候服务，可处理80%以上的常见咨询问题。
2. **人工客服**：工作时间为工作日8:00-20:00、周末及法定节假日9:00-17:00。
3. **消息推送**：依赖手机APP推送通道（APNs/FCM/各厂商推送通道），需确保手机允许APP推送通知。
4. **电子回单**：电子回单采用电子签名和防篡改技术，与纸质回单具有同等法律效力。
5. **网络要求**：在线客服和视频客服需要稳定的网络连接，建议使用WiFi或4G/5G网络。
