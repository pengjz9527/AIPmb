# 手机银行 — 客户服务与消息中心用户操作流程

## 一、智能客服使用操作流程

### 1.1 进入智能客服

1. 登录手机银行APP，在首页点击右下角悬浮的"客服"图标，或在底部导航栏选择"我的"→"客户服务"。
2. 进入智能客服对话界面，页面顶部显示"智能客服小助手"。
3. 系统自动发送欢迎语："您好！我是智能客服小助手，请问有什么可以帮您？"
4. 页面下方展示"常见问题"快捷入口列表，如"查余额"、"转账问题"、"密码修改"、"信用卡还款"等。

### 1.2 文字咨询

1. 在底部输入框中输入问题，如"怎么修改交易密码？"
2. 点击"发送"按钮或按回车键发送消息。
3. 智能客服分析问题后，以文字+图文卡片形式回复答案。
4. 如回复中涉及操作指引，系统展示步骤说明和"去办理"快捷按钮。
5. 点击"去办理"可直接跳转至对应功能页面。
6. 如回复不满意，点击回答下方的"不满意"按钮，系统追问具体原因并尝试给出更精准的答案。
7. 连续追问2-3次后，系统主动建议："是否需要为您转接人工客服？"

### 1.3 快捷入口使用

1. 在对话页面下方的"猜您想问"区域，点击常见问题快捷标签。
2. 系统自动发送该问题并展示答案。
3. 常见快捷问题包括：
   - "我的账户余额是多少？"
   - "最近有什么优惠活动？"
   - "怎么修改登录密码？"
   - "信用卡怎么还款？"
   - "转账限额是多少？"
4. 点击快捷入口等同于手动输入对应问题，获得相同的回复。

### 1.4 自助业务办理

1. 在智能客服对话中，输入"挂失"或"口头挂失"。
2. 智能客服回复挂失流程说明，并展示"立即挂失"按钮。
3. 点击"立即挂失"，系统展示需要挂失的银行卡列表。
4. 选择需要挂失的银行卡，确认挂失信息。
5. 输入交易密码完成身份验证。
6. 挂失成功后，智能客服回复"您的银行卡已成功挂失，请及时前往网点办理正式挂失和补卡"。
7. 类似流程适用于密码重置引导、信用卡临时额度调整等简单业务。

## 二、人工客服使用操作流程

### 2.1 在线文字客服

1. 在智能客服对话中，输入"转人工"或点击"转人工客服"按钮。
2. 系统显示排队信息："当前排队[X]人，预计等待[X]分钟"。
3. 排队期间，智能客服继续提供服务，可随时取消排队。
4. 人工客服接入后，页面顶部显示"人工客服[工号/姓名]为您服务"。
5. 人工客服可查看之前的智能客服对话记录，无需客户重复描述问题。
6. 与客服人员进行文字对话，客服人员可发送文字、图片、产品链接、操作指引等。
7. 如需发送截图说明问题，点击输入框旁的"+"号→"相册"→选择图片发送。
8. 问题解决后，点击页面顶部"结束会话"按钮。
9. 会话结束后，系统弹出服务评价页面（1-5星评分+文字评价）。
10. 完成评价后，会话记录自动保存，可在"历史会话"中查看。

### 2.2 视频客服

1. 在"客户服务"页面，点击"视频客服"入口。
2. 系统提示视频客服服务时间（工作日9:00-17:00）。
3. 如在服务时间内，点击"开始视频通话"。
4. 系统请求相机和麦克风权限，确认后进入视频排队。
5. 视频客服接通后，双方可进行视频通话。
6. 视频客服适用场景：
   - 远程身份核验（如手机号修改）
   - 复杂业务咨询（如跨境汇款材料审核）
   - 远程面签
7. 通话过程中，客服人员可在视频窗口中展示文件、表格等辅助材料。
8. 通话结束后，弹出服务评价页面。

### 2.3 电话客服

1. 在"客户服务"页面，点击"电话客服"。
2. 展示客服热线号码和"立即拨打"按钮。
3. 点击"立即拨打"，系统调起手机拨号界面并自动填入客服号码。
4. 也可选择"网络电话"（使用APP内VoIP通话，消耗流量但不消耗通话分钟数）。
5. 电话接通后，进入智能IVR导航：
   - 系统提示"您好，欢迎致电XX银行，请说出您的需求或按数字键选择服务"。
   - 说出需求（如"我要查余额"），系统自动识别并转接至对应服务。
   - 或按传统方式按键选择：1-账户查询，2-转账汇款，3-信用卡，4-贷款，5-理财，0-人工服务。
6. 60岁以上客户拨入后直接转人工坐席，无需IVR导航。
7. 如需预约回拨，在等待过程中按提示选择"预约回拨"，输入回拨时间。

## 三、工单提交与查询操作流程

### 3.1 提交工单

1. 在"客户服务"页面，点击"提交工单"或"问题反馈"。
2. 选择工单类型：
   - 业务咨询（对业务规则、产品详情的咨询）
   - 业务办理（需要后台处理的具体业务请求）
   - 投诉建议（对服务质量、产品体验的投诉或建议）
   - 系统问题（APP功能异常、页面报错等技术问题）
3. 选择业务子类（如"转账汇款"→"转账失败"）。
4. 填写问题描述（至少20个汉字）：
   - 详细描述遇到的问题
   - 说明操作步骤和出现问题的环节
   - 提供相关的交易时间、金额等关键信息
5. 上传附件（可选）：
   - 点击"添加附件"→选择"拍照"或"从相册选择"
   - 上传问题截图、相关文件等佐证材料
   - 支持jpg/png/pdf格式，单个文件不超过10MB，最多5个
6. 填写联系方式（默认使用注册手机号，可修改）。
7. 确认提交信息无误后，点击"提交工单"。
8. 提交成功后，页面展示工单编号和预计处理时间："您的工单[编号]已提交成功，我们将在24小时内首次回复"。

### 3.2 查询工单进度

1. 在"客户服务"页面，点击"我的工单"。
2. 展示已提交的工单列表，每条工单显示：
   - 工单编号
   - 工单类型
   - 问题摘要
   - 提交时间
   - 当前状态（待处理/处理中/已回复/已完成/已关闭）
3. 支持按状态筛选（全部/待处理/处理中/已完成）。
4. 点击具体工单查看详细处理进度：
   - 工单提交时间
   - 各阶段处理记录（时间+处理人+处理内容）
   - 客服回复内容
   - 附件信息
5. 收到工单回复后，系统推送通知，点击通知可直接查看回复内容。
6. 如工单已回复但未解决问题，可在工单详情页点击"追加回复"补充信息。

### 3.3 工单催办与评价

1. 如工单超过预期处理时间仍未回复，在工单详情页点击"催办"按钮。
2. 系统提示"催办已提交，我们将尽快处理"，催办后4小时内应有回复。
3. 每个工单最多催办3次。
4. 工单处理完成后，系统推送评价通知。
5. 点击通知或在工单详情页点击"评价"按钮。
6. 对处理时效、处理结果、服务态度分别评分（1-5星）。
7. 可选填文字评价（如"处理速度快，问题已解决"）。
8. 提交评价，完成工单闭环。

## 四、消息中心操作流程

### 4.1 查看消息

1. 在手机银行首页右上角，点击消息中心图标（铃铛图标）。
2. 未读消息数以红色数字角标显示在图标上。
3. 进入消息中心，消息按类型分为标签页：
   - 全部：所有消息按时间倒序展示
   - 交易通知：动账、转账确认等交易类消息
   - 业务提醒：还款、理财到期、证件到期等提醒类消息
   - 活动营销：优惠活动、积分、活动等营销类消息
   - 系统公告：系统维护、功能更新、费率调整等公告
4. 未读消息左侧有蓝色竖线标识，点击进入详情后自动标记已读。
5. 点击消息可查看详情，部分消息包含"查看详情"或"去办理"按钮，点击跳转至相关功能页面。
6. 例如：点击信用卡还款提醒→跳转至信用卡还款页面。

### 4.2 消息管理操作

1. 在消息中心页面，点击右上角"管理"或更多操作图标。
2. **全部已读**：点击"全部已读"，所有未读消息一键标记为已读。
3. **删除消息**：左滑消息条目（iOS）或长按消息条目（Android），点击"删除"。
4. **批量删除**：点击"编辑"→勾选需要删除的消息→点击"删除"。
5. **搜索消息**：点击搜索图标，输入关键词搜索历史消息内容。
6. **消息设置**：进入"消息设置"页面，管理各类消息的推送偏好。

### 4.3 消息推送设置

1. 进入"消息中心"→右上角"设置"或"我的"→"设置"→"消息通知"。
2. 查看各类消息的推送开关状态。
3. 按需开启/关闭各类消息推送：
   - **动账通知**（建议保持开启）：APP推送开关、短信通知开关
   - **还款提醒**：APP推送开关、短信通知开关
   - **理财到期提醒**：APP推送开关
   - **营销活动**：APP推送开关
   - **系统公告**：始终推送（不可关闭）
4. 设置免打扰时段：
   - 开启"免打扰模式"
   - 设置开始时间（默认22:00）和结束时间（默认08:00）
   - 免打扰期间营销类消息暂停推送
5. 设置动账通知阈值：
   - 最低通知金额（默认0.01元，即所有金额变动均通知）
   - 可调整阈值（如设置为100元，则100元以下交易不发送通知）
6. 设置完成后点击"保存"。

## 五、电子回单操作流程

### 5.1 单笔交易回单下载

1. 在转账/缴费/支付成功后，交易结果页面展示"下载电子回单"按钮。
2. 点击"下载电子回单"，系统生成PDF格式的电子回单文件。
3. 回单内容包含：银行名称及logo、交易日期、交易金额、收款方/付款方信息、交易流水号、银行电子印章。
4. 回单生成后，可选择：
   - "保存至手机"：保存至手机本地存储
   - "分享"：通过微信、邮件、蓝牙等方式分享
   - "打印"：连接无线打印机直接打印
5. 回单文件默认保存至手机"下载"文件夹。

### 5.2 历史交易回单下载

1. 进入"我的账户"→选择银行卡→"交易明细"。
2. 找到需要下载回单的交易记录，点击进入交易详情。
3. 在交易详情页底部点击"下载电子回单"。
4. 系统生成该笔交易的电子回单PDF文件。
5. 保存或分享回单文件。

### 5.3 批量交易明细导出

1. 进入"我的账户"→选择银行卡→"交易明细"。
2. 点击右上角"导出"按钮。
3. 选择导出时间范围：近1个月/近3个月/近6个月/近1年/自定义时间段。
4. 选择导出格式：Excel（方便筛选分析）或PDF（带电子印章）。
5. 确认导出信息，输入交易密码验证身份。
6. 系统生成文件（可能需要1-2分钟），完成后自动下载至手机。
7. 大额明细导出（超过1000条记录）将发送至客户预留邮箱。

### 5.4 电子回单验真

1. 收到他人提供的我行电子回单时，可通过验真功能确认真实性。
2. 在"客户服务"页面，点击"回单验真"。
3. 输入回单编号（回单右上角的编号）。
4. 输入验证码（回单上的6位验证码）。
5. 点击"验证"，系统比对回单信息。
6. 验证通过：显示"该回单真实有效"，并展示回单核心信息。
7. 验证失败：显示"未查询到该回单信息"，提示回单可能为伪造。

## 六、常见问题查询操作流程

### 6.1 浏览常见问题

1. 在"客户服务"页面，点击"常见问题"或"帮助中心"。
2. 展示常见问题分类列表：
   - 账户管理、转账汇款、信用卡、理财产品、贷款业务、生活缴费、安全设置、手机银行使用、外汇业务等。
3. 点击某一分类，展示该分类下的热门问题列表（按热度排序）。
4. 点击具体问题，展开答案详情。
5. 答案内容包含文字说明、操作步骤、配图说明等。
6. 部分答案底部包含"去办理"快捷按钮，直接跳转至对应功能页面。

### 6.2 搜索常见问题

1. 在常见问题页面顶部，点击搜索框。
2. 输入关键词（如"转账限额"、"密码修改"、"信用卡年费"等）。
3. 系统实时展示搜索结果（匹配标题和内容）。
4. 搜索结果按相关度排序，关键词高亮显示。
5. 点击搜索结果查看完整答案。
6. 如搜索无结果，系统提示"未找到相关问题"并建议联系在线客服。

### 6.3 问题反馈

1. 在常见问题答案底部，有"这个回答对您有帮助吗？"评价区。
2. 点击"有帮助"（大拇指向上）或"无帮助"（大拇指向下）。
3. 如选择"无帮助"，弹出反馈框，可选择原因：
   - 答案不相关
   - 答案不完整
   - 答案已过时
   - 看不懂答案
4. 可选填补充说明，描述您的具体问题。
5. 提交反馈，帮助银行优化常见问题内容。

## 七、常见问题处理

| 问题场景 | 处理步骤 |
|----------|----------|
| 智能客服无法解决问题 | 1. 尝试换一种方式描述问题 → 2. 查看智能客服推荐的相似问题 → 3. 输入"转人工"转接人工客服 → 4. 或提交工单等待处理 |
| 消息收不到推送 | 1. 检查手机系统是否允许APP推送通知（设置→通知→手机银行→允许通知） → 2. 检查APP内消息推送设置是否开启 → 3. 检查是否处于免打扰时段 → 4. 如仍收不到，联系客服 |
| 电子回单无法下载 | 1. 检查手机存储空间是否充足 → 2. 确认APP有存储权限 → 3. 尝试使用WiFi网络下载 → 4. 如仍无法下载，联系客服获取回单 |
| 人工客服等待时间过长 | 1. 查看排队人数和预计等待时间 → 2. 选择"预约回拨"，在指定时间等待客服回拨 → 3. 尝试使用在线客服（通常等待时间较短） → 4. 非紧急问题可提交工单 |
