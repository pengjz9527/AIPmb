# 自助终端业务办理流程

> 本文档按业务类型分节，详细说明各类型自助终端（STM/ATM/CRS/查询机/回单打印机）的具体操作步骤和注意事项，为厅堂服务引导和客户自助操作提供参考。

---

## 一、自助终端类型概览

| 设备类型 | 简称 | 核心功能 | 是否处理现金 |
|---------|-----|---------|------------|
| 智能柜员机 | STM | 非现金全品类业务 | 部分型号支持 |
| 自动取款机 | ATM | 取款、转账、查询 | 是（仅取款） |
| 存取款一体机 | CRS | 存取款、转账、查询 | 是 |
| 自助查询机 | QTM | 账户查询、明细打印 | 否 |
| 回单打印机 | — | 电子回单、流水打印 | 否 |

---

## 二、STM智能柜员机操作流程

### （一）个人开卡

**操作步骤**：

1. 在STM主界面点击"个人开卡"。
2. 将二代身份证放置在身份证阅读区，系统自动读取身份信息。
3. 面对摄像头进行人脸识别，系统自动比对身份证照片（相似度需≥80%）。
4. 填写个人信息（职业、联系地址、手机号），可手动输入或扫描名片识别。
5. 选择卡种（标准借记卡、特色卡等），确认卡面样式。
6. 设置6位交易密码（两次输入确认）。
7. 阅读并勾选《借记卡领用协议》《个人信息授权书》等协议。
8. 在签字板上进行电子签名。
9. 等待发卡模块制卡（约30-60秒），从出卡口取走银行卡。
10. 取走身份证，操作完成。

**注意事项**：
- 人脸识别不通过时，可请大堂经理人工审核。
- 16岁以下未成年人开户需监护人陪同，到柜面办理。
- 开户同时可一并签约手机银行、短信通知等服务。

### （二）密码重置

**操作步骤**：

1. 在主界面点击"密码重置"。
2. 插入银行卡（或手动输入卡号）。
3. 将身份证放置在阅读区进行身份核验。
4. 面对摄像头进行人脸识别验证。
5. 输入新密码（6位数字，两次确认）。
6. 打印密码重置凭条，操作完成。

**注意事项**：
- 密码重置需本人办理，不可代办。
- 如账户已被锁定（连续6次输错密码），需到STM或柜面解锁。
- 新密码不得与近3次使用过的密码相同。

### （三）理财风险评估

**操作步骤**：

1. 插入银行卡并输入密码登录。
2. 点击"理财服务"→"风险评估"。
3. 阅读《风险评估说明》。
4. 依次回答10-15道风险评估题目（涉及收入、投资经验、风险偏好等）。
5. 系统根据答题结果生成风险等级（保守型/稳健型/平衡型/进取型/激进型）。
6. 确认评估结果并签字。
7. 打印风险评估结果凭条。

**注意事项**：
- 风险评估结果有效期1年，到期需重新评估。
- 评估过程中不得中途退出，否则需重新开始。
- 风险等级决定了可购买的理财产品范围。

## 三、ATM/CRS操作流程

### （一）取款操作

**操作步骤**：

1. 将银行卡插入插卡口（芯片朝上朝前）。
2. 输入6位交易密码。
3. 在主菜单选择"取款"。
4. 选择快捷金额（100/200/500/1000/2000/5000元）或选择"其他金额"手动输入。
5. 等待出钞（约10-15秒），从出钞口取走现金。
6. 屏幕提示"是否继续操作"，选择"否"退卡。
7. 取走银行卡，如需凭条可选择打印。

**注意事项**：
- 单笔取款金额为100元的整数倍。
- 取出现金后请立即清点，30秒内未取走现金将被回收。
- 如ATM出钞异常（少钞或未出钞），请勿离开，立即拨打设备上的客服电话。

### （二）存款操作（CRS）

**操作步骤**：

1. 插入银行卡并输入密码。
2. 选择"存款"。
3. 将整理平整的纸币（100元面额）放入入钞口，一次最多100张。
4. 系统自动清点识别，屏幕显示识别结果（可接受张数、金额，退回张数）。
5. 如有退回纸币，从退钞口取回。
6. 确认存款金额，点击"确认存款"。
7. 存款实时入账，屏幕显示入账金额和账户余额。
8. 退卡并取走银行卡。

**注意事项**：
- 纸币需去除回形针、橡皮筋等异物，折角纸币需展平。
- 残损币、涂改币、假币将被退回。
- 如退回纸币较多，建议到柜面办理存款。

### （三）转账操作

**操作步骤**：

1. 插入银行卡并输入密码。
2. 选择"转账"。
3. 输入收款方卡号（手动输入或扫描二维码转账）。
4. 输入转账金额。
5. 屏幕显示收款人姓名（脱敏）、转账金额，请仔细确认。
6. 输入交易密码确认转账。
7. 打印转账凭条。

**注意事项**：
- ATM转账默认24小时后到账，24小时内可拨打客服电话撤销。
- 转账前请仔细核实收款人信息，防范电信诈骗。
- 如收款人姓名显示异常，请暂停操作并确认。

## 四、自助查询机操作流程

### （一）余额查询

**操作步骤**：

1. 在查询机主界面点击"余额查询"。
2. 插入银行卡。
3. 输入6位交易密码。
4. 屏幕显示账户余额、可用余额、冻结金额。
5. 可选"打印凭条"。
6. 退卡取走银行卡。

### （二）交易明细打印

**操作步骤**：

1. 插入银行卡并输入密码。
2. 选择"交易明细"。
3. 选择查询时间范围（近1个月/3个月/6个月/1年/自定义）。
4. 屏幕显示交易明细列表。
5. 可上下翻页浏览，选择"打印"输出纸质明细。
6. 打印的明细加盖银行电子业务章，具有法律效力。

**注意事项**：
- 自助查询机打印的明细通常限近1年内，更早的明细需到柜面办理。
- 打印的明细请妥善保管，防止个人信息泄露。

### （三）账户信息查询

**操作步骤**：

1. 插入银行卡并输入密码。
2. 选择"账户信息"。
3. 查看卡号、开户日期、账户类型、绑定手机号等基本信息。
4. 退卡完成操作。

## 五、回单打印机操作流程

### （一）电子回单打印

**操作步骤**：

1. 在回单打印机主界面选择"电子回单打印"。
2. 插入银行卡并输入密码。
3. 选择回单类型（转账回单/缴费回单/理财购买回单等）。
4. 选择交易日期范围。
5. 屏幕显示可打印的回单列表，勾选需要打印的回单。
6. 点击"打印"，从出纸口取走回单。
7. 退卡完成操作。

### （二）流水打印

**操作步骤**：

1. 选择"流水打印"。
2. 插入银行卡并输入密码。
3. 选择打印时间范围。
4. 确认打印页数和费用（部分银行超出免费页数后收费）。
5. 打印完成，取走流水单。

**注意事项**：
- 回单打印机通常部署在网点自助区，营业时间内可使用。
- 打印的回单和流水均加盖电子业务章，可作为有效凭证。
- 如打印不清晰或缺章，请到柜面重新打印。

## 六、通用注意事项

### （一）安全提示

1. 操作自助设备前观察周围环境，注意是否有可疑人员或异常装置。
2. 输入密码时用手遮挡键盘，防止被偷窥或摄像。
3. 操作完成后务必取走银行卡、现金和凭条，不要遗留个人信息物品。
4. 如遇设备吞卡，请勿轻信旁边"热心人"的帮助，直接拨打设备上的银行客服电话。

### （二）设备故障处理

| 故障类型 | 处理方式 |
|---------|---------|
| 吞卡 | 记下设备编号，拨打客服电话或到附近网点处理 |
| 未出钞但已扣款 | 保留凭条（如有），拨打客服电话登记，银行核实后补账 |
| 存款被退 | 检查纸币是否平整，重新整理后再试；多次退回请到柜面办理 |
| 设备死机 | 等待设备重启，如已扣款未出钞/未退卡，拨打客服电话 |

### （三）服务时间

| 设备位置 | 服务时间 |
|---------|---------|
| 网点自助区（24小时） | 7×24小时 |
| 网点大堂内 | 营业时间（部分网点延长至20:00） |
| 离行式自助银行 | 7×24小时 |
| 商场/社区等离行点位 | 跟随场所营业时间 |

---

## 七、附注

- 各类自助设备的界面设计因厂商（广电运通、怡化、日立等）和银行系统版本而异，操作流程大同小异。
- 建议网点在自助区域配备操作指南海报或视频，引导客户正确使用。
- 对首次使用自助设备的客户（尤其老年客户），大堂经理应主动上前协助指导。
- 自助设备的日均交易量、故障率、客户满意度等数据应纳入网点运营管理考核。
