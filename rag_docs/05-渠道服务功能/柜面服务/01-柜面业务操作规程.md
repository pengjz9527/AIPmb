# 柜面业务操作规程

> 本文档系统梳理柜面各业务类型的标准操作规程，涵盖开户、存取款、转账、银行卡、理财、外汇等全品类业务的操作标准、审核要点和凭证要求，为柜员日常操作提供规范指引。

---

## 一、概述

柜面是银行最基础、最全面的业务办理渠道，承担着客户身份核实、现金处理、复杂业务办理、合规审核等核心职能。柜面操作规程的严格执行是防范操作风险、保障客户资金安全、确保业务合规的基石。

柜面业务操作应遵循以下基本原则：

- **实名制原则**：所有业务必须核实客户真实身份。
- **当面办理原则**：关键业务环节须客户当面操作确认。
- **双人复核原则**：重要业务须双人经办复核。
- **凭证完整原则**：业务凭证填写完整、签章齐全。
- **即时记账原则**：业务完成后即时准确记账。

## 二、开户类业务

### （一）个人开户（借记卡/存折）

**操作标准**：

1. **身份核实**：
   - 使用身份证阅读器读取客户二代居民身份证。
   - 核对身份证照片与客户本人是否一致。
   - 通过联网核查系统验证身份证真伪。
   - 如非居民身份证（护照、军人证等），按相关规定办理。

2. **信息录入**：
   - 系统自动读取身份证信息（姓名、证件号、地址）。
   - 补充录入客户职业、联系电话、联系地址。
   - 选择账户类型（I类户/II类户/III类户）。
   - 客户填写《个人结算账户开户申请书》。

3. **签约与发卡**：
   - 设置6位交易密码（客户在密码键盘上输入两次确认）。
   - 询问是否同时签约电子银行（手机银行、网银）、短信通知等。
   - 系统制卡/制折，打印开户凭条。
   - 客户在开户申请书和领卡确认书上签字。
   - 将银行卡/存折、身份证、客户回单交付客户。

**审核要点**：
- 同一客户在本行开立I类户不得超过1个（全行范围）。
- 同一客户在本行开立的借记卡不超过4张（含已注销）。
- 对异常开户行为（多人集中开户、拒绝提供辅助证明材料等）应提高警惕，必要时拒绝开户。

**凭证要求**：
- 《个人结算账户开户申请书》（客户签字）。
- 身份证复印件（联网核查通过可不留存）。
- 《电子银行服务协议》（如同时签约电子银行）。

### （二）电子银行签约

**操作标准**：

1. 客户持本人有效身份证件和银行卡到柜面办理。
2. 核实客户身份（身份证阅读器+联网核查）。
3. 客户填写《电子银行服务申请表》，选择开通渠道（手机银行/网银/微信银行）。
4. 选择安全认证方式（短信验证码/U盾/动态令牌）。
5. 如申请U盾，发放U盾设备，指导客户设置U盾密码。
6. 设定转账限额（单笔/日累计），由客户在允许范围内自选。
7. 客户在申请表和《电子银行风险提示》上签字。
8. 系统完成签约，打印客户回单。

**审核要点**：
- 电子银行签约必须本人办理，不得代办。
- U盾/令牌等安全工具应当面发放，不得邮寄。
- 应向客户充分揭示电子银行使用风险。

## 三、存取款类业务

### （一）现金存款

**操作标准**：

1. 客户递交现金和银行卡/存折。
2. 柜员当面清点现金（先点大数、再点细数），与客户确认金额。
3. 在系统中选择"现金存款"交易，输入卡号/账号和金额。
4. 大额现金存款（5万元及以上）需客户出示身份证件并登记。
5. 系统记账，打印存款凭条。
6. 客户在存款凭条上签字确认。
7. 将银行卡/存折和客户回单交付客户。

**审核要点**：
- 大额现金存款需执行反洗钱规定，登记《大额交易和可疑交易报告》。
- 单日累计存款20万元（含）以上需报告大额交易。
- 假币识别：发现假币应按规定收缴，出具《假币收缴凭证》。

### （二）现金取款

**操作标准**：

1. 客户递交银行卡/存折和取款凭条。
2. 柜员刷磁/读芯片，客户在密码键盘输入交易密码。
3. 系统验证密码后，柜员输入取款金额。
4. 大额取现（5万元及以上）需客户出示身份证件，经主管授权。
5. 系统记账，柜员当面点付现金。
6. 打印取款凭条，客户签字确认。
7. 将现金、银行卡/存折、客户回单交付客户。

**审核要点**：
- 单笔取现5万元（含）以上需出示身份证件。
- 单笔取现20万元（含）以上需提前1个工作日预约。
- 大额取现需主管授权并在系统中登记取款用途。
- 对可疑取现行为（频繁大额取现、配合他人取现等）应加强审核。

**凭证要求**：
- 取款凭条（客户签字）。
- 大额取现需留存身份证件复印件。
- 大额取现预约登记表。

## 四、转账类业务

### （一）行内转账

**操作标准**：

1. 客户填写《转账凭条》，注明付款账号、收款账号、收款人姓名、金额、用途。
2. 柜员核实客户身份（刷磁/读芯片+密码验证）。
3. 在系统中录入转账信息，核对收款人姓名。
4. 金额5万元（含）以上需出示客户身份证件。
5. 系统记账，打印转账凭条。
6. 客户签字确认。

### （二）跨行转账

**操作标准**：

1. 客户填写《汇款申请书》，注明收款人姓名、收款账号、开户行、金额、用途。
2. 柜员核实客户身份。
3. 选择汇款渠道：大额支付系统（5万以上，工作日）、超级网银（5万以下，7×24小时）。
4. 录入汇款信息，系统校验收款行行号。
5. 金额5万元（含）以上需主管复核。
6. 系统记账并发送支付报文。
7. 打印汇款回单，客户签字确认。

**审核要点**：
- 核对收款人姓名、账号、开户行信息的一致性。
- 大额跨行转账需双人复核。
- 对异常转账（如老年人向陌生账户大额转账）应进行防诈骗提醒。

### （三）跨境汇款

**操作标准**：

1. 客户填写《境外汇款申请书》（中英文），注明收款人名称、账号、银行SWIFT代码、汇款金额、币种、用途。
2. 柜员核实客户身份，审核汇款用途证明材料。
3. 确认客户年度便利化购汇额度（等值5万美元）使用情况。
4. 在系统中录入汇款信息，选择代理行路线。
5. 金额等值1万美元（含）以上需主管授权。
6. 系统记账，发送SWIFT报文。
7. 打印汇款回单，客户签字确认。

**审核要点**：
- 严格执行外汇管理规定，审核汇款用途真实性。
- 关注分拆结售汇、异常大额购汇等可疑行为。
- 跨境汇款需进行反洗钱筛查（制裁名单过滤）。

**凭证要求**：
- 《境外汇款申请书》（客户签字）。
- 汇款用途证明材料（如学费通知书、合同等）。
- 身份证件复印件。

## 五、银行卡类业务

### （一）银行卡挂失

**操作标准**：

1. 客户持有效身份证件到柜面申请挂失。
2. 柜员核实客户身份，通过系统查询账户信息。
3. 确认挂失类型：口头挂失（临时）或书面挂失（正式）。
4. 书面挂失需客户填写《挂失申请书》。
5. 系统执行挂失操作，冻结账户资金。
6. 打印挂失凭条，客户签字确认。
7. 告知客户补卡时间（通常7个工作日后）。

**审核要点**：
- 挂失必须本人办理（特殊情况可代办，需双人身份证明+委托书）。
- 书面挂失前应先做口头挂失冻结资金（如尚未挂失）。
- 挂失手续费按标准收取。

### （二）银行卡销户

**操作标准**：

1. 客户持银行卡和有效身份证件到柜面申请销户。
2. 柜员核实身份，查询账户是否有未结清的贷款、理财、协议等。
3. 结清账户利息，将所有资金转出或支付给客户。
4. 解除所有签约关系（电子银行、代扣代缴、第三方存管等）。
5. 客户填写《销户申请书》。
6. 系统执行销户，回收银行卡（剪角处理）。
7. 打印销户凭条，客户签字确认。

**审核要点**：
- 销户前需确认无未结清的业务关系。
- 如账户绑定贷款还款，需先变更还款账户。
- 社保卡、公积金卡等特殊卡种销户需相关部门出具证明。

## 六、理财类业务

### （一）风险评估

**操作标准**：

1. 首次购买理财产品的客户须到柜面（或STM）完成风险评估。
2. 柜员指导客户阅读《风险承受能力评估问卷》。
3. 客户独立作答，柜员不得引导或代填。
4. 系统根据问卷结果生成风险等级。
5. 客户在评估结果确认书上签字。

### （二）理财产品销售（双录）

**操作标准**：

1. 柜员/理财经理向客户推荐适合的理财产品（产品风险等级不得高于客户风险承受能力）。
2. 向客户充分揭示产品风险，阅读《产品说明书》《风险揭示书》。
3. 启动"双录"（录音录像）系统，确保销售过程全程录音录像。
4. 双录内容须包含：产品介绍、风险揭示、客户确认、销售人员声明等关键环节。
5. 客户确认理解产品风险后签署《理财产品购买协议》。
6. 系统完成产品购买，打印确认书。

**审核要点**：
- "双录"是监管硬性要求，不得省略或简化。
- 理财销售人员须持有基金从业资格或银行从业资格。
- 不得向风险承受能力不足的客户推荐高风险产品。
- 双录录像资料保存期限不少于产品到期后6个月。

**凭证要求**：
- 《风险评估问卷》及评估结果确认书。
- 《理财产品说明书》《风险揭示书》（客户签字确认）。
- 《理财产品购买协议》（客户签字）。
- 双录音视频文件存档。

## 七、外汇类业务

### （一）结售汇

**操作标准**：

1. 客户持有效身份证件到柜面办理。
2. 填写《个人结汇/购汇申请书》，注明金额、用途。
3. 柜员核实客户身份，查询年度便利化额度使用情况。
4. 在额度内（等值5万美元/年）按当日牌价办理。
5. 超额度需提供真实性证明材料，经主管审核。
6. 系统完成交易，打印凭条。
7. 客户签字确认。

### （二）跨境汇款

（详见"转账类业务-跨境汇款"章节）

**审核要点**：
- 严格审核结售汇用途的真实性。
- 关注分拆交易行为（同一人短期内多次接近额度的结售汇）。
- 对超额度结售汇须审核交易真实性证明材料原件。

---

## 八、附注

- 柜面所有业务操作应遵循《商业银行内部控制指引》和各行内部操作规程。
- 柜员应定期参加业务培训和合规教育，确保操作技能持续达标。
- 操作规程应根据监管政策变化及时更新修订。
- 本文档为通用操作规程，各行可能有更细化的本地化操作细则。
