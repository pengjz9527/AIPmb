# 柜面适老化服务规范

> 本文档系统阐述面向老年客户的柜面服务标准，涵盖绿色通道、专人引导、适老设施、方言服务、上门服务和防诈骗提醒等内容，参照《银行营业网点适老化服务规范》等行业标准，为网点适老化服务建设提供参考。

---

## 一、概述

随着我国老龄化程度持续加深，老年客户已成为银行网点的重要服务群体。据统计，城商行/农商行网点中老年客户占比通常达30%-50%，部分社区型网点更高。做好适老化服务不仅是监管合规要求，更是银行社会责任的体现，也是赢得老年客户信任和口碑的关键。

**适老化服务原则**：

- **尊重原则**：尊重老年客户的选择权和自主权，不强制使用自助设备或电子渠道。
- **便利原则**：从硬件设施到服务流程全方位降低老年客户的使用障碍。
- **安全原则**：加强防诈骗提醒，保护老年客户资金安全。
- **温度原则**：以耐心、细致、温暖的服务态度赢得老年客户信任。

## 二、服务设施与硬件配置

### （一）绿色通道设置

| 设施要求 | 具体标准 |
|---------|---------|
| 爱心窗口 | 每个网点至少设置1个"爱心窗口"（老年客户/残障人士优先窗口），设置明显标识 |
| 优先叫号 | 叫号系统设置老年客户优先功能，65岁及以上客户刷卡后自动优先排号 |
| 无障碍通道 | 网点入口设置无障碍坡道，宽度≥1.2米，坡度≤1:12 |
| 无障碍停车位 | 有条件的网点设置无障碍停车位 |

### （二）助老设备配置

| 设备名称 | 配置要求 | 配置位置 |
|---------|---------|---------|
| 老花镜 | 至少配备2副不同度数（+1.5/+2.0/+2.5），定期清洁 | 填单台、柜面窗口 |
| 放大镜 | 至少1台台式放大镜 | 填单台 |
| 轮椅 | 至少1辆备用轮椅 | 网点入口处 |
| 血压计 | 至少1台电子血压计 | 等候区/便民服务台 |
| 急救箱 | 含常用急救药品（速效救心丸、创可贴等） | 大堂经理台 |
| 大字版计算器 | 按键大、屏幕大的计算器 | 填单台 |
| 饮水机 | 提供温热饮用水 | 等候区 |
| 休息座椅 | 带扶手和靠背的座椅（非无扶手条凳） | 等候区 |

### （三）大字版单据与回单

| 材料名称 | 适老化要求 |
|---------|-----------|
| 业务凭条 | 提供大字版打印选项（字号≥14pt），关键信息加粗 |
| 产品说明书 | 提供大字版或简化版产品说明，突出关键风险点 |
| 业务指南 | 在等候区放置大字版业务办理指南手册 |
| 回单 | 金额、账号等关键信息使用大字号打印 |
| 收费标准 | 公示大字版常见业务收费标准 |

### （四）标识与导引

| 标识项目 | 适老化要求 |
|---------|-----------|
| 功能分区标识 | 使用大字号、高对比度色彩，辅以图标说明 |
| 地面引导线 | 不同颜色引导线引导至不同功能区 |
| 防滑标识 | 湿滑区域设置醒目的"小心地滑"提示 |
| 紧急呼叫按钮 | 在卫生间、无障碍通道等位置设置紧急呼叫按钮 |

## 三、专人引导服务

### （一）迎宾引导

1. **主动识别**：大堂经理在客户进门时主动识别老年客户，上前问候并了解办理需求。
2. **全程引导**：引导老年客户至相应功能区，协助取号、填写单据。
3. **等候关怀**：老年客户等候期间主动询问是否需要饮水，告知预计等候时间。
4. **协助填单**：为有需要的老年客户代为填写业务凭条（客户口述，大堂经理代填，客户确认签字）。

### （二）柜面服务规范

| 服务要点 | 具体要求 |
|---------|---------|
| 称呼尊重 | 使用"老先生""老太太""叔叔""阿姨"等尊称，不使用编号称呼 |
| 语速语调 | 放慢语速，适当提高音量，吐字清晰 |
| 解释说明 | 用通俗易懂的语言解释业务内容，避免使用专业术语 |
| 确认理解 | 每步操作后确认客户是否理解，必要时重复说明 |
| 耐心等待 | 给予老年客户充足的反应和操作时间，不催促 |
| 双手递送 | 递送物品（卡、现金、凭条）时双手递送，便于接取 |
| 逐项交代 | 业务结束后逐项交代应取回的物品（身份证、银行卡、现金、回单），确认无遗漏 |

### （三）离柜关怀

1. 业务完成后提醒老年客户收好随身物品。
2. 对行动不便的老年客户，引导至门口或协助叫车。
3. 告知后续如有问题可随时致电网点或客服热线。

## 四、方言服务

### （一）方言服务要求

| 服务场景 | 具体要求 |
|---------|---------|
| 方言地区网点 | 至少配备2名能说当地方言的柜员/大堂经理 |
| 方言窗口标识 | 可提供方言服务的窗口设置"方言服务"标识牌 |
| 电话银行 | 客服热线提供方言服务选项（部分地方银行已实现） |
| 自助设备 | 有条件的STM支持方言语音引导功能 |

### （二）沟通技巧

1. 使用方言时避免使用地方特有的银行术语替代标准术语。
2. 涉及金额、账号等关键数字信息时，使用标准普通话和方言双重复确认。
3. 对于只会方言的老年客户，重要告知事项用方言解释后，在凭条上用文字记录。

## 五、上门服务规范

### （一）适用情形

| 适用情形 | 说明 |
|---------|------|
| 行动不便 | 因疾病、伤残等原因无法亲自到网点的老年客户 |
| 住院客户 | 住院期间需紧急办理银行业务的客户 |
| 高龄独居 | 80岁以上独居老人有紧急金融需求 |
| 特殊业务 | 密码重置、挂失等必须本人办理但客户无法到场 |

### （二）服务流程

1. **申请受理**：
   - 客户或其家属通过电话或到网点提出上门服务申请。
   - 网点登记客户信息、需求类型、服务地址、联系方式。
   - 网点负责人评估服务可行性并安排人员。

2. **服务准备**：
   - 至少安排2名员工共同上门（双人制度）。
   - 携带移动终端设备（移动PAD、便携式发卡机等）。
   - 准备相关业务单据、协议文本。
   - 准备鞋套、工牌等物品。

3. **服务实施**：
   - 到达后首先核实客户身份（身份证原件+人脸识别）。
   - 确认客户需求，当面办理业务。
   - 业务办理过程全程录音录像（移动设备录制）。
   - 客户在相关单据上签字（或按手印）。

4. **服务收尾**：
   - 确认客户对服务满意。
   - 留下网点联系电话，告知后续服务渠道。
   - 返回网点后及时补录系统信息，归档服务记录。

### （三）安全注意事项

- 上门服务必须双人同行，不得单人上门。
- 上门服务不得携带现金，仅办理非现金业务。
- 上门时间原则上安排在工作日白天（9:00-17:00）。
- 服务过程中注意保护客户隐私，避免在公共场合讨论金融信息。

## 六、防诈骗提醒义务

### （一）提醒场景

| 场景 | 提醒内容 |
|-----|---------|
| 大额取现 | "请问取款用途是什么？是否有人要求您取出现金？" |
| 大额转账 | "请问收款人您是否认识？是否有人通过电话/微信要求您转账？" |
| 新开网银 | "请注意保护好网银密码，不要将U盾/密码交给他人" |
| 购买理财 | "请注意理财有风险，收益越高风险越大，请根据自身情况选择" |
| 异常行为 | 客户神色慌张、边打电话边办理业务、有人陪同指导操作 |

### （二）防诈骗宣传

1. 在网点显著位置张贴防诈骗宣传海报。
2. 在等候区播放防诈骗宣传视频。
3. 定期开展老年客户防诈骗专题讲座。
4. 发放防诈骗宣传折页，涵盖常见诈骗手法（冒充公检法、虚假投资、保健品骗局等）。

### （三）可疑情况处置

| 可疑情形 | 处置措施 |
|---------|---------|
| 老年客户边打电话边要求转账 | 暂停办理，主动询问通话对象和转账原因 |
| 老年客户对收款人信息含糊不清 | 耐心核实，必要时联系客户家属确认 |
| 有人陪同老年客户办理大额业务且代为指示 | 要求老年客户单独确认业务意愿 |
| 老年客户要求将全部存款转至陌生账户 | 向主管报告，联系家属或社区，必要时报警 |
| 老年客户频繁到不同网点办理类似业务 | 通过系统查询交易记录，评估是否存在被骗风险 |

## 七、服务质量评估

### （一）评估指标

| 指标 | 目标值 | 评估方式 |
|-----|-------|---------|
| 老年客户满意度 | ≥90% | 定期问卷调查 |
| 爱心窗口利用率 | ≥80% | 系统数据统计 |
| 上门服务及时率 | 申请后2个工作日内 | 服务记录统计 |
| 防诈骗拦截率 | 100%（发现可疑即干预） | 内部记录 |
| 适老设施完好率 | ≥95% | 定期检查 |

### （二）持续改进

1. 每季度开展一次老年客户满意度调查。
2. 收集老年客户意见和建议，形成改善清单。
3. 定期组织员工适老化服务培训（沟通技巧、急救知识、防诈骗识别等）。
4. 对标行业优秀实践，持续提升适老化服务水平。

## 八、附注

- 本文档参照《银行营业网点适老化服务规范》（T/CBA 001-2022）、《银行业营业网点文明规范服务评价指标》等标准编制。
- 各行可根据当地监管要求和客户特点制定更细化的适老化服务实施细则。
- 适老化服务建设应纳入网点年度服务提升计划，确保经费保障和持续改进。
- 建议建立老年客户档案，记录客户的服务偏好、健康状况、紧急联系人等信息，提升个性化服务能力。
