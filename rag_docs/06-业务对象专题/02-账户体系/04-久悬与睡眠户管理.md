# 久悬与睡眠户管理

## 专题概述

久悬账户（又称"久悬户"）和睡眠账户（又称"沉睡账户"）是指长期未发生主动交易、余额较低的个人银行账户。这类账户大量存在不仅占用银行系统资源，增加管理成本，更重要的是成为电信网络诈骗和洗钱犯罪的潜在工具——被不法分子收购后用于资金转移。

2023年以来，在监管部门推动下，银行业掀起了一轮"沉睡账户"清理浪潮。多家银行发布公告，对长期不使用的账户采取限制措施。2025年10月，新一轮集中清理行动启动，范围从传统银行账户扩展到手机银行、电子账户等。

对城商行/农商行而言，久悬和睡眠户管理具有特殊意义。中小银行历史遗留的"僵尸账户"数量多（部分农商行因早期批量开卡遗留大量睡眠户），系统处理能力有限，客户通知覆盖不全，清理工作面临较大挑战。同时，如何在清理账户的同时保障客户合法权益，避免引发投诉和舆情，需要审慎平衡。

---

## 核心知识框架

### 一、久悬户与睡眠户的定义区分

| 类型 | 认定标准 | 主要成因 | 风险等级 |
|------|----------|----------|----------|
| 睡眠户 | 连续1-3年无主动交易，余额低于一定标准 | 客户遗忘、搬迁、替代使用 | 中等 |
| 久悬户 | 连续3-5年无主动交易，余额极低或为零 | 长期弃用、失联 | 较高 |
| 不动户 | 连续6个月-1年无主动交易 | 短期不用 | 较低 |

> 注：各银行认定标准不完全一致，监管未统一规定具体时限。主流银行以"3年无交易+余额≤100元"作为睡眠户认定标准，"5年无交易+余额为零或极低"作为久悬户标准。

### 二、"主动交易"的认定

以下交易视为"主动交易"，可重置无交易计时器：

| 交易类型 | 是否视为主动交易 |
|----------|----------------|
| 客户主动存取款 | 是 |
| 客户主动转账 | 是 |
| 客户主动消费支付 | 是 |
| 客户主动投资理财 | 是 |
| 银行系统自动结息 | 否 |
| 银行自动代扣费用 | 否 |
| 银行系统批量操作 | 否 |
| 他人向该账户转入 | 视银行规则而定 |

## 详解内容

### 一、认定标准与分类

#### 主流银行认定标准参考

| 账户状态 | 无交易时长 | 余额条件 | 附加条件 | 管控措施 |
|----------|-----------|----------|----------|----------|
| 不动户 | ≥6个月 | 不限 | — | 降低非柜面限额 |
| 睡眠户 | ≥1年 | ≤100元 | 无关联合约 | 暂停非柜面交易 |
| 久悬户 | ≥3年 | ≤10元或为零 | 无关联合约 | 暂停所有交易 |
| 拟清理户 | ≥5年 | 为零 | 无关联合约 | 拟销户 |

#### 豁免情形

以下账户即使满足睡眠/久悬标准，一般也不纳入清理：

- 社保/医保关联账户
- 公积金关联账户
- 代发工资账户（仍有代发记录的）
- 贷款还款关联账户
- 定期存款未到期账户
- 理财/基金持仓账户
- 签约有第三方代扣协议的账户
- 法院冻结/质押的账户
- 军人、烈士等特殊群体账户

### 二、管理流程

#### 完整管理流程（七步法）

**第一步：数据筛查（T-60天）**

- 系统按认定标准自动筛查符合条件的账户清单
- 排除豁免情形账户
- 生成待管控/待清理账户清单
- 内部审批确认

**第二步：客户通知（T-30天）**

- 通过多种渠道通知客户：
  - 短信通知（预留手机号）
  - APP站内消息推送
  - 官网/网点公告
  - 电话通知（高余额账户）
- 通知内容：账户状态、管控/清理时间、激活方式
- 通知记录留存备查

**第三步：管控实施（T日）**

| 管控级别 | 管控措施 | 适用对象 |
|----------|----------|----------|
| 一级管控 | 降低非柜面交易限额 | 不动户 |
| 二级管控 | 暂停非柜面交易，保留柜面服务 | 睡眠户 |
| 三级管控 | 暂停所有交易（含柜面），仅保留查询 | 久悬户 |
| 四级管控 | 账户注销（余额转入久悬专户） | 拟清理户 |

**第四步：余额处理**

- 账户余额≤10元：转入银行"久悬未取款专户"集中管理
- 账户余额>10元：原则上不销户，保持管控状态
- 余额为零的账户：可直接注销

**第五步：激活服务（持续）**

客户申请激活时的处理流程：

```
客户到网点/线上申请激活
    │
    ├── 身份验证 ──→ 通过 ──→ 确认账户信息 ──→ 解除管控 ──→ 恢复使用
    │                 │
    │                 └── 未通过 ──→ 补充身份证明材料 ──→ 重新验证
    │
    └── 已销户 ──→ 确认余额处理情况 ──→ 协助客户支取余额（如有）
```

**第六步：余额支取**

- 账户已管控但未销户：激活后正常支取
- 账户已销户但余额转入久悬专户：客户凭身份证件到原开户行申请支取
- 久悬专户资金长期无人认领的：按相关规定处理（最终转入营业外收入）

**第七步：持续监控**

- 定期（每半年）筛查新增睡眠/久悬户
- 监控激活和支取情况
- 评估管控效果，优化认定标准

### 三、激活方式

| 激活渠道 | 适用情况 | 流程 |
|----------|----------|------|
| 网点柜面 | 所有类型 | 身份证件+密码验证→解除管控 |
| 手机银行 | 一级/二级管控 | 登录→人脸识别→解除管控 |
| 智能柜员机 | 一级/二级管控 | 身份证+人脸→解除管控 |
| 远程视频 | 部分银行支持 | 视频面签→解除管控 |

#### 激活审核要点

- 验证客户身份真实性（防止冒名激活）
- 了解客户激活目的和后续使用计划
- 对于高风险账户，激活后设置"观察期"限额
- 记录激活原因和处理过程

### 四、与反电信诈骗的结合

久悬/睡眠户管理与"断卡行动"紧密结合：

| 关联场景 | 风险 | 管控措施 |
|----------|------|----------|
| 大量睡眠户被"唤醒" | 可能被收购用于诈骗 | 激活时加强审核，设置观察期限额 |
| 账户长期不用突然大额交易 | 可能已被转手 | 触发异常交易预警 |
| 一人多户且多个为睡眠户 | 账户囤积嫌疑 | 限制新开账户，清理多余账户 |
| 非本人操作的交易 | 账户被盗用 | 冻结账户，核实身份后解控 |

### 五、客户权益保障

在清理过程中须保障客户权益：

- **知情权**：提前充分通知客户，给予合理反应时间
- **财产权**：账户余额属于客户财产，不得侵占；即使销户，客户仍有权支取
- **选择权**：客户有权选择保留或注销账户
- **投诉渠道**：设立专门的投诉处理通道
- **合理期限**：管控措施给予合理的缓冲期

## 管理方法与工具

### 1. 账户状态管理系统

| 功能模块 | 说明 |
|----------|------|
| 自动筛查引擎 | 按规则定期筛查睡眠/久悬账户 |
| 通知管理 | 多渠道通知发送和回执管理 |
| 管控执行 | 自动执行管控措施 |
| 激活管理 | 激活申请受理和处理 |
| 余额管理 | 久悬专户管理、余额支取 |
| 报表统计 | 各类账户状态统计、趋势分析 |

### 2. 数据治理策略

| 策略 | 说明 |
|------|------|
| 增量管控 | 新开户后6个月未激活非柜面功能的，自动降级 |
| 存量清理 | 定期批量清理符合标准的存量账户 |
| 客户整合 | 引导客户将多个低活跃账户整合为一个主力账户 |
| 主动唤醒 | 通过营销活动激活有潜力的睡眠客户 |

### 3. 关键运营指标

| 指标 | 定义 | 管理目标 |
|------|------|----------|
| 睡眠户占比 | 睡眠户数量/总账户数量 | ≤15% |
| 久悬户占比 | 久悬户数量/总账户数量 | ≤5% |
| 清理完成率 | 已处理账户/应清理账户 | ≥80%/轮 |
| 激活成功率 | 通知后激活的账户占比 | 视情况 |
| 投诉率 | 因账户管控引发的投诉占比 | ≤0.5% |

## 城商行/农商行实践

### 案例一：某农商行"睡眠户集中清理专项行动"

该农商行历史遗留睡眠户数量庞大（占总账户数的40%以上），组织专项清理行动：

**准备阶段（1个月）**：
- 全面梳理存量账户数据，确定清理范围
- 制定清理方案，报总行审批
- 完善通知话术和FAQ
- 培训网点人员和客服中心

**通知阶段（1个月）**：
- 对预留手机号有效的客户发送短信通知
- 在官网、微信公众号发布清理公告
- 在网点张贴公告
- 对余额>50元的账户进行电话通知

**执行阶段（持续）**：
- 第一批：暂停非柜面交易（睡眠户）
- 第二批：暂停所有交易（久悬户）
- 第三批：注销零余额账户

**效果**：3个月内清理存量睡眠户20万户（占总睡眠户的60%），系统资源占用降低30%，客户投诉率0.3%（低于行业平均）。

### 案例二：某城商行"睡眠户唤醒营销"

该城商行将睡眠户管理与客户营销结合：

- 对"有价值睡眠户"（历史AUM较高、职业优质）开展唤醒营销
- 设计"回归有礼"活动：激活账户并完成首笔交易送积分/优惠券
- 客户经理对高价值睡眠户进行电话回访
- 将唤醒效果纳入客户经理考核

实施效果：有价值睡眠户唤醒率达到25%，唤醒后6个月留存率达到65%。

### 案例三：北京农商行"久悬户"管控实践

北京农商行于2025年宣布对久悬户进行管控，对连续3年以上无主动交易且余额为零或极低的账户采取管控措施。该行通过提前公告、多渠道通知的方式，给予客户充分反应时间，同时设置便捷的激活渠道（柜面+线上），平衡了管理效率和客户体验。

## 监管要求

### 1. 账户清理的合规要求

- 银行有权对长期不使用的账户采取管控措施，但须提前通知客户
- 通知方式应充分（短信、公告等多渠道），给予合理期限（一般不少于30天）
- 不得未经通知直接注销客户账户
- 账户余额属于客户财产，即使账户被注销，客户仍有权要求返还

### 2. "断卡行动"相关要求

- 银行应主动排查存量账户中的可疑账户
- 对一人多户、长期不用等异常情况进行重点排查
- 对可疑账户采取管控措施，同时留有客户申诉通道
- 定期向人民银行和银保监会报送排查和清理情况

### 3. 消费者权益保护

- 账户管控/清理信息须以清晰易懂的方式告知客户
- 不得因账户清理导致客户产生经济损失（如定期存款被提前支取）
- 老年人等特殊群体应提供便利的激活服务
- 客户对管控措施有异议的，应提供便捷的申诉渠道

### 4. 数据管理要求

- 已销户账户的交易记录须按规定期限保存（一般不少于5年）
- 久悬专户资金须单独核算和管理
- 长期无人认领的资金处理须符合相关规定

## 常见问题

**Q1：我的银行卡很久没用了，里面的钱还在吗？**
A：钱还在。即使账户被管控或注销，账户中的余额仍属于客户个人财产。如果账户只是被管控（暂停交易），激活后即可正常使用。如果账户已被注销，余额会被转入银行"久悬未取款专户"，客户凭身份证件到银行申请支取。

**Q2：银行通知我账户要被清理了，我该怎么操作？**
A：如果您希望保留该账户，可通过以下方式激活：到银行网点办理一笔交易（存取款、转账均可），或通过手机银行登录后进行一次操作。如果不想保留，可以不用操作，银行将按计划管控或注销。账户中有余额的，建议先取出再决定是否保留。

**Q3：银行清理睡眠账户是否侵犯了我的财产权？**
A：不构成侵犯。银行清理的是"账户功能"而非"账户资金"。即使账户被注销，资金仍然安全保管在银行（转入久悬专户），客户随时可以申请支取。银行清理睡眠户是为了防范金融风险、保护账户安全，是监管鼓励的行为。

**Q4：城商行/农商行的睡眠户比例为什么普遍较高？**
A：主要原因包括：一是历史上为完成任务而批量开卡（如代发工资、社保卡、惠农补贴卡），客户实际使用率不高；二是农村客户外出务工后原账户长期不用；三是客户在其他银行开立新账户后弃用旧账户；四是银行缺乏有效的睡眠户管理机制。

**Q5：如何从源头减少睡眠户的产生？**
A：一是严格开户审核，确保客户有真实开户需求；二是开户时引导客户完成首笔交易和产品绑定；三是定期开展账户健康检查，对低活跃客户主动触达；四是引导客户整合账户，减少不必要的多户；五是建立增量管控机制，新开户后长时间未使用的自动降级。
