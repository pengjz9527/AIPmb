# 个人贷款五级分类

## 专题概述

贷款五级分类是银行对金融资产按照风险程度进行分类管理的核心制度。2023年2月，中国银保监会和中国人民银行联合发布《商业银行金融资产风险分类办法》（以下简称"新规"），对原有的贷款五级分类制度进行了重大调整和完善，于2023年7月1日起正式施行。

新规的核心变化包括：一是将风险分类对象从"贷款"扩展到"承担信用风险的全部金融资产"；二是明确了逾期天数与分类的硬性对应关系；三是引入了"以债务人为中心"的分类理念（同一债务人在同一银行的所有金融资产应归入同一分类）；四是加强了与资产质量相关的拨备计提要求。

对城商行/农商行而言，五级分类新规的实施对资产质量认定和利润水平产生重大影响。特别是逾期天数与分类的硬性对应（如逾期超过90天必须归为不良），可能导致部分此前分类"宽松"的中小银行不良率上升。正确理解和执行新规，是中小银行当前合规管理的重点。

## 核心知识框架

### 五级分类总览

| 分类 | 风险程度 | 定义 | 逾期天数对应（硬性标准） | 拨备计提参考 |
|------|----------|------|------------------------|-------------|
| 正常 | 低 | 借款人能够履行合同，没有足够理由怀疑贷款本息不能按时足额偿还 | 未逾期或逾期≤30天 | 1%-2% |
| 关注 | 中低 | 尽管借款人目前有能力偿还，但存在一些可能对偿还产生不利影响的因素 | 逾期31-90天 | 2%-5% |
| 次级 | 中高 | 借款人的还款能力出现明显问题，完全依靠其正常营业收入无法足额偿还 | 逾期91-270天 | 25%-50% |
| 可疑 | 高 | 借款人无法足额偿还，即使执行担保也肯定要造成较大损失 | 逾期271-360天 | 50%-75% |
| 损失 | 极高 | 在采取所有可能的措施或一切必要的法律程序后，本息仍然无法收回 | 逾期>360天 | 100% |

> 注：拨备计提比例因银行和监管要求而异，上表为参考范围。

### 新规核心变化对比

| 维度 | 旧规（2007年版） | 新规（2023年版） |
|------|------------------|------------------|
| 分类对象 | 仅贷款 | 承担信用风险的全部金融资产 |
| 逾期对应 | 无硬性对应，以定性判断为主 | 逾期天数与分类硬性对应 |
| 分类单位 | 以单笔贷款为单位 | 以债务人为中心（同一债务人统一分类） |
| 重组贷款 | 重组贷款可直接归为正常 | 重组贷款至少归为关注 |
| 交叉影响 | 不考虑他行情况 | 债务人在他行不良，本行应相应下调 |
| 适用范围 | 商业银行 | 商业银行+农村合作银行+村镇银行 |

## 详解内容

### 一、各级分类详细标准

#### 1. 正常类

**定性标准**：
- 借款人经营状况正常，收入稳定
- 还款意愿良好，按时足额还款
- 担保品价值充足（如有）
- 无重大不利变化

**定量标准**：
- 未逾期或逾期不超过30天
- 借款人收入足以覆盖还款
- 资产负债率在合理范围内

**关注要点**：
- 正常类并非"零风险"，仍需持续监测
- 关注正常类中"即将到期"贷款的续贷安排
- 防止通过展期、借新还旧等方式掩盖风险

#### 2. 关注类

**定性标准**：
- 借款人还款能力出现一定程度的减弱
- 出现可能影响还款的不利因素
- 担保品价值下降或担保人能力减弱

**典型情形**：
- 逾期31-90天
- 借款人所在行业出现不景气
- 借款人收入下降但仍可维持还款
- 借款人涉及诉讼但尚未影响还款
- 借款人更换工作/失业，但有积蓄可过渡

**管理要求**：
- 密切关注借款人经营和财务状况
- 加强贷后检查频次
- 及时评估担保品价值
- 提前制定风险化解预案

#### 3. 次级类（不良贷款起点）

**定性标准**：
- 借款人还款能力出现明显问题
- 完全依靠正常经营收入无法足额偿还
- 即使执行担保，也可能造成一定损失

**典型情形**：
- 逾期91-270天
- 借款人经营严重亏损或失业
- 借款人出现重大诉讼/被执行
- 借款人资产被查封/冻结
- 借款人明显资不抵债

**硬性标准（新规）**：
- 金融资产逾期超过90天，必须归为次级及以下
- 债务人在同一银行的所有资产均须下调
- 债务人在他行出现不良，本行也应相应下调

#### 4. 可疑类

**定性标准**：
- 借款人无法足额偿还贷款本息
- 即使执行担保也肯定要造成较大损失
- 损失金额尚不确定

**典型情形**：
- 逾期271-360天
- 借款人已停产/停营
- 借款人下落不明
- 担保品价值严重不足
- 诉讼/执行程序正在进行中

#### 5. 损失类

**定性标准**：
- 在采取所有可能的措施后，贷款本息仍无法收回
- 或只能收回极少部分

**典型情形**：
- 逾期超过360天
- 借款人已破产清算完毕
- 借款人死亡且无遗产可执行
- 诉讼/执行完毕，确无财产可执行
- 贷款已过诉讼时效

### 二、逾期天数与分类的硬性对应（新规重点）

#### 硬性标准

| 逾期天数 | 最低分类 | 说明 |
|----------|----------|------|
| 0-30天 | 正常 | 短期逾期可仍为正常 |
| 31-90天 | 关注（最低） | 不得归为正常 |
| 91-270天 | 次级（最低） | 必须归为不良 |
| 271-360天 | 可疑（最低） | 必须归为可疑或损失 |
| >360天 | 损失（最低） | 必须归为损失 |

#### 特殊说明

- "逾期"是指超过合同约定的还款日未偿还应还金额
- 逾期天数从应还未还之日起计算，连续计算
- 部分还款不中断逾期天数计算（除非全额偿还当期应还金额）
- 展期/重组后重新计算逾期天数（但有特殊规定）

### 三、重组贷款的分类规则

#### 重组的定义

当借款人发生财务困难时，银行对贷款合同条款进行调整，包括但不限于：
- 展期（延长贷款期限）
- 降息（降低贷款利率）
- 减免（减免部分本金或利息）
- 变更还款方式
- 借新还旧

#### 重组贷款分类规则

| 情形 | 分类要求 |
|------|----------|
| 重组前为正常/关注 | 重组后至少归为关注类 |
| 重组前为不良 | 重组后至少归为次级类 |
| 重组后观察期满（≥1年）且正常还款 | 可上调至关注类 |
| 重组后再次逾期 | 须下调至少一级 |

#### 重组贷款"观察期"

- 重组后设置不少于1年的观察期
- 观察期内借款人须按时足额还款
- 观察期内不得上调分类
- 观察期满后可根据还款情况酌情上调

### 四、以债务人为中心的分类

#### 核心原则

同一债务人在同一银行的所有金融资产应归入同一分类等级（取最低等级）：

```
债务人在A银行的资产：
- 住房贷款：正常
- 消费贷款：关注
- 信用卡：次级

→ 该债务人在A银行的所有资产均须归为次级及以下
```

#### 跨行交叉影响

- 债务人在他行的金融资产被归为不良
- 本行应评估风险影响，酌情下调本行分类
- 新规要求银行关注债务人的整体债务状况

### 五、拨备计提

#### 拨备类型

| 拨备类型 | 说明 | 计提基础 |
|----------|------|----------|
| 一般准备 | 覆盖预期损失 | 贷款余额的一定比例 |
| 专项准备 | 覆盖特定贷款的已识别损失 | 按五级分类分别计提 |
| 特种准备 | 应对特定风险事件 | 监管要求 |

#### 拨备计提参考比例

| 分类 | 专项准备计提比例 | 说明 |
|------|----------------|------|
| 正常类 | 1%-2% | 最低拨备要求 |
| 关注类 | 2%-5% | 适度拨备 |
| 次级类 | 25%-50% | 大额拨备 |
| 可疑类 | 50%-75% | 高额拨备 |
| 损失类 | 100% | 全额拨备 |

#### 拨备覆盖率

- 拨备覆盖率 = 贷款损失准备/不良贷款余额 × 100%
- 监管要求：≥150%（部分时期有阶段性调整）
- 拨备覆盖率越高，银行抗风险能力越强，但也意味着利润被压缩

#### 贷款拨备率

- 贷款拨备率 = 贷款损失准备/贷款总额 × 100%
- 监管要求：≥2.5%（部分时期有阶段性调整）
- 拨备覆盖率和贷款拨备率两者取高者执行

## 管理方法与工具

### 1. 资产质量管理系统

| 功能模块 | 说明 |
|----------|------|
| 自动分类引擎 | 基于逾期天数和规则自动初分 |
| 人工调整模块 | 对系统初分结果进行人工复核和调整 |
| 重组管理 | 重组贷款识别、观察期跟踪 |
| 拨备计算 | 自动计算各级贷款的拨备计提 |
| 资产质量报表 | 不良率、迁徙率、拨备覆盖率等 |
| 监管报送 | 按监管要求生成报送数据 |

### 2. 资产质量分析框架

| 分析维度 | 指标 | 分析要点 |
|----------|------|----------|
| 总量分析 | 不良贷款余额和比率 | 绝对水平和趋势 |
| 结构分析 | 各级占比、产品分布、区域分布 | 风险集中领域 |
| 迁徙分析 | 正常→关注→不良的迁徙率 | 风险劣变速度 |
| Vintage分析 | 不同放款月份的逾期发展曲线 | 各时期放款质量对比 |
| 回收分析 | 不良贷款回收率、处置周期 | 处置效率 |

### 3. 逾期天数监测

建立逾期天数实时监测机制：

- T+1：逾期1天即预警，短信/电话提醒
- T+30：逾期30天，启动正式催收
- T+90：逾期90天，自动下调至不良（硬性标准）
- T+180：逾期180天，评估诉讼必要性
- T+360：逾期360天，评估核销条件

## 城商行/农商行实践

### 案例一：某城商行五级分类新规实施

该城商行在新规实施前进行了全面的资产质量"体检"：

- 逐笔排查全量贷款，识别"隐性不良"
- 按照逾期天数硬性标准重新分类
- 结果：不良率从1.5%上升至2.3%（新增不良主要来自此前归为"关注"的逾期90天以上贷款）
- 增提拨备2.5亿元，当期利润受到影响
- 但资产质量更加真实，为后续经营管理提供了准确基础

### 案例二：某农商行"五级分类+整村授信"联动

该农商行将五级分类管理与整村授信结合：

- 以村为单位统计贷款质量，计算村级不良率
- 不良率低于1%的村评为"信用村"，享受利率优惠和额度提升
- 不良率超过3%的村暂停新增授信，启动风险排查
- 建立"村级资产质量月报"，实时监控各村贷款质量变化

### 案例三：某城商行拨备策略优化

该城商行在满足监管要求的前提下优化拨备策略：

- 对正常类贷款适当降低拨备比例（从2%降至1.5%），释放利润空间
- 对关注类贷款维持较高拨备（5%），为潜在劣变做好准备
- 对不良贷款加大处置力度，通过催收和核销加快出清
- 建立拨备动态调整机制，每季度根据资产质量变化调整拨备策略

## 监管要求

### 1. 核心监管文件

| 文件名称 | 核心内容 |
|----------|----------|
| 《商业银行金融资产风险分类办法》（银保监会令2023年第1号） | 五级分类新规，2023年7月1日起施行 |
| 《商业银行贷款损失准备管理办法》 | 拨备计提要求 |
| 《商业银行不良资产管理指引》 | 不良资产管理和处置规范 |
| 《金融企业呆账核销管理办法》 | 核销条件和程序 |

### 2. 分类管理要求

- 银行应至少每季度对全部金融资产进行一次风险分类
- 分类工作应做到真实、准确、及时
- 分类结果应作为拨备计提、绩效考核、监管评价的依据
- 不得人为调整分类以操纵利润或规避监管

### 3. 信息披露要求

- 定期披露不良贷款余额、不良率、拨备覆盖率等核心指标
- 披露重组贷款规模和分类情况
- 披露逾期贷款分布情况

### 4. 过渡期安排

- 新规自2023年7月1日起施行
- 存量业务过渡期至2025年12月31日
- 过渡期内银行应逐步将存量资产按新规要求重新分类
- 过渡期结束后全面执行新规

## 常见问题

**Q1：逾期天数与分类是"硬性对应"还是可以灵活调整？**
A：逾期天数对应的是"最低分类"。例如，逾期91天的贷款至少归为次级类，但如果银行认为风险更高（如借款人已破产），可以归为可疑类或损失类。反过来，不可以将逾期91天的贷款仍归为正常或关注类。

**Q2：借新还旧/展期算不算"重组贷款"？**
A：是的。新规明确规定，当借款人发生财务困难时，银行对合同条款进行的任何调整（包括展期、借新还旧、降息等）均属于"重组"。重组后贷款至少归为关注类，且须设置不少于1年的观察期。

**Q3：五级分类新规对城商行/农商行利润影响有多大？**
A：影响因行而异。此前分类"宽松"的银行（如将大量逾期90天以上贷款仍归为关注类），新规实施后不良率将上升，需要增提拨备，利润下降。据行业估算，部分中小银行不良率可能上升0.5-1个百分点。

**Q4：同一债务人在多家银行都有贷款，一家银行分类下调会影响其他银行吗？**
A：新规要求银行关注债务人的整体债务状况。如果债务人在A银行的贷款被归为不良，B银行在知晓后也应评估风险，酌情下调分类。但实操中，银行获取他行分类信息的渠道有限，主要依赖征信报告。

**Q5：不良贷款核销的标准和流程是什么？**
A：核销标准包括：借款人死亡/失踪且无财产可执行、诉讼/执行完毕确无财产、贷款已过诉讼时效等。流程为：业务部门提出→风险管理部门审核→财务部门复核→审批委员会审批→报监管备案→账务处理（转入表外）→持续催收。
