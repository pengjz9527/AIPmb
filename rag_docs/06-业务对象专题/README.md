# 06-业务对象专题

## 分类说明

本目录系统梳理零售银行核心业务对象的知识体系，面向城商行/农商行零售业务管理者和产品设计人员，提供可落地的业务参考。内容覆盖零售银行三大核心领域：**个人客户**、**账户体系**、**风控与合规**。

每篇文档遵循统一结构（专题概述 → 核心知识框架 → 详解内容 → 管理方法与工具 → 城商行/农商行实践 → 监管要求 → 常见问题 → 关联知识），便于快速检索和交叉引用。

## 使用说明

1. **按主题查阅**：根据业务需要进入对应子目录，选择具体文档阅读。
2. **交叉引用**：各文档末尾"关联知识"部分提供相关文档的相对路径链接。
3. **持续更新**：监管政策和业务实践持续演进，建议定期复核文档时效性。
4. **适配使用**：城商行/农商行应根据自身规模和区域特点，对文档中的标准进行本地化调整。
