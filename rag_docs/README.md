# 零售银行业务知识库

## 知识库定位

本知识库面向**城市商业银行和农村商业银行**的零售业务条线，系统性地整理零售银行业务的核心知识体系。内容覆盖业务分类、产品介绍、办理流程、法规政策、渠道服务和专题研究六大维度，按业务对象细分建档，便于快速检索和引用。

## 适用人群

- 产品经理：产品设计、功能规划、需求编写
- 渠道运营：渠道功能管理、用户旅程优化
- 合规与风控：政策解读、合规审查、风险管控
- 业务管理：制度制定、流程规范、培训教材
- 技术团队：业务理解、系统设计、需求评审

## 知识深度

本知识库采用**专业级**深度，每篇文档包含：
- 业务定义与分类框架
- 操作细则与流程节点
- 政策法规依据与解读
- 风控要点与合规红线
- 城商行/农商行特色实践
- 典型案例与常见问题

## 文档模板

所有知识文档遵循统一结构：

1. **概述** — 业务定义、性质、适用客户群
2. **业务分类** — 子类型划分及特点
3. **适用对象与准入条件** — 客户条件、门槛、合规要求
4. **业务要素** — 金额/期限/利率/费用等核心参数
5. **操作流程** — 端到端流程及关键节点
6. **政策法规依据** — 监管要求与法规引用
7. **风控要点** — 风险识别、防控措施、操作红线
8. **城商行/农商行特色实践** — 中小银行差异化做法
9. **常见问题与案例分析** — FAQ + 典型案例
10. **关联知识** — 相关文档索引链接

## 内容来源

- 监管文件：国家金融监督管理总局、中国人民银行公开发布的规章和通知
- 行业规范：中国银行业协会发布的行业自律规范和服务标准
- 银行公开资料：各城商行/农商行公开的产品说明书、服务协议、收费标准
- 专业文献：零售银行转型、普惠金融、数字化转型等研究
- 渠道服务：手机银行/网银功能说明、智能柜台操作手册等

## 使用说明

1. 通过 INDEX.md 全文索引快速定位所需知识
2. 按目录层级浏览，每个分类下有 README.md 说明本分类的组织逻辑
3. 手机银行部分按业务板块细分，每个板块包含功能清单、业务规则、操作流程、风控要点四个维度
