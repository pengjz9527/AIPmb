## 关键发现

### 🔴 收入下滑预警
2024年收入峰值¥606K，2025年降至¥457K（**-25%**），主要原因是工资收入波动。2026-02出现¥21.2万异常高收入（疑为年终奖），剔除后月均约¥3.4万。

### 🟡 支出同步收缩
支出从2024年的¥591K降至2025年的¥475K（-20%），说明用户**主动控制支出**。但结余率仍为负（-4%），说明收入下降幅度更大。

### 🔴 结构性支出压力
"转账汇款"占总支出53%（¥92万），若为房贷则每月约¥2.5万，是最大的刚性支出。医疗健康消费占总消费约3%，但从安贞医院频次推断存在潜在医疗需求。

### 🟢 资产缓冲
主账户余额¥63,749，按当前月均支出计算，可支撑约**1.6个月**。

## 收入 vs 消费增长预判

```
当前趋势：
- 收入增速：-25%/年（2024→2025）
- 支出增速：-20%/年（2024→2025）

若延续此趋势：
第一年：收入 ¥343K  vs  支出 ¥380K  →  结余率 -11%
第二年：收入 ¥257K  vs  支出 ¥304K  →  结余率 -18%
```

**结论：收支缺口会以约5%的速度扩大。**

### 但这里有几点乐观因素：

1. **2026收入已回升**：前5月年化约¥444K，比2025年的¥457K下降幅度已收窄
2. **消费下降是主动行为**：信用卡月均从¥1.5万降到¥1.1万，说明消费可控
3. **公积金是潜在缓冲**：12万的公积金提取是可调用的储备

### 建议
| 优先级 | 行动 | 预期效果 |
|--------|------|----------|
| P0 | 稳定工资收入，关注职场安全 | 收入端 | 
| P1 | 审视¥92万转账汇款去向，识别优化空间 | 支出端 -53% |
| P2 | 建立6个月应急储备金（约¥24万） | 风险缓冲 |
| P3 | 减少信用卡分期，降低财务费用 | 支出优化 |
